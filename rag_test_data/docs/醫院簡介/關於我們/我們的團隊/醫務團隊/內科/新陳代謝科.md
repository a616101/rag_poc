# 新陳代謝科

## 團隊簡介
本科提供內分泌疾病及新陳代謝疾病的診斷與治療服務。擁有健全的醫療照護團隊，包括醫師、個管師、營養師，並具有完整的設備如非散瞳眼底攝影、甲狀腺超音波、四肢血流探測。提供就診前完成完整相關檢驗服務，減少等待時間，並提供就診提醒及護理營養追蹤、衛教及諮詢。定期舉辦糖尿病衛教講座及糖尿病友聯誼會，運用團體力量幫助病友達到血糖控制目標。本科與屏東縣甜園糖尿病之友協會合作，該協會於1990年成立，為正式立案的糖尿病病友社會團體，由病友主導，結合病友影響力與凝聚力，達到相互扶持與自我管理，目標是透過相互關懷與成長，達到「最佳血糖控制與併發症預防」。

## 團隊編制
- 醫師：主治醫師
- 護理/照護：專職個管師
- 醫事技術/其他：營養師

## 甜園糖尿病之友協會
### 名稱
屏東縣甜園糖尿病之友協會

### 源起
1990年（民國八十九年）成立，為正式立案的糖尿病病友社會團體。由病友主導，結合病友影響力與凝聚力，達到相互扶持與自我管理。目標是透過相互關懷與成長，達到「最佳血糖控制與併發症預防」。

### 宗旨
透過團體衛教、知識學習與經驗分享，加深病友及家屬對糖尿病的認識，建立疾病照護與控制的信心，並透過相互關懷與情感交流，幫助更多相同病況的病友達到血糖控制。

### 組織任務
1. 配合政府單位推行糖尿病共同照護網
2. 配合政府推廣糖尿病衛生教育
3. 建構糖尿病友共同支援網絡
4. 協助糖尿病友疾病照護及心理支援
5. 協助貧困糖尿病友醫療照護

### 得獎事跡
- **2003年**：行政院衛生署第一屆全國優良糖尿病病友團體表揚活動「團體精神獎」
- **2004年**：國家生技醫療品質標章
- **2005年**：中華民國糖尿病學會林瑞祥教授糖尿病傑出團隊獎
- **2006年**：行政院衛生署「全國優良糖尿病病友團體選拔活動」第三名、「糖尿病衛教創意活動」南區團體特別優秀獎
- **2007年**：行政院衛生署「第三屆全國優良糖尿病病友團體選拔活動」優良團體獎、衛生教育成效推廣活動競賽優等獎
- **2008年**：屏東縣「潛力優良糖尿病病友團體」
- **2009年**：行政院衛生署「您對血糖知多少-正常及空腹血糖值認知推廣」第二名、「胰島素注射與提升照護品質創意團體衛教競賽」第三名、糖尿病健康促進機構績效競賽第三名
- **2010年**：糖尿病健康促進機構績效競賽「傑出獎」（醫院層級，南台灣唯一）、全國糖尿病支持團體競賽「ABCDEFG模範糖尿病病友」（2名）
- **2011年**：屏東縣模範病友團體、競賽「模範糖尿病病友」（2名）
- **2012年**：屏東縣模範病友團體
- **2013年**：屏東縣模範病友團體、全國糖尿病支持團體競賽「ABCDEFG模範糖尿病病友」（2名）
- **2014年**：屏東縣模範病友團體、全國糖尿病支持團體競賽「ABCDEFG模範糖尿病病友」（2名）
