# 1701572052_383

## 主旨  
本文件說明經食道心臟超音波檢查的原理、作法、檢查前後護理事項及注意事項，以協助患者了解檢查流程並配合醫療人員。

## 重點整理

| 項目 | 內容 |
|------|------|
| **檢查原理** | 因成人胸壁較厚，傳統胸前超音波影像可能模糊；經食道檢查可從心臟後方獲得更清晰影像。 |
| **檢查方法** | 檢查前10分鐘噴喉部麻醉劑，將超音波探頭經口腔置入食道，從後方檢查心臟功能與構造。 |
| **檢查前準備** | 禁食8小時（含食物與水），填寫同意書，取下活動假牙、眼鏡、手錶等物品，勿塗口紅。 |
| **檢查後注意** | 檢查後1~2小時勿進食，待麻藥作用消退後可試喝水，無嗆咳即可正常飲食。 |
| **併發症觀察** | 注意有無出血、嘔吐、異常疼痛等症狀，如有不適應盡快就醫。 |

## 流程/步驟

#### 檢查前準備：
1. 經醫師解釋後，由醫護人員協助填妥檢查同意書。
2. 檢查前8小時禁食（含食物與水）。
3. 檢查當日請取下活動假牙、口腔整形物品、手錶、眼鏡，並避免塗口紅。
4. 檢查前10分鐘，於咽喉內噴局部麻醉劑2~3次，並請病人將藥劑含著。

#### 檢查過程：
- 將超音波探頭經由口腔放入食道內，從食道內從心臟後方進行檢查，評估心臟功能與構造。

#### 檢查後護理：
1. 檢查完後，請病人休息1~2小時，勿進食。
2. 待麻藥作用消退後，可先試喝少量水，若無嗆咳情形，即可正常飲食。
3. 護理人員將觀察病人的生命徵象及有無併發症，如出血、嘔吐、異常疼痛等。

## 注意事項/警示

- 檢查後1~2小時內請勿進食。
- 如出現不適或上述併發症，請立即就醫。

經食道心臟超音波檢查須知

一、經食道心臟超音波與傳統心臟超音波檢查有何不同？
由於成人胸壁較厚，由胸前心臟超音波檢查是較不容易穿透的；因此，經胸前超音波的影像有時會比較模糊；所以才發展出經食道從心臟後方檢查心臟超音波的影像，比經胸前超音波的要清晰。

二、經食道心臟超音波檢查的作法：
在檢查前 10 分鐘，會在病人咽喉內噴局部麻醉劑 2~3 次，並請病人將藥劑含著。而後會把超音波探頭經由口腔放入食道內，在食道內從心臟後方檢查心臟的功能和構造。

三、經食道心臟超音波檢查執行前護理人員會為您做什麼？
1. 經醫師解釋後，醫護人員會協助填妥檢查同意書。
2. 在檢查前，需禁食 8 小時（包含食物及水）。
3. 醫護人員會再次為您解說檢查過程及配合事項。
4. 檢查當日請將活動假牙或口腔整形物品、手錶、眼鏡取下，並請不要塗口紅。

四、經食道心臟超音波檢查執行後護理人員會為您做什麼？
1. 檢查完後，請病人 1~2 小時不要吃東西，等待麻藥作用消失，先試喝水不會有嗆咳情形即可正常飲食。
2. 觀察病人的生命徵象、及有無合併症如：出血、嘔吐、異常疼痛等情形。

五、執行後，請您注意的事項：
1. 注意 1~2 小時不要吃東西。
2. 如有不舒服或上述合併症發生，請盡快就醫。
