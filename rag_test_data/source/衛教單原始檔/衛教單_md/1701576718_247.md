# 1701576718_247

## 主旨  
本文件為屏東基督教醫院腹膜透析患者提供藥水保存、出國準備及使用時的注意事項，以確保用藥安全與順利。

## 重點整理  
- 藥水應至少備有7天以上的安全用量，並優先使用即將到期的藥水。  
- 儲存環境應避免陽光直射、潮濕、高溫及蟲害，並保持存放處清潔。  
- 藥水若出現混濁、變色、沉澱物或已過期，請勿使用。  
- 外箱如有受潮、變形或損壞，請勿使用。  
- 出國前應於4週至2個月前通知腹膜透析室，並提供前往地區、地址、收件人姓名、電話、藥水數量及希望送貨日期。  
- 出國時建議隨身攜帶數包備用藥水，以應對班機延誤或突發狀況。  
- 出國前應與國外收貨人確認藥水能依時送達。  
- 若需延長停留時間，應儘早通知醫院，以確保藥水供應。  
- 收貨時應核對貨品與送貨單明細是否相符，並以全名簽收。  
- 若發現藥水瑕疵或破損，應立即通報腹膜透析室，並提供品項及貨號以便補送。  
- 提供兩家藥水公司免費服務專線，以處理送貨或產品疑問。

## 流程/步驟  
#### 出國前準備流程：  
1. 若有出國需求，請於4週至2個月前通知腹膜透析室。  
2. 提供前往地區、地址、收件人姓名、電話、藥水數量及希望送貨日期。  
3. 出國時建議隨身攜帶數包藥水，以應對班機延誤或突發狀況。  
4. 出國前與國外收貨人確認藥水能依時送達。  
5. 若在國外決定延長停留時間，需二次叫貨時，應儘早通知腹膜透析中心。  

#### 藥水使用與收貨流程：  
1. 收到藥水後，立即檢查外箱是否完整，有無受潮或變形。  
2. 檢查藥水有無混濁、變色、沉澱物或過期情形，如有異常請勿使用。  
3. 收貨時，確實清點貨品與送貨單上的明細是否相符。  
4. 在藥水公司出貨單上以全名簽收，並註明日期。  
5. 若發現瑕疵或破損，請立即告知腹膜透析室，並提供藥水品項及貨號，以便後續補送。

## 注意事項/警示  
- **藥水保存**：應避免陽光直接照射、高溫環境，並存放於清潔、乾燥、無蟲害處。  
- **藥水使用**：若藥水出現混濁、變色、沉澱物或已過期，請勿使用；外箱如有受潮、變形或損壞，亦請勿使用。  
- **出國注意**：應提前規劃送貨，避免因時間不足導致藥水短缺；出國時應自備備用藥水，以應對突發狀況。  
- **緊急情況**：若在國外需延長停留時間，應儘早通知醫院，以確保藥水能及時送達。

屏基醫療財團法人屏東基督教醫院
腹膜透析藥水使用及儲存注意事項

| 編號 | ND 內-070 |
| :--- | :--- |
| 製訂日期 | 2013.07 |
| 三修日期 | 2022.03 |

一、藥水保存注意事項：
1. 藥水安全備用量應保存 7 天以上；若不足時請提早三天前與腹膜透析中心聯絡。
2. 有效期限快到的先用。
3. 藥水儲存注意事項：
   （1）避免陽光直接照射。
   （2）存放場所保持清潔，避免潮濕、蟲害。
   （3）藥水勿於高溫環境中存放。
4. 藥水若有混濁、變色、沉澱物、過期之情形，請勿使用。
5. 藥水外箱是否完整，無潮濕、變形之情形。

二、出國準備注意事項：
1. 若您有出國的需求時，請於 4 週～2 個月前通知腹膜透析室，以便安排送貨事宜；並告知欲前往地區、地址及收件人姓名、電話、數量與希望送貨日期。
2. 出國時最好能隨身自備數包藥水，以備班機延誤或遇突發狀況需更換藥水時使用。
3. 出國前先與您的國外收貨人確定藥水是否依時送達。
4. 若您在國外決定延長停留時間，需二次叫貨時，請儘早通知您的腹膜透析中心，以便有足夠的作業時間，並確保藥水能及時送達。

三、藥水使用注意事項：
1. 若有瑕疵或破損問題，請告知腹膜透析室藥水的品項及貨號，以利藥水公司後續補送。
2. 確實清點貨品與送貨單上之明細是否相符。
3. 請於藥水公司出貨單上確實以全名簽收貨品並註明日期。
4. 若有送貨或產品上之任何疑問時，請與腹膜透析室聯絡，本室當竭力為您服務。
5. 腹膜透析藥水公司免費服務專線：
   （1）百特醫療產品股份有限公司：0800-231611 轉 552
   （2）台灣費森尤斯醫療股份有限公司：0800-288258 轉腹膜透析部門
