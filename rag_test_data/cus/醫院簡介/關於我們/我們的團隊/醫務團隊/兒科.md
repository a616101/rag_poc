# 兒科

## 團隊簡介
屏基小兒科為衛福部評定的新生兒照護中度及急救責任醫院，擁有完善的小兒加護病房、中重度病房，屏基新生兒加護病房早產兒存活率達93.4%，更是成功照顧只有570公克早產兒，是屏東地區急重症和早產兒後送醫院。本科小兒科8位專任主治醫師，由5個次專科所組成，對於兒童的疾病診治預防、生長發育、身心發展提供詳盡的醫療照護及諮詢，目前科內有一般病床、小兒加護病房、新生兒加護病房、病嬰室及嬰兒室共78床，各次專科全方位同心協力合作，提供屏東縣市兒童一個優良的醫療環境為最大目標。

## 團隊編制
- 醫師：小兒科8位專任主治醫師，由5個次專科所組成
- 護理/照護：專業小兒科護理團隊
- 醫事技術/其他：檢查技術人員
- 特色分工：各次專科全方位同心協力合作

## 服務與治療
### 服務範圍（疾病/問題）
- 一般兒科常見疾病
- 新生兒科、新生兒加護及小兒加護照護
- 兒科心臟疾病、新生兒心臟科會診、先天性心臟病、心律不整
- 上下腸胃道出血、肝炎、肝硬化、消化疾病、急性及慢性腹瀉
- 早產兒照護、生長不良、新生兒黃膽、新生兒呼吸窘迫
- 小兒泌尿道感染、腎小管疾病、急慢性腎衰竭、排尿困難、血尿、蛋白尿、水腎、腎病症候群
- 小兒癲癇、神經系統感染、新成代謝異常、腦性麻痺、神經系統畸形、肌肉神經病變、發展遲緩、頭痛、眩暈、抽搐
- 常見兒童過敏疾病如氣喘、過敏性鼻炎、異味性皮膚炎、蕁麻疹
- 兒童自體免疫疾病
- 常見兒科疾病
- 過敏原篩檢

### 主要治療/處置
- 新生兒科、新生兒加護及小兒加護照護：提供完善的新生兒及小兒加護照護
- 早產兒照護：屏基新生兒加護病房早產兒存活率達93.4%，成功照顧只有570公克早產兒
- 兒科心臟疾病診治：新生兒心臟科會診、先天性心臟病、心律不整
- 小兒腸胃疾病診治：上下腸胃道出血、肝炎、肝硬化、消化疾病、急性及慢性腹瀉
- 小兒腎臟疾病診治：小兒泌尿道感染、腎小管疾病、急慢性腎衰竭、排尿困難、血尿、蛋白尿、水腎、腎病症候群
- 小兒神經疾病診治：小兒癲癇、神經系統感染、新成代謝異常、腦性麻痺、神經系統畸形、肌肉神經病變、發展遲緩、頭痛、眩暈、抽搐
- 小兒過敏免疫疾病診治：氣喘、過敏性鼻炎、異味性皮膚炎、蕁麻疹、兒童自體免疫疾病、過敏原篩檢

### 特色門診/照護計畫（若有）
