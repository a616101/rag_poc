## 主旨
本文旨在幫助民眾正確認識胃藥，了解不同類型胃藥的功用、可能的副作用、藥物交互作用，以及正確的用藥方式，避免因自行購藥或錯誤用藥而影響健康。

## 重點整理
- 許多民眾因擔心西藥傷胃而自行購買胃藥，或因症狀輕微而自行購藥，可能導致延誤就醫或用藥錯誤。
- 胃藥並非完全無副作用，短期及長期服用都可能產生不良反應。
- 胃藥種類繁多，不同藥物之間可能產生交互作用，影響療效或增加副作用。
- 服用胃藥應遵照醫囑，並了解藥物特性，特別是服用時間。
- 一旦身體不適，應就醫診斷病因，勿自行購藥，並應告知醫師或藥師所有正在服用的藥物。

## 流程/步驟
1.  **辨識症狀**：若出現胃痛、胃灼熱、消化不良等症狀，應先判斷是否為常見胃腸問題。
2.  **就醫診斷**：若症狀嚴重、持續或反覆發作，應儘早就醫，由專業醫師診斷病因。
3.  **諮詢專業**：在使用任何胃藥前，應諮詢醫師或藥師，特別是正在服用其他藥物時。
4.  **正確用藥**：遵照醫囑或藥品說明書服用，注意藥物的正確服用時間。
5.  **注意副作用**：留意用藥後是否出現腹瀉、便秘等副作用，並及時與醫療人員聯繫。

## 注意事項/警示
- **勿自行購藥**：不應因擔心西藥傷胃而自行購買胃藥，或因症狀輕微而自行用藥，以免延誤治療。
- **藥物交互作用**：
    - 制酸劑與抗生素、降血壓藥、苯妥英（Phenytoin）併用，可能影響後者的吸收與療效，建議至少間隔2小時。
    - 制酸劑與質子幫浦抑制劑（PPI）併用，可能降低PPI的吸收，影響抑酸效果。
    - 制酸劑與鐵劑併用，可能因胃酸鹼值改變而降低鐵的吸收，導致貧血。
- **注意副作用**：
    - 含鎂離子的制酸劑可能導致腹瀉。
    - 含鋁離子的制酸劑可能導致便秘。
    - 長期服用含鋁製劑可能導致鋁累積，增加結石發生率，對腎功能衰竭者尤為需注意。
- **正確用藥時間**：
    - 制酸劑一般作用時間為2-4小時。
    - 飯前半小時服用可延長作用時間至4小時。
    - 飯後立即服用，作用時間最長僅2小時。
- **避免重複用藥**：若家中備有藥品，應告知醫師或藥師，避免因重複用藥而產生副作用。

# 認識胃藥

文／藥劑科藥師 陳慧娟

凡肚子痛或胃悶痛，有些民眾為免去醫院就診或經濟考量，會自行去藥局或開架式的超市購買胃腸藥。服用這些藥物，可能只有症狀緩解不能對症下藥而延誤就醫，嚴重時胃腸出血或敗血症。另外在醫院常見的現象是，年長的病人在醫院拿藥時，因為擔心吃西藥傷胃，都會要求醫生開立胃藥，吃中藥時反而未想到也有可能傷胃。

吃胃藥真的是顧胃，不會有副作用嗎？最重要的是該不該吃胃藥？甚或考慮到自行購買的藥物是否會影響目前服用藥物的療效？為讓民眾服藥前了解藥物的性質，這次就來談胃藥。

## 胃藥類型：制酸劑、抑制胃酸分泌、胃黏膜保護劑、助消化劑。

| :--- | :--- | :--- |
| 制酸劑 | | 中和胃酸 |
| 胃酸分泌抑制劑 | H2受體拮抗劑(H2 receptor antagonist)：Cimetidine、Ranitidine | 抑制胃酸、胃蛋白酶分泌 |

### 該不該吃胃藥？

如病人被確診有十二指腸潰瘍、胃潰瘍、胃食道逆流或服用如阿斯匹靈、非類固醇止痛藥導致的胃不舒服時，有必要服用胃藥。

### 吃胃藥不會有副作用嗎？

胃藥短期及長期服用，也會有副作用。制酸劑含鎂離子，會發生腹瀉；含鋁離子常見副作用便祕，對末期腎功能衰竭者，可作為降血磷劑，但長期服用，因鋁不易排出累積於體內產生毒性，發生如骨病、腦病變；含鈣離子，一般做為鈣缺乏病人的補充劑，現今少當作制酸劑的原因是易產生酸反彈，同含鋁製劑適用於腎衰竭的高血磷患者，長期服用增加結石發生率。

### 藥物-藥物交互作用

胃腸藥種類多除了上述提及的胃藥，也包含益生菌、止瀉藥、排便藥、促腸蠕動劑等。這些藥物有時須搭配服用，但相互間也會有交互作用發生，例舉制酸劑與瀉劑bisacodyl併用時，會降低其排便效果；制酸劑與質子幫浦抑制劑併服，使得其吸收降低，抑胃酸分泌的療效也跟這受影響。

制酸劑（成份含2價金屬離子）及胃酸分泌抑制劑，由於改變胃酸鹼值或與鐵結合成不溶性鹽類，降低鐵於胃中吸收，導致貧血。

制酸劑與Phenytoin、抗生素或降血壓藥併用，這些藥物療效會受影響，須至少間隔2小時。H2受體拮抗劑尤其以Cimetidine，與許多藥物有交互作用，使用前務必諮詢醫師或藥師。質子幫浦抑制劑，會影響鐵及B12吸收，也須注意。

### 胃藥的正確用法

服藥除了依照醫囑之外，也須了解藥物的特性。制酸劑，一般作用時間是2-4小時，如飯前半小時服用可拉長至4小時，飯後立即服用作用時間最多只有2小時。有時多種胃藥或腸胃藥的併用雖有助於病情改善，但服用時間錯誤時，不一定能使療效加強，反而會有副作用的發生。

還有須提醒的是，一旦有身體不舒服時須就診，由醫生判定病因，勿自行購買成藥，即使有備藥也須告知醫師或藥師，因藥物彼此間可能存有交互作用及重複用藥。另外，藥物服用時間的長短也勿自行決定，免影響病情。
