# 1701574163_809

## 主旨  
屏基醫療財團法人屏東基督教醫院肺結核患者衛教須知，內容涵蓋住院隔離、飲食、陪病注意事項及返家後治療配合事項。

## 重點整理  
- 為防止結核菌透過飛沫傳染，需入住專屬負壓病房並配合門禁管制。  
- 病房設有兩道感應式電動門，僅能單獨開啟一道，以維持負壓環境。  
- 咳嗽或咳痰時應使用衛生紙遮住口鼻，痰液需包好並丟入感染性垃圾桶。  
- 飲食建議均衡，補充蛋白質、鈣質及維生素A、C，並避免刺激性食物。  
- 避免抽菸、喝酒，以免影響藥物代謝及肝功能。  
- 陪病家屬需配戴N95口罩並勤洗手，且6歲以下兒童不得探視。  
- 若家屬持續咳嗽5天以上，應就醫檢查。  
- 返家後不可自行停藥，需配合衛生所「送藥到手、服藥到口、吃完再走」政策。  
- 定期回診接受醫師及個管師的追蹤與衛教。

## 流程/步驟  
### 住院期間注意事項  
1. 進入病房時，由一位陪病家屬持個人證件至護理站換取門禁卡。  
2. 病房設有兩道電動門，僅能單獨開啟一道，第一道門關閉後，第二道門才可開啟。  
3. 需離開病房時（如檢查），應配戴外科口罩。  
4. 咳嗽時，以衛生紙遮住口鼻；若以手遮擋，應立即洗手。  
5. 痰液應以衛生紙包好，並投入病房內的感染性垃圾桶。  

### 飲食建議  
1. 採均衡飲食，可額外補充蛋白質（如牛奶、小魚乾）與鈣質（如鈣片）。  
2. 多攝取富含維生素A、C的食物（如菠菜、胡蘿蔔、柑橘類水果）。  
3. 若因藥物引起噁心、嘔吐，應採清淡飲食，少量多餐。  
4. 避免抽菸與喝酒，以免刺激氣管或影響肝臟代謝。  

### 陪病家屬注意事項  
1. 在病房內需配戴N95口罩並勤洗手。  
2. 住院期間限制訪客，且不得讓6歲以下兒童探病。  
3. 若家屬持續咳嗽超過5天，應配戴外科口罩前往胸腔科或感染科門診就醫。  

### 返家後注意事項  
1. 不可隨意停藥，以免產生抗藥性。  
2. 由衛生所觀察員執行「送藥到手、服藥到口、吃完再走」，確保規律服藥。  
3. 定期回診，接受醫師診療及個管師相關衛教。

## 注意事項/警示  
- 病房門禁管制嚴格，僅允許一位陪病家屬持證換卡。  
- 病房維持負壓，門禁系統設計為單獨開啟一道門，以防止空氣流通。  
- 咳嗽或咳痰時，務必使用衛生紙，並正確處理痰液，避免污染環境。  
- 避免抽菸與喝酒，以免加劇咳嗽或影響藥物效果。  
- 家屬若出現呼吸道症狀，應儘早就醫，避免交叉感染。  
- 不可自行中斷治療，否則可能導致結核菌產生抗藥性。

屏基醫療財團法人屏東基督教醫院

肺結核須知

| 編號 | ND 內-037 |
| :--- | :--- |
| 製訂日期 | 2003.08 |
| 六修日期 | 2019.07 |

一、為避免結核菌經由打噴嚏、咳嗽造成傳染，須請您住在專屬病房，請配合下列事項：

1. 隔離病房實施空氣隔離，進出有管制，一位陪病家屬可以個人證件到護理站換一張門禁卡；門禁卡不須使用時，即可換回。
2. 病房有二道門，皆為感應式電動門，同時只能開啟一道門。第一道門關閉後，第二道門才能開啟，以確保病室維持在負壓值。
3. 若需離開病房，例如：檢查等，需配戴外科口罩。
4. 咳嗽、咳痰之處理方式：
   (1) 須以衛生紙摀住口鼻；若是以手摀住口鼻，咳嗽後立即洗手。
   (2) 痰液請以衛生紙包好後，丟入病房內的感染性垃圾桶。

二、飲食注意事項

1. 建議採均衡飲食，亦可多補充蛋白質、鈣質，例如：牛奶、小魚干、鈣片及富含維生素 A、C 之食物，例如：菠菜、胡蘿蔔、柑橘類之水果。
2. 因藥物之影響，可能會有噁心、嘔吐的情形，此時可採清淡飲食、少量多餐。
3. 請避免抽菸、喝酒；因抽菸會刺激氣管引起咳嗽，喝酒會影響肝臟功能及藥物代謝，影響治療效果。

三、陪病家屬注意事項

1. 家屬在病房陪伴病人時，需戴 N95 口罩及勤洗手。
2. 住院期間須限制訪客，且小於 6 歲以下之兒童，避免探病。
3. 家屬若連續咳嗽 5 天以上，請配戴外科口罩至胸腔科或感染科門診就醫。

四、返家後注意事項

1. 請不要隨便停止藥物的服用，避免造成抗藥性。
2. 返家後將由衛生所的觀察員執行「送藥到手、服藥到口、吃完再走」，關懷結核病個案服藥治療過程，確保每一個病人規則服下每一顆藥，讓疾病能如期治癒。
3. 定期回診接受醫師的診療及個管師相關的衛教。

屏基關心您 祝您早日康復
