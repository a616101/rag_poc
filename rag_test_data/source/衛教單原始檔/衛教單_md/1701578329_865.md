# 1701578329_865

## 主旨  
本文件為膽道疾病手術後之衛教指引，內容涵蓋手術後注意事項、引流管護理、飲食建議及出院前衛教，協助患者安全恢復。術後居家照護與活動建議亦包含在內，以促進恢復並預防併發症。

## 重點整理

| 項目 | 要點 |
|------|------|
| **手術後活動** | 每1~2小時翻身，使用束腹帶支撐傷口；術後第2天可下床活動，促進循環，減少併發症。 |
| **呼吸與咳嗽** | 隨時進行深呼吸及咳嗽運動，預防肺部擴張不全及肺炎。 |
| **排便與引流** | 術後可能出現輕微腹瀉（每日1~3次），數週後改善；傷口需保持乾燥，如有潮濕應立即通知護理人員。 |
| **傷口護理** | 拆線後2天才可淋浴，避免盆浴；若紗布沾濕應即時更換，減少細菌入侵風險。 |
| **引流管管理** | 引流液正常為深綠色；引流袋應低於傷口，防止膽汁逆流；固定於同側腰際，避免牽扯。 |
| **飲食指導** | 未排氣前禁食（含水）；排氣後可先試喝水，再進食果汁、飲料，最後轉為低油高蛋白飲食。 |
| **口腔衛生** | 每日漱口、刷牙，維持口腔清潔。 |
| **出院後注意** | 若有未拔除引流管，需依醫護人員指導記錄每日引流量；保持傷口乾燥，按時服藥並回診追蹤。 |
| **返家後注意** | 若出現傷口紅、腫、熱、痛，發燒，腹痛，黃疸或灰白大便，應立即返院追蹤。 |
| **活動限制** | 手術一週後可從事輕度工作；一個月後可恢復正常活動；6週內避免提拿2公斤以上之重物或參與用力之活動；建議3個月內避免劇烈活動，以防腹壓增加及避免傷口及引流管之牽扯。 |

## 流程/步驟

#### 1. 手術後早期活動與呼吸
- 每1~2小時在床上翻身，使用束腹帶支撐傷口，雙手按住以減輕疼痛。
- 隨時進行深呼吸及咳嗽運動，促進肺部擴張，預防肺部併發症。
- 在家屬或護理人員協助下，術後第2天可下床活動，以減少術後併發症及促進血液循環。

#### 2. 引流管護理
- 觀察引流液顏色，正常為深綠色。
- 以紙膠固定引流管於與傷口同側之腰際，引流袋應低於傷口，不可高於傷口，以防膽汁逆流。
- 臥床時，用安全別針將引流管固定於衣服上，不可固定於床上，並預留足夠長度以利翻身。
- 下床時，應有適當支托，以別針固定於衣服下緣，避免牽扯或扭折。
- 若需帶引流管回家，應避免拉扯或扭曲管子，以保持管路通暢。

#### 3. 飲食與排氣管理
- 手術後尚未排氣或排便前，切勿進食（包括水）。
- 排氣後若無嘔吐，經醫師評估後可試喝清水。
- 若無不適，可進食果汁、飲料等，最後再開始進食低油、高蛋白飲食，以減輕膽囊負擔。
- 保持口腔衛生，每日漱口、刷牙。

#### 4. 出院前衛教
- 若返家後仍有未拔除的引流管，應依醫護人員指導分別倒出並記錄每日引流量。
- 保持傷口清潔乾燥，按時服藥，並依醫師指示按時回門診追蹤。

## 注意事項/警示

- **傷口疼痛**：若傷口疼痛，應立即告知護理人員。
- **傷口潮濕**：若發現傷口潮濕，應立即通知護理人員，並更換敷料。
- **引流管牽扯**：避免引流管被拉扯或扭折，特別在活動時應注意固定與支撐。
- **飲食禁忌**：術後未排氣前，切勿進食，包括水，以防腸道負擔。
- **排氣後進食順序**：應依序從清水→果汁/飲料→低油高蛋白飲食，避免過早進食加重膽囊負擔。
- **避免盆浴**：拆線後2天才可淋浴，禁止盆浴，以防感染。
- **返家後若出現以下症狀，請立即返院追蹤：**
  - 傷口紅、腫、熱、痛
  - 發燒
  - 腹痛
  - 黃疸或灰白大便
- **活動限制：**
  - 手術一週後：可從事輕度工作。
  - 一個月後：可恢復正常活動。
  - 6週內：避免提拿2公斤以上之重物或參與用力之活動。
  - 建議3個月內：避免劇烈活動，以防腹壓增加及避免傷口及引流管之牽扯。

膽道疾病手術後須知

| 編號 | ND-002 |
| :--- | :--- |
| 製訂日期 | 2003.07 |
| 六修日期 | 2022.07 |

一、手術後之注意事項：

1. 術後移動身體或每 1~2 小時在床上翻身運動，以束腹帶支托傷口雙手按住，以減輕傷口疼痛。
2. 手術後隨時做深呼吸及咳嗽運動，以利肺部擴張，預防肺部擴張不全及肺炎。
3. 在家屬或護理人員協助下，手術第 2 天可下床活動，早期下床可減少術後合併症及傷口引流，並促進血液循環。
4. 術後會有輕微腹瀉現象，每天 1 至 3 次大便，此情況會於數週或數月改善。
5. 若傷口疼痛請告知護理人員。
6. 傷口需保持清潔乾燥，若傷口潮濕，請告知護理人員，給予換藥。
7. 拆線後 2 天才可淋浴，返家後採淋浴，不要盆浴，若紗布沾濕需即時更換，以減少細菌進入傷口的機會。

二、引流管之注意事項：

1. 觀察引流液顏色，正常為深綠色。
2. 以紙膠固定引流管在與傷口同側之腰際，引流袋應低於傷口，不可高於傷口，以防膽汁逆流。
3. 臥床時引流管用安全別針固定於衣服，不可固定於床上，預留長度以利翻身避免牽扯。
4. 下床時引流管應該有適當的支托，以別針固定在衣服下緣。
5. 若需帶引流管回家，不要牽扯或扭折到管子，以保持管子通暢。

三、手術後飲食之注意事項：

1. 手術後尚未排氣或排便前，切勿進食（包括水），並於排氣後如果無嘔吐，經醫師評估後可以試著喝水，若無不適先進食果汁、飲料等，最後才開始進食低油飲食，高蛋白之飲食，以減少膽囊的工作量。
2. 保持口腔衛生，可以每日漱口、刷牙。

四、出院前衛教：

1. 返家後若身上尚有未拔除的各式引流管，請依醫護人員教導的方式分別倒出，並紀錄每日引流量。
2. 應保持傷口清潔乾燥，按時服藥，並依醫師指示按時回門診追蹤。

3. 返家後如有傷口紅、腫、熱、痛、發燒、腹痛、黃疸或灰白大便，請立即返院追蹤。

4. 手術一週後可從事輕度工作，一個月後可以恢復正常活動，6週內避免提拿2公斤以上之重物或參與用力之活動，建議3個月內避免劇烈活動，以防腹壓增加及避免傷口及引流管之牽扯。
