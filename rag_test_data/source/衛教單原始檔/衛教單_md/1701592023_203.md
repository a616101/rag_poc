# 1701592023_203

## 主旨  
本文件為屏基醫療財團法人屏東基督教醫院提供的石膏固定護理指引，旨在協助患者正確使用石膏、促進復健並預防併發症。

## 重點整理  
- 石膏材質分為石灰與樹脂兩種，各有不同特性。  
- 石膏未乾燥前應妥善保護，避免受潮或變形。  
- 保持患肢抬高，以促進血液循環，減少腫脹。  
- 在醫護人員指導下進行關節活動，預防僵硬與肌肉萎縮。  
- 避免直接將石膏著地行走，建議使用石膏鞋保護。  
- 不可自行拆除石膏棉捲或進行其他操作。  
- 出院後若出現異常症狀，應立即返院就醫。  
- 如有疑問，可隨時向護理人員諮詢。

## 流程/步驟  
1. **選擇合適的石膏材質**：  
   - 石灰石膏：初期有溫熱感，約 24–48 小時完全乾燥，較重，活動不便。  
   - 樹脂石膏：由玻璃纖維與 PU 樹脂製成，輕便，活動較方便。  

2. **石膏乾燥與保護**：  
   - 石膏未乾前，應將其露出於被單外，並放置於枕頭上，以促進乾燥。  
   - 避免受潮或變形，保持乾燥。  

3. **臥床期間護理**：  
   - 將患肢抬高至心臟以上，以促進血液循環。  
   - 可使用枕頭墊高或抬高床尾。  

4. **關節活動與復健**：  
   - 在醫護人員指導下，進行石膏上方與下方可動關節的運動。  
   - 有助於減少腫脹、關節僵硬及肌肉萎縮。  

5. **日常注意事項**：  
   - 保持石膏清潔與乾燥。  
   - 勿以石膏直接著地行走，建議購買石膏鞋保護。  
   - 不可使用棍子、鐵絲或尖銳物品伸入石膏內抓癢。  
   - 若有皮膚破損，可使用棉棒塗抹止癢，避免自行拆除石膏。  

6. **石膏棉捲使用**：  
   - 石膏棉捲用於保護皮膚，減少石膏對皮膚的磨擦。  
   - 切勿自行拆除石膏棉捲。  

7. **拆石膏與活動指導**：  
   - 依骨折部位與恢復情況，由醫師評估後決定拆石膏時間。  
   - 依醫師指示進行行走或下床活動。  

8. **返院就醫指徵**：  
   - 上石膏肢體出現冰涼、麻木、發紫。  
   - 持續性疼痛或感覺異常。  
   - 上石膏肢體嚴重腫脹或石膏龜裂。  
   - 上石膏肢體有任何異味或滲出物流出。  

9. **諮詢與服務**：  
   - 如有任何疑問，請隨時向護理人員提出諮詢。  
   - 護理人員將樂意提供協助。

## 注意事項/警示  
- 石膏未乾燥前應避免接觸水或濕氣。  
- 不可自行拆除石膏或使用尖銳物品處理。  
- 避免患肢受壓或碰撞。  
- 若出現異常症狀，應立即返院就醫，勿延誤治療。

## 全文

屏基醫療財團法人屏東基督教醫院
石膏固定須知

編號 ND 外-021
製訂日期 2003.07
六修日期 2019.07

一、石膏材質有二種：
一為石灰製成，上石膏最初 10-15 分鐘會有溫熱感，隨石膏漸漸乾燥溫熱感會改善，約 24-48 小時才完全乾燥，質量較重活動較不方便。
二為樹脂石膏：由玻璃纖維加上 PU 樹脂製成。材質較輕，活動方便。

二、在石膏未乾前，請將石膏露出於被單外，並放於枕頭上，以促進石膏乾燥及預防變形。

三、臥床期間請保持上石膏肢體高於心臟，以促進血液循環，可用枕頭墊高或將床尾抬高。

四、依醫護人員指導進行石膏上方和下方的可動關節之運動，可以幫助血液循環，減少腫脹，更可以減少關節僵硬及肌肉萎縮。

五、為預防石膏變形，請保持石膏的清潔、乾燥，並請勿以石膏直接於著地行走，可購買石膏鞋保護。

六、上石膏內肢體如有發癢時，不可用棍子、鐵絲或尖銳物品伸入石膏內抓癢，以免皮膚破皮而感染，可使用棉棒塗擦止癢。

七、石膏棉捲是用來保護皮膚，減少石膏對皮膚之磨擦，因此請勿自行拆除石膏之棉捲。

八、依骨折部位不同，癒合時間也有所不同，所以請依醫師指示行走或下床活動，醫師會評估狀況拆除石膏。

九、於出院後，如有下列症狀請立即返院診治：
1. 上石膏肢端冰冷、麻木、發紫。
2. 持續性的疼痛或感覺異常。
3. 上石膏肢體嚴重腫脹或石膏龜裂。
4. 上石膏肢體有任何異味或滲出物流出時。

十、如您有任何疑問，請隨時向護理人員提出諮詢，護理人員將會很樂意的為您服務。
