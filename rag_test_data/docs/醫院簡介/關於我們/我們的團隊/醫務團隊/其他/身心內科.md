# 身心內科

## 團隊簡介
本科負責全院精神科臨床工作、教學及研究，提供病患全人治療模式，涵蓋生理、心理及社會層面。初期發展階段主要為兼職醫師提供門診服務。1998年6~7月，兩位全職精神科專科醫師加入，招募專責臨床心理師、職能治療師、社會工作師及護理人員，擴展門診服務，開始提供每日急診及全院病患會診。1998年12月，在院內大樓建立20床住院復健病房，服務精神症狀尚未完全穩定或功能下降的病患，提供持續藥物治療、職能訓練、社交技巧訓練、心理及行為治療。2000年11月，醫院開始籌備成立兒童發展遲緩評估中心，本科負責兒童智力評估。2001年8月，醫院成立高屏青少年健康中心，本科提供專科醫師及臨床心理師支援。2004年，本科受託協助社區照顧工作人員精神健康檢查。2010年，本科協助全院建立自殺防治及安心團體。2011年，本科建立兒童智力發展特殊門診。2014年，本科協助建立兒童及青少年保護醫療服務試辦計畫。

## 歷史發展
### 初期發展
- 初期發展階段主要為兼職醫師提供門診服務

### 1998年
- **6~7月**：兩位全職精神科專科醫師加入，招募專責臨床心理師、職能治療師、社會工作師及護理人員，擴展門診服務，開始提供每日急診及全院病患會診
- **12月**：在院內大樓建立20床住院復健病房，服務精神症狀尚未完全穩定或功能下降的病患，提供持續藥物治療、職能訓練、社交技巧訓練、心理及行為治療

### 2000-2014年
- **2000年11月**：醫院開始籌備成立兒童發展遲緩評估中心，本科負責兒童智力評估
- **2001年8月**：醫院成立高屏青少年健康中心，本科提供專科醫師及臨床心理師支援
- **2004年**：本科受託協助社區照顧工作人員精神健康檢查
- **2010年**：本科協助全院建立自殺防治及安心團體
- **2011年**：本科建立兒童智力發展特殊門診
- **2014年**：本科協助建立兒童及青少年保護醫療服務試辦計畫
