# 1701600677_280

## 主旨  
本文件旨在說明兒童頭部外傷的定義、常見原因、居家照護方法、注意事項及預防措施，以協助家長正確應對與預防。

## 重點整理  
- **頭部外傷定義**：指頭皮、頭骨、腦膜或腦組織等受傷情形之統稱。  
- **常見原因**：跌倒、高處跌落、受虐、撞擊硬物、交通事故等。  
- **評估指標**：意識清醒度與活動力是評估預後的最佳指標。  
- **居家照護重點**：靜養、冰敷、密切觀察、避免自行用藥。  
- **需立即就醫情況**：意識不清、劇烈頭痛、持續嘔吐、抽筋、肢體無力、嬰兒前囟膨脹等。  
- **預防措施**：防滑墊、安全座椅、包覆尖銳物、運動防護等。

## 流程/步驟  
#### 兒童頭部外傷的居家照護方法  
1. **靜養與觀察**  
   - 禁止兒童劇烈運動或過度嬉戲。  
   - 保持環境安靜，避免刺激。  

2. **冰敷減腫**  
   - 在受傷部位冰敷，可減少腫脹。  
   - 建議在受傷後48小時內進行。  
   - 腫塊大小不代表傷勢嚴重程度。  

3. **密切觀察症狀**  
   - 受傷後72小時內，顱內壓力可能上升。  
   - 檢查頻率：  
     - 受傷後24小時內：每2小時檢查一次。  
     - 第二天：每4小時檢查一次。  
     - 第三天起：每8小時檢查一次。  
   - 注意症狀包括：意識、呼吸、睡眠狀況。  

4. **避免自行用藥**  
   - 切勿給予止痛藥、安眠藥或鎮靜劑，以免掩蓋病情。  

5. **陪睡觀察**  
   - 最好睡在兒童身邊，以便隨時觀察呼吸與睡眠狀況。  
   - 如有異常，應立即叫醒，確認是否昏迷。

## 注意事項/警示  
#### 若出現下列任何情況，請盡速回院就診：  

| 症狀 | 說明 |  
|------|------|  
| 意識漸趨不清醒（不易叫醒或叫不醒） | 可能為顱內出血或腦損傷徵兆 |  
| 劇烈頭痛、頭昏及暈眩 | 可能為顱內壓力升高 |  
| 持續性嘔吐三次以上（禁食仍嘔吐） | 可能為顱內損傷 |  
| 手腳或嘴角抽筋 | 可能為腦部受損或癲癇發作 |  
| 手腳一側較軟弱（漸趨癱瘓） | 可能為神經損傷 |  
| 一歲半前嬰幼童前囟門膨脹，晚上易哭鬧 | 可能為顱內壓力增高 |  
| 極度哭鬧或躁動不安 | 可能為不適或腦損傷 |

## 全文

屏基醫療財團法人屏東基督教醫院
小兒頭部外傷

一、什麼是頭部外傷？
是指頭皮、頭骨、腦膜或腦組織等有受傷情形之統稱。

二、頭部外傷常見原因
兒童的頭部外傷常見的原因為跌倒、從高處跌落、受虐、頭部碰到堅硬的東西撞擊或交通事故等，傷害的嚴重度與事故發生時撞擊的力度大小有關，臨床上觀察兒童的意識清醒度及活動力為目前評估預後最好的指標。

三、兒童頭部外傷的居家照顧方法
1. 頭部外傷後：靜養及仔細觀察是最重要的，禁止兒童劇烈運動或過度嬉戲。
2. 在瘀傷部位冰敷，可減少腫脹的程度，小孩頭部外傷常會出現如雞蛋般大小的腫塊，但腫塊大小並不代表受傷的嚴重程度。
3. 密切觀察兒童，如有顱內出血，在受傷後 72 小時或更長時間內，會使顱內壓力逐漸升高而出現各種症狀（請見第四點）。因此在受傷後 24 小時內，應每隔兩個鐘頭檢查病人一次，第二天 4 小時檢查一次，第三天每 8 小時檢查一次。
4. 切勿自行予以孩童服用止痛藥、安眠藥或鎮靜劑，以免耽誤病情。
5. 最好能睡在孩童身邊，以便能觀察其呼吸及睡眠狀況。如有異樣，應予以叫醒，以確知是否有昏迷情況發生。
6. 在 48 小時內可在瘀傷部位予以冰敷，可減少腫脹程度。

四、兒童頭部外傷注意事項（若有下列任何情況，請盡速回本院診察）
1. 意識漸趨不清醒（不容易叫醒或叫不醒）。
2. 劇烈頭痛、頭昏及暈眩。
3. 持續性嘔吐三次以上（禁食仍然有嘔吐）。
4. 手腳或嘴角抽筋。
5. 手腳一側比較軟弱（漸趨癱瘓）。
6. 一歲半前的嬰幼童前囟門膨脹，晚上容易哭鬧。
7. 極度地哭鬧或躁動不安。

五、如何預防及避免因意外造成頭部外傷？
1. 浴室地面鋪防滑墊，床的四周鋪安全地墊。
2. 不單獨留小朋友在獨立空間。
3. 外出時安裝正確安全座椅，並確實使用。
4. 家具有尖銳突出處需貼防撞條或用軟布適當包覆，降低碰撞後造成傷害。
5. 不可搖晃嬰兒。
6. 運動時適時給予防護（穿著護膝、安全帽等）。
