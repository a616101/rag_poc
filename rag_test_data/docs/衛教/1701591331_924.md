## 主旨  
本文件為植皮手術後的照護指引，針對受皮區、供皮區、營養補充、諮詢與回診等事項提供具體建議，以促進傷口癒合與患者康復。

## 重點整理  
| 項目 | 要點 |
|------|------|
| **受皮區照護** | 手術後患肢需抬高，預防腫脹與減輕疼痛；若植皮區在下肢，需臥床休息，經醫師同意後方可活動；四肢植皮可能需石膏固定 5–10 天，不可自行拆除；如有疼痛、蒼白、冰冷或麻木感，應立即告知醫護人員。 |
| **供皮區照護** | 傷口滲濕時需立即換藥，但非每日更換；僅更換最外層紗布，內層透明膜不需更換；不可自行撕除敷料，以免損傷皮膚；若出現腫脹、疼痛、滲血或不適，應立即通報。 |
| **營養補充** | 多進食高熱量、高蛋白飲食，如魚、肉、蛋；糖尿病患者需視情況會診營養師。 |
| **諮詢與回診** | 如有任何疑問，可隨時向醫護人員諮詢；請依醫師指示按時回診。 |

## 流程/步驟  
#### 1. 受皮區照護步驟  
1. 手術後將患肢抬高，以減輕疼痛並預防腫脹。  
2. 若植皮區位於下肢，應臥床休息，不可自行下床活動，需經醫師同意後方可活動。  
3. 若需石膏固定，應固定 5–10 天，不可擅自拆除，並依醫師指示限制活動。  
4. 石膏固定期間，若出現疼痛、皮膚蒼白、冰冷或麻木感，應立即通知醫護人員評估處理。  
5. 臉部或頭部植皮患者，應依醫師或醫護人員指示調整飲食。  

#### 2. 供皮區照護步驟  
1. 傷口若出現滲濕，應立即更換敷料，但非每日更換。  
2. 換藥時，僅更換最外層紗布，最接近傷口的透明膜不需更換。  
3. 若使用代用敷料，應依醫師指定方式進行照護。  
4. 不可自行強行撕除敷料，以免造成皮膚損傷，延遲癒合。  
5. 若出現異常腫脹、疼痛、滲血或任何不適，應立即告知醫護人員。  

#### 3. 營養與健康指導  
- 建議多攝取高熱量、高蛋白食物，如魚、肉、蛋。  
- 糖尿病患者應依醫囑調整飲食，必要時會診營養師。  

#### 4. 回診與諮詢  
- 請依醫師指示按時回診。  
- 如有任何疑問，可隨時向醫護人員提出諮詢。  

## 注意事項/警示  
- **不可擅自拆除石膏**：若植皮區在四肢，需石膏固定 5–10 天，未經醫師同意不得拆除。  
- **避免自行撕除敷料**：供皮區敷料不可自行強行撕除，以免造成皮膚損傷。  
- **異常症狀應立即通報**：如出現疼痛、蒼白、冰冷、麻木、腫脹、滲血或不適，應立即告知醫護人員。  
- **特殊族群需特別注意**：糖尿病患者應視情況會診營養師，以確保營養攝取適切。

植皮部位手術後之照護

| 編號 | ND外-009 |
| :--- | :--- |
| 製訂日期 | 2003.07 |
| 六修日期 | 2019.05 |

一、 受皮區：
1. 手術後患肢必須抬高，以減輕疼痛，預防腫脹，有利傷口癒合。
2. 若植皮區在下肢者，須臥床休息，不可任意下床活動，須取得醫師同意後才可下床活動。
3. 若植皮區在四肢可能會有石膏固定約 5~10 天，無醫師同意請勿隨意拆除，並依醫師指示限制活動。
4. 石膏固定期間有疼痛、皮膚蒼白、冰冷、麻木感等情況，請隨時告知醫護人員評估及處理。
5. 臉部或頭部植皮之患者，請依醫師或醫護人員指示使用飲食。

二、 供皮區
1. 供皮區的傷口除滲濕需立即換藥外，不需要每天換藥。
2. 換藥時，只需更換最外層的紗布，最接近傷口內層的透明膜不需更換，或依醫師指定的代用敷料來進行傷口照護。
3. 供皮區所覆蓋之敷料，不可以自行勉強撕去，以免造成皮膚損傷，延遲傷口癒合。
4. 供皮區或植皮區有異常腫脹、疼痛、滲血或任何不適時，請立即告知醫護人員。

三、 營養補充：多進食高熱量、高蛋白飲食，如：魚、肉、蛋，糖尿病病人會視需要會診營養師。

四、 如您有任何疑問，請隨時向醫護人員提出諮詢，護理人員將會很樂意為您服務。

五、 請按醫師指示按時回診。
