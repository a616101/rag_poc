# 1701574181_997

## 主旨  
本文件提供慢性阻塞性肺病（COPD）患者居家照護的相關建議，涵蓋呼吸技巧、飲食、預防感染及生活保健等面向，以幫助患者減輕症狀、提升生活品質並預防急性發作。

## 重點整理

#### 一、噘嘴呼吸（腹式呼吸法）
- **目的**：幫助減輕呼吸困難，促進氣體交換。
- **方法**：
  1. 放鬆心情，以鼻吸氣。
  2. 吸氣時短暫停頓。
  3. 用嘴唇慢慢呼氣，如同輕吹蠟燭。
  4. 呼氣時間約為吸氣時間的2倍（吸-吐-吐-吐）。

#### 二、飲食治療
- **目標**：減輕呼吸困難，增強抵抗力。
- **建議**：
  - 選擇低醣、高脂食物，如堅果、雞蛋、魚、優格、橄欖油，避免血液中二氧化碳過高。
  - 攝取足夠水果與蔬菜，養成規律排便習慣，預防便秘。
  - 避免易脹氣食物，如豆類、洋蔥、花椰菜、牛奶，以防腹脹影響呼吸。
  - 採用少量多餐，每餐不宜過飽，餐後勿立即躺下。
  - 吃東西時細嚼慢嚥，若感到呼吸困難，應休息並依醫囑使用氧氣。

#### 三、避免感染
- **措施**：
  - 避免前往人多公共場所，遠離感冒患者。
  - 保持良好營養與充足睡眠，增強免疫力。
  - 依天氣適時添加衣物，避免受寒。
- **感染徵兆**（需立即就醫）：
  - 痰量增加、痰色變黃或變黏稠。
  - 發燒、咳嗽加劇。
  - 呼吸困難惡化。
  - 意識混亂或嗜睡。

#### 四、生活保健小祕方
- **活動與移動**：
  - 活動時減少身體移動，物品盡量靠近身邊。
  - 戒菸並避免二手菸，防止肺功能進一步惡化。
  - 遠離油煙與污染空氣。
- **活動與休息**：
  - 活動中應常休息，避免過度勞累。
  - 以漸進方式增加活動量，必要時使用氧氣。
- **衣物與裝備**：
  - 穿著寬鬆衣物，避免彎腰或綁鞋帶的鞋子。
  - 氧氣治療：使用流速1~2L/min，不宜超過3L/min，以免造成呼吸抑制。
- **氧氣設備使用注意**：
  - 氧氣設備應固定，避免翻倒。
  - 周邊勿放置熱源或電器用品。
  - 使用氧氣時嚴禁抽菸，且不可在氧氣用品上塗抹任何潤滑劑。

## 注意事項/警示
- **氧氣使用**：流速不可超過3L/min，以防呼吸抑制。
- **設備安全**：氧氣設備應固定，避免傾倒；周邊勿放置火源或電器。
- **緊急情況**：若出現痰量增加、變色、發燒、呼吸困難加劇或意識改變，應立即就醫。

## 全文

屏基醫療財團法人屏東基督教醫院
慢性阻塞性肺部疾病之居家照護

下列生活的居家保健是您應該知道的：

一、噘嘴呼吸：
1. 心情放輕鬆。
2. 用鼻子呼吸，作短暫的停頓。
3. 噘起嘴唇慢慢呼出氣體，就好像輕輕的吹蠟燭一樣。
4. 呼氣的時間約為吸氣時間的 2 倍（吸-吸-吐-吐-吐）。

二、飲食治療：
適當的飲食可以減輕呼吸困難的情形，還可以增加抵抗力。
1. 選擇低醣、高脂的食材，例如：堅果、雞蛋、魚、優格、橄欖油等等，可以避免血液中二氧化碳過高，使換氣不易。
2. 攝取足夠的水果、蔬菜、養成規則排便習慣，預防便秘以免造成呼吸困難。
3. 避免食用會脹氣的食物。例如：豆類、洋蔥、花椰菜、牛奶，以免腹脹造成呼吸困難。
4. 採少量多餐的方式，每餐不宜太飽，餐後不要立即躺下。
5. 吃東西要細嚼慢嚥，若吃東西時感到呼吸困難，可先休息暫停進食或依照醫師指示用氧氣。

三、避免感染：
1. 避免到人多的公共場合，避免與感冒的人接觸。
2. 要有良好的營養狀況和充足的睡眠，以增加抵抗力。
3. 適時添加衣物避免受寒。
4. 認識感染的徵象，如果痰量增加、變黃、變黏稠、發燒咳嗽、呼吸困難加劇或意識混亂、嗜睡的情形要儘速就醫。

四、生活保健小祕方：
1. 活動時減少身體移動、物品儘量靠近身邊，多使用輔助的裝置：如推車。
2. 戒菸及避免二手菸：是防止肺功能繼續迅速惡化最根本的方法。
3. 遠離油煙及污染的空氣。
4. 活動中要常常休息，避免過度勞累；應以漸進性增加活動量為宜，必要時使用氧氣。
5. 衣服要寬鬆易穿，避免彎腰或綁鞋帶的鞋子。
6. 氧氣治療：使用流速 1~2L/min 的氧氣，不宜超過 3L/min，以免容易造成呼吸抑制。
7. 氧氣設備要固定以免翻倒，週邊勿放置熱源或電器用品。在使用氧氣時嚴禁抽菸、氧氣用品不可塗上任何潤滑劑。
