# 1701604317_431

## 主旨  
本文旨在提供糖尿病患者及三高族群在外用餐時的飲食建議，透過中西式餐點的比較與選擇原則，幫助讀者在不影響生活品質的前提下，控制油脂、糖分與鹽分攝取，維持健康飲食。

## 重點整理  
- 需先熟悉並記住自己可攝取食物的份量。  
- 多選擇低油、清淡的食物。  
- 避免選擇「吃到飽」的餐廳，自我節制。  
- 注意藥物作用時間與用餐時間的配合。  
- 均衡攝取六大類食物。  
- 定時定量進食。  
- 多攝取富含纖維質的食物。  
- 食物烹調儘量以清淡為原則。  
- 避免吃過鹹及膽固醇高的食物。  
- 儘量減少含糖的烹調方法。  

## 流程/步驟  
### 中式自助餐及麵食  
1. 夾菜時，選擇較上層的食物，因菜汁通常含油脂較高。  
2. 油炸、沾粉或黏稠菜色應少攝取，食用時應剝去外皮。  
3. 可多選擇蔬菜，以增加飽足感。  
4. 忌用任何湯汁來泡飯，因湯汁中油脂、鹽份、味精等調味品含量高。  
5. 飯類與麵類多選擇乾飯或湯麵，少選擇吸油量大的炒飯與炒麵。  
6. 吃麵時，建議點些海帶、青菜以增加蔬菜攝取量。  

### 中式宴會  
1. 選擇蒸、煮、烤、涼拌等烹調方式，以降低油脂攝取。  
2. 濃湯、勾芡湯汁（羹類）、糖醋、蜜汁食物應少食，否則應瀝去湯汁，只吃料不喝湯。  
3. 多選擇湯內的青菜或盤飾上的蔬菜，因蔬菜熱量低，富含纖維質，可增加飽足感，也可將湯汁瀝乾或用白開水把菜濾過，減少油脂攝取。  
4. 內臟類、蟹黃、卵黃類、烏魚子、蝦頭、魚頭等膽固醇含量高，應減少攝取。  
5. 碎肉製品、加工食品如火腿、香腸或其他不明成份的食物不宜食用，應選擇帶骨的肉塊，實際吃肉量不多，較有滿足感。  
6. 避免含糖量高的果汁、汽水、飲料；無糖茶是不錯選擇，或以白開水取代。  
7. 宴會上的水果應適量攝取，可用來取代飯後甜點或甜湯。  

### 西餐  
1. 各種蔬菜沙拉應適量選用，沙拉醬一般含油多糖，應將沙拉裝小碟，勿加入生菜中。  
2. 西式濃湯含有大量澱粉與油脂，熱量極高，應選擇清湯較適合。  
3. 一個小餐包 = 法國麵包一小片 = 一份主食，因此麵包攝取量應控制，不選油脂含量高的大蒜麵包，也不塗果醬或奶油。  
4. 主菜以海鮮或雞肉為主，因含油量較少，烹調方式採用「烤」為主，避免油炸菜色，也可將外皮剝去；焗烤調味因起司、牛奶、油脂高，應避免食用。  
5. 罐裝飲料、汽水或新鮮果汁都不宜選用。  

### 速食  
1. 主餐可多選擇魚、雞（如麥香雞、麥香魚），因其含油量較低。  
2. 副餐的份量應選擇小份或小杯即可。  
3. 炸雞需剝去外皮，且份量要控制。  
4. 低卡飲料或咖啡不加奶油球是好的選擇。  
5. 減少購買促銷活動（如買一送一），不要貪便宜點了多得不必要的食物。  

## 注意事項/警示  
### 小叮嚀  
1. 均衡攝取六大類食物。  
2. 定時定量進食。  
3. 多攝取富含纖維質的食物。  
4. 食物烹調儘量以清淡為原則。  
5. 避免吃過鹹及膽固醇高的食物。  
6. 儘量減少含糖的烹調方法。  

### 特別提醒  
服用口服降血糖藥物及胰島素注射的病人，嚴禁空腹喝酒，因會增加低血糖發生的危險性。

## 全文

中西式外食健康吃

文／營養師 周莉菁

「老外老外老外，三餐老是在外」這句耳熟能詳的廣告詞大家一定都聽過，但…真的確定你吞下肚的是什麼嗎？現代人除了少數有些家庭會天天煮飯之外，其他人幾乎三餐都吃外食，上班族、警察、大學生…各行各業幾乎都是到外面去解決，吃青菜的機會少之又少！外食不外乎是快餐、速食、便當、夜市小吃…等等，尤其是學生族群，為了省錢，一盤30元的青菜都捨不得叫，蔬果量攝取太少，肉食反而吃得太多，而廣告商就是看中這一點，主打只要喝了果菜汁，一天營養就都均衡了！不過真的是這樣嗎？現在外食都高油、高糖、高鈉，尤其三高患者與糖尿病友更要注意外食的技巧喔！小心！一不注意之下，代謝症候群、心血管疾病、肥胖就悄悄找上門來。

糖尿病人外食原則

（1）需先熟悉牢記自己可吃食物的份量

（2）多選擇低油及清淡的食物

（3）少選擇可以吃到飽的餐廳，自我節制

（4）注意藥物作用時間與吃飯時間上的配合

中式宴會

1. 選擇蒸、煮、烤、燉、涼拌等烹調方式的菜色，降低油脂攝取量。

2. 濃湯、勾芡湯汁（羹類）、糖醋、蜜汁食物皆含有大量太白粉、油、糖、熱量高，盡量少吃，否則應瀝去湯汁，吃料不喝湯。

3. 多選擇湯內的青菜或盤飾上的蔬菜，因蔬菜熱量低，富含纖維質，可增加飽足感，也可以將湯汁瀝乾或用白開水把菜濾過，減少油脂攝取。

4. 內臟類、蟹黃、卵黃類、烏魚子、蝦頭、魚頭，等膽固醇含量高應減少攝取。

5. 碎肉製品，加工食品如火腿香腸或其他不明成份的食物不宜食用，選擇帶骨的肉塊，剝去骨頭，實際吃入肉量不多，較有滿足感。

6. 避免含糖量高的果汁、汽水、飲料；無糖茶是不錯選擇，或以白開水取代。

7. 宴會上的水果經過切，體積較小，一口一塊，適量攝取，可用來取代飯後甜點或甜湯。

中式自助餐及麵食

1. 夾菜時，夾取較上層的食物，因菜汁（湯）通常油脂、鹽份含量高。

2. 油炸、沾粉或黏稠菜色少攝取，食用應剝去外皮。

3. 可多選擇蔬菜，增加飽足感。

4. 忌用任何湯汁來泡飯，因湯汁中油脂、鹽份、味精等調味品含量高。

5. 飯類與麵類多選擇乾飯或湯麵，少選擇吸油量大的炒飯與炒麵。

6. 吃麵時，建議點些海帶、青菜增加蔬菜的攝取量。

速食

1. 主餐可多選擇魚、雞（如麥香雞、麥香魚），因其含油量較低。

2. 副餐的份量盡量選擇小份或小杯即可。

3. 炸雞需剝去外皮且份量要控制。

4. 低卡飲料或咖啡不加奶油球是好的選擇。

5. 減少購買促銷活動（如買一送一），不要貪便宜，點了多得不必要食物。

西餐

1. 各種蔬菜沙拉適量選用，沙拉醬一般多油多糖，所以應將沙拉醬小碟另外沾取，勿加入生菜中。

2. 西式濃湯含有大量麵粉與油脂，熱量極高，應選擇清湯較適合。

3. 一個小餐包＝法國麵包一小片＝一份主食，所以麵包攝取量應控制，不選油脂含量高的大蒜麵包也不要塗果醬或奶油。

4. 主菜以海鮮或雞肉為主，因含油量較少，烹調方式採用「烤」為主，不要選擇油炸菜色，也可以將外皮剝去；焗烤調料因起司、牛奶，油脂、熱量高，應避免食用。

5. 罐裝飲料、汽水或新鮮果汁都不宜選用。

小叮嚀～

1. 均衡攝取六大類的食物

2. 定時定量

3. 多攝取富含纖維質的食物

4. 食物烹調儘量以清淡為原則

5. 避免吃過鹹及膽固醇高的食物

6. 儘量減少含糖的烹調方法

服用口服降血糖藥物及胰島素注射的病人，嚴禁空腹喝酒，因會增加低血糖發生的危險性。
