# 一般外科

## 團隊簡介
本科可執行一般外科之手術，包括腹腔鏡微創手術及傳統開腹手術，提供各種一般外科疾病的診斷與治療服務，涵蓋消化系統、內分泌系統、乳房等疾病的診斷與治療。

## 團隊編制
- 醫師：一般外科專科醫師
- 護理/照護：專業護理團隊
- 醫事技術/其他：手術技術人員
- 特色分工：提供腹腔鏡微創手術及傳統開腹手術

## 服務與治療
### 服務範圍（疾病/問題）
- 闌尾炎
- 膽管結石
- 胃癌
- 胰癌
- 肝臟疾病、肝癌
- 大腸直腸癌
- 甲狀腺疾病
- 副甲狀腺疾病
- 疝氣
- 皮下腫瘤
- 乳房腫瘤、乳癌
- 腹膜透析

### 主要治療/處置
- 腹腔鏡闌尾切除術：微創手術治療闌尾炎
- 腹腔鏡膽管取石術：微創手術治療膽管結石
- 闌尾切除術：傳統手術治療闌尾炎
- 腹腔鏡膽囊切除：微創手術治療膽囊疾病
- 胃癌手術：提供胃癌的診斷與手術治療
- 胰癌手術：提供胰癌的診斷與手術治療
- 肝臟手術及肝癌手術：提供肝臟疾病及肝癌的診斷與手術治療
- 大腸直腸癌手術：提供大腸直腸癌的診斷與手術治療
- 甲狀腺手術：提供甲狀腺疾病的診斷與手術治療
- 副甲狀腺手術：提供副甲狀腺疾病的診斷與手術治療
- 疝氣及一般皮下腫瘤切除：提供疝氣及皮下腫瘤的診斷與手術治療
- 乳房腫瘤及乳癌手術治療：提供乳房腫瘤及乳癌的診斷與手術治療
- 腹膜透析導管置入手術：為需要腹膜透析的病患置入導管

### 特色門診/照護計畫（若有）
- 腹腔鏡微創手術：提供各種腹腔鏡微創手術，減少手術創傷
- 癌症手術：提供各種癌症的診斷與手術治療
