# 社會工作室

## 團隊簡介
社會工作室成立於民國59年8月，由社會服務部演變為社會工作部，再演變為社會工作室。初期目的為解決醫療費用問題，後來擴展為解決因疾病所產生的社會、心理、經濟、家庭等相關問題，以減輕壓力，增進醫療效益。人力編制：主任1名、社工（成員）、助理，負責推動醫療社會工作，提供專業服務給住院病患及其家屬。

## 團隊編制
- 醫師：不適用
- 護理/照護：不適用
- 醫事技術/其他：主任1名、社工（成員）、助理
- 特色分工：提供個案工作、團體工作、社區工作、安寧緩和、志願服務、福利諮詢服務、社會行政工作、教育訓練等服務

## 服務與治療
### 服務範圍（疾病/問題）
- 各種社會、家庭、心理問題
- 醫療救助
- 器官捐贈
- 安寧緩和
- 預立醫療照護
- 身心障礙鑑定

### 主要治療/處置
#### 個案工作
協助門診、急診、住院病患及其家屬，評估及處理相關社會、家庭、心理問題，以減少就醫障礙。服務項目包括：
1. 協助醫療瞭解
2. 經濟評估與補助
3. 情緒支持與心理輔導
4. 協助疾病調適
5. 輔導出入院及轉院
6. 家庭問題評估與處理
7. 社會資源運用
8. 就醫陪同
9. 協助行政工作/通報
10. 追蹤輔導

#### 團體工作
運用社會團體工作方法，促進病患及家屬經驗分享及情緒支持，增進疾病調適，改善生活品質。病友團體包括：
1. 乳癌病友團體（美麗人生俱樂部）
2. 腦中風病友團體
3. 腎臟科透析病友團體

#### 社區工作
整合社區社會資源，提供病患及家屬適當的人力、物力、財力及福利服務，並推廣衛生教育，表達醫院對社區的關懷，建立良好公共關係。服務項目包括：
1. 慈善單位之轉介
2. 器官捐贈、安寧療護、病人自主權利理念宣導

#### 安寧緩和
提供「預立安寧緩和醫療暨維生醫療抉擇意願書簽署」服務。「安寧緩和醫療」定義為：為減輕或免除末期病人之痛苦，施予緩解性、支持性之醫療照護，或不施行心肺復甦術，以增進其生活品質。

如何表達您的意願？
1. 與家人溝通，獲得支持
2. 至醫院服務台或社會工作室簽署意願書；醫院將正本寄至衛生福利部或推廣單位，簽署人保留副本
3. 相關問題可諮詢醫院社會工作室

相關網站：
- 財團法人中華民國（台灣）安寧照顧基金會
- 台灣安寧照顧協會

#### 志願服務
醫院於民國50年成立志願服務隊，提供病患及家屬可近性、適當性及主動性的服務，滿足民眾、行動不便者及長者的諮詢需求。服務項目包括：
1. 志工人力招募與管理
2. 志願服務獎勵推薦
3. 辦理志工教育訓練
4. 辦理志工聯誼活動

#### 福利諮詢服務
提供醫療團隊及民眾有關全民健保及社會福利相關資訊服務。

#### 社會行政工作
方案設計，協助醫院推動品質，制定改善措施之作業程序及流程。

#### 教育訓練
承辦院內外相關訓練課程及講授，接受大學系所參訪及督導社工實習生。服務項目包括：
1. 社會工作系所實習生之督導訓練
2. 承辦相關訓練課程
3. 院內外課程講授
4. 受理學校系所學生參訪或訪談
