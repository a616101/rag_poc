## 主旨  
本文件提供急性腰扭傷的處理建議，包含緊急處理步驟、日常預防措施及就醫指引，以協助患者緩解症狀並促進康復。

## 重點整理  
- 急性腰扭傷常因腰部肌肉、筋膜或韌帶受到突然過度牽拉所致。  
- 初期應以冰敷減輕疼痛與腫脹，後期可改用溫熱敷舒緩肌肉僵硬。  
- 提重物時應保持正確姿勢，避免直接彎腰。  
- 若症狀未改善，應儘早就醫。

## 流程/步驟  
#### 一、扭傷處理  
1. **休息與姿勢調整**  
   - 讓傷者臥床休息，雙膝下放置枕頭，以舒緩下背肌肉。  
   - 避免躺在過軟的床墊上。  

2. **冰敷（受傷72小時內）**  
   - 每天3–4次，每次10–15分鐘，以減輕疼痛與腫脹。  

3. **溫熱敷（受傷72小時後）**  
   - 每天2–3次，每次約15分鐘，用於緩解肌肉酸痛與關節僵硬。  
   - 注意溫度，避免燙傷。  

4. **姿勢減痛**  
   - 抱住雙膝，使身體呈圓球狀，可幫助減輕疼痛。  

5. **就醫指引**  
   - 若症狀未緩解，應儘早就醫。  

#### 二、預防建議  
- 提重物時應雙腳張開與肩同寬，保持腰部直立。  
- 蹲下時再取物，勿直接彎腰搬取物品。

## 注意事項/警示  
- 冰敷與溫熱敷時間需控制，避免過度。  
- 溫熱敷時應注意溫度，防止燙傷。  
- 若出現持續性疼痛或活動受限，應立即就醫。

腰扭傷之處理

| 編號 | ND外-033 |
| :--- | :--- |
| 製訂日期 | 2004.10 |

急性腰扭傷是腰部肌肉、筋膜、韌帶等軟組織，因外力作用突然受到過度牽拉而引起的。

一、扭傷處理：

1. 首先讓傷者，臥床休息於雙膝下放置枕頭舒緩下背肌肉並避免躺在過軟的床墊上。

2. 為減輕疼痛，受傷72小時內可冰敷，一天3-4次，每次10-15分鐘。

3. 受傷72小時後，肌肉會出現酸痛、關節僵硬等症狀，可協助溫熱敷以舒緩肌肉不適，一天2-3次，每次約15分鐘，須注意溫度避免燙傷。

4. 抱住雙膝使身體成圓球狀，可減輕疼痛。

5. 症狀未緩解須送醫就診。

二、平時提重物時，應雙腳張開與肩同寬，保持腰部直立，才蹲下拿取物品，勿直接彎腰取物。
