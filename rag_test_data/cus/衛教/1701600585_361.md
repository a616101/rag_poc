## 主旨
本文件旨在宣導遺傳性血管性水腫（Hereditary Angioedema, HAE）的相關知識，鼓勵社會大眾認識罕見疾病，並提供高風險族群（如已患病者、有家族史者或準備生育者）進行遺傳諮詢與產前檢查的資訊，以期早發現、早因應，用愛與尊重迎接每一個獨一無二的生命。本文件由財團法人罕見疾病基金會製作，旨在介紹遺傳性血管性水腫（Hereditary Angioedema, HAE）這項罕見遺傳疾病。透過一個真實的患者案例，說明疾病的症狀、診斷過程、遺傳模式與治療方式，並鼓勵患者正向面對疾病，尋求支持與幫助。

## 重點整理
- 遺傳性血管性水腫（HAE）是一種罕見的遺傳性疾病，可能導致嚴重的血管性水腫，危及生命。
- 遺傳性血管性水腫 (HAE) 是一種罕見的體染色體顯性遺傳疾病，發生率約為 1/50,000。
- 疾病由 C1 抑制蛋白（C1-INH）功能缺陷引起，導致血管滲透性增加，引發反覆性、局部性水腫。
- 水腫可發生於皮下、黏膜下，常見部位包括四肢、面部（眼瞼、嘴唇、舌頭）、生殖器、腸胃道及呼吸道（咽喉）。
- **呼吸道水腫**可能導致窒息，是危及生命的緊急狀況。
- **腸胃道水腫**會引起噁心、嘔吐及劇烈腹痛。
- **抗組織胺藥物及皮質類固醇**對 HAE 症狀無效。
- 疾病通常在兒童期出現，青春期惡化，終生持續。
- 治療包括新鮮冷凍血漿（FFP）、抗纖維溶解劑、基因重組 C1 抑制蛋白等。
- 患者應記錄每次發作情況，定期回診，並避免誘發因素。
- 即使沒有家族病史，生育時仍有 3% 的機率產出缺陷兒，其中 0.5% 可能患有罕見遺傳疾病。
- 每個生命都獨特且珍貴，及早發現並接受專業諮詢，可幫助家庭做好準備與因應。
- 財團法人罕見疾病基金會提供相關資源與支持，並與多個醫療機構合作提供遺傳諮詢服務。
- **HAE 的呼吸道水腫可能致命**，若出現喉嚨腫脹、呼吸困難，應立即就醫。
- **HAE 的症狀無法用常見的抗過敏藥物（如抗組織胺）或類固醇改善**。
- **壓力、創傷、感染、手術或荷爾蒙變化**可能誘發發作。
- **應避免使用 ACE 抑制劑**，此類藥物可能誘發或加劇 HAE 發作。

## 流程/步驟
1.  **認識疾病**：了解遺傳性血管性水腫（HAE）的症狀、成因與風險。
2.  **認識症狀**：出現不明原因的局部腫脹，可能伴隨刺痛感，但不癢。
3.  **評估風險**：
    -   若無家族病史，仍應配合產前檢查，因有 3% 機率產出缺陷兒。
    -   若有家族病史，應進行產前遺傳檢查、諮詢與生化遺傳檢驗。
4.  **尋求醫療協助**：若腫脹反覆發生，特別是影響呼吸或腸胃道，應儘速就醫。
5.  **進行基因檢測**：確診 HAE 需透過抽血進行基因檢測。
6.  **接受治療**：根據醫生建議，使用如新鮮冷凍血漿、基因重組 C1 抑制蛋白等藥物進行急性發作治療或長期預防。
7.  **建立生活管理**：記錄發作情況，避免壓力、感染、創傷等誘發因素，定期回診追蹤。
8.  **尋求專業諮詢**：若胎兒確診為遺傳性疾病，應尋求專業諮詢，評估自身狀況與未來照護能力。
9.  **早期干預**：及早發現並進行專業諮詢，可及早因應與準備，用完整的愛迎接生命。

## 注意事項/警示
-   遺傳性血管性水腫可能導致喉頭水腫，引發呼吸困難，屬急症，需立即就醫。
-   有家族病史者應特別重視遺傳諮詢與檢查。
-   每個生命都應被尊重與珍惜，應以完整的愛迎接每一個獨一無二的小生命。
-   **HAE 的呼吸道水腫可能致命**，若出現喉嚨腫脹、呼吸困難，應立即就醫。

-   **壓力、創傷、感染、手術或荷爾蒙變化**可能誘發發作。

罕見遺傳疾病不是怪病、更不應予以歧視。
「罕見遺傳疾病」，是生命傳承中的小小意外。人體內約有兩萬五千個基因，藉著DNA（去氧核醣核酸）準確的複製，把基因代代相傳。不過，其中若有基因發生變異，就可能在傳宗接代時，把有缺陷的基因帶給子女，造成遺傳性疾病。

在台灣，罕見疾病的定義是年盛行率在萬分之一以下的疾病。全世界已知有一萬一千多種人類遺傳疾病，其中大多非常罕見。罕見遺傳疾病發生機率雖小，卻是生命傳承中無可避免的遺憾。雖然這不幸只降臨在少數人身上，卻是每個人都必須承擔的風險。

幸好，手術成功，喉嚨腫脹消退，主治醫師仔細詢問俊廷的病史，發現俊廷的媽媽和妹妹都有類似症狀，媽媽甚至更嚴重，腫的部位遍及臉和全身。醫師推測是遺傳疾病的情況下，全家人抽血驗基因，最終確診是罹患「遺傳性血管性水腫」。一場急性發作讓他多年來無解的毛病終於知道如何醫治，無疑是上帝開的明亮之窗。

確診後，俊廷了解到此疾病與免疫功能有關，大學時常熬夜、壓力大，發作較為頻繁，每2～3個月便會發作一次。現在每個月回診拿口服藥，副作用輕微，久久發作時，腫脹程度也較以往輕微、持續的時間也較短。

走過人生關卡，俊廷始終保持正向、樂觀的心態，如今疾病控制得很好，現年35歲的他，成立遺傳性血管性水腫病友聯誼會，希望能找出更多同樣疾病的病友，幫助他們在診斷與治療的過程中能更加順利，同時俊廷正擘劃人生下一階段的藍圖呢！

遺傳性血管性水腫

遺傳性血管性水腫（Hereditary Angioedema, HAE）是一種補體缺陷的免疫不全疾病。補體由許多分子組合，是存在於人體血清、組織液及細胞膜表面具有活性的蛋白質，在人體中進行免疫調節抵禦外來微生物的侵襲。C1抑制蛋白（C1 esterase inhibitor, C1-INH）是補體的調節因子之一，當C1抑制蛋白的功能發生缺陷，影響人體免疫調節，使血管滲透性增加，造成發炎反應。HAE發生率為1/50,000，大多為體染色體顯性遺傳，第一與第二型HAE其中約有20-25%患者為新突變（de novo）。

主要臨床特徵為：反覆發作的皮下或黏膜下水腫，主因於過量血管外積液，阻塞血液或淋巴液的流動導致四肢、面部（眼瞼、嘴唇、舌頭）、生殖器、腸胃道或呼吸道（咽喉）腫脹。發作時皮膚會刺痛但不癢、或有邊緣性紅斑與局部硬腫。當有腸胃道腫脹會使患者出現噁心、嘔吐及劇烈腹痛的症狀；呼吸道（咽喉）腫脹造成疼痛、吞嚥困難和發音障礙，嚴重導致窒息而危及性命。此外抗組織胺藥物及皮質類固醇治療無法有效改善HAE症狀也為特徵之一。

通常在兒童時期出現症狀，於青春期階段惡化，終生持續，平均每1～2週會發作一次，約3～5天症狀才會緩解。HAE症狀發作的誘因可以為壓力、創傷、感染、雌激素及過度疲勞。HAE目前分為三種型別：

| 型別 | HAE C1-INH 缺乏 | HAE C1-INH 正常 |
| :--- | :--- | :--- |
| | 第一型 | 第二型 | 第三型 |
| 患病比例 | 85% | 15% | 非常稀少 |
| 影響基因 | SERPING1 | SERPING1 | FXII, PLG, ANGPTI |
| 補體 C4 濃度 | 下降 | 正常或上升 | 正常 |
| C1-INH 濃度 | 下降 | 正常或上升 | 正常 |
| C1-INH 功能 | 下降 | 下降 | 正常 |

治療包括：新鮮冷凍血漿（fresh frozen plasma, FFP）、雄性激素（androgen）、抗纖溶劑（antifibrinolytic agents）、血漿製成C1抑制蛋白（plasma-derived C1 inhibitor）、基因重組C1抑制蛋白（recombinant human C1 inhibitor）、激肽系統調節劑、單株抗體等作為緩解HAE急性發作、短期或長期預防，目前多種新藥尚在研發中。

建議患者記錄每次發作情況，在正常生活狀況下盡量避免誘發因子，定期回診追蹤並配合治療，以改善生活品質。
