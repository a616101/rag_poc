# 耳鼻喉科

## 團隊簡介
現有專科醫師3人，聽力檢查技術員1人。人雖少卻滿懷熱情，為屏東人打拼。

## 團隊編制
- 醫師：專科醫師3人
- 護理/照護：專業護理團隊
- 醫事技術/其他：聽力檢查技術員1人
- 特色分工：人雖少卻滿懷熱情，為屏東人打拼

## 服務與治療
### 服務範圍（疾病/問題）
- 耳鼻喉領域內所有門診治療項目
- 各項聽覺障礙
- 眩暈症
- 耳科疾病：外耳及耳道疾病、中耳疾病
- 鼻科疾病：鼻腔良性腫瘤、鼻中膈、鼻甲、鼻息肉、鼻竇炎、鼻咽癌、過敏性鼻炎
- 口腔疾病：舌唇懸雍垂疾病、打鼾
- 咽喉疾病：鼻咽、口咽、喉咽病變、喉頭疾病、扁桃腺疾病、聲帶疾病

### 主要治療/處置
- 耳鼻喉領域內所有門診治療項目及各項聽覺障礙、眩暈症等專業檢查（附上所有皆可做之項目：純音聽力檢查、耳聲傳射、助聽器評估、聽音阻抗檢查、眼振圖檢查）及鑑定
- 耳鼻喉科內各項手術：
  - 耳科：外耳及耳道之各式手術、中耳穿刺術及通氣管置入術
  - 鼻科：鼻腔良性腫瘤、鼻中膈、鼻甲、鼻息肉、鼻竇炎、鼻咽癌切片、過敏性鼻炎高頻熱凝治療，等各式手術
  - 口腔（含舌唇懸雍垂）：各式病變之切片檢查、良性腫瘤切除；打鼾之高頻熱凝治療及懸雍垂咽整形術等
  - 各式鼻咽、口咽、喉咽病變之切片檢查；喉頭內視鏡檢查；良性腫瘤切除；扁桃腺摘除；聲帶結節或息肉之顯微手術切除等

### 特色門診/照護計畫（若有）
- 聽力檢查：提供純音聽力檢查、耳聲傳射、助聽器評估、聽音阻抗檢查
- 眩暈檢查：提供眼振圖檢查
- 過敏性鼻炎治療：提供過敏性鼻炎高頻熱凝治療
- 打鼾治療：提供打鼾之高頻熱凝治療及懸雍垂咽整形術

## 檢查項目
### 聽力及內耳功能檢查
1. 純音聽力檢查
2. 耳聲傳射
3. 助聽器評估
4. 聽音阻抗檢查
5. 眼振圖檢查

## 設備與特色
1. 對應上述門診及手術之各種必備器械
2. 聽力及內耳功能之各項裝備：
   - 純音聽力檢查
   - 耳聲傳射
   - 助聽器評估
   - 聽音阻抗檢查
   - 眼振圖檢查
