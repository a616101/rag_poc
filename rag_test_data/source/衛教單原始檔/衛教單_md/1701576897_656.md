# 1701576897_656

## 主旨
屏基醫療財團法人屏東基督教醫院提供急性腸胃炎（嘔吐、腹瀉）病人返家後的注意事項與照護指引。

## 重點整理
- 經醫師診斷為急性腸胃炎，返家後應注意個人衛生、飲食與藥物使用。
- 飲食應由清淡、易消化的食物開始，並依症狀漸進恢復。
- 若出現特定警示症狀，應立即就醫。

## 流程/步驟
1.  **初期禁食與休息**：除口服藥外，禁食4-8小時，讓腸胃休息。
2.  **症狀出現時**：
    -   嘔吐後請漱口，維持口腔清爽。
    -   若有嘔吐，半小時內暫不進食。
    -   避免食用油膩、辛辣或刺激性食物。
    -   可以白稀飯為主食，避免肉鬆等油膩食物。
3.  **補充水分**：發生嘔吐或腹瀉的第一天，應多補充水分，如溫開水、稀釋的運動飲料或低糖果汁，暫時勿喝乳製品以防脹氣。
4.  **漸進恢復飲食**：從第二天起，觀察症狀，可從流質（如稀飯）過渡到軟質食物。
5.  **服藥注意**：遵照醫師指示服藥，特別是止瀉藥，應留意排便狀況，若排便變成糊狀，應停止服用。

## 注意事項/警示
如有以下任何情況，請儘速就醫或回到本院複診：
1.  持續腹瀉超過三日。
2.  發冷、發燒。
3.  劇烈腹痛。
4.  嚴重嘔吐，以致完全無法進食。
5.  嘔吐物出現血絲。
6.  皮膚乾燥，小便變少。
7.  出現黑便、血便或是草莓醬狀大便。

屏基醫療財團法人屏東基督教醫院
嘔吐、腹瀉病人返家注意事項

編號 ND 急-009
製訂日期 2003.08
六修日期 2019.06

經醫師診斷您的嘔吐、腹瀉是屬於急性腸胃炎，返家後請注意下列事項：

一、用餐前及上完廁所後，務必使用肥皂洗手，避免疾病的傳播。

二、除了口服藥外，請禁食 4-8 小時讓腸胃適當的休息。若仍有嘔吐情形請繼續禁食，嘔吐後請漱口，維持口腔清爽，增加舒適。

三、若有嘔吐情形，半小時內暫不進食，並避免吃油膩、辛辣或刺激性食物，可先以白稀飯為主（注意：肉鬆屬於油膩食物，應避免食用）。

四、發生嘔吐或腹瀉症狀的第一天請多補充水分，如：溫開水、加水稀釋的運動飲料、含糖量較低的果汁等，暫時勿喝乳製品以防脹氣不適。

五、第二天起觀察腹瀉改變情況，可漸進恢復飲食吃流質→稀飯→軟質。

六、遵照醫師指示服藥：服用止瀉藥時，為避免用藥過量造成便秘，故應小心留意排便狀況，如果排便變成糊狀時，可停止服止瀉劑。

七、如果有以下情況，請儘速就醫或回到本院複診：

1. 持續腹瀉超過三日。
2. 發冷、發燒。
3. 劇烈腹痛。
4. 嚴重嘔吐，以致完全無法進食。
5. 嘔吐物出現血絲。
6. 皮膚乾燥，小便變少。
7. 出現黑便、血便或是草莓醬狀大便。

屏基關心您 祝您早日康復
