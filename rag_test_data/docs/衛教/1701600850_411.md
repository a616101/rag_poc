# 1701600850_411

## 主旨  
兒童因好奇心強，容易誤食藥品、硬物或小玩具，導致呼吸道阻塞或中毒。若未及時處理，可能造成腦部缺氧、神經損傷，甚至危及生命。家長應掌握正確的急救步驟，並從日常防護做起，預防意外發生。

## 重點整理  
- 小孩誤食或異物吸入是急診常見問題，尤其三歲以下兒童風險最高。  
- 一旦出現呼吸困難，應立即施以哈姆立克急救法。  
- 預防措施包括妥善存放危險物品、加強安全教育與家庭環境管理。  
- 第一時間正確處置，可挽救寶寶生命，避免長期後遺症。

## 流程/步驟  
### 一歲以下小孩呼吸道阻塞處理步驟：  
1. 將嬰兒俯臥於大人手臂上，大腿固定。  
2. 用另一手的手腕拍打嬰兒背部五下。  
3. 打開嬰兒嘴巴，檢查是否有異物。  
4. 若看到異物，立即用手指將其清除。  

### 一歲以上小孩呼吸道阻塞處理步驟：  
1. 成人站在小孩身後，一手握拳，置於肋骨與肚臍之間。  
2. 另一手環握拳頭，向內向上施力，進行五次快速按壓。  
3. 重複動作，直至異物排出。

## 注意事項/警示  
- 避免讓幼兒接觸果核、藥品、硬幣、小玩具等易誤吞物。  
- 家中熱源、電源、瓦斯開關應隨時關閉，並避免孩童觸及。  
- 不要用空寶特瓶盛裝清潔劑、農藥等化學物品，以免孩子誤認為飲料。  
- 端熱湯或放熱水時，應特別注意孩子的動向，防止燙傷。  
- 家長應從小進行安全教育，以身作則，教導孩子辨識危險。  
- 若發現小孩誤食藥品或錢幣，應立即送醫處理。  
- 若出現呼吸困難，應立即使用哈姆立克急救法，爭取黃金救援時間。

小孩誤食意外
把握第一時間處理

成長中的幼兒對世界充滿好奇、對所有事務感到新鮮，甚麼東西都想要「嚐」試一下，也因此小孩的意外事件中，「誤食」成為醫院急診室最常處理的問題之一。尤其三歲以下的小孩很容易看到東西就往嘴裡塞，小孩若誤食藥品，依藥性的不同，可能帶來心臟、肝、腎等器官功能的損傷，或是誤食硬幣、玩具及其他不易咀嚼的食物（如：蒟蒻果凍）被噎住時，一旦無法呼吸，腦部缺氧超過四分鐘，就可能留下神經病變的後遺症，影響孩子日後的發育，因此為人父母者絕不可掉以輕心。

對兒童來說，家中也是個危險的地方。例如燙傷、燒傷、跌倒、氣管嗆到異物、誤食中毒等多數是發生在家裡。屏基急診室提出防治兒童意外的幾項原則供父母參考：

- 避免果核或含有韌性物質食物給幼兒吞下，以免嗆入或堵塞呼吸道。
- 注意家中擺設，任何危險物品或藥品不要讓幼兒伸手可及，最好能上鎖存放。
- 注意家中的熱源、電源、瓦斯等開關，應隨時關好，並且避免孩童觸及。
- 從小施予安全教育，要有良好的身教，並從小教導孩子如何避免危險。
- 端熱湯或放熱水時應特別注意孩子的動向。
- 勿用喝光飲料的寶特瓶盛裝清潔劑、農藥等化學物品，小孩無法分辨內容物，很容易誤以為是茶飲、可樂、汽水。

家長一旦發現小孩誤食藥品或錢幣等物，應立即送醫處理，若是被噎到，導致呼吸困難時，應運用哈姆立克急救法，處理幼兒呼吸道阻塞意外，搶在第一時間挽救寶寶的性命。

○一歲以下的小孩梗塞，處理步驟是：
大人手臂放在大腿上讓嬰兒俯臥，用另一手手腕拍背五下，接著打開嬰兒嘴巴查看，若看到異物再用手指將清出。

○一歲以上的孩子梗塞，處理步驟是：
大人站在小孩身後，一手握拳置於小孩肋骨與肚臍間，另一手環握，向內而上搓揉五下，促使異物吐出。

沒有自我保護能力的幼兒，很可能因照顧者的疏忽而造成一生的遺憾；因此，如何預防兒童意外事件的發生，並且在第一時間做出正確且立即的處置，是每位家長的必修課！
