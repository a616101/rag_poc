# 整形外科

## 團隊簡介
整形外科醫師具備豐富理論基礎、細膩思維、敏銳判斷、非凡技術、強烈使命感，安慰病患心靈並重建完美外觀。本院提供腫瘤、車禍外傷或各種外傷病患的切除及重建手術，並提供各種慢性疾病造成的頑固傷口專責照護。
