## 主旨
本文件為中風患者出院後的照護規劃指引，提供居家、機構及外籍看護等多種照護選項，以協助患者及家屬做好出院準備。

## 重點整理
- 本院提供慢性住院個案的連續性服務，中風患者在急性期後需進行長期照護與復健。
- 家屬應在住院期間學習照護技巧，並提前做好心理與環境準備。
- 若在家照顧有困難，可選擇以下四種照護方式：居家照護資源、機構照護、外籍看護工，或洽詢出院準備服務組。

## 流程/步驟
1.  **評估需求**：根據患者病情穩定度及家屬照護能力，評估是否需要外部支援。
2.  **選擇照護方式**：
    -   **居家照護資源**：包括居家服務、復能、喘息服務、日間照護、交通接送、輔具租借、居家無障礙改善等。
    -   **機構照護**：針對病情穩定但家屬無法照顧的個案，提供24小時住宿、復健及生活照顧。
    -   **外籍看護工**：經主治醫師評估符合資格者，可由家屬委託合法仲介公司申請。
3.  **申請與轉介**：依需求向相關單位申請，並可洽詢本院出院準備服務組取得詳細資訊。

## 注意事項/警示
- 居家照護資源的申請與轉介，請洽「屏東縣長期照護管理中心」（簡碼 1966）。
- 外籍看護工的申請需經主治醫師評估並開立相關證明。
- 所有照護方式的詳情，可向本院「出院準備服務組」查詢。
- 諮詢時間為週一至週五，上午8:00至下午17:00。

屏基醫療財團法人屏東基督教醫院
中風個案之社會資源簡介

| 編號 | ND 內-036 |
| :--- | :--- |
| 製訂日期 | 2003.11 |
| 八修日期 | 2019.03 |

• 前言：
中風是一種來得很突然，讓人措手不及的疾病，本院社會工作員及出院準備服務人員，在院內與各科醫護人員組成醫療團隊，經常就患者之問題與病情相互聯繫討論，針對患者之需要提供相關的社區資源與福利，並以專業知識與技能來協助患者改善有關疾病適應、家庭、情緒、經濟等問題。

• 健保相關規定：
依現行「全民健康保險重大傷病範圍」，腦中風限急性發作後一個月內免部份負擔，於出院時會提供「重大傷病證明單」；並將相關資訊鍵入健保卡內。

• 殘障手冊：
中風患者有肢體癱瘓、失語……等神經後遺症者（由醫師評估鑑定時機），可持個案一吋照片 3 張、身分證及印章（委託他人者須攜帶委託人之身分證及印章），至戶籍所在縣市之任一鄉、鎮、區公所申請身心障礙鑑定表，經本院專科醫師及第二鑑定人員進行鑑定後，本院將鑑定表送至衛生局進行審核，後由公所核發身心障礙證明，並提供身障福利相關資訊。

• 出院計劃：

本院為提供慢性住院個案連續性服務，然而中風個案在急性期過後常需要時間照顧及復健，家屬需在患者住院時配合醫護指導，習得照顧技巧，及早做好出院的心理及環境準備，若回家照顧有困難者，以下幾項照顧方式可供參考：

1. 居家照顧資源：居家服務、居家復能、喘息服務、日間照護、交通接送、輔具請購或租借、居家無障礙環境改善服務；依需求提供相關資訊與轉介申請諮詢可洽屏東縣長期照護管理中心（簡碼 1966）。

2. 機構照顧：針對病情穩定而家屬無暇照顧之個案，提供其 24 小時住宿、復健及生活照顧等綜合性服務；可依需要選擇養護機構或護理之家。

3. 外籍看護工：經主治醫師評估符合申請者會開立相關證明給家屬，可委託合法的仲介公司代辦申請。

4. 以上照顧方式詳情，可洽本院出院準備服務組。
