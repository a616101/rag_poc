# 藥劑科

## 團隊簡介
本科提供完整的藥事服務，包括門診調劑、病房調劑、臨床藥學、特殊處方調劑及藥品管理等服務，確保病患用藥安全及品質。

## 團隊編制
- 醫師：不適用
- 護理/照護：不適用
- 醫事技術/其他：藥師、調劑人員
- 特色分工：分為門診調劑組、病房調劑組、臨床藥學、特殊處方組、藥品管理組

## 服務與治療
### 服務範圍（疾病/問題）
- 各種疾病用藥

### 主要治療/處置
#### 門診調劑組
1. 負責門診就診病人調劑發藥，包括拿連續慢箋領藥患者及慢處預約包藥
2. 為高齡者及身心障礙者的設置便捷的領藥博愛窗口
3. 在病人領藥同時，給予基本的用藥指導
4. 門診領藥區，隨時會有相關藥物訊息的公告/衛教螢幕

#### 病房調劑組
1. 負責住院病患的單一劑量調劑及出院時的用藥衛教
2. 住院病人的訪視用藥指導
3. 負責急診病人藥物調劑

#### 臨床藥學
1. 提供病人、醫療人員及一般民眾相關用藥諮詢/社區衛教
2. 臨床訪視用藥評估/藥物不良反應通報
3. 編輯醫院處方集
4. 負責新進人員/實習生/科內學術教學

#### 特殊處方組（負責：化療、PCA、全靜脈營養調劑）
1. 調配TPN（全靜脈營養輸注）
2. 負責臨床藥物血中濃度監測計算
3. 審核、調配化療處方
4. 編撰化療藥品處方集

#### 藥品管理組
- 負責院內所有藥品的庫存管理、效期之控管

### 特色門診/照護計畫（若有）
- 高齡者及身心障礙者便捷領藥博愛窗口
- 用藥指導及用藥諮詢服務
- 臨床訪視用藥評估
- 藥物不良反應通報

## 檢查項目
- 用藥評估：臨床訪視用藥評估
- 藥物血中濃度監測：臨床藥物血中濃度監測計算

## 設備與特色
- 調劑設備：提供門診及病房調劑服務
- 包藥設備：提供慢處預約包藥服務
- TPN調配設備：調配全靜脈營養輸注
- 化療調配設備：審核、調配化療處方
- 門診領藥區設有相關藥物訊息的公告/衛教螢幕

## 品質與安全
- 負責門診就診病人調劑發藥，確保用藥安全
- 在病人領藥同時，給予基本的用藥指導
- 住院病患的單一劑量調劑，確保用藥正確性
- 臨床訪視用藥評估，確保用藥適當性
- 藥物不良反應通報，確保用藥安全
- 負責院內所有藥品的庫存管理、效期之控管

## 就醫資訊
- 門診：提供門診調劑發藥服務，包括連續慢箋領藥及慢處預約包藥
- 住院：提供住院病患單一劑量調劑及出院用藥衛教
- 急診/24 小時：提供急診病人藥物調劑
- 轉介：提供用藥諮詢服務
