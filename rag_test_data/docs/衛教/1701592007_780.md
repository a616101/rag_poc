## 主旨
本文件為肩關節康復運動指導，包含七項針對患側上肢及肩部的復健運動，旨在促進關節活動度與肌肉控制，適用於特定醫療情境（如中風後復健）。運動內容涵蓋鐘擺運動、爬牆運動、甩手轉圈運動、肩關節內轉與外轉運動、肩胛骨加強運動、肩關節內轉伸展運動及肩關節外轉伸展運動，均建議每日進行3次（早、中、晚），每次重複10次。

## 重點整理
- 本文件提供七項自我運動，均需每日進行3次（早、中、晚）。
- 每項運動均包含具體動作要領與圖示，以幫助正確執行。
- 運動強調患側手臂的活動，並可利用對側手協助完成動作。
- 所有運動應在無痛或輕微不適的範圍內進行。
- 建議在專業醫療人員指導下開始執行。
- 如運動中出現疼痛、腫脹或不適，應立即停止並諮詢醫師。
- 提供諮詢電話與QR碼，以供進一步協助。

## 流程/步驟
### 1. 鐘擺運動
- 身體前傾，對側手扶於固定物上。
- 患側手自然下垂，模擬鐘擺。
- 依以下方向各擺動十次：
  - 前後
  - 左右
  - 順時鐘
  - 反時鐘
- 每日3回（早、中、晚）。

### 2. 爬牆運動
- 面對牆壁，將患側手指沿著牆面漸漸向上爬（圖二-1）。
- 側對牆壁，手心朝上，將患側手指沿著牆面漸漸向上滑（圖二-2）。
- 每日3回（早、中、晚）。

### 3. 甩手轉圈運動
- 雙腳前後站立。
- 患側手臂放鬆，自然甩動兩下。
- 向前轉一圈。
- 再自然甩動兩下，向後轉一圈。
- 連續重複十次。
- 每日3回（早、中、晚）。

### 4. 肩關節內轉與外轉運動
- 將一條毛巾置於身後。
- 右手上、左手下，同時將毛巾在背後上下拉動十次（圖四-1）。
- 換為左手上、右手下，上下拉動十次（圖四-2）。
- 每日3回（早、中、晚）。

### 5. 肩胛骨加強運動
1. 站立，雙手交疊置於頭後。
2. 手肘儘量向後張開（圖五-1）。
3. 再將手肘向前置於頭部兩側（圖五-2）。
4. 重複此動作十次。
5. 每日進行3回（早、中、晚）。

### 6. 肩關節內轉伸展運動
1. 將患側手臂置於身後。
2. 儘量向上伸展至對側肩胛骨處；若無法達成，可由對側手協助向上抬高。
3. 慢慢進行內轉伸展運動（圖六）。
4. 連續做十次。
5. 每日進行3回（早、中、晚）。

### 7. 肩關節外轉伸展運動
1. 將患側手抬起，經由頭部後面慢慢滑動至對側耳朵，甚至嘴巴。
2. 進行外轉伸展運動（圖七）。
3. 連續做十次。
4. 每日進行3回（早、中、晚）。

## 注意事項/警示
- 所有運動應在無痛或輕微不適的範圍內進行。
- 建議在專業醫療人員指導下開始執行。
- 若無法自行完成動作，可請對側手協助。
- 如運動中出現疼痛、腫脹或不適，應立即停止並諮詢醫師。
- 如有不適，應立即停止並諮詢專業人員。

一、什麼是五十肩？
「五十肩」又稱冷凍肩、黏連性肩關節炎，是肩關節發生發炎、沾黏以及活動困難。主因不明，但因好發於中年人，故又稱五十肩。事實上其他年齡層的人也會得此病。年輕患者致病因與肩旋轉腱肌腱炎有關，通常是先有肌腱炎後，因為肩膀疼痛而不敢活動，使得關節逐漸沾黏形成五十肩。

二、五十肩症狀有哪些？
剛發生時，常感肩膀疼痛，局部怕冷，在晚上或活動時更痛，夜間睡眠時因肩膀壓迫而痛醒，疼痛感會向頸部及上肢擴散以致於肩關節活動受限，導致穿衣、搔背、梳頭髮都有困難，因此患者會藉著減少活動來減輕疼痛因而使肩關節活動逐漸喪失。

三、哪些人容易發生五十肩？
1. 性別：女性多於男性，尤其常見於家庭主婦和停經婦女。
2. 退行性病變及免疫、代謝性疾病：常見於高血壓、糖尿病、心血管硬化、頸椎疾病等病患。
3. 職業：常見於非體力勞動者，如從事電腦及文書工作者。
4. 外傷史：約有 30% 的患者肩關節曾有輕度創傷，肩部或上肢骨折或肩部手術後。

四、常見治療方式有哪些？
1. 口服非類固醇抗發炎藥物。
2. 局部關節內注射類固醇。
3. 物理治療：急性期以冰敷緩解疼痛，之後採溫熱敷減少肌肉痙攣。
4. 運動：「五十肩」的治療，除了熱療、吃藥或打針之外，合併運動是最有效的方法。

五、自我運動介紹：

1. 鐘擺運動
身體前彎，對側手扶於一固定物上，患側手自然下垂，有如鐘擺，以前後、左右、順時鐘、反時鐘等方向之擺動各十次，每日 3 回（早、中、晚）。（圖一）

2. 爬牆運動
面對牆壁，將患側手指沿著牆面漸漸向上爬（圖二-1）。側對牆壁手心朝上，將患側手指沿著牆面漸漸向上滑（圖二-2），每日 3 回（早、中、晚）。

3. 甩手轉圈運動
雙腳前後站立，患側手臂放鬆，自然甩動兩下，向前轉一圈，再自然甩動兩下，向後轉一圈，如此連續做十次（圖三），每日 3 回（早、中、晚）。

4. 肩關節內轉與外轉運動
一條毛巾置於身後，右手上左手下，同時將此毛巾在背後上下拉動十次（圖四-1），再換左手上右手下，上下拉動十次（圖四-2），每日 3 回（早、中、晚）。

5. 肩胛骨加強運動

站立，雙手交叉置於頭後，手肘儘量向後張開（圖五-1），再向前置於頭部兩側（圖五-2），重複運動十次，每日3回（早、中、晚）。

6. 肩關節內轉伸展運動

將患側手臂置於身後，儘量向上至對側肩胛骨處，若不能達到，則以對側手幫忙向上抬高，慢慢做內轉伸展運動（圖六），連續做十次，每日3回（早、中、晚）。

7. 肩關節外轉伸展運動

將患側手抬起，經由頭部後面慢慢滑動至對側的耳朵，甚至嘴巴，做外轉伸展運動（圖七），連續做十次，每日3回（早、中、晚）。
