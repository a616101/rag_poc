## 主旨  
本文件旨在協助家長了解兒童點滴注射的照護要點，以減少「漏針」情況，提升治療順利度與兒童舒適度。

## 重點整理  
- 兒童點滴通常每4天更換注射部位。  
- 护理人員會定期檢查點滴注射部位及流速。  
- 由於兒童血管細，難以穿刺，需注意避免「漏針」導致重新注射。  
- 家長應配合照護，協助減少不適與意外。

〔無法辨識〕

## 注意事項/警示  
1. **點滴瓶高度**：應高於注射部位90公分以上。若發現點滴不滴、滲水、滲血，或注射部位腫脹、發紅、疼痛，請立即告知護理人員處理。  
2. **使用點滴控制器**：勿自行調整按鍵，以免點滴流速錯誤；於病房時，點滴控制器需隨時插電使用。  
3. **避免活動與拉扯**：禁止兒童自行站立在活動點滴架車輪上，以防跌倒受傷。  
4. **防止點滴脫落**：避免兒童注射部位過度活動或拉扯，且不要讓兒童在床上跳躍。

兒童點滴注射照顧

編號 ND 兒-026
製訂日期 2011.06
三修日期 2022.10

親愛的家長平安：

由於您的孩子生病了，經醫師診斷評估後需注射點滴，小孩點滴通常每4天會更換注射部位，期間護理人員會定期檢視點滴注射部位及點滴滴注流速，但由於小孩血管細較難打上，為減少小孩因「漏針」而需重新注射的機會，並讓您在後續的照顧更得心應手，以下幾點事項需要您共同注意。

- 點滴瓶高度應高於注射部位90公分以上，若發現點滴不滴、滲水、滲血，或注射部位有腫脹、發紅、疼痛之情況時，請立即告知護理人員處理。
- 使用點滴控制器，勿自行調整按鍵，以避免點滴流速錯誤；另於病房時，點滴控制器需隨時插電使用。
- 禁止讓小孩自行站立在活動點滴架之車輪上，以免跌倒受傷。
- 避免小孩注射部位過度活動或拉扯，且不要讓小孩在床上跳躍，避免點滴管脫落。
