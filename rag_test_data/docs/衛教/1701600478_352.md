## 主旨  
本文件為屏基醫療財團法人屏東基督教醫院提供的照護指引，說明如何正確為嬰幼兒進行拍背排痰，以幫助鬆動濃稠呼吸道分泌物，促進痰液排出。內容涵蓋拍背的必要性、合適時間、操作方法、注意事項及相關資訊。

## 重點整理

| 項目 | 說明 |
| :--- | :--- |
| **目的** | 幫助嬰幼兒鬆動濃稠痰液，促進呼吸道分泌物排出。 |
| **合適時間** | 每天至少 7~8 次，或每 2 小時一次，每次 5~10 分鐘。飯前半小時及飯後 1 小時內避免拍背，以防嘔吐。 |
| **拍背方法** | 手掌彎成杯狀，利用手腕關節自然活動，穩穩拍打。可使用拍痰器。 |
| **拍背聲音** | 正確姿勢：發出低沉「碰碰」或「ㄩㄩㄩ」聲，表示痰液被引出。錯誤姿勢：發出「啪啪」或「ㄊㄧㄚ ㄊㄧㄚ」聲。 |
| **拍背部位** | 左右肺下葉前底，側躺並抬高臀部約 20 公分。 |
| **注意事項** | 避免過輕或過重，方向由下往上、由內向外；避免拍擊脊椎骨、下背部及臟器。 |

## 流程/步驟

### 一、為什麼要拍背？  
嬰幼兒咳嗽機能較差，當有呼吸道疾病導致痰液滯留時，可透過拍背幫助鬆動濃稠分泌物，促進痰液排出。

### 二、拍背合適的時間  
1. 每天至少 7~8 次，或每 2 小時一次，每次 5~10 分鐘，可依兒童情況與忍耐程度調整。  
2. 飯前半小時及飯後 1 小時內不要拍背排痰，以免造成嘔吐。

### 三、拍背要怎麼做？  
- **手掌握法**：將手指併攏彎成杯狀，讓氣體充滿手部，利用手腕關節自然活動彎曲的力量，穩穩拍打。  
- **拍痰器使用**：選用拍痰器，重覆拍打胸壁（一般藥局或醫院皆可購買）。  
- **正確聲音**：拍打時應發出低沉的「碰碰」聲或「ㄩㄩㄩ」聲，表示痰液被順利引出。  
- **錯誤聲音**：若發出「啪啪」或「ㄊㄧㄚ ㄊㄧㄚ」聲，表示姿勢錯誤。

### 四、合適的拍背部位與姿勢  
- **部位**：左右肺下葉前底。  
- **姿勢**：側躺，手臂提起，俯臥時臀部抬高約 20 公分。

## 注意事項/警示  
1. 拍背力量應適中，避免太輕（無效）或過重（造成傷害），方向應由下往上、由內向外規律移動，並避免拍擊脊椎骨、下背部（腰部以下）及臟器部位。  
2. 衣服勿過厚，建議只穿 1-2 件薄衫，先拍患側再拍健側，以達最佳排痰效果。  
3. 拍痰過程中，若發現兒童有呼吸困難、臉色蒼白、發紺等情況，應立即停止拍痰，並請醫護人員處理。  
4. 拍背後應鼓勵小孩咳嗽，住院嬰幼兒則可依情況請護理人員協助抽痰。  
5. 拍背過程中若有任何疑問，應隨時詢問護理人員。

屏基醫療財團法人屏東基督教醫院

輕輕鬆鬆學拍背排痰

一、為什麼要拍背？

嬰幼兒咳嗽機能較差，當有呼吸道疾病而導致痰液滯留呼吸道不易排除時，則以拍背方法幫助鬆動濃稠的呼吸道分泌物，以促進呼吸道痰液排出。

二、拍背合適的時間

1. 每天至少 7～8 次，或每 2 小時一次，每次 5～10 分鐘，可視兒童情況與忍耐程度調整。

2. 飯前半小時及飯後 1 小時內不要拍背排痰，以免造成嘔吐。

三、拍背要怎麼做？

當拍打時，發出低沉的「碰碰」聲或是「ㄅㄆㄇ」，會聽到被引流的痰所發出空而深的聲音，代表相當順利且姿勢正確；但若發出「啪啪」、「ㄊㄧㄚ ㄊㄧㄚ」聲，就是錯誤囉！！

手掌形狀

用手掌拍痰時，必須將手指併攏彎成杯狀，讓氣體充滿手部。利用手腕關節自然活動彎曲的力量，穩穩的拍打。

拍痰器

選用拍痰器拍痰，重覆地拍胸壁（一般藥局或醫院都買的到喔！）

四、合適的拍背的部位與姿勢

左右肺下葉前底：側躺手臂提起

肺下葉底：俯臥臀抬高約 20 公分

五、拍背的注意事項

1. 拍背之力量避免太輕或過重，以免拍痰無效或造成病童傷害，拍痰方向應由下往上，由內向外做規律性的移動，且應避免拍擊脊椎骨、下背部（腰部以下）、臟器部位。

2. 衣服勿過厚，建議只穿 1-2 件薄衫，先拍患側再拍健側，以達拍痰效果。

3. 拍痰中隨時注意兒童有無呼吸困難、臉色蒼白、發紺等。如有前述情況請停止拍痰，並請告知醫護人員處理。

4. 拍背後應鼓勵小孩咳嗽，住院中嬰幼兒則依情況可請護理人員予以抽痰。

5. 拍背過程中若有任何疑問，應隨時詢問護理人員。
