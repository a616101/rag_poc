# 骨科

## 團隊簡介
本院骨科在1970、1980年代，由外籍宣教士醫師引進脊椎側彎矯正手術，造福無數小兒麻痺及脊椎側彎病患，名聞全省。髖關節及膝關節置換手術由國內外訓練的資深醫師執行，享有良好聲譽。目前為骨科專科醫師訓練醫院。

## 團隊編制
- 醫師：6位專任主治醫師
- 護理/照護：專業護理團隊
- 醫事技術/其他：手術技術人員
- 特色分工：由國內外訓練的資深醫師執行髖關節及膝關節置換手術

- 髖關節置換手術：由國內外訓練的資深醫師執行，享有良好聲譽
- 膝關節置換手術：由國內外訓練的資深醫師執行，享有良好聲譽
- 各種骨科手術：提供各種骨科疾病的診斷與手術治療
