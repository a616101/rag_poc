# 1701577012_777

## 主旨  
本文件旨在介紹胰臟炎的定義、常見病因及主要症狀，幫助民眾認識此疾病，並提高對急性胰臟炎的警覺。同時提供急性胰臟炎患者的治療與出院後生活注意事項衛教指引，涵蓋急性期處置、病因治療、飲食建議及長期照護重點。

## 重點整理

### 一、 何謂胰臟炎？  
胰臟在人體中扮演重要角色，主要負責分泌消化酵素。胰臟炎是指胰臟因自身分泌的酵素而導致發炎的現象。臨床上，胰臟炎可分為急性與慢性兩種，其中以急性胰臟炎最為常見。

### 二、 病因  
胰臟炎的成因多樣，常見原因包括：

| 序號 | 病因 | 說明 |
|------|------|------|
| 1 | 膽道結石 | 膽管結石阻塞主胰管出口，導致胰液無法排出，引發胰臟發炎。 |
| 2 | 酗酒 | 酒精會使胰臟發炎腫大，造成胰管阻塞，導致胰液「自體消化」，破壞胰臟組織，並波及周邊器官。 |
| 3 | 腹部創傷、病毒感染、十二指腸潰瘍穿孔、膿瘍 | 外傷或感染可能直接損傷胰臟，引發炎症。 |
| 4 | 藥物 | 如類固醇、口服避孕藥、磺胺類藥物、四環黴素等，可能誘發胰臟炎。 |
| 5 | 代謝性疾病 | 包括高血脂症（特別是三酸甘油脂過高）及副甲狀腺機能亢進。 |
| 6 | 手術損傷 | 手術過程中可能意外損傷胰臟。 |
| 7 | 其他 | 極少數情況下，可能因自體免疫性胰臟炎、胰臟癌、先天性膽管或胰管異常，或在進行「內視鏡逆行性膽胰管攝影術」(ERCP)後發生。 |

### 三、 症狀  
胰臟炎的症狀多樣，嚴重程度不一，常見症狀如下：

1. **腹痛**：為主要症狀，位於左上腹或上腹中央，常放射至背部。疼痛為突然、持續且劇烈，坐起或向前彎時可稍緩和。  
2. **進食或飲酒加劇疼痛**：進食含脂肪食物、酒精或斜躺時，疼痛會加劇。  
3. **消化系統症狀**：包括腹脹、噁心、嘔吐，或因脂肪分解不良導致糞便浮油，甚至出現脂肪瀉。  
4. **內分泌功能喪失**：嚴重時可能導致高血糖。  
5. **全身性症狀**：輕微發燒、白血球上升、血壓下降；嚴重時可能出現腸胃道出血、敗血症、休克等。  
6. **腹部體徵**：腹部肌肉僵硬，且有壓痛。

## 流程/步驟

### 急性期治療步驟  
1. **禁食**：急性期需完全禁食，以減少胰臟分泌。  
2. **靜脈補液**：透過靜脈滴注補充體液與電解質，維持生理平衡。  
3. **臥床休息**：減少活動，必要時臥床休息。  
4. **症狀管理**：  
   - 若出現噁心、嘔吐或腹脹等症狀，可放置鼻胃管引流以緩解不適。  
5. **藥物治療**：  
   - 必要時使用止痛劑與抗生素。  
6. **病因治療**：  
   - 膽石性胰臟炎：需處理膽管結石。  
   - 出血性或壞死性胰臟炎：必要時需接受外科手術。  
7. **血糖控制**：糖尿病患者應密切控制血糖，避免高血糖。

### 出院後生活注意事項  
1. **飲食調整**：  
   - 避免刺激性食物，如咖啡因飲料。  
   - 避免產氣食物，如奶類、豆類、韭菜、洋蔥、花椰菜、青椒。  
   - 避免高油脂食物，如油炸食品、魚蝦、肉製品。  
   - 病況好轉後，應從清淡飲食開始。  
2. **生活方式**：  
   - 避免酒精。  
   - 適度運動，配合飲食控制，維持理想體重。  
3. **藥物與追蹤**：  
   - 遵從醫師指示服用藥物。  
   - 定期返診追蹤，特別是高血脂症患者應配合降血脂藥物治療。

## 注意事項/警示
- 若出現持續性劇烈腹痛，特別是放射至背部，應盡速就醫。  
- 避免過量飲酒與高脂飲食，以降低胰臟炎風險。  
- 有膽結石、高血脂、糖尿病或曾接受ERCP手術者，應定期追蹤檢查。  
- **禁忌**：禁止使用含咖啡因飲料與酒精。  
- **慎用食物**：避免奶類、豆類、韭菜、洋蔥、花椰菜、青椒等易產氣食物。  
- **高風險食物**：避免油炸、高油脂食物，如油炸食品、魚蝦、肉製品。  
- **藥物依從性**：必須遵從醫師指示服藥，不可自行停藥或調整劑量。  
- **定期追蹤**：應定期返診，特別是糖尿病或高血脂症患者。

一、 何謂胰臟炎？
胰臟在人體的分泌功能佔有相當重要的角色；胰臟炎是指胰臟受到本身分泌的酵素所消化，而導致發炎的現象。一般胰臟炎可分為急性及慢性，臨床上，以急性胰臟炎最為常見。

二、 病因：
1. 膽道結石：因膽管結石造成主胰管出口阻塞，進而阻塞胰液的排空造成胰臟發炎。
2. 酗酒：可引起急性或慢性胰臟炎，酒精會使胰臟發炎腫大而讓胰管阻塞，胰液回流造成胰臟「自體消化」破壞，然後波及臨近的組織與器官。
3. 腹部創傷，病毒感染，十二指腸潰瘍穿孔，膿瘍。
4. 藥物：如類固醇，口服避孕藥，磺胺類藥物，四環黴素等。
5. 代謝性疾病：高血脂症（特別是三酸甘油脂過高），副甲狀腺機能亢進。
6. 手術時胰臟受到損傷。
7. 極少數發生自體免疫性胰臟炎、胰臟癌、先天性膽管或胰管異常個案，或是做完「內視鏡逆行性膽胰管攝影術」(ERCP)之後發生。

三、 症狀：
1. 腹痛為主要症狀，部位在左上腹或上腹中央，且會輻射到背部，為突然、持續且劇烈的疼痛，坐起軀幹向前彎時疼痛會稍微緩和。
2. 進食含脂肪食物，酒精或斜躺會使疼痛加劇。
3. 腹脹、噁心、嘔吐，或因脂肪分解不良導致糞便浮油甚至腹瀉情形。
4. 更嚴重時內分泌功能喪失，導致高血糖。
5. 輕微發燒、白血球上升、血壓下降；發炎嚴重時有腸胃道出血、敗血症、休克等症狀。
6. 腹部肌肉呈現僵硬狀態、壓痛。

四、治療：
1. 急性期需禁食，由靜脈滴注補充體液與電解質，讓胰臟獲得休息。
2. 急性期應臥床休息，減少不必要的活動，必要時需禁食，如發生噁心、嘔吐或腹脹等症狀嚴重時，一般會先放置鼻胃管引流，緩解不適。
3. 藥物使用：必要時使用止痛劑、抗生素等。
4. 針對引起胰臟炎原因治療，如：膽石性胰臟炎，應處理膽管結石問題；若有出血性或壞死性胰臟炎時，必要時必須接受外科手術。
5. 糖尿病者應控制血糖，避免高血糖情形。

五、出院後日常生活注意事項
1. 規律生活，避免攝取刺激性的食物，禁止使用含咖啡因飲料；對於酒精性胰臟炎病人應戒酒。
2. 應多利用蒸、煮、涼拌等各種不必加食用油的烹調方式。
3. 避免產氣食物，如奶類、豆類食物、韭菜、洋蔥、花椰菜、青椒。
4. 病況好轉後開始進食需採清淡飲食，避免高油脂食物，如油炸或富含油脂的魚蝦、肉製品；高血脂症病人應配合降血脂藥物治療。
5. 適度運動或飲食控制，維持理想體重。
6. 遵從醫師指示服用藥物，定期返診追蹤。
