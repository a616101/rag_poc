# 1701601542_337

## 主旨
本文件旨在協助家長了解兒童「發展性口吃」的常見現象，並提供六項具體策略，以營造放鬆、安全的溝通環境，幫助孩子克服口吃問題。

## 重點整理
- 「發展性口吃」是學語階段常見的語言不流暢現象，通常在數週至半年內會自然改善。
- 家長應避免催促、打斷、糾正或替孩子說話，以免增加孩子的焦慮。
- 透過用心聆聽、減輕壓力、給予正面鼓勵、減少修正及細心觀察，可有效支持孩子。
- 若口吃持續未改善，應尋求復健科醫生與語言治療師的專業協助。

## 流程/步驟
1.  **用心聆聽**  
    - 不要讓孩子有被催促的感覺。  
    - 讓孩子知道你正在專注聆聽，並對他說的話感興趣。

2.  **與孩子聊天**  
    - 平時多給予孩子練習和你對話的機會。  
    - 避免單向的嘮叨碎念。  
    - 請家人不要打斷、糾正或幫孩子說話。

3.  **減輕壓力**  
    - 避免讓孩子處於緊張的環境。  
    - 了解孩子的發展里程，不應過早要求其完成超出能力範圍的事。

4.  **輕鬆積極的態度**  
    - 當孩子說得好時，給予正面鼓勵。  
    - 鼓勵孩子發展多方面的興趣，幫助其建立自信。

5.  **減少修正**  
    - 不要因孩子口吃而焦慮，更不要要求他「再說一遍」。  
    - 給孩子時間，以他自己的方式說完話。

6.  **細心觀察**  
    - 提供適當的語言刺激，營造放鬆、有安全感的溝通環境。  
    - 以正向態度看待口吃，並為孩子示範不急不徐、輕鬆愉快的說話方式。  
    - 若口吃持續未改善，應尋求專業醫療協助。

## 注意事項/警示
- 避免在孩子口吃時直接糾正或打斷，這可能加劇其焦慮與自我意識。
- 不應將口吃視為「壞習慣」或「故意為之」，應以同理心看待。
- 若口吃持續超過半年，或影響孩子的社交與學習，應儘早尋求專業評估與治療。

## 全文

如何協助口吃孩童

您的孩子最近突然開始有口吃的現象嗎？其實在學語的孩子中常常會有此現象，我們稱之為「發展性口吃」，也就是發展階段中的正常言語不流暢現象，還不到病理上口吃的嚴重程度，而隨著語言能力和環境刺激增加，大部分的孩子在幾週或半年過後此現象便會自動消失。這時候家長所能夠做的的是給予陪伴和支持，營造出一個能讓他感到很放鬆、有安全感、被信任溝通環境，以下有幾個方法在家能協助口吃孩童：

1. 用心聆聽
不要讓兒童有被催促的感覺，並讓他知道你在注意聽，且對他說的話有興趣。

2. 與孩子聊天
平時多給孩子時間練習和你對話的機會，而不只是單向談話（嘮叨碎念）；並請家人不要打斷、糾正或幫他說話。

3. 減輕壓力
也不要讓孩子處於緊張的環境裡，家長要了解孩子的發展里程，切勿揠苗助長、提前要求孩子做到不合能力的事情。

4. 輕鬆積極的態度
當孩子說得好時要給予正面鼓勵，並鼓勵他發展多方面的興趣，藉此幫助他消除缺乏自信的心理。

5. 減少修正
大人不要比孩子更焦慮，當口吃發生時，給孩子時間讓他以自己的方式說完，不要叫他「再說一遍」因為可能使孩子更意識到自己有問題，而開始排斥說話。

6. 細心觀察
除了提供適當的語言刺激及製造放鬆、有安全感的溝通情境外，最重要的是以正向態度來看待口吃，並且為孩子示範如何不急不徐、輕鬆愉快地說話！

常見口吃的現象/例子：

1. 重覆字句 /「吃…吃…吃飯」、「我要…我要吃飯」
2. 插入語詞 /「我想要『嗯…』吃飯。」
3. 拉長音 /「我——要吃。」
4. 修正 /「我要，我不要吃飯」

先在家試試以上的方法，且用耐心和細心觀察孩子的情形，若孩子的口吃仍然未改善，可就近找到復健科醫生和語言治療師尋求諮詢和幫助喔。
