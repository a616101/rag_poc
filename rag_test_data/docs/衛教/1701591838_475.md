## 主旨  
本文件由屏基醫療財團法人屏東基督教醫院製作，旨在介紹骨質疏鬆症的定義、影響及預防與治療方法，幫助高風險族群（如老年人及停經後婦女）了解如何透過飲食、運動、生活習慣調整來預防骨質流失與骨折。

## 重點整理  
- **骨質疏鬆症定義**：骨質流失導致骨骼變薄、脆弱，形成多孔隙結構，常見於70歲以上長者及停經後女性。  
- **主要影響**：易導致脊椎壓迫性骨折，引發背痛、身高變矮及駝背。  
- **預防核心**：增加鈣質攝取、適度運動與日曬、減少咖啡因與酒精攝取、預防跌倒。  
- **關鍵營養素**：鈣質與維生素D是維持骨密度的重要營養素。  
- **生活安全**：改善居家環境，如浴室、廚房、樓梯等易跌倒區域，應加強照明與防滑措施。  

## 流程/步驟  
### 骨質疏鬆症的預防與治療方法：  
1. **增加鈣質攝取**  
   - 鈣是骨骼主要成分，骨質流失後難以完全恢復。  
   - 建議食物：肉類、蛋類、豆類、小魚乾、黑芝麻等。  

2. **適度運動與日曬**  
   - 陽光促進維生素D合成，幫助腸道吸收鈣質。  
   - 適量運動可增強肌力、關節柔軟性與身體協調能力，預防跌倒。  

3. **減少咖啡因、菸與酒精攝取**  
   - 過量攝取可能影響鈣吸收或增加骨質流失風險。  

4. **預防跌倒，避免骨折**  
   - 改善居家環境：  
     - 浴室、廚房、廁所及樓梯應有充足照明。  
     - 保持地面乾燥，鋪設防滑墊。  
     - 加裝扶手以提供支撐。  
   - 維持適當運動，增進身體機能與平衡感。  

## 注意事項/警示  
- 骨質疏鬆症早期無明顯症狀，常在骨折後才被發現，應重視長期預防。  
- 女性停經後骨質流失速度加快，應特別關注骨密度變化。  
- 高齡者應定期進行骨質密度檢查，及早發現問題。  
- 防止跌倒不僅是環境改善，也需透過運動提升身體機能。

一 何謂骨質疏鬆症？

常見於 70 歲以上老年人及停經後婦女。身體骨骼的骨質自出生後會隨著年紀而增加，大約在二十至三十歲會達到最高峰，之後骨質逐漸減少，而女性停經後，骨質減少的速度會加快，如果骨質流失過多，使得原本緻密的骨骼形成許多孔隙，呈現中空疏鬆的現象，稱為骨質疏鬆。

二 骨質疏鬆症的影響是什麼？

骨頭因疏鬆而變薄、變脆弱、容易造成骨折。最明顯的症狀就是脊椎壓迫性骨折，它會引起背部酸痛，身高變矮及駝背。

圖摘自 衛生福利部福利部 認識骨質疏鬆 https://www.hpa.gov.tw/Pages/Detail.aspx?nodeid=634&pid=1196

三 預防與治療骨質疏鬆症的方法：

1. 增加鈣質攝取：

鈣質是人體骨頭最主要的成份，骨質一旦流失就很難再完全恢復，預防骨質有效方法包括增加鈣質攝取，如：肉類、蛋類、豆類、小魚乾、黑芝麻等食物。

2. 適度的運動及日曬：

陽光能使身體產生維生素 D，維生素 D 可以加強腸胃對鈣的吸收，持續而且適量運動，可以防止骨質疏鬆、幫助睡眠、維持活力。

3. 減少咖啡、菸及酒的攝取。

4. 預防跌倒，以免發生骨折：

(1) 改善居家環境浴室、廚房、廁所及樓梯是較容易跌倒的地方，應有充足的照明，保持地面乾燥或鋪設防滑墊、加裝扶手。

(2) 維持適當的運動，增進肌力及關節柔軟性並改善身體的協調能力，減少跌倒的機會。
