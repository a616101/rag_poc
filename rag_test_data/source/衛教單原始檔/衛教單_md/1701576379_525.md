# 1701576379_525

## 主旨
本文為腦中風急性後期復健的衛教資訊，由復健科主任醫師王重元撰寫。內容涵蓋中風復健的定義、目標、團隊組成、適應時機、分期治療重點，以及各項復健方法的詳細說明，並提供預防併發症與安全注意事項。

## 重點整理
- **中風復健定義**：透過醫學、物理、社會、教育、職業訓練等方法，幫助中風病友改善功能，達到最佳狀況。
- **中風復健目標**：追求最大的功能獨立性、最小的殘疾不便性，順利回歸家庭、社區與職場，重建有意義且有品質的生活。
- **復健團隊**：由復健科醫師、護理師、物理治療師、職能治療師、語言治療師、心理師及社工組成，提供全人照護。
- **復健時機**：復健應儘早開始，「復健黃金治療時間」為中風後前3個月，錯過此階段，復原效果會大打折扣。
- **復健分期**：分為急性期、復健期/適應期，各階段有不同治療重點。
- **復健內容**：主要分為物理治療、職能治療、語言治療及吞嚥訓練四項。

## 流程/步驟
### 中風復健的分期與治療重點
1.  **急性期**：
    1.  避免併發症，如肺炎、尿路感染、褥瘡、關節攣縮及足下垂。
    2.  床上適當擺位，避免患肢水腫與關節攣縮。
    3.  協助移除鼻胃管、尿管及氣切管。
    4.  協助安撫情緒，提供心理輔導。
2.  **復健期或適應期**：
    1.  進行動作與感覺的再教育。
    2.  進行神經肌肉的誘發。
    3.  處理異常張力。
    4.  強化健側肌力。
    5.  進行平衡與功能訓練，包括轉位技巧、行走訓練、步態矯正及日常功能訓練。
    6.  進行吞嚥與語言訓練。

### 各項復健方法
- **物理治療 (PT)**：
    1.  進行被動或主動關節運動、翻身、轉位、步行、爬樓梯及肌力訓練。
    2.  利用本院新購的懸吊系統，誘發病人患側的自主動作。
    3.  使用電療儀器改善中風後的疼痛或感覺異常。
- **職能治療 (OT)**：
    1.  訓練日常生活能力，如吃飯、洗澡、穿衣、走路、轉位。
    2.  訓練較複雜的功能，如洗衣、作菜、服藥、打掃環境及處理財物。
    3.  評估與運用輔具，如一般輪椅、電動輪椅、腳拐杖、助行器。
    4.  製作與使用副木，如中風病患常用的垂足板或手部固定副木。
- **語言治療 (ST)**：
    1.  針對失語症、失用症、失認症及語言理解障礙進行專業語言治療。
    2.  透過專業語言治療，幫助病患改善溝通能力。
- **吞嚥訓練 (Swallowing training)**：
    1.  針對中風後可能伴隨的吞嚥困難進行訓練。
    2.  預防因吞嚥困難導致的吸入性肺炎。

## 注意事項/警示
- **適合積極性復健的條件**：生命徵象穩定、無嚴重內外科併發症、意識清楚、心智正常、有積極復健的意願。
- **不適合積極性復健的病患**：應進行維持性或預防性復健。
- **病患吞嚥叮嚀**（注意三項原則）：
    1.  **姿勢**：不要在臥姿餵食，宜採坐姿，避免頭前傾，宜略收下巴姿勢。
    2.  **專心**：病患應意識清楚時才餵食，不要邊吃邊講話，也不宜邊吃邊看電視。
    3.  **速度與食量**：不要餵得太快或太大量，液體應難控制，所以餵食性肺炎時，應暫停自行吞嚥練習。
- **預防併發症**：應注意避免跌倒、骨折、意外傷害、感染（如吸入性肺炎、褥瘡、尿路感染）。
- **控制中風危險因子**：如糖尿病、高血壓、高血脂、心血管疾病，以避免再次中風。

## 全文

急性後期腦中風復健

文／復健科主任醫師 王重元

「醫生啊，我爸爸中風了，以後怎麼辦？」
「他會恢復嗎？以後會不會走路？會不會自己吃飯？會不會講話？」
腦中風病患與家屬遇到這突來的病況以及後續的安置問題，千頭萬緒卻總是有一堆問題要問醫師。

今年更因健保署為了提升全民健保急性後期照護品質所推行的「腦中風急性後期照護試辦計畫」（Post-acute Care Cerebrovascular Disease，簡稱 PAC-CVD），而有不少病友及家屬詢問相關復健服務。這計畫的最大特色是讓具有積極復健潛能之腦中風病友透過整合性醫療照護與積極性高強度復健，以增加腦中風病友的功能恢復，減輕失能程度，減少家人負擔和減少後續再住院醫療支出。為了對腦中風病友提供更專業的全人照護，本院集結急診、神經內科、神經外科、復健科、中醫科、影像醫學科、藥劑科、社工室和營養室之專業人員，跨團隊組成腦中風中心，設置神經疾病專屬病房，更以全新復健設備建置全方位多功能復健室，是台灣少數專為住院病友提供專屬復健場地的醫療院所，與門診復健完全區隔。

以下針對腦中風復健提供一些衛教資訊：

中風復健的定義：透過醫學、物理學、社會、教育、職業訓練等方法，訓練病人使其功能改善至最佳狀況。

中風復健的目標：最大的功能獨立性，最小的殘疾不便性，順利回歸家庭、社區、職場，進而重新建立一個有意義、有品質的人生。

復健團隊：復健科醫師、護理師、物理治療師、職能治療師、語言治療師、心理師、社工。

何時開始中風復健？

復健最好在中風6個月內施行，尤其前3個月是所謂的「復健黃金治療時間」。

如果錯過黃金治療時間，以後的復原狀況會差很多，復健效果也會較差。目前的觀念是強調早期介入，可縮短住院時間，併發症較少，預後較好。本院的神經內外科醫護團隊與復健科和中醫科具有絕佳默契和合作經驗，住院期間便開始復健訓練，部分病友會銜接急性後期腦中風復健（PAC），出院後轉介至本院或他院繼續門診復健訓練。

何時可以開始積極復健？

因為每位病患的嚴重程度不同，所以並無絕對開始復健日期。但一般而言，生命徵象及神經狀況穩定後就可開始復健訓練，循序漸進，切勿操之過急。

哪些人適合積極性中風復健？

早期介入積極性復健的先決條件在於生命徵象穩定、無嚴重內外科併發症、意識清楚、心智正常、有積極復健的意願。不符上述條件的病患，則作維持性或預防性復健。

中風復健的分期與治療重點：

一、急性期
1. 避免併發症：例如肺炎、尿路感染、褥瘡、關節攣縮。
2. 床上適當的擺位：避免患肢水腫、關節攣縮、垂足板。
3. 協助儘早移除鼻胃管、尿管、氣切管。
4. 協助安撫情緒，心理輔導。

二、復健期或適應期
1. 動作與感覺的再教育。
2. 神經肌肉的誘發。
3. 異常張力的處理。
4. 健側肌力的強化。
5. 平衡與功能訓練（轉位技巧、行走訓練、步態矯正、日常功能訓練）。
6. 吞嚥與語言訓練。

復健計劃的制定：強調因人而異，量身訂作。

中風復健的內容：主要可分為物理治療、職能治療、語言治療、吞嚥訓練。

一、物理治療（PT，Physical therapy）
1. 被動或主動關節運動、翻身、轉位到步行、爬樓梯及肌力訓練。除了傾斜床，本院新購懸吊系統是另項特色治療。
2. 以專業手法來誘發病人患側的自主動作。這是腦中風病友復健的成功關鍵。
3. 電療儀器以改善中風後所引起的疼痛或是感覺異常。如干擾波、經皮電刺激、超音波、短波等等。

二、職能治療（OT，Occupational therapy）
舉凡最基本的日常生活（例如吃飯、洗澡、穿衣服、走路、轉位），到較為複雜的功能（如洗衣、作菜、服藥、打掃環境或是處理財物）等等，都是職能治療的範疇。職能治療比物理治療更著重在實際日常功能的訓練，例如教導右側偏癱的中風患者如何用左手拿筷子吃飯，穿衣服甚至寫字。

輔具（例如一般輪椅、電動輪椅、四腳柺杖、助行器）的評估與運用，及副木（例如中風病患常用的垂足板，或是手部的固定副木）的製作。

三、語言治療（ST，Speech therapy）
左腦中風的病患可能伴隨失語症。但不表示右腦中風的人絕不會有語言障礙。失語症患者在語言理解、流暢度、覆誦能力或物品命名能力會有不同程度的損失，造成他們與家屬或是照護者溝通上的困難。輕則影響復健動機與意願，重則會造成憂鬱症。透過專業的語言治療，可使失語症患者在反覆的訓練中，增加語言理解力與說話構音能力。嚴重的運動型失語症病患甚至需要溝通板來改善其溝通能力。

四、吞嚥訓練（Swallowing training）
一半以上的中風病患在急性期有吞嚥異常的情形，為了防止吸入性肺炎的發生，許多病患都會被放置鼻胃管。但最後不能拔除鼻胃管而需長期灌食者只有少數。

吞嚥訓練目標：兼顧吞嚥安全以及充足且均衡的營養。

訓練方法可分為兩種：
1. 直接訓練法：使用實際食物或液體來訓練病人吞嚥，利用各種代償性的技巧，增加吞嚥效率，提升吞嚥的安全性。
方法：進食時姿勢的調整、食物質地的選擇、進食環境的改善、餵食者的教育。
2. 間接訓練法：利用模擬食材或特殊方式進行訓練，例如：(1) 低溫刺激以促進吞嚥反射。(2) Mendelsohn 氏手法以增進吞嚥時喉部升高程度。(3) 各種口腔及咽喉運動以增加口腔、舌頭以及咽部的肌力及協調性。(4) 吞嚥誘發電刺激器。

目前屏東有三位專業語言／吞嚥治療師（全縣最多），且為少數提供吞嚥誘發電刺激儀的醫療院所。

病房吞嚥叮嚀：要注意三項原則。
1. 姿勢：不要在臥姿餵食，宜採坐姿。不要仰頭餵食，宜略收下巴姿勢。
2. 專心：病患意識不清楚時不要餵食。不要邊吃邊講話，也不宜邊吃邊看電視。
3. 速度與食量：不要餵太快或餵太大口。液體難難控制，所以餵食性肺炎時，應暫停自行吞嚥練習。

結論：
1. 預防勝於治療：如果能控制中風危險因子（例如糖尿病、高血壓、高血脂、心血管疾病）就可能可以避免下次腦中風的機會。
2. 把握黃金治療時間：有疑似腦中風徵象，應儘快前往醫療院所就醫。
3. 住院期間或出院後應主動尋求復健治療安排。
