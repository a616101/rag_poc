# 泌尿外科

## 團隊簡介
本科提供泌尿系統疾病的診斷與治療服務，專長包括：治療泌尿系統結石（包括體外震波碎石及內視鏡碎石手術）、提供攝護腺肥大病人藥物及手術治療、針對不同病因提供各種內視鏡及逆行腎盂輸尿管攝影檢查、透過泌尿系統切片檢查診治泌尿系統各種良性及惡性腫瘤、提供尿路動力學檢查診治排尿功能障礙並予治療、對於各種泌尿系統感染及性功能障礙提供治療。本科成立至今，不斷精進醫療技術，引進各種最新的設備和技術，提供病患最優質的臨床服務。除了臨床技術的精進外，本科亦與高雄榮總醫研部合作，研究關於體外震波之生物效應，致力於將臨床服務與研究結合，期望能提供病患高品質的服務，精益求精，止於至善。

## 團隊編制
- 醫師：泌尿外科專科醫師
- 護理/照護：專業護理團隊
- 醫事技術/其他：檢查技術人員
- 特色分工：提供各種泌尿系統疾病的診斷與治療

## 服務與治療
### 服務範圍（疾病/問題）
- 泌尿道感染：膀胱炎、尿道炎、腎盂腎炎、副睪丸炎等
- 泌尿系統結石：腎結石、輸尿管結石、膀胱結石等
- 泌尿道腫瘤：腎臟腫瘤、輸尿管腫瘤、膀胱腫瘤等
- 男性生殖系統腫瘤：睪丸癌、陰莖癌
- 攝護腺疾病：攝護腺肥大、攝護腺癌、攝護腺炎
- 性病：梅毒、皰疹、淋病、菜花、披衣菌、陰蝨、愛滋病等
- 男性性功能異常：陽萎、早洩、性慾減退、人工陰莖
- 小兒先天異常：隱睪症、疝氣等
- 婦女尿失禁：急迫性尿失禁、應力性尿失禁等
- 包皮過長及包莖、包皮龜頭炎
- 男性不孕治療、男性結紮、輸精管重接
- 精索靜脈曲張、疝氣、陰囊積水、睪丸扭轉
- 其它症狀：血尿、頻尿、解尿困難、尿滯留、血精、小孩或成人尿床

### 主要治療/處置
- 治療泌尿系統結石：包括體外震波碎石及內視鏡碎石手術
- 提供攝護腺肥大病人藥物及手術治療
- 針對不同病因，提供各種內視鏡及逆行腎盂輸尿管攝影檢查
- 透過泌尿系統切片檢查，診治泌尿系統各種良性及惡性腫瘤
- 提供尿路動力學檢查，診治排尿功能障礙並予治療
- 對於各種泌尿系統感染及性功能障礙提供治療

### 特色門診/照護計畫（若有）
- 體外震波碎石：提供體外震波碎石治療
- 內視鏡碎石手術：提供內視鏡碎石手術
- 攝護腺手術：提供攝護腺肥大及攝護腺癌的手術治療

## 檢查項目
### 特殊檢查
1. 內視鏡檢查：腎臟、輸尿管、膀胱、尿道鏡檢查
2. 尿路動力學檢查：尿流速、膀胱壓、尿道括約肌及尿道壓之測定
3. 超音波檢查：腎臟、陰囊及經直腸攝護腺超音波檢查
4. 逆行性腎盂輸尿管攝影

### 手術及特殊治療項目
1. 尿路結石手術：傳統開刀、輸尿管鏡體內碎石手術、經皮穿腎造口截石術
2. 泌尿道癌症根除術及經尿道內視鏡切除手術（TURBT）
3. 應力性尿失禁手術：膀胱懸吊術、經閉孔尿道中段懸吊術
4. 攝護腺刮除手術（TURP）、攝護腺雷射手術
5. 人工陰莖植入術
6. 體外震波碎石術
7. 其他：如男性輸精管結紮手術、腹股溝疝氣修補手術、小兒隱睪固定手術等

- 膀胱鏡、輸尿管鏡、腎臟鏡：提供各種內視鏡檢查
- 經尿道前列腺切除儀器：提供攝護腺手術
- 攝護腺雷射氣化設備：提供攝護腺雷射手術
