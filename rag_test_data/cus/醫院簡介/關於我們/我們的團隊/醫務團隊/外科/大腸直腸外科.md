# 大腸直腸外科

## 團隊簡介
大腸直腸肛門科為一般外科的分支，專研大腸及肛門之疾病與處置，在較具規模之醫學中心或教學醫院大致上皆有此專科。本科提供大腸及肛門疾病的專業診斷與治療服務。
