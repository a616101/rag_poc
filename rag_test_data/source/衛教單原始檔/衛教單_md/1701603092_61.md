# 1701603092_61

## 主旨
看病時，應清楚向醫師說明自身狀況，以協助診斷與治療。女性在就醫或接受治療前，需主動告知醫療人員其是否懷孕、正準備懷孕，或正在餵食母奶，以確保用藥安全。

## 重點整理
為讓醫師能有效評估病情，就醫時應主動提供以下五項資訊：

1. **不適部位與症狀**：說明身體不適的具體位置、症狀表現及開始時間。
2. **特殊體質與飲食習慣**：包括藥物過敏史、食物過敏反應，以及特殊飲食習慣（如素食、低糖、低鹽等）。
3. **既往病史**：提供曾接受的手術、住院紀錄、醫學檢查結果，以及慢性疾病（如糖尿病、高血壓、肝病、腎病）與家族遺傳病史。
4. **目前用藥情況**：列舉正在使用的藥物，包括醫師處方的西藥、中藥、自行購買的藥品、維他命、賀爾蒙、減肥藥、保健食品及酒精等。
5. **生活與工作狀況**：說明近期是否參與考試，以及平時的工作性質，特別是是否需操作機械或駕駛車輛。

女性應主動告知醫療人員其生育相關狀況，涵蓋三種情況：是否懷孕、正準備懷孕、正在餵食母奶。此資訊對於評估藥物安全性至關重要。

## 流程/步驟
- 就醫前準備自身健康與用藥資訊。
- 向醫療人員主動說明不適部位、症狀、體質、飲食、病史、用藥、生活與工作狀況。
- 女性應特別告知是否懷孕、正準備懷孕或正在餵食母奶。

## 注意事項/警示
- 某些藥物可能引起嗜睡，影響注意力與反應能力，若工作需高度專注（如開車），務必告知醫師。
- 避免隱瞞用藥或健康資訊，以免影響治療安全與效果。
- 若女性處於懷孕、準備懷孕或哺乳期，可能需調整用藥方案。
- 醫療人員應根據此資訊評估藥物對母體與胎兒或嬰兒的潛在風險。

## 全文

看病時，要向醫師說什麼？

清楚表達自己身體的狀況

看病時，要瞭解自己身體狀況並能清楚向醫師說明下列事項

1. 哪裡不舒服

包括身體不舒服的部位、情形及大約何時開始不舒服等

2. 是否有特殊體質或飲食習慣

含使用藥物會不會過敏、曾經吃什麼食物發生過敏現象、有沒有特殊飲食習慣（例如素食、低糖、低鹽等）

3. 我生過的病

包括開刀、住院、醫學檢驗報告結果、糖尿病、高血壓、肝病、腎臟病及家族性遺傳疾病等

4. 目前正在使用的藥品

包括醫師開的西藥或中藥、自己買的藥、維他命、賀爾蒙、減肥藥、保健食品、菸酒等。

5. 最近有沒有考試、平時有沒有開車、工作性質有沒有需要操作機械

有些藥品吃了之後會想睡覺，可能會降低您的注意力及應變反應。若您的工作需要很小心（比如開車），一定要告訴醫師。

6. 女性需告知是否懷孕、正準備懷孕、正在餵食母奶
