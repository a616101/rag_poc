## 主旨  
本文件為腸阻塞手術前、術後及照護者應注意事項之衛教指引，旨在協助患者與家屬了解手術相關準備、恢復期照護重點，以促進術後順利康復。

## 重點整理

| 項目 | 內容 |
|------|------|

## 流程/步驟

#### 手術前準備步驟：
1. 禁食8小時，不可進食任何東西，包括開水、口香糖、檳榔。
2. 放置鼻胃管。
3. 進行腹部X光檢查。
4. 抽血檢查。
5. 檢查電解質並進行補充。
6. 給予抗生素。
7. 進行腹部檢查。

#### 手術後照護步驟：
1. 每2~4小時評估生命徵象、意識狀態及腸蠕動音。
2. 深呼吸並咳嗽，必要時使用束腹帶或手固定手術部位，以減輕疼痛。
3. 監測鼻胃管是否處於正確位置，並觀察引流管輸出量、顏色與性質。
4. 保持傷口敷料乾燥，如有滲濕應立即通知醫護人員更換。
5. 維持半坐臥位（搖高床頭），以減輕腹脹對橫隔膜的壓力。
6. 術後禁食3~5天（含水），但需每日刷牙及漱口，保持口腔清潔，預防細菌感染。
7. 待腸蠕動排氣恢復後，由醫師評估是否可開始喝水；若無腹脹或嘔吐，隔日可進食清流飲食（如過濾果汁、碳酸飲料）及低渣飲食（如米湯、豆漿）。

## 注意事項/警示

- **術前**：禁食時間必須嚴格遵守，避免因進食導致手術延誤或中止。
- **術後**：若出現腹部絞痛、腹脹、噁心、嘔吐、便秘或發燒等症狀，應立即就醫，以防止腸阻塞復發。
- **飲食調整**：術後應依醫囑逐步恢復飲食，避免過早進食難消化食物。
- **活動與排便**：鼓勵病人多活動，並攝取高纖食物，每日至少飲用2000c.c液體，配合緩瀉劑與高纖維藥物，搭配240c.c溫水，以促進排便，減少便秘風險。

一、手術前準備及注意事項
1. 禁食 8 小時（不可吃任何東西，包括開水、口香糖、檳榔。）
2. 放置鼻胃管
3. 腹部 X 光檢查
4. 抽血檢查
5. 電解質的檢查及補充
6. 給予抗生素
7. 腹部檢查

二、手術後注意事項
1. 每 2~4 小時評估生命徵象、意識狀態及腸蠕動音。
2. 為預防手術後併發症，請隨時深呼吸及咳嗽，將痰液咳出，以促進肺擴張，咳嗽時可使用束腹帶或以手固定手術部位，可減輕因震動牽扯傷口而引起疼痛。
3. 監測鼻胃管是否保持在適當的位置及引流管輸出量、顏色、性質。
4. 傷口敷料應保持乾燥，若有滲濕，請告知醫護人員協助換藥。
5. 維持舒適的臥位，搖高床頭給予半坐臥，減輕腹脹對橫隔膜的壓迫。
6. 術後因腸道功能尚未恢復，需禁食 3~5 天（包括水），在此期間仍必需刷牙及漱口，保持口腔清潔，減少細菌感染。
7. 在腸蠕動排氣恢復後，醫師會評估是否讓你開始喝水，若無腹脹嘔吐，隔日再採清流（過濾果汁、碳酸飲料）、低渣飲食（米湯、豆漿）。

四、照顧者注意事項：
1. 病人若出現腹部絞痛、腹脹、噁心、嘔吐、便祕或發燒情形，應立即就醫，及早阻止腸阻塞再度發生。
2. 鼓勵病人多進食高纖食物，每日至少飲用 2000c.c 的液體及適當的運動，減少便秘的情形發生，依醫師指示給予緩瀉劑或高纖維藥物，與約 240c.c 溫水一起服用，以利排便。
