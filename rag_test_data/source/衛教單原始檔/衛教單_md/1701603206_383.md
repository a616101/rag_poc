# 1701603206_383

## 主旨  
本文件說明陰道栓劑的正確使用方法，提供安全、衛生的使用步驟，以確保藥物有效施用。

## 重點整理  
- 使用前後應徹底清潔雙手。  
- 建議佩戴手套以維持衛生。  
- 栓劑可先以溫水濕潤，並置於給藥器前端。  
- 建議採用側躺姿勢，膝蓋微彎，方便將給藥器輕柔置入陰道。  
- 使用時應緩慢擠壓給藥器，將栓劑送入陰道內。  
- 施用完畢後，應保持給藥器壓緊狀態，以確保栓劑完全釋放。

## 流程/步驟  
1. 使用前後，用肥皂及清水充分清潔雙手。  
2. 去除藥物栓劑的包裝。  
3. 建議戴上手套。  
4. 可將栓劑以溫水迅速濕潤，若使用給藥器，將栓劑放入給藥器前端。  
5. 人躺下，小腿弓起，膝蓋朝上，輕輕將給藥器前端伸入陰道內，在舒適情況下，盡量深入。  
6. 慢慢擠壓給藥器，將栓劑送入陰道內。在給藥完全後，保持給藥器壓緊的狀況，將給藥器緩慢抽出。

## 注意事項/警示  
- 建議使用前後清潔雙手，以避免感染。  
- 使用給藥器時，應輕柔操作，避免造成不適或損傷。  
- 施用後應保持給藥器壓緊狀態，以確保栓劑完全釋放。
