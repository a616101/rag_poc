# 1701601735_236

## 主旨
本文旨在介紹針對輕度至中度脊椎側彎的運動治療方法，強調透過正確的姿勢訓練、呼吸配合與自我覺察，改善脊椎排列與身體對稱性，達到預防與輔助治療的目的。

## 重點整理
- 脊椎側彎會影響全身排列，包括頭部、頸椎、胸椎、內臟、肋骨、骨盆及下肢，形成一個錯綜複雜的綜合體。
- 核心肌群失衡是導致脊椎側彎的關鍵因素，應透過運動恢復身體的對稱性。
- 治療重點在於「體態矯正」與「呼吸運動」，並結合意象訓練，幫助身體意識到正確姿態。
- 建議每日持續進行，而非僅在固定時間運動，應在日常活動中加強自我身體感覺意識。
- 若側彎角度過大（大於40度），應尋求物理治療師評估並執行更詳細的處理方式。

## 流程/步驟
以下是針對脊椎側彎的四項運動治療步驟，建議每項動作持續15–20秒，重複3–5次為一個循環。

### （1）脊椎向上伸展拉長脊椎
- 姿勢：雙手合十，雙腳與肩同寬，雙膝跪地。
- 動作：將整個脊椎向上拉長，頭部向上、枕骨向後用力（縮下巴），想像脊椎被拉長。
- 呼吸：吸氣時延長脊椎，呼氣時放鬆。
- 圖示：圖一（放鬆姿態）、圖二（拉長姿態）。

### （2）貓背式伸展（四足跑姿）
- 姿勢：雙手與肩同寬，雙膝與髖同寬，手腕與肩成一直線，膝關節與膝蓋呈90度。
- 動作：
  - 背部低沉時，頸部先抬起，胸椎至腰椎逐節下沉（圖三）。
  - 背部拱起時，由頸部開始縮下巴，胸椎至腰椎拱起（圖四）。
- 呼吸：配合動作，吸氣時放鬆，呼氣時拱起。
- 圖示：圖三（背部低沉）、圖四（背部拱起）。

### （3）跪姿身體左右伸展
- 姿勢：雙手與肩同寬，雙膝跪地，身體呈四足跑姿。
- 動作：
  - 若為C型側彎，凸點在左側，則執行圖七（向左伸展）。
  - 若為反C型側彎，凸點在右側，則執行圖八（向右伸展）。
- 動作要領：將身體向凹側伸展，同時利用意象訓練，想像模膈膜下降，凹側肺部充滿氣體。
- 圖示：圖七（C型側彎拉向左）、圖八（反C型側彎拉向右）。

### （4）體態矯正與呼吸運動
- 原理：脊椎側彎影響全身排列，需從頭部至骨盆進行整體調整。
- 動作：站在鏡子前，學習在脊椎延長的狀態下，將「走針」的身體逆轉回來。
  - 例如：頸部偏左 → 調整為偏右；左肩偏高 → 下壓；右側胸椎向後旋轉 → 轉向前。
- 呼吸配合：吸氣時，將氣吸入凹側，同時利用意象訓練，讓側彎最凸點往身體中線移動。
- 圖示：X光片顯示脊椎側彎（圖九），並標示拉向中線的方向。

## 注意事項/警示
- 每日應持續進行運動，並在身體活動中加強自我覺察。
- 動作應輕柔、緩慢，避免過度拉伸造成不適。
- 若側彎角度大於40度，應尋求專業物理治療師評估，並依建議執行相關牽拉或治療。
- 建議配合呼吸訓練，以提升運動效果與身體意識。

# 預防勝於治療—淺談脊椎側彎
## 運動治療篇

文／物理治療師 廖邦淳

當身體脊椎旁的核心肌群失去了平衡，身體的排列及重心將會改變，想像而知這是一個不好的改變，身體的脊椎有小面關節、韌帶、椎間盤、神經等組織，都會因為脊椎側彎的關係全面性的影響身體狀態，也因此身體「姿勢的對稱性」才是關鍵。以下列舉出在輕度及中度脊椎側彎的牽張運動治療。

※（1）脊椎向上伸展拉長脊椎：學會如何自我向上牽拉整個脊椎，即是將整個脊椎延長，動作為雙手合十，往天花板的方向往上做牽拉，此時尾骨向下、頭部向上、且枕骨向後用力的方式（縮下巴）將脊椎延長。此時利用意象訓練，意思就是想像整個脊椎被拉長的畫面（如圖二）1次15-20秒，在回復放鬆的狀態（如圖一）重複3-5次為一個循環。

※（2）貓背式伸展—動作四足跪姿，雙手與肩關節同寬，雙膝與髖關節同寬，肩膀手肘腕部呈一直線，髖關節與膝蓋呈90度夾角，背部保持平面（如圖三）。背部低沉時，頸部先啓動抬起，再來胸椎至腰椎往下沉，想像一節一節的放鬆（如圖四）。背部拱起時，由頸部啓動縮下巴後低沉，再來胸椎至腰椎拱起（如圖五），胸背部拱起時1次15-20秒，在低沉放鬆，重複3-5次為一個循環。

※（3）跪姿，身體下壓左右伸展—動作為跪姿，胸碰膝，雙手再往前延伸（如圖七），如果是C型側彎凸點在左邊，請執行（圖七），反C型凸點在右邊則執行（圖八），1次15-20秒在回中線，重複3-5次為一個循環；若側彎角度過大（大於40度），則再請物理治療師做更詳細的評估執行相關牽拉之動作或後續可能的處理方式。

※（4）體態矯正、呼吸運動：脊椎側彎是一個錯綜複雜的一個綜合體，為甚麼？原因為上至頭部、頸椎、胸椎、內臟（心肺）、肋骨、肩膀的排列，下至腰椎、骨盆、下肢等一連串的影響，最重要的是要學會了解自己體態，有脊椎側彎的患者，站在鏡子面前，學會在脊椎延長的狀況下把「走針」的身體逆轉回來，例如說頸部偏左、左邊肩膀偏高、右側胸椎向後旋轉，在這樣的情況下頸部偏右、左邊肩膀下壓右邊肩膀擡起、右側胸椎向前旋轉；以上動作若再配合呼吸的話為最佳，試著在吸氣時吸入在凹側的部分，同時利用前述的意象訓練想像著橫膈膜下降、凹側肺部充滿氣體，側彎最凸點往身體中線移動，而凹側往身體的右外側移動），如下圖。

每天矯正自己的體態，了解自己偏移的方向，執行正確的運動，持續且反覆的動作再學習，一而再再而三的提醒自己這樣的動作有沒有對稱，而不是每天認真個兩個小時運動，而是在身體活動的狀況下加強自我身體感覺意識，在足夠的時間下，才能有效且有感知的改善脊椎側彎。
