# 1701577179_64

## 主旨  
本文件旨在介紹高血糖危象的定義、成因、症狀、處理方式及預防方法，幫助患者與照護者辨識與應對高血糖相關的緊急狀況。糖尿病患者應注意高血糖症狀，並採取適當措施以避免再次發生。

## 重點整理  
### 什麼是高血糖？  
當體內嚴重缺乏胰島素時，血液中的葡萄糖無法進入細胞內被有效利用，因而囤積於血液中，導致高血糖。高血糖危象主要包括：  
- 酮酸中毒（Diabetic Ketoacidosis, DKA）  
- 高滲透壓高血糖症（Hyperosmolar Hyperglycemic State, HHS）  

### 高血糖的常見原因  
- 擅自停藥或減藥  
- 飲食不當或過量攝取  
- 生活壓力或情緒不穩  
- 疾病因素，如手術、感染或急性 illness  
- 未確知自己患有糖尿病，或未能接受適當治療  

### 高血糖的症狀  
| 症狀類型 | 具體表現 |  
|----------|----------|  
| **輕微症狀** | 多吃、多喝、多尿、口渴、體重快速減輕 |  
| **中度症狀** | 惡心、嘔吐、食慾不振、腹痛、皮膚乾燥、心跳加快、姿勢性低血壓、軟弱無力、脫水、視力減退或模糊 |  
| **嚴重症狀** | 精神恍惚、反應遲鈍、嗜睡，甚至昏迷，嚴重時可導致死亡 |  
| **酮酸中毒者** | 呼吸深而快，呼氣有水果味（acetone） |  
| **高血糖高滲透壓非酮酸性昏迷者** | 呼吸深而緩慢 |  

### 高血糖的處理步驟  
1. **立刻檢驗血糖**：確認血糖值是否異常。  
2. **補充水分**：防止脫水，可飲用清水或含電解質的飲品。  
3. **多休息**：避免活動，減少身體負擔。  
4. **儘快就醫**：若症狀持續或惡化，應立即前往醫院就診。  

### 預防高血糖的方法  
- 正確攝取食物，避免過量飲食  
- 定時服用降血糖藥物或注射胰島素  
- 定期監測血糖值，掌握血糖變化  
- 若出現上述高血糖症狀，應立即就醫治療  
- 對於高血糖的症狀，應提高警覺  
- 當有高血糖情形發生時，應找出發生原因，避免再次發生  
- 盡量避免感染  
- 保持規律運動  

## 流程/步驟  
1. **立刻檢驗血糖**：確認血糖值是否異常。  
2. **補充水分**：防止脫水，可飲用清水或含電解質的飲品。  
3. **多休息**：避免活動，減少身體負擔。  
4. **儘快就醫**：若症狀持續或惡化，應立即前往醫院就診。  

## 注意事項/警示  
- 高血糖危象可能迅速惡化，特別是酮酸中毒與高滲透壓狀態，屬於醫療急症。  
- 不可自行調整藥物劑量，應依醫囑服藥。  
- 糖尿病患者應定期追蹤血糖，並與醫療團隊保持溝通。  
- 若有高血糖症狀，應立即注意並尋找原因。  
- 預防感染，特別是對於糖尿病患者而言至關重要。  
- 定期運動有助於控制血糖。

認識高血糖危象

編號 ND-029

壹、 你知道什麼是高血糖嗎？
體內嚴重缺乏胰島素時，血液中葡萄糖無法進入細胞內有效被利用，而囤積於血液中，則造成高血糖。包括酮酸中毒、高滲透壓高血糖症。

貳、 高血糖的原因
一、擅自停藥或減藥。
二、飲食不當或過量。
三、生活壓力、情緒不安。
四、疾病造成，例如：開刀、生病、感染等。
五、不知道自己患有糖尿病，或未能適當治療。

參、 高血糖的症狀如下：
一、輕微者：多吃、多喝、多尿、口渴及體重迅速減輕。
二、噁心、嘔吐、食慾不振、腹痛、皮膚乾燥、心跳加快、姿勢性低血壓、軟弱無力、脫水、視力減退及模糊。
三、嚴重者：精神恍惚、反應遲鈍、嗜睡，甚至昏迷，嚴重時導致死亡。
四、酮酸中毒者，呼吸深而快、呼氣有水果味；高血糖高滲透壓非酮酸性昏迷者，呼氣則深而緩慢。

肆、 高血糖的處理
一、立刻檢驗血糖。
二、補充水分。
三、多休息。
四、儘快就醫。

伍、 預防高血糖的方法
一、正確攝取食物及飲食勿過量。
二、定時服用降血糖藥物或注射胰島素。
三、定期監測血糖。
四、如有上述高血糖的症狀時，應立即就醫治療。

五、對於高血糖的症狀，應提高警覺。

六、當有高血糖情形發生時，應找出發生原因，避免下次再發生。

七、盡量避免感染。

八、規律運動。
