# 核子醫學科

## 團隊簡介
核子醫學（Nuclear Medicine）是利用放射線同位素（Radioisotope）來做診斷造影及治療的一門醫學專科。核醫造影的優點是一種非侵襲性的檢查，只需注射同位素藥物即可進行造影，檢查過程只需靜靜平躺，病患並不會有不適感。利用放射同位素配製的各種藥物來進行身體各種器官功能方面的檢查，與放射診斷科偏重解剖學的影像相輔相成，可提供臨床醫師做出最正確的診斷及最完整的評估。屏基核子醫學科成立於2000年11月，為屏東地區早成立的核子醫學科，目前使用荷蘭飛利浦（Philips BrightView）核醫閃爍攝影儀，提供與醫學中心同級的各種先進功能照影服務，包括：心肌灌注掃描、全身骨骼掃描、腦部多巴胺神經元斷層掃描、甲狀腺掃描、肺血流灌注掃描、腎功能評估、Ga-67腫瘤發炎掃描、原子碘治療等，提供臨床醫師在診斷上有用的資訊。本院也提供屏東各醫院轉檢的服務。

## 團隊編制
- 醫師：核子醫學科專科醫師
- 護理/照護：專業護理團隊
- 醫事技術/其他：核醫技術人員
- 特色分工：提供核醫造影及原子碘治療服務

- 各種疾病功能檢查

### 主要治療/處置
- 核醫造影：提供各種核醫造影服務，包括心肌灌注掃描、全身骨骼掃描、腦部多巴胺神經元斷層掃描、甲狀腺掃描、肺血流灌注掃描、腎功能評估、Ga-67腫瘤發炎掃描等
- 原子碘治療：提供原子碘治療服務
- 院外轉檢服務：提供屏東各醫院轉檢的服務

## 檢查項目
- 心肌灌注掃描：用於評估心臟功能
- 全身骨骼掃描：用於評估骨骼疾病
- 腦部多巴胺神經元斷層掃描：用於評估腦部功能
- 甲狀腺掃描：用於評估甲狀腺功能
- 肺血流灌注掃描：用於評估肺部功能
- 腎功能評估：用於評估腎臟功能
- Ga-67腫瘤發炎掃描：用於評估腫瘤及發炎

## 設備與特色
- 荷蘭飛利浦（Philips BrightView）核醫閃爍攝影儀：提供與醫學中心同級的各種先進功能照影服務
- 核醫造影的優點是一種非侵襲性的檢查，只需注射同位素藥物即可進行造影
- 檢查過程只需靜靜平躺，病患並不會有不適感
- 利用放射同位素配製的各種藥物來進行身體各種器官功能方面的檢查
- 與放射診斷科偏重解剖學的影像相輔相成，可提供臨床醫師做出最正確的診斷及最完整的評估
