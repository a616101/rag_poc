# 胸腔外科

## 團隊簡介
過去屏東地區缺乏完整的胸腔外科照護，肺部及食道疾病需要手術的病患往往需要到高雄就醫，往往導致可治療的疾病惡化。為解決此問題，本院聘請來自醫學中心的胸腔外科專科醫師，同時引進胸腔鏡微創手術，讓屏東地區民眾可以接受與醫學中心同等級的醫療照護。醫師每年積極參與國內外胸腔外科學術活動，持續更新知識與技術，與醫學中心及國際醫療實務保持連結。

## 團隊編制
- 醫師：胸腔外科專科醫師（來自醫學中心）
- 護理/照護：專業護理團隊
- 醫事技術/其他：手術技術人員
- 特色分工：每年積極參與國內外胸腔外科學術活動，持續更新知識與技術

## 服務與治療
### 服務範圍（疾病/問題）
- 肺部腫瘤
- 食道癌
- 氣胸
- 血胸
- 胸部創傷
- 膿胸
- 縱膈腔疾病

### 主要治療/處置
- 肺部腫瘤手術：提供肺部腫瘤的診斷與手術治療
- 食道癌手術：提供食道癌的診斷與手術治療
- 食道重建手術：提供食道重建手術
- 胸腔鏡微創氣胸手術：微創手術治療氣胸
- 血胸處理：處理血胸
- 胸部創傷照護：提供胸部創傷的診斷與治療
- 胸管置放及照護：置放胸管並提供照護
- 葉克膜（ECMO）放置及照護：放置葉克膜並提供照護
- 胸腔鏡膿胸手術：微創手術治療膿胸
- 縱膈腔手術：提供縱膈腔疾病的手術治療
- 氣切手術：提供氣切手術
- 氣管內管插管：提供氣管內管插管
- 化療用人工血管注射座放置：為需要化療的病患置入人工血管注射座

### 特色門診/照護計畫（若有）
- 胸腔鏡微創手術：提供各種胸腔鏡微創手術，減少手術創傷
- 胸部創傷照護：提供胸部創傷的完整照護
- 葉克膜（ECMO）照護：提供葉克膜的放置及照護

## 檢查項目
- 影像學檢查：X光、電腦斷層、磁振造影
- 病理檢查：組織切片檢查

## 設備與特色
- 胸腔鏡設備：提供胸腔鏡微創手術
- 手術設備：提供傳統開胸手術
- 葉克膜（ECMO）設備：提供葉克膜的放置及照護
- 引進胸腔鏡微創手術，讓屏東地區民眾可以接受與醫學中心同等級的醫療照護

## 品質與安全
- 聘請來自醫學中心的胸腔外科專科醫師
- 引進胸腔鏡微創手術
- 讓屏東地區民眾可以接受與醫學中心同等級的醫療照護
- 醫師每年積極參與國內外胸腔外科學術活動，持續更新知識與技術
- 與醫學中心及國際醫療實務保持連結
