# 1701602176_502

## 主旨  
本文件為電腦斷層導引組織切片術（CT-guided biopsy）的衛教指引，涵蓋手術目的、術前準備、操作步驟、術後及出院後護理，以協助患者與家屬了解整個處置流程與注意事項。

## 重點整理  
| 項目 | 內容 |
|------|------|
| **手術目的** | 利用電腦斷層導引，將穿刺針置入疑似病灶，抽取組織進行病理檢查。適用於肺、肝、胰、淋巴結、後腹腔、骨盆腔、脊椎或其他病灶部位。 |
| **優點** | 可精準定位，取得足夠組織樣本，並減少傳統手術併發症風險。成功率約 69%～93%。 |
| **成功率影響因素** | 病灶太小（尤其小於 1 公分）是導致無法取得足夠組織的主要原因。 |
| **患者配合要求** | 需絕對配合醫師指令，進行吸氣與閉氣動作。 |
| **術後注意事項** | 切片完成後，穿刺部位以枕頭加壓止血 2 小時，之後需平躺休息 4 小時。應避免說話、劇烈活動，以防氣胸。待隔日胸部 X 光確認無氣胸後方可恢復說話。 |
| **出院後注意事項** | 保持傷口乾燥，留意感染徵兆（如紅腫、疼痛、灼熱感或發燒），應立即就醫。保持生活作息規律，遵醫囑服藥與回診。 |

## 流程/步驟  
1. 請病人平躺於檢查台上。  
2. 醫師消毒處置部位，並蓋上無菌治療巾。  
3. 醫師於穿刺處注射局部麻醉劑，以降低疼痛不適。  
4. 醫師以電腦斷層導引，將穿刺針置入病灶，抽取部分病灶組織進行病理檢查。  
5. 處置結束後，醫師將傷口覆蓋紗布。  

## 注意事項/警示  
- **術前準備**：  
  - 由臨床醫師向患者及家屬說明手術目的、可能風險與併發症。  
  - 抽血檢查凝血功能及血小板值。  
  - 填寫檢查同意書及說明書。  
  - 空腹時間至少 4 小時以上；當天可服用降血壓藥物，但應避免服用降血糖藥物，以防低血糖。  
  - 更換檢查服，並移除所有金屬飾品（如項鍊、手錶、含鋼絲胸罩）。  
  - 處置前請排空膀胱，必要時使用尿套或紙尿褲。  
  - 若需打點滴注射，請家屬陪同並推床送至電腦斷層檢查室。  

- **術中配合**：  
  - 患者需依醫囑配合吸氣與閉氣動作，以利影像定位與穿刺準確性。  

- **術後注意事項**：  
  - 切片完成後，穿刺部位以枕頭加壓止血 2 小時，之後需平躺休息 4 小時。  
  - 請在床上使用尿壺或便盆，避免下床活動。  
  - 護理師將密切監測生命徵象變化。  
  - 觀察穿刺部位是否有滲液或出血，如有異常應立即通知醫護人員。  
  - 接受肺部切片後應避免說話或發出聲音，並避免劇烈活動，以防造成氣胸。  
  - 待隔日追蹤胸部 X 光檢查結果，確認無氣胸後方可恢復說話。  
  - 檢查結束若無其他限制，即可進食。  
  - 護理師將觀察傷口狀況，並於 24 小時後進行傷口換藥。

電腦斷層導引組織切片術
Computed Tomography Guide Biopsy

| 編號 | ND 內-113 |
| :--- | :--- |
| 製訂日期 | 2020.10 |

一、處置目的：
1. 利用電腦斷層攝影的導引，將穿刺針置入疑似病灶，抽取該病灶的一部分，進行病理檢查。本處置可應用在肺、肝、胰、淋巴結、後腹腔、骨盤腔、脊椎或其他病灶所在部位。
2. 本處置的優點是能在經過高度準確的電腦斷層定位下，獲取足夠的組織樣本，而又能減少傳統手術後可能引起的併發症。獲取足夠組織樣本的成功率約為 69%～93%；而不能成功獲取足夠組織樣本的原因是病灶太小，尤其小於 1 公分的病灶。
3. 本處置需要病人絕對合作配合醫師指令吸氣、閉氣。

二、處置前準備：
1. 臨床醫師向家屬及病人解釋處置之目的及可能發生的危險性及併發症。
2. 抽血檢驗凝血功能及確認血小板值。
3. 填寫檢查同意書及說明書。
4. 空腹時間 4 小時以上：處置當天可服用降血壓藥物，但降血糖藥物則避免服用，以免造成低血糖。
5. 更換檢查服並需將身上金屬飾品包括：項鍊、手錶、含鋼絲胸罩脫掉。
6. 處置前請先上廁所，排空膀胱，視需要使用尿套或紙尿褲；執行打針點滴注射，請家屬陪同並推床送至電腦斷層檢查室。

三、處置步驟：
1. 請病人平躺於檢查台上。
2. 醫師消毒處置部位，蓋上無菌治療巾。
3. 醫師予穿刺處注射局部麻醉劑，以降低疼痛不適。
4. 醫師以電腦斷層方式作引導，將穿刺針置入病灶，抽取一部份病灶進行病理檢查。
5. 醫師處置結束後將傷口覆蓋上紗布。

四、處置後注意事項：

1. 切片完成後穿刺部位以枕頭加壓止血 2 小時，之後要平躺休息 4 小時，請在床上使用尿壺、便盆，避免下床活動。

2. 護理師將密切測量生命徵象變化。

3. 觀察穿刺部位是否有滲液或出血狀況，若有發生請立刻向醫護人員反應。

4. 接受肺部切片後應避免說話或發出聲音，劇烈活動，避免造成氣胸；直到隔日追蹤胸部 X 光檢查結果，未發現氣胸才可以說話。

5. 檢查結束若無其他限制即可進食。

6. 護理師會觀察傷口狀況並於 24 小時之後傷口換藥。

五、出院後注意事項：

1. 保持傷口乾燥。

2. 若傷口周圍出現疼痛、紅腫、灼熱感或發燒情形，表示發炎徵兆，應立即就醫。

3. 生活作息規律，充足的睡眠與休息並有適度的活動，避免過度勞累。

4. 依照醫師指示按時服藥並定期回門診複診。
