# 1701602753_167

## 主旨  
本文件列舉本院不建議或不宜磨粉的口服藥物，提醒醫療人員與患者注意，避免因錯誤服用方式影響藥效或造成不適。  

## 重點整理  
- 本院不建議磨粉的口服藥物多為緩釋劑型（如 SR、CR、LA、XL）或特殊劑型（如腸溶錠、軟膠囊、發泡錠等）。  
- 磨粉可能導致藥物釋放過快、劑量不準、胃腸道刺激或藥效降低。  
- 部分藥物需整粒吞服，不可掰開、咀嚼或磨粉。  
- 某些藥物雖可掰半，但不可磨粉。  
- 許多藥品為持續釋放劑型，不可磨粉、咀嚼或掰半，以免破壞藥效。  
- 部分藥品需整粒吞服，不可壓碎或磨粉，以避免藥物過早釋放。  
- 有藥品可溶於水或於水中崩散，但需注意是否可磨粉或咀嚼。  
- 腸溶錠或腸溶微粒膠囊不可磨粉，以免破壞腸溶包衣。  
- 軟膠囊內含油狀物，應整粒吞服，不可咬碎。  
- 多數藥品不可磨粉、咀嚼或掰開，以免破壞藥物釋放機制。  
- 部分藥品需冷藏保存，以防潮解。  
- 某些藥品可能刺激黏膜，需注意口腔使用方式。  

## 流程/步驟  
- 請依藥品劑型判斷是否可磨粉、咀嚼或掰半。  
- 持續釋放劑型（如 SR / CR / LA / XL）不可壓碎、咀嚼或磨粉。  
- 腸溶錠或腸溶微粒膠囊不可磨粉，以免破壞腸溶包衣，導致藥物在胃中釋放。  
- 軟膠囊或油狀物藥物不可磨粉，以免造成胃部刺激或藥物流失。  
- 部分藥物可溶於水服用，但不可磨粉。  
- 藥品如需冷藏，應置於冰箱保存以維持藥效。  
- 藥品如需舌下含服，不可吞服。  

## 注意事項/警示  
- 所有緩釋劑型（如 SR / CR / LA / XL）均不可壓碎或咀嚼服用。  
- 腸溶錠不可磨粉，否則會破壞腸溶包衣，導致藥物在胃中釋放，影響療效。  
- 軟膠囊或油狀物藥物不可磨粉，以免造成胃部刺激或藥物流失。  
- 部分藥物因易吸濕、潮解或有特殊氣味，不建議磨粉。  
- 不可磨粉：多數持續釋放劑型、腸溶錠、軟膠囊等，磨粉會導致藥物快速釋放，增加胃腸道刺激風險。  
- 不可咀嚼或掰半：如 Fosamax plus、Imdur、Iressa 等，破壞劑型可能導致藥效降低或副作用增加。  
- 需整粒吞服：如 Imigran、Kaletra、Ofev 等，應完整吞服，不可壓碎或磨粉。  
- 可溶於水服用：如 Lamictal，可溶於水後服用，但不可磨粉。  
- 需冷藏保存：如 Leukeran，應冷藏以維持藥效。  
- 可能引起口腔變色：如 Pyridine，服用後可能導致口腔變色。  
- 不可管灌：如 Nitrostate，不可經鼻胃管給藥。  
- 需溶於水後服用：如 Neoral，需先溶於水後再吞服。  
- 不可磨粉：多數腸溶或持續釋放劑型藥物，磨粉會破壞藥物包膜，導致藥物提前釋放，影響療效或增加副作用。  
- 不可咀嚼：特別是持續釋放劑型，咀嚼會導致藥物快速釋出，可能引發毒性反應。  
- 不可掰半：除非標示「可撥半」，否則應整顆服用。  
- 特殊使用方式：如 Temegesic（Buprenorphine）需舌下含服，不可吞服。
