# 放射科

## 團隊簡介
本科提供各種影像診斷服務，擁有先進的影像設備，包括384切及1152切電腦斷層掃描儀、磁振造影、數位乳房攝影、骨質密度檢查、血管攝影等。品質政策為：安全（以病人安全為中心的就醫環境）、專業（嚴謹的態度與先進的設備）、親切（視病猶親的服務精神）、卓越（追求與全球同步的醫療品質）。

## 團隊編制
- 醫師：放射科主任1名、主治醫師5名
- 護理/照護：護理組
- 醫事技術/其他：總技師、組長、各組技術人員（一般攝影組、特殊組、電腦斷層組、磁振造影組、乳房攝影組、骨質密度組、血管攝影組）
- 特色分工：各組分工合作，提供各種影像檢查服務

## 服務與治療
### 服務範圍（疾病/問題）
- 各種疾病影像診斷

### 主要治療/處置
- 一般攝影：提供一般診斷X光檢查
- 特殊攝影：提供特殊攝影檢查
- 電腦斷層掃描：提供電腦斷層掃描檢查，包括常規、急診及自費健檢（如心臟鈣化分數、冠狀動脈攝影、虛擬大腸鏡、低劑量電腦斷層等）
- 磁振造影：提供磁振造影檢查，為非侵入性影像技術，無輻射線，提供清晰的腦部、脊椎、關節、內臟器官影像，用於偵測病灶如腫瘤、發炎等
- 數位乳房攝影：提供數位乳房攝影檢查，用於早期發現乳癌
- 骨質密度檢查：提供雙能量骨質密度儀檢查，測量骨質密度、脂肪量、肌肉量，提供詳細的身體組成分析
- 數位透視攝影：提供數位透視攝影檢查，用於特殊檢查如上消化道攝影、下消化道攝影、食道攝影及侵入性切片檢查
- 血管攝影：提供雙平面數位血管攝影檢查，用於各種診斷及介入治療（如經皮穿肝膽道引流、導管血管攝影、經皮穿肝膽道攝影及引流等），整合電腦斷層進行血管影像及治療，為屏東第一台執行急性缺血性中風機械取栓術及進階神經血管治療

### 特色門診/照護計畫（若有）
- 自費健檢：提供自費健檢服務，如心臟鈣化分數、冠狀動脈攝影、虛擬大腸鏡、低劑量電腦斷層等
- 血管介入治療：提供各種血管介入治療服務

## 檢查項目
- 一般診斷X光：提供一般診斷X光檢查
- 電腦斷層掃描：提供電腦斷層掃描檢查
- 磁振造影：提供磁振造影檢查
- 數位乳房攝影：提供數位乳房攝影檢查
- 雙能量骨質密度儀：提供骨質密度檢查

## 設備與特色
### 電腦斷層掃描儀
- 384切螺旋式電腦斷層掃描儀2台：用於常規、急診及自費健檢
- 1152切雙源雙能量電腦斷層掃描儀（Siemens SOMATOM Force）1台：為高屏地區第一台低劑量雙源雙能量電腦斷層，用於常規、急診及自費健檢（如心臟鈣化分數、心臟動態影像、虛擬大腸鏡、低劑量輻射）

### 磁振造影
- 磁振造影2台：為非侵入性影像技術，無輻射線，提供清晰的腦部、脊椎、關節、內臟器官影像

### 數位乳房攝影X光機
- 用於早期發現乳癌，50歲以上女性檢測率高，建議40歲以上女性每年進行乳房攝影，45-69歲女性及有家族史的女性建議每2年進行一次

### 數位一般診斷X光機
- 4台：用於非侵入性影像檢查，檢查內部結構如骨骼、肌肉等

### 雙能量骨質密度儀
- 使用兩種低能量X光測量骨質密度、脂肪量、肌肉量，提供詳細的身體組成分析，對早期發現停經後女性及代謝疾病患者的骨質疏鬆症很重要，可預防骨折並指導治療

### 數位透視攝影X光機
- 1台：用於特殊檢查如上消化道攝影、下消化道攝影、食道攝影及侵入性切片檢查

### 血管攝影室
- 雙平面數位血管攝影機1台：用於各種診斷及介入治療（如經皮穿肝膽道引流、導管血管攝影、經皮穿肝膽道攝影及引流等），整合電腦斷層進行血管影像及治療，為屏東第一台執行急性缺血性中風機械取栓術及進階神經血管治療
