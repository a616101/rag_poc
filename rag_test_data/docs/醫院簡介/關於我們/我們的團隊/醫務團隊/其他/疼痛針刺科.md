# 疼痛針刺科

## 團隊簡介
疼痛針刺療法為結合傳統中醫針灸（體針、頭皮針、鼻針、耳針）與東西方醫學研究心得的方法，不依循傳統經絡，專注於「肌肉的扳機點」。此療法可處理過去難以治療的疾病，包括「現代文明病」（如頭暈、腰痛、慢性頭痛、不明原因胃痛、腹瀉、失眠）及「難治疾病」（如持續流淚、視力模糊、耳鳴、耳朵癢、不明原因胸悶、呼吸不順、手麻、腹脹、打嗝）。人體有400多條肌肉，每條肌肉都有扳機點，疼痛往往來自附近肌肉而非患部本身，治療扳機點可有效處理疾病。肌肉扳機點的存在有基礎及臨床醫學實驗如肌電圖及紅外線熱像圖支持。梁氏頭皮針為整合醫學方法，融合各種有效頭皮針法（如朱氏、方氏、焦氏、國際標準、戴氏、山元式頭皮針），包含原始點療法，如按摩頭部原始點（從翳風穴到風池穴）。整合中西醫優勢及現代科學發現，將知識轉化為頭皮針治療點。隨著對大腦認識的進步，梁氏頭皮針持續演進，提供更多樣且有效的治療。創新組合療法包括微針系統、越南面部針灸、鼻針、雙向刺激（後溪穴、Loo's Point）、頭皮針+扳機點乾針。也包含「膽經的包打聽功能」及「三焦經監視五臟六腑功能」。臨床應用涵蓋30多種疾病，包括中風後遺症、視神經萎縮、巴金森氏症、三叉神經痛、顏面神經麻痺、老年失智、兒童智能障礙、注意力不集中、記憶力差、理解力差等困難疾病。

### 主要治療/處置
#### 治療方法
1. 針灸、腳底按摩、耳針、鼻針、頭皮針、梁氏針法等
2. 中西醫針灸方法：先進之肌肉扳機點、體壁敏化點、雙向刺激法、皮內針、動態針灸、小針刀等
3. 其他方法：吃藥、打針、神經阻斷等

### 特色門診/照護計畫（若有）
- 疼痛針刺療法：結合傳統中醫針灸與東西方醫學研究心得，專注於「肌肉的扳機點」
- 梁氏頭皮針：整合醫學方法，融合各種有效頭皮針法
- 原始點療法：按摩頭部原始點（從翳風穴到風池穴）
