# 1701578301_723

## 主旨  
本文件為經尿道攝護腺切除手術後之居家照護指引，包含手術後注意事項與返家照護建議，以促進恢復並預防併發症。

## 重點整理

| 項目 | 內容 |
|------|------|
| **手術後注意事項** | 導尿管固定於大腿內側，需保持腳伸直，避免拉扯；次日拔除導尿管後可下床活動，尿袋應置於大腿以下，防止尿液倒流。 |
| **飲水與排便** | 每日飲水 2000–3000cc，多攝取蔬果，避免用力排便，以防傷口出血。 |
| **排尿與症狀** | 拔管後可能出現尿急、尿失禁、小便次數多、疼痛或微血尿，屬暫時現象，通常會逐漸改善。 |
| **活動限制** | 三個月內避免騎車、提重物、蹲久、溫水坐浴、跑步等劇烈運動，應緩步上下樓梯。 |
| **性生活與併發症** | 一個月內避免行房，一個月後可漸恢復；部分患者可能出現陽萎或逆行性射精，應回診諮詢。 |
| **飲食與習慣** | 一個月內避免辛辣、薑、酒類等刺激性食物，養成定時解尿習慣，勿憋尿。 |
| **藥物與就醫** | 出院後依醫囑服藥，若出現連續大量血尿、發高燒、無法排尿、膀胱膨脹或尿失禁，應立即返院處置。 |

## 流程/步驟

1. **手術後至隔天早上**  
   - 導尿管進行牽引並固定於大腿內側。  
   - 牽引時腳需保持伸直，避免拉扯，以減少出血。

2. **手術後第二天**  
   - 醫師將導尿管拔除。  
   - 可下床活動，尿袋應維持在大腿以下，防止尿液倒流引發感染。

3. **每日照護**  
   - 每日飲水 2000–3000cc。  
   - 觀察小便量及顏色，若出現減少或顏色改變，立即通知醫護人員。  
   - 多攝取蔬菜水果，避免用力解大便，排便不順時應告知醫護人員。

4. **返家照護**  
   - 拔管後若出現尿急、尿失禁、小便次數多、疼痛或微血尿，屬暫時現象，通常會在三個月內逐漸消失。  
   - 白天可多飲水 2000–3000cc，晚上則減少攝取，以免影響睡眠。  
   - 三個月內避免騎腳踏車、摩托車、提重物、蹲太久、溫水坐浴及跑步等劇烈運動。  
   - 少數患者可能出現陽萎或逆行性射精，如有類似症狀，應回門診洽詢主治醫師。  
   - 一個月內避免行房，一個月後可漸恢復性生活；若行房後小便有混濁現象，屬正常現象，不影響健康。  
   - 一個月內避免刺激性食物，如辣、薑、酒類等。  
   - 養成解尿習慣，勿憋尿。  
   - 出院後依醫囑服藥，若出現連續大量血尿、發高燒、無法排尿、膀胱膨脹或尿失禁，應盡快返院處置。

## 注意事項/警示

- **導尿管管理**：牽引時腳需保持伸直，勿拉扯，防止出血。
- **尿袋位置**：活動時尿袋應置於大腿以下，避免尿液倒流導致感染。
- **排便注意**：避免用力排便，以防傷口出血。
- **症狀觀察**：若出現小便減少、顏色改變、血尿、發燒、無法排尿等異常，應立即就醫。
- **飲食禁忌**：一個月內避免辛辣、薑、酒類等刺激性食物。
- **性生活限制**：一個月內避免行房，以減少併發症風險。
- **運動限制**：三個月內避免劇烈運動，應緩步活動。

屏基醫療財團法人屏東基督教醫院
經尿道攝護腺切除手術後須知

| 編號 | ND外-004 |
| :--- | :--- |

一、手術後注意事項：
1. 手術後至隔天早上，導尿管會進行牽引並固定於大腿內側，牽引時腳需保持伸直、尿管勿拉扯，目的為減少出血。
2. 手術後第二天醫師會將尿管牽引取下，即可下床活動，活動時尿袋須維持大腿以下，以免造成尿袋內小便倒流引發感染。
3. 每天需多喝水約 2000-3000cc，隨時觀察小便量及顏色，如：小便減少或尿液顏色改變時，請立即告知醫護人員。
4. 多攝取蔬菜水果，不要太用力解大便，若排便不順，告知醫護人員，以免傷口出血。

二、返家照護
1. 手術後拔除導尿管時，如有尿急、尿失禁、小便次數多、小便疼痛或輕微血尿，是手術後暫有之現象，症狀會逐漸消失，三個月內繼續門診治療。
2. 白天可多飲水約 2000cc-3000cc，晚上則減少水份攝取，以免影響睡眠。
3. 三個月內避免騎腳踏車、摩托車、提重物、蹲太久、溫水坐浴及避免跑步等劇烈運動，應緩步上下樓梯。
4. 少數病人手術後會有一些合併症，例如：陽萎、逆行性射精，如有類似症狀，請回門診洽詢主治醫師。
5. 一個月內避免行房，一個月後可漸恢復性生活，若行房後小便有混濁現象，不用驚慌，這是射精時逆流入膀胱的原因，解尿時會隨小便流出，不影響健康。
6. 一個月內避免刺激性食物，如辣、薑、酒類…等。
7. 養成解尿習慣，請勿憋尿。
8. 出院後，請依照醫師指示，按時服藥，如有不適情況，例如：連續大量血尿、發高燒、無法排尿、膀胱膨脹、有尿失禁現象，應盡快返院處置。
