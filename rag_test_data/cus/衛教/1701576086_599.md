# 1701576086_599

## 主旨  
本指引旨在協助腦血管疾病患者透過飲食調整，控制血壓，降低心血管風險。建議減少鈉攝取，增加鉀與鈣的攝取，並避免多種高鈉加工食品。同時，提供飲食調理原則，包括蛋白質、纖維質、油脂、飽和脂肪酸及膽固醇的攝取建議，並介紹低鹽烹調方法，以提升食物風味，促進健康與康復。

## 重點整理  
- **核心原則**：控制鈉攝取，預防高血壓。  
- **建議飲食**：選擇低鈉食品，如「健康低鈉鹽」；多攝取新鮮蔬果以補充鉀，並攝取牛奶、乳製品、小魚、魚乾、蝦類、蛤及牡蠣等含鈣食物。  
- **應避免食品**：包含多種加工肉品、調味品、罐頭及高鈉蔬果等。  
- **蛋白質來源**：建議多選擇魚類、雞鴨、豆類及其製品。  
- **膳食纖維來源**：選擇未加工的豆類、蔬菜、水果及全穀類。  
- **油脂攝取**：避免油炸、油煎或油酥等高油脂烹調方式；優先選擇單元不飽和脂肪酸含量高的油品（如花生油、菜籽油、橄欖油），並減少使用飽和脂肪酸含量高的油品（如豬油、牛油）。  
- **膽固醇攝取**：建議少攝取內臟、蟹黃、蝦卵、魚卵等高膽固醇食物。  
- **每日用油與肉類攝取量**：烹調用油不應超過 2~8 茶匙，肉類總攝取量不應超過 5~7 兩。  
- **腎功能不佳者注意**：應避免攝取高鉀食物。  
- **中藥材與辛香料使用**：應注意個人體質，避免過量。  
- **低鈉調味品使用**：應依營養師指導使用。  

## 流程/步驟  
### 低鹽烹調七步法  
1. 選擇新鮮的食物。  
2. 利用具有特殊風味的蔬菜（如：青椒、蕃茄、洋蔥、香菇、九層塔）與味道清淡的食物一起烹煮。  
3. 利用蔥、薑、蒜等，經沙拉油爆香後產生的油香味來增加食物可口性。  
4. 在烹調時，使用檸檬、鳳梨、柳丁汁等各種酸味來增加食物味道。  
5. 採用易保持食物原味的烹調方式。  
6. 使用當歸、枸杞、川芎、紅棗、黑棗等中藥材，以及肉桂、五香、八角、花椒等辛香料，以減少鹽的添加。  
7. 使用含鈉量較低的醬油或食鹽來代替調味，但須依照營養師指導使用。  

### 飲食調整步驟  
1. 選擇蛋白質來源：優先選擇魚類、雞鴨、豆類及其製品。  
2. 攝取富含纖維的食物：包括未加工的豆類、蔬菜、水果及全穀類。  
3. 控制油脂攝取：  
   - 避免油炸、油煎或油酥的烹調方式。  
   - 選用單元不飽和脂肪酸含量高的油品（如花生油、菜籽油、橄欖油）。  
   - 減少使用飽和脂肪酸含量高的油品（如豬油、牛油）。  
4. 限制膽固醇攝取：避免內臟、蟹黃、蝦卵、魚卵等高膽固醇食物。  
5. 控制每日用油與肉類攝取量：  
   - 烹調用油：不超過 2~8 茶匙。  
   - 肉類總攝取量：不超過 5~7 兩。  

## 注意事項/警示  
- 鈉是鹹味的主要來源，除了食用鹽，也常隱藏於多種調味品與加工食品中。  
- 高鈉飲食會加重高血壓，增加腦血管疾病復發風險。  
- 建議使用「健康低鈉鹽」作為替代品，以減少鈉攝取。  
- 飲食調整應配合醫師建議，並定期追蹤血壓與營養狀況。  
- 腎功能不佳者應避免攝取高鉀食物。  
- 使用中藥材與辛香料時，應注意個人體質，避免過量。  
- 低鈉調味品雖可替代，但使用前應諮詢專業營養師，以確保營養均衡。  
- 飲食應避免過於油膩，以減少油脂、飽和脂肪酸及膽固醇的攝取。  
- 烹調方式應以清蒸、水煮、涼拌等為主，以降低油脂攝取。  
- 每日用油與肉類攝取量應嚴格控制，以維持健康飲食。

屏基醫療財團法人屏東基督教醫院
腦血管疾病患者飲食指導

編號 ND 內-035
製訂日期 2003.11
八修日期 2020.05

一、腦血管疾病常合併有高血壓，
建議您減少飲食中鈉的攝取量，
增加鉀、鈣的攝取量。

二、鈉—鹹味的來源，是食用鹽的主要成份。
除了食用鹽，鈉還存在許多調味品中。預防高血壓，建議您減少下列食品之食用：

1. 蛋豆魚肉類：火腸、香腸、燻雞、滷味、豆腐乳、魚肉鬆、肉醬、各式罐頭、炸雞等。

2. 麵包、蛋糕及鹹甜餅乾、奶酥、油麵、速食麵、速食米粉、速食冬粉等。

3. 油脂類：奶油、瑪琪琳、沙拉醬、蛋黃醬等。

4. 蔬菜類：醃製榨菜、酸菜、醬菜、豌豆莢、青豆仁及各種加鹽的加工蔬菜汁和蔬菜罐頭。

5. 水果類：蜜餞、脫水水果及各類加鹽的罐頭水果和加工果汁。

6. 調味品：味精、豆瓣醬、辣椒醬、沙茶醬、甜麵醬、蠔油、烏醋、蕃茄醬等。

7. 其他：雞精、牛肉精、炸洋芋片、爆米花、米果、運動飲料。

三、鉀—有助於血壓的控制

1. 新鮮食物中蔬菜、水果均含有大量鉀離子應多攝取。

2. 腎功能不好者不宜使用高鉀食物。

四、鈣—預防骨質疏鬆又能降血壓

1. 牛奶及乳製品為鈣質最好的來源，建議每天最好飲用一杯（240cc）。

2. 小魚（連骨進食）、魚乾、蝦類、蛤及牡蠣等，都是鈣質的食物來源。

五、低鹽烹調（如何食之有味？）

1. 選擇新鮮的食物。

2. 強烈風味蔬菜的利用—選用具有特殊風味的蔬菜（如：青椒、蕃茄、洋蔥、香菇、九層塔等）和味道清淡的食物一起烹煮，如蕃茄炒蛋。

3. 油脂的利用—利用蔥、薑、蒜等，經沙拉油爆香後所生的油香味來增加食物的可口性。

4. 酸味的利用—在烹調時，使用檸檬、鳳梨、柳丁汁等各種酸味，來添加食物的味道。

5. 採用易保持食物原味之烹調方式。

6. 中藥材與辛香料的利用—使用當歸、枸杞、川芎、紅棗、黑棗等中藥材及肉桂、五香、八角、花椒等辛香料，可以減少鹽量的添加。

7. 低鈉調味品的利用—使用含鈉量較低的醬油或食鹽來代替調味，但須依照營養師指導使用。

六、含蛋白質與纖維質的食物選用：

1. 蛋白質的來源：請多選擇魚類、雞鴨、豆類及其製品。
2. 選用富含纖維質的食物：如未加工的豆類、蔬菜、水果及全穀類。

七、減少食物中油脂量，飽和脂肪酸及膽固醇的攝取：

1. 飲食不可太油膩：避免油炸、油煎或油酥的烹調方式，少吃含油多的食物。
2. 炒菜宜選用單元不飽和脂肪酸高者（如花生油、菜籽油、橄欖油、大豆油、玉米油、葡萄籽油、胡麻油等）；少用飽和脂肪酸含量高者（如豬油、牛油），烹調宜多採用清蒸、水煮、涼拌等方式。
3. 少吃膽固醇含量高的食物：如內臟、蟹黃、蝦卵、魚卵等。
4. 每人每日的烹調用油，最好不要超過 2~8 茶匙，肉類總攝取量不要超過 5~7 兩。
