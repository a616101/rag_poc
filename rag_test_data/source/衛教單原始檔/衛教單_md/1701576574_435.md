# 1701576574_435

## 主旨  
本文件為住院及出院後雙腔靜脈導管照護的注意事項與應對措施，旨在協助患者避免感染、導管脫落或移位等併發症，並提供緊急情況下的處理指引。

## 重點整理  
- 住院期間由透析護理人員執行換藥，避免自行處理。  
- 若導管周圍出現紅腫、熱痛、滲血、發燒或異常分泌物，應立即通知護理人員。  
- 勿自行撕開膠布、觸碰傷口或拉扯導管。  
- 保持導管通暢，避免壓迫傷口，坐姿彎曲不超過 90 度，避免過度牽扯或伸展患部肢體。  
- 沐浴應採用擦拭方式，避免弄濕傷口，並穿著寬鬆衣物。  
- 導管夾子應維持關閉，勿自行打開或更換藥物。  
- 出院後若出現導管周圍發紅、腫脹、發熱、疼痛、滲血，或導管脫出，應立即就醫。  
- 提供諮詢電話與緊急聯絡方式。

## 流程/步驟  
- 住院期間：由透析護理人員執行換藥，定期檢查導管周圍狀況。  
- 出院後：如發現導管周圍發紅、腫脹、發熱、疼痛、滲血，或導管脫出，應立即就醫。  
- 導管脫出時，應以手壓迫傷口止血，並立即至醫院求治。

## 注意事項/警示  
#### 一、住院中雙腔靜脈導管照護注意事項：  
1. **避免感染**：住院期間由透析護理人員執行換藥。  
2. **異常症狀處理**：若固定導管的膠帶脫落、傷口出現異常疼痛、明顯紅腫熱痛、滲血、發燒或異常滲出液，請立即通知護理人員。  
3. **勿自行操作**：請勿自行撕開膠布、觸碰患部，或抓撓、拉扯導管。  
4. **保持導管通暢**：  
   - 勿壓迫傷口處。  
   - 避免側睡壓迫插管處。  
   - 若導管置於下肢，活動時應避免過度牽扯及伸展。  
   - 坐姿時，勿彎曲大於 90 度。  
5. **活動限制**：可活動，但應避免過度牽扯及伸展患部肢體。  
6. **個人衛生**：  
   - 沐浴應採用擦拭方式，避免弄濕傷口。  
   - 避免因弄濕造成傷口發炎感染。  
7. **穿著建議**：宜穿著寬鬆衣褲，避免更衣時造成導管牽扯或移位。  
8. **導管夾子管理**：導管上的夾子應維持關閉，勿自行打開或換藥，以避免感染。  

#### 二、出院返家後應就醫情形：  
1. 觀察導管留置處是否有發紅、腫脹、發熱、疼痛及滲血，須即刻至醫院求治。  
2. 導管不慎脫出時，應以手壓迫傷口止血，並立即至醫院求治。

屏基醫療財團法人屏東基督教醫院
認識雙腔靜脈導管照護

| 編號 | ND 內-051 |
| :--- | :--- |
| 製訂日期 | 2003.11 |
| 三修日期 | 2021.02 |

一、住院中雙腔靜脈導管照護注意事項：

1. 避免患部感染：住院期間由透析護理人員為您執行換藥。
2. 若固定導管的膠帶脫落或暴露在外或傷口異常疼痛，明顯紅腫熱痛、瘀血、滲血、發燒、異常滲出液，請立即通知護理人員。
3. 請勿自行將膠布撕開及以手碰觸患部及抓癢或拉扯導管。
4. 保持導管通暢：勿壓迫傷口處，應避免側睡壓迫插管處。若放置位置於下肢時，可活動，勿過度牽扯及伸展，坐姿勿彎曲大於 90 度，坐姿儘量勿彎曲大於 90 度。
5. 可活動，但勿過度牽扯及伸展患部肢體。
6. 注意個人衛生，沐浴宜採擦澡方式，避免弄濕傷口造成發炎感染情形。
7. 宜穿著寬鬆衣褲，避免更衣時造成導管牽扯移位，如弄濕隨時請護理人員更換。
8. 導管上的夾子，需維持關閉，勿自行打開夾子及換藥避免造成感染。

二、出院返家如有下列情形請至醫院求治：

1. 觀察導管留置處是否有發紅、腫脹、發熱、疼痛及滲血，須即刻至醫院求治。
2. 導管不慎脫出時，以手壓迫傷口止血，須即刻至醫院求治。
