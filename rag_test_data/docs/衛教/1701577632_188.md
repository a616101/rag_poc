## 主旨
本文件為屏基醫療財團法人屏東基督教醫院提供的「人工血管 Port-A」照護指引，說明其安裝原因、使用方法、日常注意事項及相關注意事項，以協助患者安全使用。

## 重點整理
- **安裝原因**：為降低化學治療時藥物外滲或外漏導致血管及周邊組織壞死的風險。
- **使用方式**：
  - 住院期間：由醫護人員以特製角針穿刺，並以生理食鹽水或抗凝劑沖洗，每5天消毒並更換透明敷料，每7天更換角針。
  - 出院後：每月回門診掛號沖洗，以避免阻塞或凝血。
- **日常注意事項**：
  - 保持傷口乾燥，避免弄濕。
  - 注意是否有紅、腫、熱、痛或分泌物等感染跡象，應立即就醫。
  - 避免過度活動裝置手臂，如360度旋轉、擴胸或提重物，以防移位。
- **置入位置**：通常位於左前胸或右前胸。

## 流程/步驟
### 住院期間使用流程
1. 醫護人員使用特製角針，穿刺「人工血管」中間鼓起的軟橡皮。
2. 感到些許刺痛，類似打針。
3. 角針打入後，以生理食鹽水或抗凝劑沖洗，確保管路暢通。
4. 每5天消毒注射部位，並更換透明敷料。
5. 每7天重新更換一次角針。
6. 避免弄溼敷料。

### 出院後與完成治療後
- 每月回門診掛號沖洗人工血管，以防止阻塞或凝血。

## 注意事項/警示
- **傷口護理**：
  - 手術後傷口約4-5公分，術後三天以紗布覆蓋，每日更換。
  - 依醫師指示決定是否拆線。
- **消毒與保護**：
  - 出院返家時，若傷口未癒合，每天使用無刺激性的優碘藥水消毒。
  - 使用無菌紗布覆蓋傷口，並暫時避免弄溼。
- **感染警訊**：
  - 注意傷口是否有紅、腫、熱、痛或分泌物。
  - 若有體溫變化，應立即回醫院就醫。
- **活動限制**：
  - 避免360度旋轉手臂、擴胸動作或提過重物品，以防人工血管移位。
  - 一般日常活動不受影響。

人工血管 Port-A 照護

一、 為什麼要裝「人工血管」？

需要執行化學治療時，肢體上的血管除了可能會變硬，若不慎也可能會造成化學藥物的外滲或外漏，使局部血管以及週邊組織壞死。而「人工血管」則可大幅的降低化學治療時外滲或外漏的危險性。

二、 「人工血管」如何使用？

（一）住院期間

醫護人員會使用特製的角針，插入「人工血管」中間鼓起的軟橡皮，這時就像打針一樣會感到些許的刺痛。等到角針打入後，再以生理食鹽水或抗凝劑沖洗人工血管，來確保管路的暢通。之後每5天護理人員就會消毒注射部位，並更換透明敷料，每7天重新更換一次角針。請避免弄溼敷料（如圖一）。

圖一 人工血管置入處通常位於左前胸或右前胸

（二）出院後與完成治療後

為避免人工血管有阻塞、凝血的情形發生，故必須每個月回門診掛號沖洗人工血管。

三、 「人工血管」日常注意事項

1. 人工血管手術後，傷口約4-5 公分，術後三天以紗布覆蓋每日換藥，依醫師指示傷口是否拆線。

2. 出院返家傷口未癒合時，每天使用沒有刺激性的優碘藥水消毒並用無菌紗布覆蓋傷口，並暫時避免弄溼傷口。

3. 注意傷口有無紅、腫、熱、痛或分泌物及體溫的變化等情形，若有以上情形則表示可能有感染的現象，此時應立即回到醫院給醫師視診。

4. 裝置「人工血管」的手臂勿過度的活動，如：避免360度的旋轉手臂與擴胸動作或提過重的物品避免人工血管移位，除此以外，一般的日常活動並不受影響。
