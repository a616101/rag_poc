# 牙科

## 團隊簡介
牙科照護的範圍不僅是20顆乳牙及32顆恆牙，包括口腔黏膜、咬合功能重建、頭頸部相關疾病等都是守護口腔健康的第一關。本著全人照護原則，以病患為中心提供全方位治療建議與診治。科內每位醫師皆為專科醫師，包含口腔病理及診斷專科、牙周病專科、牙髓病科、顱顏齒列矯正專科、植牙專科等，以團隊合作方式，為病患提供專業建議及計劃，完成「全人照護」。本科同時也是家庭牙醫學科專科醫師訓練機構，除臨床醫療照護外，亦具專科醫師培訓能力。

## 團隊編制
- 醫師：科內每位醫師皆為專科醫師，包含口腔病理及診斷專科、牙周病專科、牙髓病科、顱顏齒列矯正專科、植牙專科等
- 護理/照護：專業護理團隊
- 醫事技術/其他：檢查技術人員
- 特色分工：以團隊合作方式，為病患提供專業建議及計劃，完成「全人照護」

## 服務與治療
### 服務範圍（疾病/問題）
- 口腔黏膜、口腔病理診斷
- 牙周病
- 阻生牙、多生牙、囊腫
- 缺牙
- 根管疾病
- 齒列不整
- 咬合功能問題

### 主要治療/處置
1. 口腔黏膜、口腔病理診斷：提供口腔黏膜疾病及口腔病理診斷
2. 牙周病、牙周骨移植、引導骨再生術、牙齦移植：提供牙周病的診斷與治療
3. 門診手術、阻生牙多生牙拔除、囊腫摘除：提供門診手術服務
4. 植牙：提供植牙服務
5. 根管顯微治療：提供根管疾病的顯微治療
6. 齒列矯正：提供齒列矯正服務
7. 補綴及咬合功能重建：提供補綴及咬合功能重建服務
8. 家庭牙醫：提供家庭牙醫服務

### 特色門診/照護計畫（若有）
- 特殊需求者特診：自101年9月份起，牙科設立「特殊需求者特診」服務
- 全人照護：本著全人照護原則，以病患為中心提供全方位治療建議與診治
- 家庭牙醫學科專科醫師訓練機構：除臨床醫療照護外，亦具專科醫師培訓能力
