# 1701592227_163

## 主旨  
本文件為屏基醫療財團法人屏東基督教醫院提供的腸阻塞手術相關衛教資訊，內容涵蓋手術前準備、手術後護理要點、照顧者注意事項，以及相關諮詢管道。

## 重點整理  
- **手術前準備**：禁食8小時、放置鼻胃管、腹部X光與抽血檢查、電解質補充、抗生素使用、腹部檢查。  
- **手術後護理**：每2~4小時評估生命徵象與腸蠕動音；使用束腹帶減輕疼痛，並固定傷口以減少活動時的牽扯；監測鼻胃管與引流管狀況；維持半坐臥位以減輕腹脹；術後禁食3~5天，但需保持口腔衛生；待腸蠕動恢復後，依醫囑逐步恢復飲食。  
- **飲食與排便**：腸蠕動恢復後，依醫囑從清流飲食（如過濾果汁、碳酸飲料）過渡至低渣飲食（如米湯、豆漿）；鼓勵多進食高纖食物，每日飲用至少2000c.c液體，並配合運動與藥物（如緩瀉劑或高纖維藥物）以預防便秘。  
- **緊急情況**：若出現腹部絞痛、腹脹、噁心、嘔吐或便秘，應立即就醫，以防止腸阻塞復發。

## 流程/步驟  
### 手術前準備  
1. 禁食8小時（不可進食任何東西，包括開水、口香糖、檳榔）。  
2. 放置鼻胃管。  
3. 進行腹部X光檢查。  
4. 抽血檢查。  
5. 檢查電解質並進行補充。  
6. 施打抗生素。  
7. 進行腹部檢查。  

### 手術後護理  
1. 每2~4小時評估生命徵象、意識狀態及腸蠕動音。  
2. 使用束腹帶，並在深呼吸或咳嗽時用手固定傷口，以減少疼痛與裂開風險。  
3. 監測鼻胃管位置及引流管的輸出量、顏色與性質。  
4. 維持半坐臥位，以減輕腹脹對橫隔膜的壓迫。  
5. 術後禁食3~5天，期間仍需刷牙或漱口，保持口腔衛生，減少細菌感染。  
6. 待腸蠕動恢復後，由醫師評估是否可開始喝水；若無腹脹或嘔吐，隔日可開始清流飲食（如過濾果汁、碳酸飲料），並逐步過渡至低渣飲食（如米湯、豆漿）。

## 注意事項/警示  
- **緊急就醫**：若病人出現腹部絞痛、腹脹、噁心、嘔吐或便秘，應立即就醫，以防止腸阻塞復發。  
- **飲食與排便**：鼓勵病人多進食高纖食物，每日至少飲用2000c.c液體，並配合適當運動。依醫師指示服用緩瀉劑或高纖維藥物，並與約240c.c溫水一同服用，以促進排便。

一、手術前準備及注意事項
1. 禁食 8 小時（不可吃任何東西，包括開水、口香糖、檳榔）。
2. 放置鼻胃管。
3. 腹部 X 光檢查。
4. 抽血檢查。
5. 電解質的檢查及補充。
6. 給予抗生素。
7. 腹部檢查。

三、手術後注意事項
1. 每 2~4 小時評估生命徵象、意識狀態及腸蠕動音。
2. 術後傷口需使用束腹帶，可減輕活動時或咳嗽時引起的疼痛感，防止傷口裂開。執行深呼吸、咳嗽運動時，以手固定手術傷口部位，減少因震動牽扯傷口而引起疼痛。
3. 監測鼻胃管是否保持在適當的位置及引流管的輸出量、顏色、性質。
4. 維持舒適的臥位，搖高床頭給予半坐臥，減輕腹脹對橫隔膜的壓迫。
5. 術後依醫囑需禁食 3~5 天，在此期間仍必須刷牙或漱口，保持口腔衛生，減少細菌感染。
6. 在腸蠕動排氣恢復後，醫師會評估是否讓你開始喝水，若無腹脹嘔吐，隔日再採清流（過濾果汁、碳酸飲料）、低渣飲食（米湯、豆漿）。

四、照顧者注意事項：
1. 病人若出現腹部絞痛、腹脹、噁心、嘔吐或便秘情形，應立即就醫，及早阻止腸阻塞再度發生。
2. 鼓勵病人多進食高纖食物，每日至少飲用 2000c.c 的液體及適當的運動，減少便秘的情形發生，依醫師指示給予緩瀉劑或高纖維藥物，與約 240c.c 水溫和一起服用，以利排便。
