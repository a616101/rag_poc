# 採購室

## 團隊簡介
本科負責院內各種採購業務，包括預算設備、藥品、醫材、衛耗材、牙材、文具、消耗品、布品、電器設備、辦公傢俱、檢驗、印刷品、圖書、器械等。提供新供應商評鑑、醫療特材申請、供應商受款帳戶資料卡、自費特材品項用途說明書等表單服務。資材組負責一般衛耗材、特殊醫材、心導、放射科材料、耗材、牙材、檢驗、印刷品、點滴等收貨作業。

## 團隊編制
- 醫師：不適用
- 護理/照護：不適用
- 醫事技術/其他：採購室人員、資材組人員
- 特色分工：分為採購室及資材組，各負責不同採購項目
