## 主旨
當患者面臨呼吸衰竭，且需長期依賴呼吸器時，應考慮是否接受氣管切開手術（氣切手術），以減少長期插管可能引發的併發症。本文件為醫病共享決策工具，協助患者或家屬在面臨呼吸衰竭需插管時，評估是否應接受氣管切開手術（氣切），並做出治療選擇。最終治療決策應由患者、家屬與醫療團隊共同協商決定。

## 重點整理
- **呼吸衰竭**：指呼吸系統無法有效進行氣體交換，導致代謝需求無法滿足，嚴重時可能威脅生命。
- **機械通氣**：用於支持呼吸，分為非侵入性（如陽壓或雙壓呼吸器）與侵入性（經氣管內管置入）。
- **適用對象**：需長期使用呼吸器的患者，包括：
  - 使用呼吸器超過3週以上。
  - 無法成功脫離呼吸器。
  - 曾經拔管試驗失敗者。
- **氣切手術目的**：建立人工氣道，減少長期氣管內插管所導致的併發症。
- **長期插管併發症**：包括口腔、咽喉至氣管的損傷，如表皮黏膜損傷、水腫、牙齒損傷、聲帶麻痺、接觸性肉芽腫、吞嚥困難、氣管軟化、氣管食道瘻管等。
- **氣管內插管**：經鼻或經口置入，通常無法說話，無額外費用，居家照護需求為0%。
- **氣管造口術**：傳統手術於床旁進行，可提高病人舒適度，有脫離呼吸器後說話的可能性，但需自付費用（約 NT$6,000~10,000，特殊型號可能更高），居家照護需求約為8%。
- **氣管造口術優勢**：在舒適度、呼吸器脫離率、呼吸器相關肺炎風險等方面優於氣管內插管。
- **氣管造口術風險**：需手術及麻醉，有手術創口及照護需求，但脫管風險低，且可訓練從口進食。
- **決策工具**：本文件為「醫病共享決策」流程的一部分，用於協助患者與醫療團隊共同決定治療方案。透過回答一系列問題，評估不同治療方式的優缺點。

## 流程/步驟
1. **評估病情**：確認患者是否屬於需長期依賴呼吸器的呼吸衰竭患者。
2. **評估脫離可能性**：判斷患者是否能成功脫離呼吸器。
3. **評估併發症風險**：考慮長期插管可能導致的併發症。
4. **討論治療選項**：與患者及家屬討論是否接受氣切手術。
5. **實施氣切手術**：若決定進行，建立人工氣道。
6. **評估對治療方式的了解**：
   - Q1. 若接受氣管造口術，可以增加病人的舒適度？ 是
   - Q2. 若接受氣管造口術，發生風險合併症機率很高？ 否
   - Q3. 若接受氣管造口術，提高呼吸器脫離可能？ 是
   - Q4. 若接受氣管造口術，有講話的可能性？ 否
   - Q5. 若接受氣管造口術，有臥床常態性需要照護？ 是
7. **選擇治療方式**：
   - [ ] 我選擇維持「經鼻/口氣管內插管」治療
   - [ ] 我選擇接受「氣管造口術（氣切）」治療
   - [ ] 我希望與家人朋友進一步討論
   - [ ] 我希望與醫療人員進一步討論
   - [ ] 我不願意做任何治療
8. **完成文件內容**，並攜帶至診間與醫師共同討論適合的治療方案。

## 注意事項/警示
- **長期插管風險**：長期使用氣管內管可能導致多種併發症，包括口腔、咽喉至氣管的損傷。
- **氣切手術優點**：可減少長期插管併發症，提高患者舒適度與生活品質。
- **術後護理**：需注意氣切口的護理，防止感染與其他併發症。
- **氣管內插管風險**：呼吸器脫離率為0%，呼吸器相關肺炎風險高達67%。
- **氣管造口術風險**：雖需手術及麻醉，但舒適度較高，且呼吸器脫離率及相關肺炎風險較低。
- **兩種方法均有輕微併發症風險**，需根據患者具體情況評估。
- **本文件為決策輔助工具，非醫療建議**。治療選擇應根據患者具體病情、醫療團隊評估及個人意願綜合決定。
- **氣管造口術為重大醫療決策**，建議充分了解手術的效益、風險與替代方案。
- **決策應綜合考慮醫療專業意見、個人健康狀況、家庭支持與個人價值觀**。
- **請務必與主治醫師、護理人員及相關醫療團隊充分溝通**。

(Version Number & Date : Jul. 31, 2017)

我/我的家人面對呼吸衰竭插管的狀況時，我該接受氣切手術嗎？

疾病介紹

急性或慢性呼吸衰竭指的是呼吸系統的氣體交換，因無法應付人的代謝需求，而發生的急性生理失衡。嚴重時可能會危及生命。一般來說呼吸衰竭的病人必須用機械通氣支持的方式取代自主呼吸。而機械通氣的途徑可區分為非侵襲性(如陽壓或雙壓呼吸器)及侵襲性(經氣管內管置入後使用之機械通氣形式)。而臨床上認為「使用呼吸器大於3週以上」，「無法成功訓練脫離呼吸器」，或「曾經經過拔管嘗試後失敗者」，為需要長時間呼吸器使用或有潛在脫離困難者，建議應考慮建立氣管造口，以減少長時間氣管內管插管使用之合併症產生。

適用對象/適用狀況–需長期依賴呼吸器之患者

當病人呼吸衰竭，需使用侵入性呼吸器輔助呼吸時，就必須先建立人工氣道，插管對需緊急建立人工氣道者，是最迅速簡單的方法。

然而並非所有病人皆能順利脫離呼吸器，若長期插管會造成口腔、咽喉至氣管的損傷，包含表皮黏膜損傷、水腫、牙齒損傷、聲帶麻痺、接觸性肉芽腫、吞嚥困難、氣管軟化、氣管食道瘻管等合併症產生。

為減少長時間氣管內管插管使用之合併症產生，醫生會建議病人或家屬考慮接受氣切手術。

A. 氣切管置入後，空氣就可經氣切管流入肺。

B. 肺部空氣中的氧氣，會被吸收然後在全身循環。
