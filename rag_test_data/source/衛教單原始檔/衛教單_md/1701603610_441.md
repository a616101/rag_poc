# 1701603610_441

## 主旨
屏東基督教醫院檢驗科提供門診血液檢驗報告寄發便民服務，適用於僅需查詢特定檢驗項目、不需看診或領藥的患者。患者需自備回郵信封並填寫收件資訊，並簽署同意書以寄送報告。

## 重點整理
- 本服務適用於僅需查詢特定血液檢驗項目者。
- 患者需自備回郵信封，並填妥收件人姓名與地址。
- 需簽署「同意檢驗報告寄發」同意書。
- 可寄發的檢驗項目包含肝功能、腎功能、血脂、血糖、血常規等多項。

## 流程/步驟
1. 確認需查詢的血液檢驗項目。
2. 自備回郵信封，並填妥收件人姓名與地址。
3. 簽署「同意檢驗報告寄發」同意書。
4. 將填妥的信封與同意書交至檢驗科櫃檯。

## 注意事項/警示
- 僅限於所列可寄發項目，其他項目不提供寄發。
- 患者需自行負擔郵寄費用。
- 報告寄發後，恕不接受退件或更正。
