## 主旨  
屏東基督教醫院提供的尿路感染病人返家注意事項，包含治療建議、日常保健方法及相關注意事項。

## 重點整理  
- 急診處置僅開3天藥物，需持續門診追蹤，療程通常為7至14天。  
- 水分攝入是緩解排尿灼熱感的最快方法，建議每日飲用2000~3000毫升（約10~15杯），白天飲用，並於晚上解尿1~2次，以減少尿液積聚與膀胱過度膨脹。  
- 若經3天治療後症狀未改善，或出現解尿疼痛、頻尿、小便灼熱、血尿、發燒、發冷等情況，應立即就醫。  
- 提供七項日常保健方法，以幫助預防與緩解尿路感染。

## 流程/步驟  
1. **藥物治療與追蹤**  
   - 急診處置僅開3天藥物。  
   - 需依醫師指示持續門診追蹤，療程通常為7至14天。  
2. **症狀惡化時的應對**  
   - 若治療3天後仍解尿疼痛、頻尿、小便灼熱，或出現血尿、發燒、發冷，應立即前往急診。  
   - 亦可至內科、泌尿科或家庭醫學科門診就醫。  
3. **日常保健方法**  
   1. 每日飲用2000~3000毫升水，水源包括開水、飲料及湯品。可多攝取酸性食物，如肉類、乳酪、蛋、梅子、蔓越莓汁、葡萄柚汁、柑橘類，以降低細菌繁殖速度。  
   2. 有尿意時應儘快上廁所解尿，避免憋尿，以防細菌滋生。  
   3. 養成正確清潔習慣：女性排尿排便後，應由尿道口向肛門方向擦拭。  
   4. 採淋浴，勿泡澡。  
   5. 性行為後應立即排尿，並多喝水，以增加排尿量；若出現尿道感染症狀，應暫時避免性行為。  
   6. 不穿太緊衣物或束褲，建議選擇棉質內褲，保持通風。  
   7. 若有放置導尿管，需每日進行尿管護理。

## 注意事項/警示  
- 除非有特殊情況（如腎功能不良、心臟衰竭），否則應每日飲用2000~3000毫升水。  
- 避免憋尿，特別是女性應注意正確的擦拭方向，以防止細菌從肛門傳至尿道。  
- 若出現血尿、發燒、發冷等症狀，應立即就醫，不可延誤。

屏基醫療財團法人屏東基督教醫院
尿路感染病人返家注意事項

| 編號 | ND內-019 |
| :--- | :--- |
| 製訂日期 | 2003.07 |
| 六修日期 | 2018.03 |

一、急診處置只能開 3 天藥物，請您一定要繼續到門診追蹤。由醫師決定您何時可以停藥，通常一個療程為 7~14 天。

二、水分攝入是解除排尿灼感的最快途徑，除非有特殊情況（如：腎功能不良、心臟衰竭），每天喝約 2000~3000ml（以紙杯計算為 10~15 杯），平均在白天內喝完，以增加排尿量，晚上也應該解尿 1~2 次，減少尿液鬱積和膀胱過度膨脹。

三、如果經過 3 天藥物治療後，仍發生尿痛、頻尿、小便灼熱、或解完尿會痛，有血尿、發燒、發冷等情況請立即到急診就醫。也可到內科、泌尿科或家醫科門診治療。

四、以下為日常保健方法：

1. 每天喝水 2000~3000ml，水份攝取來源包括開水、飲料及湯品等，可以多攝取酸性食物及飲料，如：肉類、乳酪、蛋、梅子、蔓越梅汁、葡萄柚汁、柑橘類，可以降低細菌繁殖速度。

2. 當有尿意時，應儘快上廁所解尿，避免憋尿，以免細菌滋生。

3. 養成正確的清潔衛生習慣：女性排尿排便後，應由尿道口擦到肛門。

4. 採淋浴，勿泡澡。

5. 性行為後馬上排尿，並多喝水，增加排尿；當出現尿道感染症狀時，暫時勿有性行為。

6. 不穿太緊的衣服或束褲，可選擇棉質內褲較通風。

7. 有放置導尿管時，須每天做尿管護理。
