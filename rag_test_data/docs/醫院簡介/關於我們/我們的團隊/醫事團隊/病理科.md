# 病理科

## 團隊簡介
為了因應日益漸增之醫療作業，提昇醫療診斷之正確性及水準，並促進臨床醫師及病理科診斷之雙向溝通及教學，並以醫學中心之病理診斷水準，樹立區域級教學醫院之典範，故於民國86年成立病理科。本院病理科主要分為外科病理及細胞病理兩組。本科成員包括合格之病理專科醫師、合格醫檢師及行政人員所組成。並且在病理主任及主治醫師親自參與及督導下，負責所有外科檢體、子宮頸抹片及非婦科細胞病理之診斷及判讀。就醫療功能而言，本科不斷地更新改善病理切片及染色技術，並且不斷充實外科病理及細胞病理之專業知識，以期達到精確的診斷判讀，作為臨床診療之依據。就教學訓練工作而言，本科積極參與住院醫師、醫檢生及見習生之訓練活動，以期培育高素質之專科醫師及有志一同之外科病理醫檢師及細胞醫檢師，以提昇及改善本院醫療品質，以期達到醫病雙方最大的精確診斷及診療效益。

## 團隊編制
- 醫師：合格之病理專科醫師（病理主任及主治醫師）
- 護理/照護：不適用
- 醫事技術/其他：合格醫檢師、行政人員
- 特色分工：主要分為外科病理及細胞病理兩組

## 服務與治療
### 服務範圍（疾病/問題）
- 各種疾病病理診斷

### 主要治療/處置
- 外科病理診斷：負責所有外科檢體之診斷及判讀
- 細胞病理診斷：負責子宮頸抹片及非婦科細胞病理之診斷及判讀
- 院外一般院所病理組織切片相關服務：提供院外一般院所病理組織切片相關服務
- 外科病理診斷之相關免疫染色服務：提供外科病理診斷之相關免疫染色服務，歡迎地區醫院級以上之醫院加入合作策略聯盟
- 細胞病理子宮頸抹片之篩檢服務：提供細胞病理子宮頸抹片之篩檢服務，歡迎院外相關院所及各衛生所加入我們的服務行列

### 特色門診/照護計畫（若有）
- 院外服務：提供院外一般院所病理組織切片相關服務、外科病理診斷之相關免疫染色服務、細胞病理子宮頸抹片之篩檢服務
- 教學訓練：積極參與住院醫師、醫檢生及見習生之訓練活動

## 檢查項目

- 子宮頸抹片：提供子宮頸抹片篩檢
- 非婦科細胞病理：提供非婦科細胞病理診斷
- 免疫染色：提供外科病理診斷之相關免疫染色服務

- 染色設備：提供病理染色服務
- 免疫染色設備：提供免疫染色服務
