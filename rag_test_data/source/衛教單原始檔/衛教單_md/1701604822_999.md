# 1701604822_999

## 主旨
碘-131（I-131）治療甲狀腺機能亢進，適用於對抗甲狀腺藥物治療反應不佳的患者，透過碘-131的β射線破壞過度活躍的甲狀腺組織，達到治療目的。

## 重點整理
- **目的**：利用碘-131的β射線破壞機能亢進的甲狀腺組織，治療甲狀腺機能亢進。
- **適應症**：甲狀腺機能亢進。
- **禁忌症**：懷孕、計畫懷孕或正在哺乳者，需事先告知並與醫師討論治療必要性。
- **治療後注意事項**：服藥後身上會帶有微量輻射，需避免與孕婦、嬰幼兒親密接觸，哺乳婦女應停止哺乳至少1個月。
- **替代方案**：抗甲狀腺藥物。

## 流程/步驟
1. 服藥前至少禁食2小時。
2. 按預約時間，攜帶身分證、健保卡及本說明書，至韓偉大樓地下室1樓（急診室地下1樓）的核醫科櫃檯報到。
3. 確認身分後，由工作人員發放一顆碘-131膠囊。
4. 服藥後即可離開，並依預約時間回原開單醫師門診複診。

## 注意事項/警示
- **副作用**：少數患者可能出現喉嚨不適或腫脹感，可透過含酸梅或嚼口香糖增加唾液分泌，以減少口乾副作用。
- **長期影響**：治療後可能導致甲狀腺素分泌不足，需長期補充甲狀腺素，並定期追蹤甲狀腺功能。
- **輻射安全**：
  - 接受治療後會有少量輻射，但對身體傷害輕微，治療效益大於風險。
  - 服藥後應多喝水、多排尿，以加速體內碘-131排出，減少不必要的輻射劑量。
- **接觸限制**：
  - 建議在治療後1週內，避免與孕婦或嬰幼兒親密接觸，保持至少1公尺距離。
  - 哺乳婦女應在服藥後停止哺乳至少1個月。

## 全文

核醫/z910 碘-131 治療甲狀腺機能亢進
(I-131 treatment)

目的：甲狀腺機能亢進病人若對於抗甲狀腺藥物治療反應不佳，可選擇使用碘-131 治療，利用碘-131的β射線，來破壞機能亢進的甲狀腺組織，以達到治療的效果。

適應症：甲狀腺機能亢進。

禁忌症：若有懷孕、計畫懷孕或正在哺乳的情形，請事先告知核醫科櫃檯服務人員，或與您的醫師討論此治療的必要性。

處置流程：
(1) 碘-131治療甲狀腺機能亢進服藥前需要禁食至少2小時。
(2) 請依照預約時間，攜帶身分證、健保卡及此說明書至韓偉大樓地下室1樓（急診室地下1樓）的核醫科櫃檯報到。
(3) 確認您的身分後，本科工作人員會讓您服下一顆碘-131膠囊。
(4) 服藥後即可離去，依照預約掛號時間回原開單醫師的門診複診。

併發症/風險：
(1) 使用碘-131治療甲狀腺機能亢進，少數病人會有喉嚨不適、腫脹感，可利用冰敷減輕症狀，另外可含酸梅或嚼口香糖，來增加唾液分泌，以減少唾液腺受到的輻射劑量，避免口乾的副作用發生。
(2) 使用碘-131治療甲狀腺機能亢進，可能會造成甲狀腺素分泌不足，必須長期補充甲狀腺素，建議定期回門診追蹤甲狀腺功能。
(3) 接受碘-131治療會接受到少量的輻射劑量，此輻射對身體的傷害非常輕微，當醫師由臨床上評估您必須接受碘-131治療時，治療對您的健康帶來的效益遠大於這少量輻射的風險。服用碘-131後多喝水，多排尿，可加速身體組織的碘-131排出，減少不必要的輻射劑量。

替代方案：抗甲狀腺藥物。

檢查後處置：服用碘-131後您身上會帶有微量的輻射，建議在1週內避免與孕婦或嬰幼兒親密接觸（保持距離1公尺以上）。哺乳婦女建議在服藥後停止哺乳1個月。
