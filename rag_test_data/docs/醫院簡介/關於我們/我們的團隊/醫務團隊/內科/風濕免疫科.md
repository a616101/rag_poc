# 風濕免疫科

## 團隊簡介
本科設有實驗室，提供各種血清免疫學檢查，用於風濕免疫疾病的完整診斷與治療。本科參與山地醫療，提供原住民常見疾病痛風的診斷及治療。風濕免疫科主要診治範圍包括：痛風性關節炎、全身性紅斑性狼瘡、硬皮症、皮肌炎、多發性肌炎、混合性結締組織疾病、血管炎、各種自體免疫疾病。過敏科主要診治範圍包括：藥物過敏、皮膚過敏、蕁麻疹、過敏性鼻炎、過敏原鑑定。

## 團隊編制
- 醫師：風濕免疫專科醫師
- 護理/照護：專業護理團隊
- 醫事技術/其他：實驗室技術人員
- 特色分工：提供各種血清免疫學檢查

## 服務與治療
### 服務範圍（疾病/問題）
- 風濕免疫疾病：痛風性關節炎、全身性紅斑性狼瘡、硬皮症、皮肌炎、多發性肌炎、混合性結締組織疾病、血管炎、各種自體免疫疾病
- 過敏疾病：藥物過敏、皮膚過敏、蕁麻疹、過敏性鼻炎、過敏原鑑定

### 主要治療/處置
- 風濕免疫疾病診斷及治療：提供完整的風濕免疫疾病診斷與治療服務
- 過敏疾病診斷及治療：提供過敏疾病的診斷、治療及過敏原鑑定
- 山地醫療服務：參與山地醫療，提供原住民常見疾病痛風的診斷及治療

### 特色門診/照護計畫（若有）
- 山地醫療服務：提供原住民常見疾病痛風的診斷及治療

## 檢查項目
1. 單鈉尿酸鹽結晶及焦磷酸鈣二水化合物結晶鑑定
2. ANA（抗核抗體）、RF（類風濕因子）、CRP（C反應蛋白）、ASLO（抗鏈球菌溶血素O）、IgG、IgA、IgM、C3、C4、銅藍蛋白檢測
3. Anti-dsDNA、Anti-Sm、Anti-RNP、Anti-SSA、Anti-SSB、Anti-Scl-70、Anti-Jo-1、抗磷脂抗體、冷凝球蛋白檢測
4. HLA-B27檢測
5. Phadiatop檢測及總IgE
6. HPRT mRNA突變分析
7. HLA-B1502檢測
