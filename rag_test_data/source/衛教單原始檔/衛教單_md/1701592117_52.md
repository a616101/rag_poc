# 1701592117_52

## 主旨  
本文件為泌尿道結石術後之衛教指引，內容涵蓋結石定義、常見原因、症狀、治療方式、術後注意事項及返家照護建議，以協助患者術後恢復並預防復發。

## 重點整理

| 項目 | 內容 |
|------|------|
| **結石定義** | 泌尿系統內形成結晶體，堵塞輸尿管、腎臟或膀胱等管道，稱為尿路結石。 |
| **好發年齡** | 約 50 歲上下。 |
| **常見類型** | 輸尿管結石最常見。 |
| **復發率** | 治療後五年內再發機率為 35%。 |
| **常見症狀** | 間歇性腰部與腹部痙攣性疼痛、血尿、噁心、嘔吐、全身乏力、發燒與寒顫。 |
| **治療方式** | 依結石大小、位置與成分決定，小於 0.4 公分者可自行排出，大於 0.5 公分者建議手術。 |
| **飲水建議** | 术后每日飲水量至少 3000–4000cc，以促進排石與預防復發。 |
| **飲食建議** | 減少鹽分、鈣、磷、草酸鹽及高普林食物攝取，多攝取蔬菜與水果。 |

## 流程/步驟

1. **內科治療**  
   - 適用於結石小於 0.4 公分者。  
   - 建議多喝水（每日至少 2000–3000cc），並使用止痛劑或解痙劑。  
   - 約八成以上結石可在 6 週內自行排出。

2. **外科治療**  
   - 適用於結石大於 0.5 公分者，難以自行排出。  
   - 醫師將建議手術治療。

## 注意事項/警示

1. **雙側置管注意**  
   - 术后置放雙側引流管，偶爾會出現腰酸、頻尿、急尿、偶發性血尿。  
   - 症狀多隨引流管拔除後自然消失。  
   - 避免過度用力解便、解尿、仰臥起坐、搬重物或跳躍，以防引流管移位或脫落。

2. **尿袋使用注意**  
   - 若尿袋內尿液超過 1/3 或 1/2 滿時，應以尿壺將尿液倒出，避免尿液逆流造成感染。

3. **飲水量控制**  
   - 若無水分限制，請依醫師指示每日飲水量至少 3000cc 以上。

4. **應立即就醫的症狀**  
   - 腰部異常疼痛。  
   - 畏寒、高燒。  
   - 解尿有灼熱感、血尿。  
   - 解尿不順暢或尿不出來。

泌尿道結石術後須知

一、什麼是泌尿道結石：
指泌尿系統內有結石產生，可能出現在腎臟、輸尿管、膀胱，堵塞了這個管道，人體水分大部分需由腎臟代謝，經由輸尿管、膀胱、尿道排出。若尿液出現沉積性的結晶體，堵塞了這個管道，即形成尿路結石。

二、發生常見原因
(一)與體質、遺傳、飲食、喝水量以及代謝症候群有著密切關係，好發年齡層約在50歲上下。
(二)泌尿道結石當中，最常見的為輸尿管結石。
(三)結石經治療後五年內再發機率為35％。

三、症狀
間歇性腰部和腹部的痙攣性疼痛、血尿、噁心、嘔吐、全身乏力發燒和寒顫。

四、治療方式
醫師會依您的症狀、位置、大小及成分決定治療方式：
1. 內科療法：結石小於0.4公分以下者，鼓勵多喝水每天至少2000-3000cc、多活動及使用止痛劑或解痙劑，約八成以上機會可在6週內自行排出。
2. 外科療法：結石大於0.5公分以上者，很難自行排出依醫師建議手術治療。

五、手術後注意事項/合併症
1. 於雙側腰部放置引流管，偶爾會有腰酸、頻尿、急尿、偶發性血尿情形，其症狀大多會隨引流管拔除後自然消失，故置放期間避免過度用力解便、解尿、仰臥起坐、搬重物、跳躍等運動，以防引流管移位掉落。
2. 蓄尿袋避免與地面接觸及不可高過膀胱，避免尿液逆流造成感染。
3. 蓄尿袋內尿液若超過1/3或1/2滿時，請以尿壺將尿液倒出。
4. 若無水分限制，請依醫師指示多喝水，每日飲水量至少3000cc以上。

六、返家後照護
1. 鼓勵多攝取水分，每日3000-4000c.c.。
2. 不可憋尿、適度運動，有利尿路通暢。
3. 保持身心愉悅，避免抑鬱，因精神長期抑鬱，會分泌過度的甲狀腺素，致使血中鈣、磷含量升高，增加結石機率。
4. 飲食減少鹽分、鈣、磷、草酸鹽或高普林食物攝取(表一)、多攝取蔬菜與水果。

(1) 腰部異常疼痛。

(2) 畏寒、高燒。

(3) 解尿有灼熱感、血尿。

(4) 解尿不順暢或尿不出來。
