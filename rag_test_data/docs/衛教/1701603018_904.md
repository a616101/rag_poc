# 1701603018_904

## 主旨
本文件說明 Aldara cream（Imiquimod）的正確使用方法與注意事項。

## 重點整理
- 藥品名稱：Aldara cream 5%（每包 250mg，每克含 50mg）
- 主要成分：Imiquimod
- 儲存條件：不可超過 25℃，禁止冷凍
- 使用注意：治療部位不可包紮或覆蓋紗布
- 清潔方式：塗藥後 6-10 小時，以中性肥皂和清水洗淨

## 流程/步驟
- 塗抹 Aldara cream 5% 於治療部位。
- 塗藥後 6-10 小時，以中性肥皂和清水洗淨。
- 治療部位不可使用繃帶包紮，或覆蓋紗布。

## 注意事項/警示
1. 儲存溫度不可超過 25℃，不可冷凍。
2. 治療部位不可使用繃帶包紮，或覆蓋紗布。
3. 塗藥後 6-10 小時，應以中性肥皂和清水洗淨。
