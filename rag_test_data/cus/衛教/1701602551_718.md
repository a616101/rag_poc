## 主旨
本文件由屏基醫療財團法人屏東基督教醫院製作，旨在說明抽菸對身體的負面影響，並介紹戒菸後身體各項指標的改善進程。同時提供免費戒菸諮詢管道，鼓勵吸菸者及早戒菸，促進健康。

## 重點整理
### 一、抽菸的影響
抽菸會對身體多個系統造成負面影響，具體如下：
- **心臟及血管系統**：干擾氧氣交換，減少氧氣供應，導致缺氧現象。
- **肺部**：降低肺部清除分泌物的能力。
- **傷口**：易阻礙傷口癒合，且會加深疤痕顏色。
- **其他**：手術後常用的鴉片類止痛藥，在吸菸者的需求量會較無吸菸者高。

### 二、戒菸的好處
戒菸後，身體會在不同時間點出現明顯改善。根據國民健康局戒菸教戰手冊，具體進程如下：
| 時間 | 身體改善項目 |
|------|--------------|
| 20分鐘 | 心跳及血壓恢復正常 |
| 8小時 | 血液中的尼古丁和一氧化碳濃度回到正常值 |
| 24小時 | 一氧化碳排除，肺部開始清除痰液及菸品殘渣 |
| 48小時 | 味覺和嗅覺改善，肺功能改善，呼吸變得容易 |
| 2-12週 | 血液循環大幅改善 |
| 3-9個月 | 減少咳嗽、哮喘等呼吸問題，肺功能增加10% |
| 1年 | 心臟病發作機會減半 |
| 10年 | 肺癌發生機會減半 |
| 15年 | 心臟病發作機率和非吸菸者相同 |
