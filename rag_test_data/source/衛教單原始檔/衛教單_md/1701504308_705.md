# 1701504308_705

## 主旨
屏基醫療財團法人屏東基督教醫院提供的氣喘病人返家注意事項，旨在協助患者正確管理病情，預防急性發作，並促進健康生活。

## 重點整理
- 氣喘患者應持續服藥並定期就醫追蹤。
- 使用吸入劑者需遵醫囑，並隨身攜帶緊急藥物。
- 若症狀未改善或出現嚴重症狀，應立即就醫。
- 建立健康的生活方式，包括飲食、情緒、環境與運動。
- 注意天氣變化及流感季節，做好防護。

## 流程/步驟
### 1. 藥物與治療管理
- 按時服用藥，並定期至胸腔內科門診追蹤。
- 使用吸入劑者，應依醫師指示正確使用。
- 隨時攜帶緊急藥物（如擴張劑）以備不時之需。

### 2. 症狀惡化時的應對
若出現以下情況，應立即就醫或返回本院複診：
- 說話困難
- 嘴唇或指甲變灰或變藍
- 呼吸時鼻孔張大
- 脈搏加快
- 發燒、畏寒、感冒或咳嗽
- 氣喘發作

### 3. 建立健康生活方式
#### (1) 飲食
- 攝取均衡飲食。
- 避免高蛋白、高脂肪、高熱量食物。
- 避免刺激性食物，如酒精、咖啡及冰冷食物。

#### (2) 情緒
- 保持適當休息與睡眠。
- 避免熬夜及過大壓力。

#### (3) 環境
- 長期抽菸者應戒菸。
- 避免處於空氣不流通、二手煙、濕氣、灰塵、花粉等環境。
- 避免養寵物，因動物唾液、排泄物、毛髮可能引發過敏。
- 避免接觸絨毛玩具、豬、狗等動物。
- 定期打掃環境，減少過敏原。

#### (4) 運動
- 不宜參與突發性、刺激性活動。
- 可進行規律運動，但避免劇烈活動。

## 注意事項/警示
- 連續使用吸入性藥物三次後症狀未改善，應立即就醫，以免生命危險。
- 天氣溫度變化時，外出應保暖並戴口罩，保護氣管及肺部。
- 流行感冒季節，應儘量減少出入公共場所，避免接觸感冒患者，以防感染。

屏基醫療財團法人屏東基督教醫院
氣喘病人返家注意事項

| 編號 | ND 急-005 |
| :--- | :--- |
| 製訂日期 | 2003.08 |
| 七修日期 | 2022.10 |

1. 請按時服藥並定時到胸腔內科門診接受追蹤治療，不要自行停藥。
2. 如果有在使用吸入劑者，請按照醫師指示正確適時使用吸入劑，隨時攜帶緊急藥物（擴張劑）以備不時之需。
3. 連續使用吸入性藥物三次沒有改善，應立即就醫，以免有生命危險。
4. 建立健康生活方式。
   (1) 飲食：平時攝取均衡飲食，避免高蛋白、高脂肪和高熱量，不要吃刺激性如酒精、咖啡和冰冷的食物。
   (2) 情緒：適當休息及睡眠，避免熬夜及壓力。
   (3) 環境：
     ☆ 若有長期抽菸者要改掉抽煙習慣，避免處於空氣不流通、二手煙、濕氣、灰塵、花粉…之環境。
     ☆ 不要養寵物，很多人會對動物的唾液、排泄物、毛過敏，避免接觸絨毛玩具或豬、狗…動物，定期打掃環境減少過敏原。
     運動：不宜參與突發性、刺激性之活動。
   (4) 有下列情況儘早就醫：說話很困難、嘴唇或指甲變灰或變藍色、呼吸時鼻孔會張得很大、脈搏很快、發燒、畏寒、感冒、咳嗽或是氣喘發作，請儘速就醫或是回本院複診。
5. 以下為日常保健原則：
   (1) 天氣溫度變化時，外出記得保暖、戴口罩來保護您的氣管及肺部。
   (2) 在流行感冒的季節，儘量減少出入公共場所，接觸感冒的人，以免被傳染。
   (3) 運動：規律運動但不要做劇烈的活動。

屏基關心您 祝您早日康復
