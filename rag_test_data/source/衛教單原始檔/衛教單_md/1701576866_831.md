# 1701576866_831

## 主旨
本文件為B型肝炎帶原者提供健康照護指引，說明發現帶原後應如何處理、定期追蹤檢查項目，以及日常生活中需注意的事項，以維護肝臟健康。B型肝炎是導致國人肝癌的主要原因，肝癌死亡率長期位居台灣癌症死因前二。大部分人在感染B型肝炎後可自癒，但少數人（尤其出生時感染）會成為慢性帶原者。慢性B型肝炎帶原者若持續HBsAg陽性超過6個月，且e抗原陽性者，罹患肝癌風險顯著增加。感染途徑包括垂直傳染（母親傳給新生兒）、輸血、共用針具、不安全的醫療或美容行為等。多數帶原者無明顯症狀，易忽略健康風險，導致肝臟持續受損。

## 重點整理
- B型肝炎帶原者通常無明顯症狀，但肝臟可能已受損，因此定期追蹤檢查至關重要。
- 藥物治療成效良好，全民健康保險已提供給付，是否接受治療應由專科醫師評估決定。
- 帶原者應養成健康生活習慣，並避免加重肝臟負擔的行為。
- 大部分人感染B型肝炎後可自癒，但少數人（尤其出生時感染）會成為慢性帶原者。
- 慢性B型肝炎帶原者若持續HBsAg陽性超過6個月，且e抗原陽性者，罹患肝癌風險顯著增加。
- 多數帶原者無症狀，但可能隱性損害肝臟。
- 感染B型肝炎的高風險群包括：母親為帶原者、醫療從業人員、接受手術或輸血者、共用針具者。
- 接種B型肝炎疫苗是預防感染最有效的方式。

## 流程/步驟
1. **感染B型肝炎病毒**：透過血液或體液接觸，如垂直傳染、輸血、共用針具等。
2. **成為慢性帶原者**：若HBsAg持續陽性超過6個月，即為慢性帶原。
3. **肝臟持續受損**：慢性感染導致肝臟反覆發炎，可能演變為慢性肝炎。
4. **肝臟結構改變**：長期發炎導致肝纖維化，進而發展為肝硬化。
5. **可能轉變為肝癌**：肝硬化或慢性肝炎可能進一步演變為肝癌。
6. **發現帶原**：若經檢查確認為B型肝炎帶原者，應尋求專業醫療協助。
7. **就醫評估**：前往肝膽腸胃科就診，由醫師評估肝臟狀況。
8. **定期追蹤檢查**：
   - **無肝硬化**：每6至12個月進行一次抽血檢查肝功能及甲型胎兒蛋白，並依醫師建議接受超音波或其他影像學檢查。
   - **有肝硬化**：每3至6個月進行一次抽血檢查肝功能及甲型胎兒蛋白，並依醫師建議接受超音波或其他影像學檢查。
9. **治療決策**：根據檢查結果與醫師建議，決定是否接受藥物治療。

## 注意事項/警示
- **避免吸菸、嚼檳榔及過量飲酒**：若有吸菸習慣，應盡量戒菸。
- **飲食注意**：避免食用含人工添加物、醃漬或可能被黃麴毒素污染的食物，例如花生製品、豆瓣醬、豆腐乳等發酵食品。
- **慎用藥物**：避免服用成分不明的藥物或偏方，以免增加肝臟負擔。
- **性行為防護**：若性伴侶一方帶原，另一方未帶原，建議接種B型肝炎疫苗，或在性行為時使用保險套以預防感染。
- 多數B型肝炎帶原者無症狀，但可能隱性損害肝臟。
- 建議定期進行肝功能檢查與B型肝炎相關檢測，特別是高風險族群。

## 全文

5. 發現自己是B型肝炎帶原者應該怎麼辦？

應找肝膽腸胃科醫師檢查，以確認自己肝臟的狀況

• 定期追蹤 B型肝炎帶原者的肝臟損傷演變過程中通常是没有明顯症狀，因此，定期追蹤檢查是相當重要的！由於每個人的個別情況不同，其追蹤時間仍應由醫師決定，原則上：

  • 無肝硬化 每6-12個月抽血檢驗肝功能及甲型胎兒蛋白，並依醫囑接受超音波或其他影像學檢查。

  • 有肝硬化 每3-6個月抽血檢驗肝功能及甲型胎兒蛋白，並依醫囑接受超音波或其他影像學檢查。

• 藥物治療 B型肝炎的藥物治療成效很好，目前全民健康保險已經有條件的給付該治療，可以進一步請教專科醫師決定是否接受治療。

• 正常生活作息、均衡飲食及遠離菸酒 B型肝炎帶原者在生活飲食上並沒有特別的限制，只要生活作息正常及均衡飲食即可。

提醒帶原者應注意事項：

1. 避免吸菸、嚼檳榔及過量飲酒。若有吸菸者，請戒菸！

2. 避免食用含人工添加、醃漬、或有可能被黃麴毒素污染的食物（如花生製品、豆瓣醬、豆腐乳等發酵食品）。

3. 避免服用成分不明的藥物或偏方，以免增加肝臟負擔。

4. 如果性伴侶一方帶原，另一方沒有帶原，最好是接種B型肝炎疫苗，或是性行為時使用保險套以預防感染。

B型肝炎？

1. 肝癌與B型肝炎的關係？
肝癌死亡率近10年來一直位居台灣癌症死因之第1或第2位。根據死因統計，每年約有8千人死於肝癌。調查顯示，死於肝癌的病患中，大部分的人為B型肝炎帶原者，所以B型肝炎是引起國人肝癌的主要原因。

我國民國73年前出生的成人約有15~20%為B型肝炎帶原者，帶原者發展成肝癌的機率是非帶原者的10倍左右，如果30歲以上e抗原仍為陽性時，則高達60倍。

2. 何謂B型肝炎帶原者？
大部分人感染B型肝炎病毒之後會痊癒，但少數的人，尤其是出生就感染的嬰兒，無法將病毒清除，B型肝炎表面抗原（HBsAg）持續6個月以上呈陽性，即成為帶原者，也就是慢性感染B型肝炎病毒的人。如果B型肝炎e抗原（HBeAg）也陽性，表示病毒量多，其傳染力強且罹患肝癌的機會亦較高。

3. 為何會感染B型肝炎病毒？
主要是經由血液或其他體液感染。過去B型肝炎有一半是母親生產時，經由胎盤或是產道將病毒傳染給新生兒，我國因為推行嬰兒B型肝炎疫苗注射之後，帶原率已大幅降低。另，輸血、打針、刺青或穿耳洞的器械消毒不完全、共用牙刷或刮鬍刀、毒癮者共用針具與稀釋液，或性行為，接觸到感染者之血液或體液亦會導致感染。

4. B型肝炎帶原對肝臟有什麼影響？
帶原者有可能演變為慢性肝炎、肝硬化或肝癌，很多帶原者並不知道自己帶原，加上沒有明顯症狀，因而容易忽略帶原可能對肝臟的影響。

● 何謂慢性肝炎、肝硬化及肝癌？
慢性肝炎：如果連續6個月以上肝功能（GOT及GPT值，或稱AST與ALT值）異常。
肝硬化：肝臟反覆發炎，導致纖維化，有很多人沒有症狀，而有些人則是出現肝功能異常或衰竭。
肝癌：肝臟出現癌症病變，有一部份人會由肝炎至肝硬化再演變成肝癌（約佔85%），但也有部分人不經由肝硬化，直接從肝炎變成肝癌（約佔15%）。
