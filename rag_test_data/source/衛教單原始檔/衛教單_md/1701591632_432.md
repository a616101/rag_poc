# 1701591632_432

## 主旨  
本文由神經外科主治醫師廖振傑撰寫，旨在澄清「骨刺刺到腎臟就要洗腎」的常見誤解。骨刺是正常老化現象，並非直接導致腎臟病或需洗腎的原因。文章透過案例說明，強調應尋求專業醫療評估，並提供預防與改善骨刺的綜合建議。

## 重點整理  
- **骨刺並非「刺」**：醫學上稱為骨質增生，常因長期姿勢不良、營養不均或受傷等因素加速生成，並非如字面般「刺」到器官。  
- **骨刺不直接導致洗腎**：骨刺本身與腎臟病無直接關聯，「骨刺刺到腎臟」是誤解。若骨刺壓迫神經，可能引發嚴重症狀，但與洗腎無關。  
- **應尋求專業診斷**：身體不適應尋求合格醫師診察，找出真正病因，再安排合適治療。  
- **綜合管理是關鍵**：透過調整姿勢、運動、飲食與睡眠習慣，可有效緩解骨刺症狀，避免病情惡化。

## 流程/步驟  
1.  **發現症狀**：出現肩頸痠痛、手臂麻痺等症狀，且反覆不愈。  
2.  **尋求專業評估**：前往醫院，透過神經學檢查與影像學（如X光、MRI）確認是否為骨刺壓迫神經。  
3.  **制定治療計畫**：根據診斷結果，由醫師建議合適的治療方式，可能包括物理治療、藥物治療或手術。  
4.  **積極復健與生活調整**：術後或治療期間，配合醫師建議進行復健，並調整日常生活習慣。

## 注意事項/警示  
- **勿輕信偏方**：避免輕信「整復脊椎可消除骨刺」等非科學說法，可能延誤治療或造成傷害。  
- **勿過度恐懼**：骨刺是常見的老化現象，不必過度焦慮，但也不可忽視症狀。  
- **避免錯誤行為**：  
  - 避免負重超過2公斤。  
  - 避免過度彎曲或伸展脊椎。  
  - 站姿應直立，頭部與胸部勿前傾。  
  - 坐姿應雙腳自然著地，後方需有腰椎支撐。  
- **勿因病忌醫**：若出現症狀，應及時就醫，避免小問題演變成複雜疾病。

## 全文

骨刺刺到腎臟，就要洗腎？
無稽之談！

文／神經外科主治醫師 廖振傑

帶、椎間盤突出或破裂、骨質增生、小關節過度增生等問題醫師為方便與民眾解釋通稱為骨刺，但是多半長的都不像一根刺，僅是其引起症狀不適，如芒刺在背，如此說明容易獲得民眾理解。

二、身體不適，應尋求專業合格醫師之診察，找到病因之後再安排後續治療，不論中醫或西醫，只要安全有效且符合病情需要的治療，都是好選擇。

三、脊椎不好的民眾需注意避免負重（超過兩公斤），過度彎曲或伸展的動作都不適合做，站姿宜直立頭部勿前傾，坐姿宜雙腳自然著地，後方需有靠背且正坐為佳，勿彎腰駝背，以減輕腰椎負擔。

四、運動方面以游泳，健走，騎腳踏車等對脊椎較友善，目的在促進血液循環，增強其自我修復療癒之功能，其中訓練核心肌群尤為重要。

五、飲食方面需增加鈣質、維生素D、維生素B群之攝取，以提供骨骼及神經足夠養分。常見食物有牛奶、小魚乾、芝麻糊、深綠色蔬果等。若有疼痛症狀，可適量攝取薑、薑黃、魚油、蝦紅素、鳳梨酵素等天然抗發炎及減痛食品。

六、睡眠方面床墊宜軟硬適中具相當支撐力，枕頭不宜太高，一般約10~15公分左右，後頸部最好有所支撐不要懸空，避免長時間收縮肌肉疲乏容易痠痛。平躺或側睡皆可，但是習慣側睡者枕頭高度要比平躺稍高，避免頸椎呈現側彎姿勢。

「骨刺刺到腎臟，就要洗腎？」關於骨刺坊間有許多說法，本院神經外科主治醫師廖振傑強調骨刺是正常老化現象，民眾無需過度恐懼，但在臨床上也發現不少年輕人因姿勢不良，久坐久站，年紀輕輕即出現骨刺。

王先生長期肩頸及手臂痠痛麻，嘗試過各種民俗及傳統療法，時有改善，然症狀始終反覆無法根治，已嚴重影響生活品質，從親友處聽聞有師傅可整復脊椎，配合藥物消除骨刺，便前往一試，經診視後王先生被告知確有骨刺壓迫神經，必須治療，若不積極處理此骨刺刺到腎臟即需洗腎。王先生驚嚇之餘不能自己，馬上決定接受治療，整復後起初頗有改善，信心大振，然經過月餘症狀加劇，無奈之下才至屏基神經外科門診。經神經學及影像檢查後確診為頸椎椎間盤突出，需進一步手術治療，王先生手術後恢復良好，已可回歸正常生活不再受症狀所苦。

在此要提醒民眾幾個觀念：

一、骨刺乃正常老化過程中必然出現的結果，但是長期姿勢不良、營養不均衡、缺乏運動、曾經受傷等因素可能加速骨刺的生成。沒有任何證據顯示骨刺與洗腎、心臟病或胃穿孔有關聯，勿過度聯想自己嚇自己。骨刺是一般而言過度肥厚增生的勸稱，一般而言過度肥厚增生的問題變複雜。因生活型態改變，現代人脊椎退化性疾病相當常見，提醒民眾勿諱疾忌醫，聽取專業合格醫師的建議並不吃虧，以免簡單的問題變複雜。
