# 領導與經營理念

## 概要

本文件說明屏基醫院的領導與經營理念，包括醫院的宗旨、願景、經營理念、目標與核心價值，幫助了解醫院的文化與服務精神。

## 適用對象

- 病人
- 家屬
- 訪客
- 員工

## 內容說明

### 宗旨

「藉由神的愛與能力,經由醫療等服務,恢復人的健康與尊嚴。」

### 願景

成為倡導與實踐全人關懷的卓越醫院

### 經營理念

- 專業設備優質化
- 醫療服務人性化
- 健康促進社區化
- 經營管理合理化

### 目標

- 財務
- 社會公益
- 顧客
- 內部流程
- 學習成長

### 核心價值

| 核心價值 | 說明 |
|---------|------|
| **尊嚴** | 尊重生命,並視病人及員工均為獨特、具潛力、有價值的 |
| **服侍** | 隨時提供熱心、愛心及謙和的服務 |
| **更新** | 內在更新、服務創新 |
| **卓越** | 致力於個人專業的成長與團隊的合作 |
