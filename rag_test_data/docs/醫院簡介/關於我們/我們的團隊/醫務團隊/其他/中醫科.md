# 中醫科

## 團隊簡介
本科以「神愛世人」精神及中醫經典為基礎，整合現代醫學知識，達到中西醫結合目標。強調全人照護（身、心、靈），非僅依賴藥物治療，強調身體自癒能力及正確生活方式與藥物治療並重。中醫醫療團隊具備中西醫專業知識，利用現代診斷及影像系統，提供高屏地區民眾優質服務。目前中醫醫療團隊包括中醫師2名、藥師2名、診間護理師3名。

- 特色分工：中醫醫療團隊具備中西醫專業知識，利用現代診斷及影像系統
