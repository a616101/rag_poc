# 1701601528_280

## 主旨
本表說明兒童構音發展的關鍵年齡階段，並提供語音清晰度的發展指標與治療建議，幫助家長了解孩子語言發展的正常進程。

## 重點整理
- 兒童構音發展有明確的年齡節奏，從2歲開始逐步成熟。
- 語音清晰度隨年齡增長而提升：2歲約50%，3歲約75%，4歲約90%。
- 4至6歲為構音治療的黃金期。
- 若孩子說話不清，應先排除聽力、理解能力或動作發展等問題。

## 流程/步驟
兒童構音發展依年齡分為以下階段：

- **2歲**  
  所有韻母及四聲聲調在2至3歲之間發展完成，孩子大致可順利發出這些聲音，並在表達上產生變化。

- **3歲**  
  聲母陸續出現，並搭配不同韻母，產生多樣的語音與語彙。

- **4歲**  
  塞音和塞擦音等較複雜的構音技巧開始表現，語音更清晰。

- **5歲**  
  送氣音表現清晰且明確，語句長度更完整。

- **6歲**  
  正常構音發展的孩子至此階段，構音已成熟穩定且清晰，能正確仿念大部分語音。

## 注意事項/警示
- 若家長發現孩子講話不清楚，建議先就醫小兒科，以排除聽力、理解能力或動作發展等問題。
- 若孩子到4歲仍無構音改善，應尋求語言治療師進行諮詢、評估與治療。

## 表格/比較
| 年齡 | 語音清晰度 | 發展重點 |
|------|------------|----------|
| 2歲 | 約50% | 韻母與聲調發展完成，可發出基本聲音 |
| 3歲 | 約75% | 聲母出現，語音與語彙逐漸豐富 |
| 4歲 | 約90% | 塞音、塞擦音等技巧出現，語音更清晰 |
| 5歲 | - | 送氣音清晰，語句更完整 |
| 6歲 | - | 構音成熟穩定，可正確發音 |

## 全文

兒童構音發展時程表

2 歲 所有韻母及四聲聲調會在 2~3 歲之間發展完成，也就是說孩子大致都可以順利的發出這些聲音和表達的聲調上也可以產生不同的變化。

3 歲 ㄅ、ㄆ、ㄉ、ㄊ、ㄋ、ㄌ，這些聲母會陸續的在這個時期慢慢的表現出來，並且搭配不同的韻母產生不同的語音和說出不同的語彙。

4 歲 ㄐ、ㄑ、ㄒ，塞音和塞擦音的部分會在年齡更大時，有較成熟的構音技巧下慢慢的表現出來。

5 歲 ㄗ、ㄘ、ㄙ，送氣音的表現會在這個年齡後表現的清晰及明確，語彙量及語句的長度也會更加完整。

6 歲 ㄓ、ㄔ、ㄕ，若是正常構音發展的孩子到達這段時期，構音的發展大致上已經成熟穩定且清晰了，可以正確仿念大部分的語音。

★ 語音清晰度：2 歲 50%，3 歲 75%，4 歲 90%

★ 4-6 歲為構音治療的黃金期

家長注意

- 若覺得孩子講話不清楚，建議先掛小兒科門診以排除是聽力、理解能力、動作發展等其他問題。

- 孩子若是四歲了構音沒有改善，建議找語言治療師諮詢、評估或治療。
