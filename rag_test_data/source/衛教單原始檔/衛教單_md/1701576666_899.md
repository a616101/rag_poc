# 1701576666_899

## 主旨  
本指引提供動靜脈瘻管手術後的護理注意事項，幫助患者促進瘻管成熟、預防併發症，並促進康復。

## 重點整理  
- **傷口護理**：保持傷口完整，避免抓傷或破皮；傷口癒合約 10–14 天可拆線。若出現紅腫、熱痛、分泌物或出血，應立即返院就診。  
- **患肢管理**：手術後應將患肢墊高並伸直，促進血流，避免阻塞。  
- **避免壓迫與操作**：勿於手術手臂抽血、打點滴、量血壓；避免配戴飾物、穿太緊袖子，不可用手當枕頭。  
- **日常檢查**：定期觸摸瘻管，檢查是否有震顫或血流聲，若靜止無聲，應立即就醫。  
- **手部運動**：手術 24 小時後，若無疼痛、出血或腫脹，可進行握軟球運動，持續 3–5 秒後放開，交替進行，促進血管成熟。  
- **清潔注意**：使用瘻管前，應以肥皂清洗皮膚，預防感染。

## 流程/步驟  
**手部運動指導（手術 24 小時後）**  
1. 握緊軟球，持續 3–5 秒。  
2. 放開，休息 3–5 秒。  
3. 重複數次，若手部感到疼痛或疲憊，應立即休息。  
4. 建議每日進行，以促進血管成熟。

## 注意事項/警示  
- **避免壓迫**：勿將手臂當枕頭，避免壓迫瘻管導致血流不暢。  
- **避免操作**：禁止在手術手臂進行抽血、打點滴或量血壓。  
- **警惕異常**：若發現瘻管無震顫、無血流聲，應立即就醫。  
- **感染預防**：使用瘻管前，務必用肥皂徹底清潔皮膚。  
- **立即就醫**：出現紅腫、熱痛、分泌物、出血或傷口破損，應立即返院診治。

屏基醫療財團法人屏東基督教醫院
動靜脈瘻管手術後護理指導

| 編號 | ND 內-050 |
| :--- | :--- |
| 製訂日期 | 2007.09 |
| 四修日期 | 2020.07.21 |

動靜脈瘻管術後注意事項:

一、保持傷口皮膚完整性，避免抓傷或破皮；傷口癒合良好約 10~14 天可拆線。若有紅腫熱痛及分泌物、或是受傷出血的情形，應立即返院診治。

二、手術後應每天觀察傷口情形，輕微腫脹為正常現象，可用枕頭將手術患肢墊高及伸直，促進血流，以免瘻管血流不順而導致阻塞。

三、避免於手術的手臂上抽血、打點滴及量血壓。

四、動靜脈瘻管的手，請勿配戴飾物、穿太緊的袖子且不可以用手當枕頭，以免壓迫血管，導致瘻管血流不順暢。

五、平時多觸摸瘻管，檢查血液是否通暢，其要領是用手觸摸時會有電流感或觸動感，若發現靜止無聲，請到門診就醫或告知醫護人員並返院就診。

六、手術 24 小時後，若傷口不痛，無出血及腫脹情形，可執行手部運動握軟球持續 3~5 秒，心中默唸「1 秒、2 秒、3 秒、4 秒、5 秒」後放開，交替動作可數次，若手部感到疼痛疲憊時需休息，此動作可促進血管成熟(如下圖示)。

七、瘻管使用前可用肥皂執行打處皮膚清潔，以預防感染。
