## 主旨
本文為營養師周莉菁撰寫的「中西式外食健康吃」衛教資訊，針對現代人普遍外食的現象，提供糖尿病患者及三高族群在外用餐時的飲食原則與建議，並以中式宴會為例，說明如何在不影響營養均衡的情況下，選擇較健康的外食方式。本文件亦涵蓋中式自助餐、西餐及速食等常見餐飲類型的健康選擇建議，強調均衡飲食、控制份量與選擇清淡烹調方式的重要性，並提醒服用降血糖藥物者應避免空腹喝酒。

## 重點整理
- 現代人外食普遍，但常導致蔬果攝取不足、油脂與糖分過高。
- 外食易導致「三高」（高血壓、高血糖、高血脂）及代謝症候群、心血管疾病、肥胖等問題。
- 糖尿病患者應特別注意外食的技巧，以維持血糖穩定。
- 建議選擇低油、清淡、高纖維的烹調方式與食物。
- 中式自助餐及麵食應避免高油脂、高鹽的菜汁與湯汁，選擇乾飯、湯麵，並多攝取蔬菜以增加飽足感。
- 西餐應控制沙拉醬與麵包份量，選擇清湯、海鮮或雞肉主菜，避免油炸、焗烤及高脂調味品。
- 速食建議選擇低脂魚類、控制副餐與炸物份量，並以低卡飲料取代含糖飲品。
- 一般飲食原則應均衡攝取六大類食物，定時定量，多選高纖、清淡烹調，並避免高膽固醇與高糖食物。
- 服用口服降血糖藥物或胰島素注射的病人，應嚴禁空腹喝酒，以防低血糖。
- 不應依賴「果菜汁」來補充營養，應直接攝取新鮮蔬果。
- 避免因省錢而捨棄青菜，導致蔬果攝取不足。
- 選擇「吃到飽」餐廳時，應自我節制，避免過量攝取。

## 流程/步驟
### 糖尿病人外食原則
1. 熟悉並記住自己可攝取食物的份量。
2. 多選擇低油及清淡的食物。
3. 避免選擇「吃到飽」的餐廳，以自我節制。
4. 注意藥物作用時間與進餐時間的配合。

### 中式宴會健康吃法
1. 選擇蒸、煮、烤、燉、涼拌等烹調方式的菜色，以降低油脂攝取。
2. 避免濃湯、勾芡湯汁（如羹類）、糖醋、蜜汁等高油、高糖、高熱量食物，若需食用應瀝去湯汁，只吃料不喝湯。
3. 多選擇湯內的青菜或盤飾上的蔬菜，因蔬菜熱量低且富含纖維，可增加飽足感；亦可將湯汁瀝乾或用白開水清洗，以減少油脂攝取。
4. 減少內臟類、蟹黃、卵黃類、烏魚子、蝦頭、魚頭等膽固醇含量高的食物攝取。
5. 避免碎肉製品、加工食品（如火腿、香腸）及不明成分的食物；應選擇帶骨肉塊，剝去骨頭，以控制實際攝取的肉量。
6. 避免含糖量高的果汁、汽水、飲料；可選擇無糖茶或以白開水取代。
7. 宴會上的水果體積小，應適量攝取，可用來取代飯後甜點或甜湯。

### 中式自助餐及麵食
1. 夾菜時，優先選擇較上層的食物，避免菜汁（湯）因油脂與鹽份含量高。
2. 少攝取油炸、沾粉或黏稠菜色，並去除油炸食物的外皮。
3. 多選擇蔬菜，以增加飽足感。
4. 忌用任何湯汁泡飯，因湯汁含油脂、鹽份與味精。
5. 飯類與麵類選擇乾飯或湯麵，避免吸油量大的炒飯與炒麵。
6. 吃麵時，建議搭配海帶、青菜以增加蔬菜攝取。

### 西餐
1. 各種蔬菜沙拉可適量選用，但沙拉醬應裝小碟，避免直接沾取。
2. 西式濃湯含大量麵粉與油脂，應選擇清湯。
3. 一個小餐包（如法國麵包一小片）視為一份主食，應控制攝取量，避免選擇高脂的大蒜麵包或塗抹果醬、奶油。
4. 主菜以海鮮或雞肉為主，因含油量較少；避免油炸菜色，可將外皮剝去；焗烤類因使用起司、牛奶，油脂與熱量高，應避免。
5. 罐裝飲料、汽水或新鮮果汁均不建議選用。

### 速食
1. 主餐可多選擇魚類（如麥香雞、麥香魚），因其含油量較低。
2. 副餐份量盡量選擇小份或小杯。
3. 炸物需剝去外皮，且控制份量。
4. 選擇低卡飲料或咖啡，不加奶油球。
5. 減少購買促銷活動（如買一送一），避免點多不必要的食物。

## 注意事項/警示
- 外食時應留意食物的油、糖、鈉含量，特別是三高患者與糖尿病病友。
- 不應依賴「果菜汁」來補充營養，應直接攝取新鮮蔬果。
- 避免因省錢而捨棄青菜，導致蔬果攝取不足。
- 選擇「吃到飽」餐廳時，應自我節制，避免過量攝取。
- 均衡攝取六大類食物。
- 定時定量進食。
- 多攝取富含纖維質的食物。
- 食物烹調以清淡為原則。
- 避免食用過鹹及膽固醇高的食物。
- 儘量減少含糖的烹調方法。
- 服用口服降血糖藥物及胰島素注射的病人，應嚴禁空腹喝酒，以免增加低血糖發生的危險性。

糖尿病人外食原則

（1）需先熟悉牢記自己可吃食物的份量
（2）多選擇低油及清淡的食物
（3）少選擇可以吃到飽的餐廳，自我節制
（4）注意藥物作用時間與吃飯時間上的配合

中式宴會

1. 選擇蒸、煮、烤、燉、涼拌等烹調方式的菜色，降低油脂攝取量。
2. 濃湯、勾芡湯汁（羹類）、糖醋、蜜汁食物皆含有大量太白粉、油、糖、熱量高，盡量少吃，否則應瀝去湯汁，吃料不喝湯。
3. 多選擇湯內的青菜或盤飾上的蔬菜，因蔬菜熱量低，富含纖維質，可增加飽足感，也可以將湯汁瀝乾或用白開水把菜濾過，減少油脂攝取。
4. 內臟類、蟹黃、卵黃類、烏魚子、蝦頭、魚頭，等膽固醇含量高應減少攝取。
5. 碎肉製品，加工食品如火腿香腸或其他不明成份的食物不宜食用，選擇帶骨的肉塊，剝去骨頭，實際吃入肉量不多，較有滿足感。
6. 避免含糖量高的果汁、汽水、飲料；無糖茶是不錯選擇，或以白開水取代。
7. 宴會上的水果經過截切，體積較小，一口一塊，適量攝取，可用來取代飯後甜點或甜湯。

中式自助餐及麵食

1. 夾菜時，夾取較上層的食物，因菜汁（湯）通常油脂、鹽份含量高。
2. 油炸、沾粉或黏稠菜色少攝取，食用應剝去外皮。
3. 可多選擇蔬菜，增加飽足感。
4. 忌用任何湯汁來泡飯，因湯汁中油脂、鹽份、味精等調味品含量高。
5. 飯類與麵類多選擇乾飯或湯麵，少選擇吸油量大的炒飯與炒麵。
6. 吃麵時，建議點些海帶、青菜增加蔬菜的攝取量。

西餐

1. 各種蔬菜沙拉適量選用，沙拉醬一般多油多糖，所以應將沙拉醬裝小碟，另外沾取，勿加入生菜中。
2. 西式濃湯含有大量麵粉與油脂，熱量極高，應選擇清湯較適合。
3. 一個小餐包 = 法國麵包一小片 = 一份主食，所以麵包攝取量應控制，不選油脂含量高的大蒜麵包也不要塗果醬或奶油。
4. 主菜以海鮮或雞肉為主，因含油量較少，烹調方式採用『烤』為主，不要選擇油炸菜色，也可以將外皮剝去；焗烤烹調因利用起司、牛奶，油脂、熱量高，應避免食。
5. 罐裝飲料、汽水或新鮮果汁都不宜選用。

速食

1. 主餐可多選擇魚、雞（如麥香雞、麥香魚），因其含油量較低。
2. 副餐的份量盡量選擇小份或小杯即可。
3. 炸雞需剝去外皮且份量要控制。
4. 低卡飲料或咖啡不加奶油球是好的選擇。
5. 減少購買促銷活動（如買一送一），不要貪便宜，點了多得不必要食物。

小叮嚀～

1. 均衡攝取六大類的食物
2. 定時定量
3. 多攝取富含纖維質的食物
4. 食物烹調儘量以清淡為原則
5. 避免吃過鹹及膽固醇高的食物
6. 儘量減少含糖的烹調方法

服用口服降血糖藥物及胰島素注射的病人，嚴禁空腹喝酒，因會增加低血糖發生的危險性。
