# 1701591365_435

## 主旨  
本文件為胃腫瘤手術前後之護理指引，包含術前準備、術後注意事項及相關護理措施，以促進患者術後恢復並預防併發症。術後居家照護亦包含飲食、傷口護理、生活作息及用藥指導，以促進恢復。

## 重點整理

| 項目 | 內容 |
|------|------|
| **術前準備** | 教導正確的深呼吸與咳嗽方法，預防術後肺部併發症；進行腸道準備（如甘油球灌腸）；手術前一天晚上開始禁食，避免吸入性肺炎。 |
| **術後注意** | 每小時進行 5~10 次深呼吸與咳嗽運動；採半坐臥姿勢，保持安靜環境；使用止痛針或靜脈鎮痛藥物控制疼痛；妥善管理鼻胃管、傷口引流管及導尿管；保持傷口乾燥，防止感染；進食需待腸蠕動恢復後，由醫師指示。 |
| **術後居家照護** | 術後可能出現「傾倒症候群」，應注意飲食與姿勢；飲食應循序漸進，由流質至軟質，再至正常飲食；避免刺激性、高脂、高糖食物，選擇易消化、低殘渣飲食；避免攝取牛奶及奶製品；進食時應細嚼慢嚥，採端坐姿勢，飯後30分鐘才可休息；每日觀察傷口狀況，保持乾燥並清潔消毒；安排規律生活，保持心情愉悅；依醫囑服藥並定期門診追蹤，如有不適應立即就醫。 |

## 流程/步驟

#### 一、手術前準備及注意事項

1. **深呼吸運動方法**  
   - 緩慢由鼻吸氣，腹部膨出。  
   - 再緩慢由口吐氣，腹部凹陷。  
   - 持續三秒鐘，重複一次。

2. **咳嗽運動方法**  
   - 坐立姿勢，身體往前傾。  
   - 以雙手壓住傷口部位。  
   - 慢慢吸氣，閉氣三秒鐘，連續咳嗽兩次。

3. **腸道準備**  
   - 依醫師指示執行，如甘油球灌腸或清潔灌腸。

4. **禁食**  
   - 手術前一天晚上開始禁食，不可進食任何食物或飲品，包括開水、口香糖、檳榔，以預防麻醉時嘔吐導致吸入性肺炎。

5. **使用尿盆及便盆**  
   - 教導患者在床上使用尿盆及便盆。

#### 二、手術後注意事項

1. **呼吸與咳嗽運動**  
   - 每小時進行 5~10 次深呼吸及咳嗽運動，促進肺部擴張。

2. **姿勢與休息**  
   - 依醫師指示採半坐臥式，提供安靜環境，並適度休息，減輕傷口疼痛。

3. **止痛管理**  
   - 依醫囑注射肌肉止痛針，通常每 4~6 小時一次，避免呼吸抑制。  
   - 或使用自費靜脈鎮痛控制藥物。

4. **鼻胃管引流**  
   - 目的：引流胃液，預防腹脹、噁心、嘔吐。  
   - 醫師每日評估引流量及腸蠕動情況，決定拔管時間。  
   - 若有不適，應立即告知醫護人員。

5. **傷口引流管**  
   - 目的：引流腹腔內殘餘血水，預防傷口化膿、紅腫發炎。  
   - 醫師每日評估引流液量、性質與顏色，決定拔管時間。  
   - 翻身時避免壓迫或反折引流管，並使用固定夾妥善固定。

6. **導尿管護理**  
   - 導尿管放置期間，護理人員每日執行尿管護理，降低泌尿道感染風險。  
   - 待醫師評估病情穩定後，即可拔除尿管。

7. **傷口護理**  
   - 保持傷口乾燥，防止感染。  
   - 若傷口紗布滲濕，應立即告知護理人員更換敷料。

8. **進食指導**  
   - 手術後至排氣前，禁止進食任何食物或飲水，以免胃部傷口受損及腹脹。  
   - 若口乾，可使用棉棒沾開水潤口。  
   - 待腸蠕動恢復、排氣後，由醫師告知可進食的餐點種類。

9. **活動建議**  
   - 若無頭暈、乏力等不適症狀，可提早下床活動，促進腸蠕動恢復及全身血液循環。

#### 三、術後居家飲食調配步驟

1. 由清淡流質開始，例如米湯。  
2. 逐漸過渡至軟質食物，例如稀飯。  
3. 最終恢復正常飲食。

## 注意事項/警示

- **避免深呼吸與咳嗽**：術後因怕痛而避免深呼吸或咳嗽，易導致肺炎或術後發燒。  
- **禁食期間**：手術前一天晚上起需完全禁食，包括開水與口香糖，以防吸入性肺炎。  
- **止痛藥使用**：不可自行調整止痛藥劑量，以免引起呼吸抑制。  
- **引流管管理**：翻身時應注意避免壓迫或反折引流管，確保引流通暢。  
- **傷口感染預防**：保持傷口乾燥，一旦發現滲濕或紅腫，應立即通知醫護人員。  
- **進食時機**：必須等待腸蠕動恢復、排氣後，經醫師評估後方可進食。  
- **飲食注意事項**：  
  - 採用「少量多餐」原則。  
  - 水份應在兩餐之間飲用，避免飯中飲水。  
  - 避免辛辣、油膩、高糖、高蛋白及易產生氣體的食物。  
  - 避免牛奶及奶製品。  
  - 进食時應細嚼慢嚥，採用端坐姿勢。  
  - 飯後至少30分鐘才可臥床休息。  
- **傷口護理**：每日觀察傷口有無紅、腫、熱、痛等發炎症狀；每日清潔消毒傷口，保持乾燥。  
- **生活與用藥**：安排規律生活步調；保持心情愉悅；依醫師指示服用藥物；定期門診追蹤病情變化；如有身體不適，應立即返院就診。

一、手術前準備及注意事項

1. 教導正確的深呼吸、咳嗽，預防術後怕痛而不做深呼吸、咳嗽動作併發術後相關併症，如肺炎或術後發燒。

2. 深呼吸運動的方法：緩慢由鼻吸氣，腹部膨出，再緩慢由口吐氣，腹部凹陷，並閉氣三秒鐘。

3. 咳嗽運動的方法：坐立姿勢，身體往前傾，以雙手壓住傷口部位，開始慢慢吸氣，閉氣三秒鐘，連續咳嗽二次。

4. 教導您在床上使用尿壺及便盆。

5. 依醫師指示必要時執行腸道準備，如甘油球灌腸或清潔灌腸。

6. 依醫師指示於手術前一天晚上開始禁食，不可吃任何東西包括開水、口香糖、檳榔，預防麻醉時嘔吐造成吸入性肺炎。

二、手術後注意事項

1. 每小時做 5~10 次深呼吸及咳嗽運動，以利肺部擴張。

2. 依醫師指示採半坐臥式及提供安靜的環境，並適度的休息，減輕傷口疼痛。

3. 手術後傷口疼痛可注射肌肉止痛針，通常二次止痛劑給予的時間需要相隔 4~6 小時，以免藥物引起呼吸抑制等副作用，或自費並依醫師指示予裝置靜脈疼痛控制藥物。

4. 鼻胃管引流：目的是引流胃液，預防腹脹、噁心、嘔吐，醫師會評估每天引流量及腸蠕動情形而決定拔管時間，若有不適請告知醫護人員。

5. 傷口引流管：目的是引流手術後腹腔內殘餘的血水，預防傷口化膿、紅腫發炎，醫師會評估每日引流液量、性質、顏色，決定拔管時間。翻身時避免壓迫或反折到引流管，傷口引流管會有固定夾，可作適當的固定。

6. 導尿管：導尿管放置期間，護理人員會每天執行尿管護理，以降低發生泌尿道感染，待醫師評估病情穩定，即可拔除尿管。

7. 手術後保持傷口乾燥很重要，如果傷口紗布滲濕，請告知護理人員換藥，以預防傷口感染。

8. 進食：手術後至排氣前，不可以吃任何食物、喝水，以免胃部傷口受到損害及腹脹不適，如果覺得口乾，可以棉棒沾開水潤口。當腸蠕動排氣後，醫師將告訴您可以進食那些種類的餐點。

9. 手術後如無特別的頭暈、身體乏力等不適症狀，可提早下床活動，以幫助促進腸蠕動的恢復及活絡全身血液循環暢通。

五、術後居家照護注意事項

1. 手術後有可能發生「傾倒症候群」，吃入的食物未經胃適當的消化、稀釋，就快速的進入空腸，會出現腹部疼痛痙攣、飽脹感、噁心、嘔吐等不適現象。飲食注意事項：

(1) 食物調配須採循序漸進方式，由清淡而濃稠，由流質（米湯）→軟質（稀飯）→正常飲食。

(2) 須少量多餐。

(3) 不要喝過多的水（含湯），水份應在兩餐之間飲用。

(4) 應避免刺激性的飲食，如辛辣食物，而以低殘渣、體積小、易消化、高蛋白、適量脂肪、低糖、溫和性的飲食為原則（烹調方式例如：蒸蛋、魚湯）。

(5) 應避免牛奶及奶製品食物之攝取。

(6) 進食時應細嚼慢嚥，採端坐姿勢。

(7) 飯後 30 分鐘才可臥床休息。

2. 每日觀察傷口有無紅、腫、熱、痛等發炎症狀，並每日清潔消毒傷口，保持傷口乾燥。

3. 安排規律的生活步調，並保持心情愉悅。

4. 依照醫師的指示服用藥物治療，並定期門診追蹤病情變化，如有身體不適，應立即返院就診。

屏基關心您 祝您早日康復
