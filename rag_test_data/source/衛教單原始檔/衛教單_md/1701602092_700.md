# 1701602092_700

## 主旨
本文件旨在向患者說明區域麻醉的定義、適用對象、準備工作、常見類型（如脊髓麻醉與硬脊膜外麻醉）及其操作步驟，並提醒相關注意事項與潛在風險。文件亦涵蓋臂神經叢阻斷麻醉的兩種方法（斜角肌間進入法與腋下進入法），並列出可能的併發症及麻醉風險分級標準。

## 重點整理
- **區域麻醉**：在患者意識清醒、無痛的情況下，透過局部麻醉劑阻斷手術部位神經傳導，減少全身麻醉的副作用。
- **優點**：相較於全身麻醉，區域麻醉對心肺功能影響較小，術後疼痛較輕，且意識清晰。
- **常見類型**：包括半身麻醉、硬脊膜外麻醉、臂神經叢阻斷等。
- **適用對象**：需篩選不適合接受區域麻醉的患者，如敗血症、凝血功能不良、對局部麻醉藥過敏者。
- **準備工作**：確認患者身份、假牙是否移除、禁食時間是否充足，並進行心電圖、血壓、血氧監測。
- **操作姿勢**：多採側臥位，以減少穿刺併發症。
- **潛在風險**：包括腰痛、頭痛、神經損傷（發生率約 0.015%），以及硬脊膜穿刺後頭痛。
- **臂神經叢阻斷麻醉**：可用於手及前臂手術，麻醉後約6至10分鐘，手術部位開始麻木，藥效可持續1至3小時。
- **可能併發症**：包括低血壓、局部麻醉藥過量中毒、神經損傷、局部血腫，以及意外將大量麻醉藥注入血管或硬脊膜腔導致休克。
- **麻醉風險分級**：根據美國麻醉醫學會（ASA）標準分為五級，風險隨患者健康狀況惡化而增加。

## 流程/步驟
### 脊髓麻醉（Spinal Anesthesia）
1. **定位**：在第三至第五腰椎間隙進行穿刺。
2. **消毒**：以消毒液對穿刺部位進行消毒，形成無菌區。
3. **穿刺**：使用 25 至 27 號細針（較打点滴的 20 號針細）穿過硬脊膜，注入局部麻醉劑。
4. **姿勢**：患者採側臥位，以減少術中不適。
5. **效果**：麻醉後可達到良好的麻醉及肌肉鬆弛效果。

### 硬脊膜外麻醉（Epidural Anesthesia）
1. **定位**：使用 18 號硬脊膜針經椎間隙穿刺至硬脊膜腔。
2. **置管**：將細塑膠軟管置入，以便持續輸注局部麻醉藥。
3. **消毒**：對穿刺部位進行消毒，形成無菌區。
4. **姿勢**：患者採側臥位。
5. **注意**：因硬脊膜腔狹窄，技術較困難，有時可能刺破硬脊膜，引發頭痛。

### 臂神經叢阻斷麻醉方法
1. **斜角肌間進入法**  
   - 用於手及前臂手術的麻醉。
   - 麻醉劑注入後數分鐘，手臂開始麻木及無力。
   - 藥效可維持1至3小時。

2. **腋下進入法**  
   - 用於手及前臂手術的麻醉。
   - 麻醉劑注入後數分鐘，手臂開始麻木及無力。
   - 藥效可維持1至3小時。

## 注意事項/警示
- **禁忌症**：
  - 敗血症、凝血功能不良、服用抗凝血劑。
  - 神經性疾病、神經阻斷部位感染、對局部麻醉藥過敏。
- **術前準備**：
  - 確認患者身份，並確認假牙是否已移除。
  - 詢問禁食時間，避免吸入性肺炎。
  - 詢問過去疾病史及麻醉經驗。
- **術中注意**：
  - 穿刺時應使用細針（25–27 號）以減少組織及神經損傷。
  - 建議採側臥位，以降低併發症風險。
- **術後注意**：
  - 半身麻醉或硬脊膜外麻醉後，可能出現腰痛、頭痛或神經損傷。
  - 硬脊膜穿刺後頭痛發生率較高，需密切觀察。
- **若患者不適合區域麻醉，或手術時間超過預期，或出現併發症，應改為施行全身麻醉。**
- **麻醉可能導致的嚴重併發症包括：低血壓、局部麻醉藥過量中毒、脊髓損傷、以及因大量麻醉藥意外注入血管或硬脊膜腔而導致休克。**

壹、什麼是區域麻醉
區域麻醉是病患在意識清醒及無痛下接受手術，藉由局部麻醉劑來阻斷手術部位的神經傳導。區域麻醉所得到的好處是少了全身麻醉後的意識不清、嗜睡、術後止痛強度比全身麻醉低、對心臟血管及肺功能的影響較少等。區域麻醉可分為半身麻醉、硬脊膜外麻醉、臂神經叢阻斷等。

貳、準備工作
1、篩選不適合接受區域麻醉的患者（如敗血症、凝血功能不良、服用抗凝劑、神經性疾病及神經阻斷之下針部位感染及對局部麻藥過敏等）。對病人身份再確定，假牙是否移除，並詢問患者是否有適當的禁食時間（若禁食時間不足會有吸入性肺炎的危險），及過去曾患疾病或麻醉經驗等。
2、心電圖、血壓、血中含氧量等監測器設立好後立即進行。

參、脊髓麻醉
1、其方法是在第三至第五腰間空隙下針，當穿過硬脊膜後注入適量的局部麻醉劑，即可達到良好的麻醉及肌肉鬆弛效果。
2、麻醉姿勢有側臥及坐姿兩種，其中以側臥最常用。以消毒液消毒下針範圍形成一個無菌區，再使用脊髓針（使用的脊髓針以27至25號為主，比打點滴的20號針細很多，為的是減少組織及神經的傷害，避免腰酸背痛及頭痛的發生率）作穿刺。
3、半身麻醉或其他區域麻醉病患有發生腰痛或頭痛的機會，且有極少數可能導致短期或長期之神經傷害（發生率0.015%）。半身麻醉後有可能發生硬脊膜穿刺後頭痛的情形。

1、其方法是用18號硬脊膜針經椎間空隙至硬脊膜腔（並沒有像脊髓麻醉那樣直接注入藥物），然後將米粉般細的塑膠軟管置入，以便持續加局部麻醉藥。因硬脊膜腔很窄，硬脊膜外麻醉技術上比較困難，有時會刺破硬脊膜引發頭痛。

柒、麻醉風險

對於部份接受麻醉之病人而言，無論是全身麻醉或區域麻醉，均有可能會發生副作用或是併發症，嚴重者甚至可能死亡，這種因麻醉而來的危險與您身體狀況有關。根據美國麻醉醫學會所制定的標準可分為以下等級：

第一級：正常、健康。（手術麻醉死亡率為0.06-0.08%）

第二級：有輕微的全身性疾病，但無功能上的障礙。（手術麻醉死亡率為0.27-0.4%）

第三級：有中度至重度的全身性疾病且造成部份的功能障礙。（手術麻醉死亡率為1.8-4.3%）

第四級：有重度全身性疾病，具有相當的功能障礙且時常危及生命。（手術麻醉死亡率為7.8-23%）

第五級：瀕危狀態，不管有無手術預期在24小時內死亡。（手術麻醉死亡率為9.4-51%）
