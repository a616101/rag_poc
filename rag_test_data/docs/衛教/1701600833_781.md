## 主旨  
本文件為屏東基督教醫院提供的小兒腹股溝疝氣術後護理指導，說明腹股溝疝氣的成因、治療方式，並提供術後飲食與傷口照護的具體建議。

## 重點整理  
- **腹股溝疝氣**：俗稱「脫腸」或「墜腸」，是因胚胎發育時腹股溝環未閉合，導致腹腔內組織（如小腸、腹膜）經小洞滑出，可發生於任何年齡，需盡快就醫並進行疝氣修補術。  
- **術後照護**：重點在於飲食管理與傷口護理，以促進恢復並預防復發。  
- 手術後若出現哭鬧、咳嗽或打噴嚏，應用手輕壓傷口，以減輕疼痛並保護傷口。  
- 學齡兒童在術後兩週內應避免劇烈運動。  
- 迴家後一周，需依醫師指示至門診追蹤檢查。  

## 流程/步驟  
### 術後飲食步驟  
1. 手術後完全清醒，即可進食。  
2. 先讓小孩喝半杯開水，若無嘔吐、咳嗽、嗆水等反應，即可正常進食。  
3. 若喝半杯開水後出現上述不適反應，需等待一小時後再試飲，直至無反應才可進食。  
4. 建議攝取高纖維食物（如水果、蔬菜）並多喝水，養成規律排便習慣。  
5. 避免排便時用力或久坐，以防疝氣復發。  

### 傷口照護步驟  
1. 若傷口疼痛或哭鬧不安，可依醫師指示服用止痛藥。  
2. 讓小孩穿著寬鬆衣服，避免摩擦傷口；幼童需隨時更換乾燥尿布，減少污染。  
3. 若傷口持續出血、高燒（>38.5°C）、膿狀分泌物或紅腫，需立即回急診。  
4. 輕微發燒（約38°C）時，多喝水並服用退燒藥即可。  
5. 傷口會貼上透氣敷料，以擦澡方式清潔身體，術後一週後才可正常淋浴。  
6. 平時需保持傷口乾燥，不需特別換藥或拆線。  
7. 陰囊疝氣者，術後二至三天內陰囊輕微水腫屬正常，可平躺時冰敷並抬高陰囊，以促進血液回流。  
8. 冰敷傷口周圍及腹股溝處，一天7-8次，每次2-3分鐘，且勿弄濕傷口。  

## 注意事項/警示  
- **立即就醫**：若出現傷口持續出血、高燒（>38.5°C）、膿狀分泌物或患部紅腫等情況，應立即回急診處理。  
- **避免動作**：術後應避免排便用力、久坐等動作，以防疝氣復發。  
- **保持乾燥**：傷口需保持乾燥，避免弄濕，特別是術後一週內。  
- **冰敷注意**：冰敷時應避免直接接觸傷口，並注意頻率與時間。  
- 迴家後若傷口無異常，則無需更換藥物。  
- 若出現以下症狀，應立即就醫：  
  - 傷口周圍紅腫或裂開  
  - 高燒不退  
  - 持續嘔吐

小兒腹股溝疝氣術後護理指導

編號 ND 兒-023
製訂日期 2010.06
二修日期 2018.05

腹股溝疝氣就是俗稱的「脫腸」或「墜腸」，是因腹部下方在胚胎發育時，腹股溝環未閉合而留下一小洞，腹腔內的組織或器官的某部份，常見的是小腸、腹網膜或部份大腸，由此小洞下滑而出鼓起。腹股溝疝氣可發生在任何年齡層，一經發現應盡快就醫，治療方法為進行疝氣修補術。

術後注意事項：

（一）飲食
1. 手術後完全清醒，即可進食，可先讓小孩喝半杯開水，如無嘔吐、咳嗽、嗆水等反應，即可如平常般進食。
2. 喝半杯開水時，如有嘔吐、咳嗽、嗆水等反應，需再等一小時後再試飲開水，直到無上述不適反應發生，才可進食。
3. 建議可以攝取高纖維食物（水果、蔬菜等）並多喝水，養成規律的排便習慣；此外，若大便解不出來，請避免排便用力或執行久坐等動作，以免疝氣復發。

（二）傷口照顧
1. 手術後若因傷口疼痛，哭鬧不安時，可依醫師處方服用止痛藥。
2. 儘量讓小孩穿著寬鬆衣服，以避免摩擦傷口；較小的幼童，必須隨時更換尿布，盡可能保持尿布乾燥，以減少傷口的汙染。
3. 返家後，傷口有持續出血，高燒（>38.5℃）、膿狀分泌物，患部紅腫屬害等情況，需立即回急診處理，若輕微發燒（38℃左右），只要多喝開水及服用退燒藥即可。
4. 手術後傷口會貼上透氣敷料，以擦澡方式清潔身體，一週以後才可照常淋浴，平時需保持傷口乾燥，傷口不需特別換藥及拆線。
5. 鼠蹊部疝氣者，手術後二至三天內陰囊輕微水腫是正常的，可於平躺時冰敷並抬高陰囊，以促進血液回流，減輕腫脹。冰敷傷口周圍及腹股溝處，一天 7－8 次，每次 2－3 分鐘，且勿弄濕傷口。

6. 手術後若哭鬧、咳嗽、打噴嚏時，可用手壓住傷口，以減輕疼痛且保護傷口，學齡兒童兩週內要避免劇烈運動。

(三) 其他注意事項

1. 返家後，傷口若無異常則不需換藥，若傷口周圍紅腫裂開、高燒不退、持續嘔吐等，請速回醫院檢查。

2. 返家後一周，依醫師醫囑於門診追蹤檢查。
