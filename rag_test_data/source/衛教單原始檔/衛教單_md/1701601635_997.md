# 1701601635_997

## 主旨  
當孩子尚未具備語言能力時，家長可透過非語言溝通、提升聽懂指令能力、降低說話壓力、模仿及使用誇張語調等策略，幫助孩子建立說話的基礎，並鼓勵其願意開口說話。

## 重點整理  
- 對於還不會說話的孩子，直接要求說正確的音會造成壓力，應避免強迫仿說。  
- 可先使用手勢、指物、聲音或圖卡等非語言方式，幫助孩子表達需求。  
- 提升孩子聽懂口語指令的能力是語言發展的基礎，可透過遊戲方式訓練。  
- 降低說話時的壓力，利用孩子喜愛的活動（如扮家家酒、玩車子）引導發聲。  
- 家長可透過模仿孩子的動作、聲音與口型，引導其發出有意義的語音。  
- 使用誇張語調、放慢語速、明顯口型，並在孩子發聲時立即給予鼓勵，幫助其建立說話的正向經驗。

## 流程/步驟  
1. **使用非語言溝通方式**  
   - 教導孩子使用手勢、指物、聲音或圖卡來表達需求。  
   - 若孩子已會說一些詞語，應創造機會引導其說出，以增強信心與語詞穩定性。  

2. **提升聽懂口語指令的能力**  
   - 確認孩子是否能理解口語指令（無手勢提示）。  
   - 利用家中常見物品（如水果、餐具、衣服、玩具）進行遊戲，發出口語指令，讓孩子遵行。  

3. **降低說話壓力**  
   - 避免過度要求孩子仿說，以免造成焦慮。  
   - 利用孩子喜愛的活動（如扮家家酒、玩車子、娃娃屋）或玩偶互動，引導其自然發出聲音。  

4. **模仿孩子的行為與聲音**  
   - 當孩子做出身體動作（如玩玩具、跳舞）、口腔動作（如打哈欠、喝水）或發出類似說話的聲音時，家長應立即模仿。  
   - 將孩子的聲音轉化為有意義的語音，例如孩子發出「夕一」，可模仿說「夕一夕一，車車來了」。  

5. **使用誇張語調與放慢語速**  
   - 用語調明顯的聲音吸引孩子注意，等孩子看著你時再說話。  
   - 放慢語速，並誇張口型，引發孩子對說話聲音的注意。  
   - 使用易模仿的音節，如「狗狗」說「久尤」、「車子」說「夕入勺久」。  
   - 當孩子用聲音回應時，立即給予鼓勵，並避免糾正發音，以建立說話的快樂經驗。

## 注意事項/警示  
- 不要因孩子不會說話而過度要求仿說，以免造成負面情緒。  
- 避免在孩子說話時糾正發音，應以鼓勵為主，建立正向的說話經驗。  
- 家長應耐心觀察孩子的反應，並根據其發展階段調整引導方式。  
- 若孩子長期不說話，建議尋求專業語言治療師評估，以排除語言發展遲緩或其他潛在問題。

當孩子不說話的時候可以怎麼做

文／復健科語言治療師 林學民

對於一個還不會說話的孩子，一開始即要求說出正確的音是相當困難的，不斷要求孩子說話或跟著仿說，常常造成親子雙方不開心，讓孩子更不喜歡開口，甚至害怕或拒絕說話。其實對還不會說話的孩子，除了仿說以外，還有許多方法可促進孩子的先備能力並願意開口說話，請參考以下幾項引導孩子說話的策略：

1、先使用口語以外的方式來溝通

很多孩子在不會說話時會用哭和搶東西等不當方式表達需求，在學會說話之前，可以先教導孩子使用他人可以接受的溝通方式，如手勢、手指指物、聲音或圖卡等。如果已經會說一些語詞，則製造機會引導孩子說出，增加說話的信心及語詞的穩定性。

2、提升孩子聽懂口語指令的能力

除口語表達外，認知與聽理解都是語言的發展的根本，但家長往往只看到孩子怎麼不會說話，而不知道孩子可能還聽不懂。所謂的聽懂指令，就是只聽到口語（沒有手勢提示）指令就可以遵行，例如：從三至五樣物品或圖片中正確選取目標物。家長可利用家中常用的物品像水果、餐具、衣服、玩具？等和孩子玩遊戲，從中發號口語指令讓孩子去拿出大人所說的物品。

3、降低說話時的壓力

無論成人或幼兒，壓力情境會對各種動作表現產生負面影響，對於口語的影響包括呼吸、肌肉顫抖、構音器官的靈活度改變等。仿說是引導孩子說話的方法之一，但切忌過度要求孩子仿說，最好利用孩子喜歡的活動，如扮家家酒、玩車子、娃娃屋或利用孩子喜歡的玩偶和孩子互動，一起發出聲音。

4、模仿

孩子不太懂模仿的概念時，家長可以先模仿孩子，包括孩子的身體動作（玩玩具、跳舞）、口腔動作（如打哈欠、喝水的聲音、吃東西）及聲音等。當孩子發出類似說話的聲音時，家長先模仿孩子發出的聲音，再變成有意義的語音，如：孩子玩車發出「ㄅㄧ」，可立即模仿說「ㄅㄧ ㄅㄧ ㄅㄧ 或車車來了」。

5、使用誇張的語調和放慢速度說話

很多孩子不看人，家長可以使用語調明顯的聲音吸引孩子的注意力，等孩子看著妳/你時，再對著孩子說話。放慢說話的速度並誇張嘴型，可引發孩子對說話聲音的注意，如哇啊、喔喔等，並可先使用「狗狗」說「ㄨㄤ ㄨㄤ」、「車子」說「ㄅㄨ ㄅㄨ」…等，嘴型明顯的音較容易模仿，當孩子用聲音回應時，應立即予以鼓勵增強。在語言發展初期，讓孩子感覺說話是快樂的，不要糾正孩子的發音，而是鼓勵多發言。
