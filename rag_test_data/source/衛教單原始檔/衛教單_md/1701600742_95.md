# 1701600742_95

## 主旨
本文件為屏基醫療財團法人屏東基督教醫院提供的新生兒眼睛感染照護指引，說明新生兒眼部分泌物的常見原因、護理方式及注意事項，以預防與處理化膿性結膜炎。

## 重點整理
- 新生兒於出生第二天後，可能出現單眼或雙眼的化膿性結膜炎，症狀為黃綠色分泌物。
- 常見病因為葡萄球菌或多種革蘭氏陰性細菌感染，或於生產時經母體產道黏膜暴露感染。
- 護理重點包括：勤洗手、正確清潔眼部分泌物、避免交叉感染。
- 若分泌物持續增加或呈綠色，應儘速就醫。
- 必要時依醫師指示使用眼藥膏治療。

## 流程/步驟
### 眼部分泌物清潔方式
1. **清潔前準備**：照顧寶寶前後應確實洗手。
2. **使用工具**：使用無菌棉棒沾生理食鹽水。
3. **清潔方向**：由內眼角（眼頭）往外眼角（眼尾）方向擦拭。
4. **避免重複使用**：棉棒不可重覆使用。
5. **防止交叉感染**：毛巾勿交替使用。

## 注意事項/警示
- 常注意眼部分泌物的顏色、量有無增加，以及眼睛有無紅腫情形。
- 若分泌物持續增加，且呈現綠色，應立即帶寶寶就醫。
- 必要時，依醫師指示使用眼藥膏治療。
- 請勿自行使用藥物，應遵照醫囑。

新生兒眼睛感染之照護

| 編號 | ND 兒-011 |
| :--- | :--- |
| 製訂日期 | 2003.07 |
| 六修日期 | 2019.07 |

一、新生兒於出生第二天後，兩眼或單眼出現化膿性結膜炎（黃綠色分泌物）。

二、病因與危險因子

（一）因葡萄球菌或多種革蘭氏陰性細菌引起。

（二）新生兒於生產時經母體產道時黏膜暴露感染。

三、眼睛分泌物之護理

（一）照顧寶寶前後確實洗手。

（二）清潔眼睛方式：

| 有分泌時之清潔方式 | 洗臉時之清潔方式 |
| :--- | :--- |
| ![](image1.png) | ![](image2.png) |
| 1. 無菌棉棒沾生理食鹽水<br>2. 由內眼角（眼頭）往外眼角（眼尾）方向擦拭<br>3. 棉棒不可重覆使用。 | 1. 先洗沒有受感染的眼睛再洗有感染的眼睛<br>2. 由內眼角（眼頭）往外眼角（眼尾）方向擦拭<br>3. 毛巾勿交替使用以防感染。 |

（一）常注意分泌物顏色及量有無增加及眼睛有無紅腫情形。

（二）眼睛分泌物若持續增加，且呈現綠色分泌物時，請帶寶寶看醫生。

（三）必要時，依醫師指示使用眼藥膏治療。
