# 1701603035_741

## 主旨  
本文為預防流感與諾羅病毒感染的衛教資訊，提醒民眾在冬季及節慶假期期間，應注意個人衛生與健康防護，特別是高風險族群，並提供五項預防與應對措施。

## 重點整理  
- 流感與諾羅病毒在冬季及節慶期間易於傳播，尤其在旅遊與探親訪友頻繁時。  
- 各年齡層皆可能感染，老年人、嬰幼兒及慢性病患者感染後易出現嚴重症狀。  
- 預防感染的「五要」措施包括：接種疫苗、警覺症狀、及時就醫、按時服藥休息、保持良好個人衛生習慣。  
- 諾羅病毒主要透過污染的水、食物或接觸病人排泄物傳播，應注意飲食與手部清潔。  
- 有嘔吐或腹瀉症狀者，應避免上班或上學，並儘速就醫。  
- 症狀解除後至少 48 小時，方可返工或返校。  
- 餐飲業者（如廚工）需特別注意，應在症狀完全消除後才可工作。  
- 嘔吐物及糞便可能含有傳染病原，處理時務必做好防護。  

## 流程/步驟  
### 預防流感與諾羅病毒感染的「五要」  
1. **要儘早打流感疫苗**  
   接種流感疫苗是預防流感最經濟有效的方式，特別建議免疫力較差的長者、幼兒、慢性病患者、國小學童及醫護人員等高風險族群接種。  

2. **要警覺症狀**  
   流感與感冒症狀相似，但流感常伴隨明顯頭痛、發熱、倦怠、肌肉酸痛，且症狀發作較突然，病程較長（約5-10天甚至數週），易引發併發症，甚至導致死亡。  

3. **危險徵兆勿輕忽，要儘速送醫**  
   若出現呼吸急促、呼吸困難、發紺（缺氧）、嘴唇發紫或變藍、血痰、胸痛、意識改變、低血壓、高燒持續72小時等症狀，應立即就醫。  

4. **生病要按時服藥及休息**  
   一般感染流感者應依醫師診斷服藥並多休息，約1-2週可痊癒。切勿未完成療程就自行停藥，以免產生抗藥性；若兒童有服藥困難，家長應與醫師溝通。  

5. **保持個人衛生，要有好習慣**  
   - 勤洗手，打噴嚏時以手帕或衣袖掩住口鼻。  
   - 咳嗽或打噴嚏時戴口罩。  
   - 保持均衡飲食、正常作息與適量運動。  
   - 生病應就醫，依醫師指示服藥並在家休息。  

### 諾羅病毒預防注意事項  
- 諾羅病毒引發病毒性腸胃炎，主要透過攝食污染的水或食物、與病人密切接觸或吸入其嘔吐物及排泄物產生的飛沫而感染。  
- 食物處理與進食前，應徹底洗手。  
- 所有食物（特別是貝類海產）應徹底煮熟後才食用。  

### 嘔吐或腹瀉症狀處理與返工指引  
1. 一旦出現嘔吐或腹瀉症狀，立即停止工作或上學，並就醫。  
2. 症狀解除後，至少等待 48 小時，方可返回工作或學校。  
3. 若出現嘔吐物或排泄物，立即清理。  
4. 使用清潔劑徹底消毒被污染的表面。  
5. 清理期間，安排其他人遠離污染區域。  
6. 處理時應配戴手套及口罩。  
7. 事後必須徹底洗手。  

## 注意事項/警示  
- 餐飲業者（如廚工）屬於高風險群體，應嚴格遵守返工時間。  
- 嘔吐物及糞便可能含有傳染病原，處理時務必做好防護。  
- 未經妥善消毒的污染區域，可能導致疾病傳播。

## 全文

如何預防流感和諾羅病毒的感染

根據疾病管制局監視資料顯示，冬天一般是為流感，以及病毒性腸胃炎（如最近國內流行的諾羅病毒）的流行期。由於民眾國內外旅遊及訪友頻繁，增加疾病傳播的機會，因此提醒民眾歡度假期時，仍應保持良好個人衛生習慣，勤洗手，注意飲食衛生，以免生病而破壞過節興致。

各年齡層都有可能感染流感，特別是老年人、嬰幼兒及具有慢性疾病患者感染後容易出現較嚴重之症狀。為了避免感染流感，提醒大家做好「五要」，才能降低感染及傳播流感之機會：

1. 要儘早打流感疫苗
   接種流感疫苗是預防流感最經濟及有效方式，尤其是免疫力較差的長者、幼兒與慢性病患者等高危險群及國小學童與醫護人員等高傳播族群。

2. 流感不是感冒，要警覺症狀
   流感與感冒都會有喉嚨痛、咳嗽及流鼻水等症狀，但流感容易出現明顯的頭痛、發熱、倦怠、肌肉酸痛等，通常症狀發作較突然，痊癒的時間也比感冒長，約需 5-10 天甚至數週才能恢復，而且容易引發併發症，甚至導致死亡。

3. 危險徵兆勿輕忽，要儘速送醫
   感染流感若出現呼吸急促、呼吸困難、發紺（缺氧、嘴唇發紫或變藍）、血痰、胸痛、意識改變、低血壓、高燒持續 72 小時等症狀時，需儘速送醫。

4. 生病要按時服藥及休息
   一般人感染流感，原則上依醫師診斷服藥、多休息，約 1-2 週會痊癒。另外，不要未完成療程就自行停藥，容易產生抗藥性，若孩子有服藥的困難，家長應多與醫師溝通。

5. 保持個人衛生，要有好習慣
   保持肥皂勤洗手、打噴嚏以手帕或衣袖掩口鼻、咳嗽戴口罩、多運動、均衡飲食、正常作息等生活好習慣；生病即就醫，依醫師指示服用藥物並在家休息。

另外，諾羅病毒等引發之病毒性腸胃炎，主要透過攝食污染的水或食物、與病人密切接觸或吸入病人嘔吐物及排泄物所產生的飛沫而感染。故防治上應注意：

1. 處理食物或進食前、如廁後須徹底洗淨雙手
2. 所有食物（特別是貝殼類海產）應該徹底煮熟才食用

3. 有嘔吐或腹瀉症狀的人，勿上班或上學，並就醫診治，尤其是餐飲業者(廚工)，應於症狀解除至少 48 小時後才可上班。

4. 立即清理糞便及嘔吐物，徹底消毒被污染的地方；清理期間應安排其他人士遠離嘔吐物及被污染的地方，處理時須戴上手套及口罩，事後必須徹底洗手。
