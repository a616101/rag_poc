# 1701601451_895

## 主旨
本文旨在鼓勵家長與孩子進行親子共讀，透過繪本閱讀促進孩子的語言理解與表達能力，並培養其對閱讀的興趣。同時提供具體的共讀技巧，以提升共讀的趣味性與教育成效。

## 重點整理
親子共讀對孩子有以下助益：

- **增加詞彙量**：繪本內容提供多種類型的詞彙，幫助孩子學習與理解。
- **讀寫萌發**：透過指讀文字，幫助孩子理解文字與聲音的對應關係，並認識書本的結構與使用方式。
- **增加視覺素養**：孩子能從圖像中發現文字未提及的細節，培養觀察力。
- **培養閱讀興趣**：在愉快的互動中建立對「書」的美好感受，進而自然產生閱讀喜好。

## 流程/步驟
建議家長在共讀時可採用以下步驟，以增強互動與學習效果：

1. **引導孩子命名與討論**  
   - 讓孩子填空，例如：「當我們收到禮物後，我們要跟對方說 ______。」
   - 提問「什麼」、「哪裡」、「為什麼」等問題，例如：「為什麼他們吵架了？」
   - 回想故事內容，並詢問相關問題，例如：「今天小青蛙遇到哪些好朋友呢？」
   - 聯繫生活經驗，例如：「你有喝過南瓜湯嗎？那你還喜歡喝什麼湯呢？」

2. **讚美與教育**  
   - 當孩子正確回答時，給予肯定與讚美。
   - 若孩子回答錯誤，提供正確答案，並利用機會教導新詞彙。

3. **擴展與重述**  
   - 重複孩子的話，並添加更多資訊，以擴展句子內容。
   - 鼓勵孩子重述經擴展後的完整句子。

## 注意事項/警示
- 家長應保持輕鬆愉快的氛圍，避免過度要求或批評，以免影響孩子的閱讀興趣。
- 共讀應以孩子為主，尊重其理解節奏與參與方式。

## 全文

繪本共讀

親愛的家長您好：

透過親子共讀，對於孩子有以下助益。因此，非常鼓勵家長與孩子共讀繪本，以促進孩子的語言理解及表達，並藉此培養孩子對閱讀的興趣。

| 增加詞彙量 | 繪本中的內容可以提供許多不同類型的詞彙。 |
| --- | --- |
| 讀寫萌發 | 一起閱讀時，可以指著字，讓孩子慢慢去理解一字搭配一音的意義，通過這個過程去識別：哪些是文字、圖像以及他們的功能為何。或是讓孩子了解該怎麼去操作一本書，比如知道拿書的正確方向。 |
| 增加視覺素養 | 孩子能從圖像閱讀去發現許多文字沒提到的小細節。因此，鼓勵各位爸媽在親子共讀時，可以和孩子一起仔細找找圖片中的秘密唷！ |
| 培養閱讀興趣 | 閱讀興趣會在開心的互動當中產生，孩子會把共讀時美好的感受，跟「書」連結起來，自然而然的就會漸漸喜歡上閱讀。 |

另外，亦提供一些繪本共讀小撇步。當家長在與孩子共讀時，可使用以下的技巧，讓共讀的過程更加趣味，同時增進共讀的成效。

1. 引導孩子命名繪本中的插圖和談論故事內容。
   (1) 讓孩子填空，如「當我們收到禮物後，我們要跟對方說 ______。」
   (2) 問「什麼」、「哪裡」和「為什麼」的問題，如「為什麼他們吵架了？」
   (3) 回想故事內容，並詢問相關問題，如「今天小青蛙遇到哪些好朋友呢？」
   (4) 詢問孩子與故事情節相關的生活經驗問題，比如，繪本提到南瓜湯，可以詢問孩童「你有喝過南瓜湯嗎？那你還喜歡喝什麼湯呢？」

2. 讚美孩子的正確回應，或是在孩子回應錯誤時，提供正確答案，以利用機會教育孩子新的詞彙。

3. 重複孩子的話，並添加更多訊息，以擴展內容，再鼓勵孩子重述經擴展後的句子。
