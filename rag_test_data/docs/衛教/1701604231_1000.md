## 主旨
本文件為屏東基督教醫院提供的腫瘤患者飲食指導，強調透過均衡營養飲食來增強體力、抵抗力與免疫力，並針對不同治療階段（如化療前、中、後及休息期）提供具體飲食建議。內容涵蓋治療期間的症狀管理、飲食禁忌、營養補充方式，以及治療結束後的飲食恢復與生活習慣調整，旨在協助患者改善營養狀態，提升生活品質。

## 重點整理
- 腫瘤無法單靠特殊飲食或秘方治癒，應依靠日常新鮮食物中的營養素來支持身體。
- 高蛋白、高熱量飲食有助於維持體力與免疫力，肌肉是體力與免疫力的關鍵。
- 避免過度限制飲食，應盡量攝取營養，特別是肉類，其營養價值高於湯類。
- 化療期間味覺可能改變，可適度食用重口味食物，但需確保衛生。
- 面對噁心、口乾、便秘等症狀，應以對症處理為主，勿過度驚慌。
- 治療前後應調整飲食，以適應身體需求，並預備體力應對醫療處置。
- 每日應攝取五份蔬果，其中三份為蔬菜，兩份為水果。
- 主食建議選擇胚芽糙米飯，並多選用粗糧與全穀類。
- 飲食以清淡為主，少油少鹽，多採用蒸、煮方式，避免油炸與烤炙。
- 保持規律進餐，定時定量，避免零食，維持三餐正常。
- 避免攝取添加劑，減少醃漬、燻烤類食物。
- 每餐應攝取新鮮魚肉與豆類，建議每餐至少二兩。
- 鼓勵攝取平時不愛吃但無毒的時令蔬果，以增加營養攝取。
- 貧血應多攝取紅肉類、豬血、蚵仔、海藻類及豆類等富含鐵質的食物。
- 鉀與鐵會互相抵銷，因此吃紅肉時應避免同時攝取奶類。
- 維生素C有助於鐵的吸收，但櫻桃、黑棗、桑椹等水果含糖量較高，補「血」效果不如其他選擇。
- 生機飲食強調未使用農藥或化肥所種植的有機農作物，並以生食蔬果或榨汁為主，但長期生食可能導致總熱量不足，易造成營養不良。
- 有機飲食指經認證的有機農產品，不強調「生食」，建議以均衡飲食為基礎，搭配有機蔬果。
- 可提高免疫力的食物包括十字花科蔬菜（如青(白)花椰菜、高麗菜、蘿蔔）、菇菌類（如香菇、蘑菇、柳松菇、鮑魚菇）及新鮮海產與肉類。
- 日常生活提升免疫力的方法包括：保持正常三餐，定時定量，進行負重運動；攝取時令新鮮蔬果，避免偏食與熬夜；少油、少糖、少鹽，多攝取纖維與深海魚；避免食用燒烤、醃漬及過度加工的食物；戒除菸、酒、檳榔等不良習慣。

## 流程/步驟
### 當您發現生病後：
1. 認識腫瘤的惡性特質，不依賴特殊飲食或秘方。
2. 依靠日常新鮮食物中的營養素來加強體力、抵抗力與免疫力。
3. 避免只喝湯類（如魚湯、肉湯），因普林會升高尿酸，真正營養在肉中。
4. 保持生活規律，盡量攝取營養，避免過度限制。
5. 鍛鍊肌肉，因肌肉等於體力，也等於免疫力。

### 當您在化療手術中：
1. 味覺可能改變，若想吃重口味食物，確認乾淨衛生後即可食用，無需有罪惡感。
2. 若口腔有金屬味，可口含微酸糖果，或使用糖醋、檸檬、香草、香酥等調味。
3. 針對噁心、口破、口乾、便秘等症狀，進行對症處理。
4. 請勿過度驚慌，保持穩定心態。

### 療程前一週：
1. 保持生活規律，盡量攝取營養。
2. 增加負重運動，例如使用小瓶礦泉水（約半瓶）進行鍛鍊。
3. 肌肉是體力與免疫力的來源，應持續維持。

### 療程後一週：
1. 攝取易消化的餐食。
2. 若吃不下，可用均衡營養品代替。
3. 水果應洗淨、剝皮，並立即食用。
4. 多補充水分，如檸檬片泡水、菊花茶、麥茶等。

### 休息的時段：
1. 一天安排六餐，以高熱量、高蛋白為主。
2. 若吃不下，可改以喝的方式攝取。
3. 貯備體力以應對醫療處置。
4. 可將牛肉作為補血點心，每週兩次，每次約半個手掌大小。

### 面對常見副作用的飲食與護理措施：
- **噁心反胃**：食用檸檬、鳳梨、糖醋、酸梅、薑汁、洛神等酸味食物；使用市售均衡營養配方，可搭配杯湯或濃湯調整口味；利用九層塔、香菜、蔥、薑、八角等調味增香。
- **嘴破**：食物以常溫或低溫食用，或使用吸管避免刺激傷口；每日補充維生素B群，或食用綠豆湯；進食後立即漱口並塗抹口內藥膏；使用麩醯胺酸（如速養療、基速得、三多、益富等）泡冷水，含在口中3～5分鐘後吞下。
- **口乾**：擦抹護唇膏（如凡士林），並準備茶水或飲料，口乾時即飲，以溫潤口咽；可飲用肉汁、筍湯以增加食物濕度；食用酸味果汁，或口含烏梅、薄荷糖、冰棒。
- **便秘**：起床後喝300～500 cc室溫涼開水；食用木瓜、香蕉、帶皮蘋果（用綠色菜瓜布刷洗），並由右往左繞圈按摩腹部；睡前服用1茶匙苦茶油，或攝取異麥芽寡糖、乳酸菌等。
- **腹瀉**：補充2000～3000 cc水分，可飲用去皮蘋果泥、柳丁汁等；若有脫水情形，應喝淡鹽水或稀釋等量運動飲料；食用稀飯、饅頭等低纖維食物，建議一天六餐。

## 注意事項/警示
- 避免過度限制飲食，應盡量攝取營養。
- 高蛋白飲食有助於維持體力與免疫力。
- 避免只喝湯類，因普林會升高尿酸，且營養主要來自肉類。
- 化療期間味覺改變屬常見現象，可適度調整口味，但需注意衛生。
- 面對症狀應對症處理，勿過度驚慌。
- 保持規律生活，並適時進行負重運動。
- 營養品以綜合維生素及魚油為主，不建議信賴昂貴或未經證實的秘方。
- 中藥如枸杞、黃耆（生薑）、紅棗等對體力恢復有幫助，建議在療程完全結束後再使用。
- 飲品建議多喝白開水、檸檬片泡冷水、麥茶、菊花茶等，有助於身體代謝。
- 類黃酮食物如蜂王乳、蜂膠、山藥、靈芝、葡萄籽、月見草油、大豆異黃酮等含類黃酮物質，可能類似女性荷爾蒙，女性腫瘤患者應慎用；但豆腐、豆漿可適量食用。
- 海產不新鮮可能引發食物中毒，需特別注意新鮮度。
- 過敏食物如茄子、竹筍、雞肉、蝦子、螃蟹、鴨肉、海鮮等可能因個人體質引起過敏，但無需特別禁食。
- 白血球低時應避免生菜沙拉、生魚片等生食，新鮮水果應選擇可剝皮後立即食用的種類，如橘子、香蕉、葡萄。
- 所有飲食建議應根據個人狀況調整，如有疑問應諮詢醫療團隊。
- 食用帶皮水果時，務必徹底清洗，以避免農藥殘留。
- 腹瀉期間應避免高纖、油膩或刺激性食物。
- 嘴破時應避免食用過燙、過辣或過酸的食物，以免加重症狀。

屏東基督教醫院
財團法人 PINGTUNG CHRISTIAN HOSPITAL

腫瘤飲食建議

當您發現生病後：

請明白腫瘤其惡質特性，無法靠『某些特殊飲食限制』或秘方來轉變，此時您所需的體力、抵抗力和免疫力，都等著您提供日常新鮮食物中的各類營養素來加強！

熬湯能熬出來的多是「普林」，普林會使尿酸升高，魚湯、肉湯不夠營養，真正的營養素在『肉』裡面。

生活規律，能吃盡量吃，勿過度限制，可以鍛鍊肌肉更好，因為肌肉等於體力，也等於免疫力。

當您在化療手術中：

味覺會改變，若特別想吃某些重口味食物，確認乾淨衛生後即可食用，無須有罪惡感。若覺口腔有金屬味，口含微酸糖果，可用糖醋、檸檬、香草、香酥…增加風味。

只要針對出現的症狀，如（噁心、口破、口乾、便秘…等）作處理即可，請勿過度驚慌。

療程前一週：生活規律，能吃盡量吃，增加荷重運動（小瓶礦泉水的重量），肌肉等於免疫力。

療程後一週：吃好消化的餐食，吃不下喝均衡營養品代替，盡量吃熟的，水果以洗淨剝皮立刻食用為佳，多補充水份（檸檬片泡冷水、菊花茶、麥茶…）。

休息的時段：一天六餐高熱量高蛋白，能吃盡量吃，吃不下用喝的，要貯備體力進入醫療處置（吃牛肉當補血點心一週兩次，每次半個手掌大即可）。

腫瘤飲食建議

當您療程結束後：

1. 每日五蔬果，三份蔬菜兩份水果。
2. 胚芽糙米飯，多選粗糙之全穀類。
3. 清淡少油鹽，多用蒸煮少用炸烤。
4. 定時兼定量，少吃零食固定三餐。
5. 不碰添加劑，少吃醃漬燻烤食物。
6. 新鮮魚肉豆，每餐盡量要吃二兩。
7. 多吃平時不愛吃的無毒時令蔬果。

備註：

1. 營養品以綜合維生素及魚油為主，勿信昂貴秘方。
2. 中藥如枸杞、黃耆（生芪）、紅棗，對體力恢復雖有一定效果，建議以療程完全結束後給予為佳。
3. 多喝白開水或檸檬片泡冷水、麥茶、菊花茶等。
4. 蜂王乳、蜂膠、山藥、靈芝、葡萄籽、月見草油、大豆異黃酮…等含類黃酮（似女性荷爾蒙），女性腫瘤患者慎用，但豆腐、豆漿可適量食用。
5. 不新鮮的帶殼海產，可能引發食物中毒，請注意。
6. 茄子、竹筍、雞肉、蝦子、螃蟹、鴨肉、海鮮…等，因個人體質，可能引起過敏，但無須特別禁食。
7. 白血球無法只靠飲食提昇，早睡早起適量運動即可，白血球過低時，該時段避免吃生菜沙拉、生魚片…等的生食，新鮮水果請選剝皮後可立即食用的種類，如（橘子、香蕉、葡萄）。

3. 利用九層塔、香菜、蔥、薑、八角…等增加風味。

4. 麩醯胺酸(速養療、基速得、三多、益富…)泡冷水，含嘴裡 3～5 分鐘再吞下去。

口乾
1. 擦護唇膏(凡士林)，備茶水飲料口乾即飲，溫潤口咽。
2. 用肉汁、芍藥、濃湯增加食物的濕度。
3. 酸味果汁或口含烏梅、薄荷糖、冰棒。

便秘
1. 起床喝一杯 300～500 cc 室溫的涼開水。
2. 木瓜、香蕉、蘋果帶皮吃(用綠色菜瓜布刷洗)…等。
3. 自己的方向，由右往左繞圈按摩腹部。
4. 睡前吞 1 茶匙苦茶油，異麥芽寡糖、乳酸菌…等。

腹瀉
1. 補水 2000～3000cc，去皮蘋果泥、柳丁汁…等。
2. 有脫水情形-喝淡鹽水、稀釋等量運動飲料。
3. 吃稀飯、饅頭…等纖維少的食物，一天六餐。

腫瘤飲食建議

貧血-易疲倦、煩躁、抵抗力減弱
1. 多吃紅肉類、豬血、蚵仔、海藻類、豆類等。
2. 鈣跟鐵會互相抵銷，吃紅肉那餐，不要同時喝奶類。
3. 雖然生素 C 可幫助鐵吸收，但櫻桃、黑棗、桑椹…等水果，補“血糖”比較快。

生機飲食？
『生食』不用農藥或化肥培養的有機農作物，強調可提升免疫力。由於直接生食蔬果或搾汁，故能保留完整微量營養素。如長期以此為主，雖「微量維他命」夠，但『總熱量』不足，反易導致嚴重的營養不良。

有機飲食？
跟生機飲食類似，只是不特別強調『生食』，一般有認證的以吉園圃…等為主。建議以均衡飲食為基礎，再搭配此類作為蔬果的來源，才是適合的飲食方式。

可以提高免疫力的食物有哪些？
1. 十字花科，青(白)花椰菜、高麗菜、蘿蔔…等。
2. 菇菌類，香菇、蘑菇、柳松菇、鮑魚菇…等。
3. 新鮮海產肉類食物，可以提高細胞自我修復力。

日常生活怎樣才能提高免疫力？
1. 正常三餐、定時定量、負重運動。
2. 吃時令新鮮蔬果、不偏食不熬夜。
3. 少油少糖少鹽、多纖維多深海魚。
4. 少吃燒烤、醃漬及過度加工食物。
5. 戒除菸、酒、檳榔…等不良習慣。
