# 1701591799_502

## 主旨  
本文件旨在指導接受脊椎手術的病人正確姿勢與禁忌姿勢，以促進術後恢復，避免腰部受傷。正確的坐姿、躺姿與正確的起床姿勢，以預防腰背壓力及跌倒。

## 重點整理  
- 依醫囑正確穿著背架並進行活動。  
- 避免搬重物、舉重，減少腰部壓力。  
- 避免久站、久坐、突然伸展或彎曲背部肌肉，以防扭傷。  
- 避免彎腰，應保持腰部平直。  
- 坐姿應保持背部貼緊椅背，避免坐沙發或矮椅。  
- 躺姿應選擇正確姿勢以減輕腰背壓力。  
- 起床時應依正確步驟進行，特別是下床活動時需注意，避免因頭暈或四肢無力導致跌倒。  

## 流程/步驟  
### 正確的起床姿勢  
1. 由平躺姿勢，先轉為側躺。  
2. 將身體移至床邊，雙手撐起上半身。  
3. 將雙腿移至床邊，雙手撐住床緣。  
4. 利用雙手支撐，將身體慢慢站起。  

## 注意事項/警示  
- 嚴格依照醫師指示穿著背架並下床活動。  
- 避免任何可能增加腰部負擔的動作，如搬重物、舉重。  
- 避免長時間站立或坐著，並避免突然伸展、彎曲或搖動背部肌肉。  
- 避免彎腰動作，應保持腰部平直，以減少腰部壓力。  
- 坐姿注意：坐椅子時應背貼椅背，不可坐沙發或矮椅。  
- 躺姿注意：  
  - 平躺時應放鬆身體，膝下墊枕頭。  
  - 側躺時應兩膝稍彎曲，中間可夾一個枕頭。  
- 避免錯誤姿勢：  
  - 平躺直腿會增加背部壓力。  
  - 趴睡會增加腰部壓力。  
- 下床活動注意：應先坐起，待無頭暈或四肢無力時再站起、行走，且必須有家屬在旁陪同，以預防跌倒。
