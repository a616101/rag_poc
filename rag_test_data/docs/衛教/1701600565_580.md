## 主旨
本文件為屏基醫療財團法人屏東基督教醫院提供的新生兒眼睛感染照護指引，說明常見的新生兒化膿性結膜炎症狀、病因、護理方式及注意事項，並提供正確的清潔步驟以防止感染擴散。

## 重點整理
- 新生兒出生後，若出現單眼或雙眼黃綠色分泌物，可能為化膿性結膜炎。
- 常見病因為葡萄球菌或多種革蘭氏陰性細菌感染，或生產時經母體產道黏膜暴露感染。
- 護理時應注意手部清潔，並使用正確方法清潔眼睛。
- 清潔時應由內眼角向外眼角擦拭，避免交叉感染。
- 若分泌物持續增加或呈綠色，應儘速就醫。
- 必要時依醫師指示使用眼藥膏治療。

## 流程/步驟
### 眼睛分泌物護理步驟
1. 照顧寶寶前後確實洗手。
2. 使用無菌棉棒沾生理食鹽水。
3. 由內眼角（眼頭）往外眼角（眼尾）方向擦拭。
4. 棉棒不可重複使用。
5. 洗臉時，先清潔未受感染的眼睛，再清潔受感染的眼睛。
6. 毛巾勿交替使用，以防感染。

## 注意事項/警示
- 注意分泌物的顏色、量有無增加，以及眼睛有無紅腫情形。
- 若眼睛分泌物持續增加，且呈現綠色，請立即帶寶寶就醫。
- 必要時，依醫師指示使用眼藥膏治療。
- 勿使用已用過的棉棒或毛巾，以免造成交叉感染。

## 全文

屏基醫療財團法人屏東基督教醫院

新生兒眼睛感染之照護

一、新生兒於出生後，兩眼或單眼出現化膿性結膜炎(黃綠色分泌物)。

二、病因與危險因子

(一)葡萄球菌或多種革蘭氏陰性細菌引起。

(二)新生兒於生產時經母體產道時黏膜暴露感染。

三、眼睛分泌物之護理

(一)照顧寶寶前後確實洗手。

(二)清潔眼睛方式:

| 有分泌物時之清潔方式 | 洗臉時之清潔方式 |
| :--- | :--- |
| 1. 無菌棉棒沾生理食鹽水<br>2. 由內眼角(眼頭)往外眼角(眼尾)方向擦拭(如圖示中箭頭方向)<br>3. 棉棒不可重覆使用 | 1. 先洗沒有受感染的眼睛<br>2. 由內眼角(眼頭)往外眼角(眼尾)方向擦拭(如圖示中箭頭方向)<br>3. 毛巾勿交替使用以防感染 |

(三)注意分泌物的顏色、量有無增加及眼睛有無紅腫情形。

(四)眼睛分泌物若持續增加，且呈現綠色分泌物時，請帶寶寶看醫生。

(五)必要時，依醫師指示使用眼藥膏治療。
