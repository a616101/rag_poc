## 主旨  
本文件旨在介紹嬰幼兒語言與溝通能力的發展階段，並提供照護者可採取的互動策略，以促進嬰幼兒的語言與社會溝通能力發展。

## 重點整理  
- 嬰幼兒的語言發展從胎內開始，透過聽覺與身體動作回應外界。  
- 語言發展與社會溝通能力密切相關，照護者的互動方式對其發展至關重要。  
- 照護者應建立良好的溝通基礎，並透過模仿、眼神接觸與肢體語言，促進嬰幼兒的語言與情緒發展。  

## 流程/步驟  
### 嬰兒社交與溝通能力發展量表  
| 年齡 | 發展特徵 |  
|------|----------|  
| 1-3 個月 | 喜歡看人臉和聽聲音，開始認人，出現社會性微笑，有眼神注視。若照護者在嬰兒發出聲音後立即回應，嬰兒會跟著再次發出聲音。 |  
| 4-6 個月 | 聲音可能伴隨著情緒出現，開始有手勢動作的模仿。 |  
| 7-9 個月 | 試著用手勢溝通，會模仿更複雜的手勢動作，注意力會隨著照護者的手勢方向轉移。 |  
| 10-12 個月 | 能回應照護者一半的要求，開始使用詞彙替代手勢作為功能性溝通。 |  

### 照護者可以怎麼做？  
1. **建立溝通基礎**  
   照護者應在嬰兒看得見的地方，並確認嬰兒處於愉快的情境下進行溝通。說話時應有節奏性，並可模仿寶寶的動作或聲音。  

2. **臉部表情及眼神的注視**  
   眼神注視有助於集中嬰兒的注意力，並促進理解能力。利用不同的臉部表情來補充語言，增強溝通效果。  

3. **頭部動作**  
   點頭及搖頭可傳遞訊息，突然的頭部動作也能維持嬰兒的注意力。  

4. **空間關係**  
   隨著嬰兒成長，與照護者的距離會增加，觸摸的行為減少，但眼神注視的頻率會增加。  

### 親子活動舉例  
#### 模仿貓 (Copy Cat)  
照護者根據嬰兒的行為進行模仿，模仿方式分為三種：  
- **誇大的模仿**：用較大動作或聲音吸引嬰兒注意。  
- **縮小的模仿**：模仿嬰兒的行為，以吸引其參與照護者的活動。  
- **修改的模仿**：當嬰兒情緒不穩時（如大哭），照護者以柔和的方式模仿，幫助緩和情緒。  

## 注意事項/警示  
- 照護者的互動應以嬰兒的興趣與情緒為基礎，避免強迫溝通。  
- 模仿應根據嬰兒的行為進行，並適時調整模仿方式，以維持其參與度。

# 嬰幼兒語言誘發

嬰兒從在媽媽肚子裡時就開始學著聽外面世界的聲音，利用身體的動作做回應，而當他從媽媽肚子裡呱呱墜地時，照顧者應該要如何跟他互動促進溝通的發展呢？讓我們來一起瞭解兒童的溝通發展和建議的互動方式吧！

> 嬰兒社交與溝通能力發展量表

1-3 個月：喜歡看人臉和聽人的聲音、開始會認人，出現社交性的微笑，可以有眼神注視，如果照顧者在嬰兒發出聲音後立即給予回應，嬰兒會跟著再次發出聲音。

4-6 個月：聲音可能是伴隨著情緒出現，有手勢動作的模仿。

7-9 個月：試著用手勢溝通，會模仿更複雜的手勢動作，注意力會跟隨著照顧者的方向轉移。

10-12 個月：回應照顧者一半的要求，開始使用詞彙替代手勢作為功能性的溝通。

> 照顧者可以怎麼做？

1. 建立溝通基礎
   照顧者要是在嬰兒看得到的地方，並確認嬰兒是在愉快的情境下進行溝通，說話時緩和並有節奏性，可模仿寶寶的動作或聲音。

2. 臉部表情及眼神的注視
   眼神注視可以讓嬰兒的注意力集中在照顧者臉部，利用不同表情來補充談話，有助於維持溝通時注意力的維持及促進理解能力。

3. 頭部動作
   點頭及搖頭可傳遞訊息，突然的頭部動作也可以維持嬰兒注意力。

4. 空間關係
   當嬰兒越來越大，與照顧者溝通的距離會變大，觸摸的行為變少，而眼神注視變多。

> 親子活動舉例

模仿貓 copy cat
照顧者根據嬰兒的行為進行模仿，模仿方式分為三種：誇大的模仿（喚起嬰兒注意力）、縮小的模仿（吸引嬰兒參與照顧者的行為中）、修改的模仿（如：嬰兒大哭，照顧者柔和的模仿有助於緩和嬰兒情緒）。
