# 1701600618_725

## 主旨
本文件為財團法人罕見疾病基金會製作的衛教宣導資料，主題為「泛酸鹽激酶關聯之神經退化性疾病」（Pantothenate Kinase Associated Neurodegeneration, PKAN），旨在提升大眾對此罕見遺傳疾病的認識，並提供相關遺傳諮詢與醫療資源，幫助家庭掌握生命價值，實現「愛與尊重」的照護理念。文件透過介紹PKAN的成因、病理機制、典型與非典型症狀，並以個案「蓉嬤」的經歷，傳達罕見遺傳疾病應予理解與尊重，鼓勵以積極樂觀的態度面對生命挑戰。

## 重點整理
- **疾病名稱**：泛酸鹽激酶關聯之神經退化性疾病（PKAN），是一種罕見的神經退化性疾病。
- **疾病定義**：PKAN是一種由PANK2基因突變引起的體染色體隱性遺傳病，導致泛酸（維生素B5）代謝異常。
- **病理機制**：泛酸代謝異常會影響輔酶A的生成，進而干擾能量與脂質代謝，並導致有害物質在神經細胞中堆積，引發神經退化。
- **典型影像特徵**：在基底核的蒼白球（globus pallidus）和黑質（substantia nigra）出現鐵質異常堆積，於MRI的T2影像中可見「虎眼徵」（the “eye of the tiger” sign）。
- **遺傳模式**：為體染色體隱性遺傳，需父母雙方皆為帶因者，子代才可能患病。
- **核心訊息**：雖然無法完全避免遺傳疾病，但透過早期發現與專業諮詢，可掌握百分之百的生命價值。
- **目標對象**：
  - 未罹患罕見疾病的社會大眾：應認識罕見疾病，給予患者關懷與尊重。
  - 已罹患罕見疾病的患者：應正確認識病因，把握治療契機。
  - 準備孕育下一代的準父母：應進行產前遺傳檢查與諮詢，特別是對於有家族病史者。
- **預防與照護**：早期發現與專業諮詢是關鍵，可及早進行療育，並在懷孕初期追蹤胎兒遺傳狀況，以做出生育決策。
- **個案分享**：個案「蓉嬤」從幼年發展遲緩，到小學時病情快速惡化，最終需依賴鼻胃管和輪椅，但她始終保持樂觀，並渴望回校讀書，展現了堅韌的生命力。

## 流程/步驟
1.  **認識疾病**：了解泛酸鹽激酶關聯之神經退化性疾病（PKAN）的成因與症狀。
2.  **基因傳遞**：人體內的基因透過DNA複製代代相傳。若其中一個基因發生變異，可能在傳宗接代時將缺陷基因傳給子女，造成遺傳性疾病。
3.  **疾病診斷**：當出現發展遲緩、運動功能退化、吞嚥困難等症狀時，應尋求專業醫療評估，進行基因檢測與影像學檢查（如MRI）以確診。
4.  **進行篩查**：準備孕育下一代的準父母應配合產前檢查，特別是對於有家族病史者，應進行遺傳諮詢與生化遺傳檢驗。
5.  **專業諮詢**：若懷孕，應於初期追蹤胎兒遺傳狀況，並根據結果與專業人員討論是否生育。
6.  **治療與照護**：目前尚無根治方法，治療以症狀管理為主，包括物理治療、職能治療、藥物（如肉毒桿菌素、肌肉鬆弛劑）以及營養支持（如鼻胃管餵食）。
7.  **心理支持**：患者與家屬應獲得心理輔導與社會支持，以應對疾病帶來的身心挑戰。
8.  **早期療育**：若確診，應及早進行專業療育，以提升生活品質。

## 注意事項/警示
- **風險提示**：即使沒有家族病史，仍有3%的機率產出缺陷兒；若家族中有病史，則有0.5%~1%的機率可能患有罕見遺傳疾病。
- **早期發現**：PKAN的症狀可能在幼年時就已出現，但初期可能被誤認為發展遲緩，應提高警覺。
- **遺傳諮詢**：若家族中有罕見遺傳病史，建議進行遺傳諮詢，了解再發風險。
- **避免歧視**：應以同理心看待罕見疾病患者，避免歧視與偏見，給予他們應有的尊重與支持。
- **決策考量**：在遺傳諮詢後，應充分評估自身狀況與未來照護能力，再決定是否生育。
- **及早行動**：每個小生命都獨特且珍貴，及早發現與專業諮詢是把握契機的關鍵。

認識罕見遺傳疾病

罕見遺傳疾病不是怪病、更不應予以歧視。
別因它「罕見」，就「輕忽」它。

「罕見遺傳疾病」，是生命傳承中的小小意外。人體內約有兩萬五千個基因，藉著DNA（去氧核醣核酸）準確的複製，把基因代代相傳。不過，其中若有基因發生變異，就可能在傳宗接代時，把有缺陷的基因帶給子女，造成遺傳性疾病。

在台灣，罕見疾病的定義是年盛行率在萬分之一以下的疾病。全世界已知有一萬一千多種人類遺傳疾病，其中大多非常罕見。罕見遺傳疾病發生機率雖小，卻是生命傳承中無可避免的遺憾。雖然這機率只降臨在少數人身上，卻是每個人都必須承擔的風險。

罕見遺傳疾病個案

用笑容面對人生

蓉嬤剛出生時，飲食及發育都與同年齡孩子相仿，但走路、語言能力卻較為緩慢。3歲時，媽媽發現她的發展呈現停滯狀態，雖然可以對話溝通，但心裡總是覺得哪裡不對勁，蓉嬤有時會無法及時反應，像是卡住了一樣，有時手拿水瓶喝東西，水瓶會不自覺的掉落到地上。

由於蓉嬤症狀一直不明顯，檢查被認定為發展遲緩，雖然可以走路，但狀況總是時好時壞，走路不穩容易跌倒，有時遇到小水溝腳也無法跨越，走路與跑步姿勢明顯與別人不一樣，經過長期復健治療，蓉嬤的狀況一直不見好轉，直到要上小學前，媽媽決定帶她再到醫院檢查，才獲得神經科醫師確診為泛酸鹽激酶關聯之神經退化性疾病（PKAN）。

蓉嬤小學時，身體退化速度特別明顯，一年內，一次流感，在家躺了數日後，她便不能再走路了；漸漸的，蓉嬤說話越來越口齒不清，喝水常會嗆到，隨後又因腸病毒關係，高燒不退，症狀快速退化到她無法再自己吃東西；一次的缺氧休克及胃出血，住院將近1個月，讓蓉嬤歷經一場人生大關。痊癒後，吞嚥能力變得困難，必須靠鼻胃管飲食，生活開始要仰賴他人的協助，疾病也讓她四肢與脖子的肌肉張力過高，呈現不自然的姿勢，「不是痛，像是身體找不到重心」蓉嬤說，她必須經注射肉毒桿菌素，及服用肌肉鬆弛劑，讓身體四肢肌肉獲得舒緩。

蓉嬤一直是個討人喜歡的愛笑女孩，面對每況愈下的身體，她總是笑笑地面對，即使疾病受限了她的行動，但她仍保有積極樂觀、努力向學精神。住院時，媽媽曾問過她：「此時此刻，妳最想去哪裡？最想做什麼呢？」蓉嬤的答案不像一般孩子一樣想去遊樂園搭摩天輪，或是河濱公園騎腳踏車，卻是說：「我想要回學校讀書。」大病後，轉學到了新環境，蓉嬤總能適應得非常好。媽媽常笑說：「她是運氣好，即便生病了，身邊總遇到很好的人，都對她很好，特別照顧，真的很感激。」轉眼間，蓉嬤已經是個18歲的高中少女了，家人不變的相互陪伴，成為支持彼此最大的力量。

泛酸鹽激酶關聯之神經退化性疾病

泛酸鹽激酶關聯之神經退化性疾病（Pantothenate Kinase-Associated Neurodegeneration，PKAN）是PANK2（Pantothenate Kinase 2）基因突變的體染色體隱性遺傳疾病，導致泛酸代謝功能異常。

泛酸即維生素B5，是人體必需維生素，合成生理代謝必要物質－輔酶A。當泛酸鹽激酶功能受損造成輔酶A的生成及泛酸代謝異常，不僅影響人體的能量與脂質代謝，亦使潛在有害的化合物堆積於神經細胞，產生毒性造成神經退化。

常見特徵為：鐵質異常堆積於功能性腦區，多沉積在基底核（basal ganglia）之蒼白球（globus pallidus）和黑質（substantia nigra）位置，從核磁共振（Magnetic Resonance Imaging，MRI）的T2影像，可發現「虎眼徵」（the “eye of the tiger” sign），屬於腦部鐵質積聚之神經退化性疾病症候群（Neurodegeneration with Brain Iron Accumulation, NBIA）中最常見的型別。

臨床上會出現的症狀，每位個案不盡相同。依典型PKAN和非典型PKAN整理如下：

| | 典型 PKAN | 非典型 PKAN |
|---|---|---|
| 發病年齡 | 通常在6歲之前（平均年齡：3.4歲） | 10歲之後（平均年齡：13.6歲） |
| 退化速度 | 快速，症狀嚴重 | 較緩慢，症狀因人而異 |
| 症狀差異 | 經常性跌倒、吞嚥困難、認知障礙、約2/3患者有色素性視網膜病變影響視野 | 肌張力不全症、巴金森症、衝動或憂鬱等情緒精神症狀、構音困難或說不清、認知功能下降、步態不穩、複語症（palilalia）或急語症（tachylalia）等言語問題 |

PKAN為進行性神經退化性疾病，病患失去的功能通常無法恢復，疾病進展速度與發病年齡有關，越早出現症狀，其退化速度越快。患者疾病進展狀況不一，可能經歷一段快速惡化階段後，進入較長的穩定期，但仍在持續退化狀態。目前依照疾病個案的症狀進行治療，如使用肌肉鬆弛劑、抗痙劑、注射肉毒桿菌素、復健治療或深層腦部電刺激術，改善患者肌張力障礙。透過物理和職能治療以維持關節活動性；語言治療增進溝通能力，適當地使用輔具幫助提升患者的生活品質。
