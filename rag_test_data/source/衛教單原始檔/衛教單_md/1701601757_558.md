# 1701601757_558

## 主旨
本文探討孩子不說話的可能原因，並提供提升孩子溝通動機的方法。透過理解孩子說話的動機，家長可運用「需求」作為催化劑，創造更多溝通情境，促進語言發展。

## 重點整理
### 孩子不說話的可能原因
1. **腦神經損傷**：如母親懷孕或孩子出生時有特殊狀況，影響腦部發展。
2. **情緒上的障礙**：如自閉症兒童。
3. **智能發展問題**：認知發展較慢，間接影響語言發展能力。
4. **聽力損失**：聽力狀況不佳或聽不清楚，影響語音辨識及學習力。
5. **家庭環境影響**：父母過於忙碌，缺乏互動；或過度溺愛，導致孩子無需主動溝通即可滿足需求。

### 人類說話的動機
1. **有需求**：例如「我想要玩具汽車，買給我。」
2. **要告狀**：例如「小華剛剛用腳踢我，好痛。」
3. **要分享**：例如「今天老師稱讚我。」
4. **要幫忙**：例如「幫我搬這重重的箱子。」
5. **要傳遞訊息**：例如「爸爸請你兩點半去接他。」
6. **要增進人際互動**：例如「你要和我們一起玩遊戲嗎？」

## 流程/步驟
### 提升孩子溝通動機的方法
除了增加溝通情境，可使用以下方法作為「需求」的催化劑，促使孩子產生溝通意願：
1. **看得到，拿不到**：將孩子喜愛的玩具或糖果放在透明罐子裡。
2. **別一次滿足需求**：將餅乾分成多份，讓孩子要求「還要」。
3. **暫停喜愛的活動**：遊戲玩到一半就暫停，讓孩子要求繼續。
4. **假裝搞錯**：例如平常用湯匙吃飯，突然給予叉子。

## 注意事項/警示
- 若家長發現孩子有語言發展遲緩的跡象，應儘早尋求專業協助。
- 可至小兒復健科、小兒耳鼻喉科、語言治療所等單位諮詢語言治療師。

如何提升孩子的溝通動機

文／語言治療師 林學民

「我的孩子為什麼都不說話？」
「他都聽得懂，但就是不願意開口說。」
「我的孩子兩歲半了，卻還只會叫爸爸、媽媽。」

首先，我們可以先探討，孩子不說話的可能原因。

1. 腦神經損傷—如：媽媽懷孕或孩子出生時是否有特殊的狀況影響到腦部發展。
2. 情緒上的障礙—如：自閉症兒童
3. 智能發展出現問題—認知發展較慢間接影響到語言發展的能力。
4. 聽力損失—聽力狀況不佳、聽不清楚，影響語音聽辨及學習力。
5. 家庭環境影響—如：父母過於忙碌，無暇與孩子互動或是父母過度溺愛，剝奪孩童主動溝通的意圖，即使不開口也可滿足所有需求等。

再來，我們要了解：一個人為什麼需要說話呢？

1. 有可能是有需求—如：我想要玩具汽車，買給我。
2. 有可能是要告狀—如：小華剛剛用腳踢我，好痛。
3. 有可能是要分享—如：今天老師稱讚我做得很好。
4. 有可能是要幫忙—如：幫我搬這重重的箱子。
5. 有可能是要傳遞訊息—如：爸爸請你兩點半去接他。
6. 有可能是要增進人際互動—如：你要和我們一起玩遊戲嗎？

諸如此類等等。

除了增加孩子的溝通情境之外，我們還可以怎麼做？

「需求」會促使孩子產生溝通的意願，我們可以使用一些方法作為催化劑，讓孩子提出需求。

1. 看得到，拿不到—如：將小朋友喜愛的玩具、糖果，放在透明的罐子裡。
2. 別一次滿足孩子的需求—如：將餅乾分成好多口，讓小朋友要求「還要」。
3. 暫停喜愛的活動—如：遊戲玩到一半就暫停，讓小朋友要求繼續。
4. 假裝搞錯了—如：平常用湯匙吃飯，突然給予叉子。

若家長想要了解更多訊息，可以到哪裡尋求資源或協助呢？請到小兒復健科、小兒耳鼻喉科、語言治療所等單位諮詢語言治療師。
