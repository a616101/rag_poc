# 1701602373_627

## 主旨
本文件為屏東基督教醫院中醫科提供的考季保健指南，旨在協助考生應對「考試症候群」，透過中醫養生觀點，提供飲食、穴位按摩及藥膳等方法，幫助考生維持身心健康，順利應對考試壓力。

## 重點整理
- **考試症候群常見症狀**：注意力不集中、頭痛、暈眩、記憶力衰退、失眠、腸胃不適、呼吸急促、肌肉酸痛緊繃，女性可能出現經期不規則。
- **生活調理原則**：保持規律作息，避免熬夜，適度運動，均衡飲食，充分休息，以維持良好體力與清晰頭腦。
- **專業建議**：若出現身體不適，應尋求專業中醫師協助，避免自行用藥。
- **中醫保健方法**：
  - **藥膳**：提供「天麻醒腦湯」與「天麻燉烏雞」兩款藥膳，有助於醒腦、增強記憶、益智。
  - **穴位按摩**：介紹百會穴、風池穴、合谷穴的位置與按摩方法，可緩解頭痛、壓力，活絡氣血。
  - **簡單運動**：建議每天進行「小周天按摩」，如按摩眼周、眼球，以舒緩用眼過度。

## 流程/步驟
### 天麻醒腦湯製作步驟
1.  將雞翅洗淨，放入沸水中汆燙去血水。
2.  將雞翅與以下藥材（天麻三錢、人蔘鬚三錢、洋蔘二錢、枸杞五錢、黃耆十錢、川芎一錢、五彩龍骨三錢）一同放入鍋中。
3.  加入少許米酒，並加入適量的水。
4.  用電鍋或燉鍋，以大火煮沸後，轉小火燉煮約30分鐘。

### 天麻燉烏雞製作步驟
1.  將烏雞去除羽毛、爪、內臟，並清洗乾淨。
2.  將天麻切片，放入雞肚內。
3.  將整隻雞放入鍋中，加入薑片、鹽、黃酒及適量的清水。
4.  先用武火將鍋內水燒開，再轉為文火，燉煮至雞肉熟爛。
5.  加入適量的味精調味後即可食用。

## 注意事項/警示
- **藥膳與食補需因人而異**：應視個人體質而定，不可一體適用。
- **穴位按摩注意**：若出現反覆性頭痛，可使用手指在百會穴、風池穴、合谷穴進行環形按揉或垂直按壓，每次約10分鐘。
- **尋求專業幫助**：若症狀嚴重或持續，應儘早尋求專業中醫師診斷與治療，以對症下藥。

## 全文

屏基醫療財團法人屏東基督教醫院 PINGTUNG CHRISTIAN HOSPITAL

中醫陪您順利度過考季

隨著考季的到來，考生們容易出現所謂的「考試症候群」，而在身體方面出現了注意力不集中、頭痛、暈眩、記憶力衰退、失眠、腸胃不適、腹瀉、頻尿、心悸、呼吸急促、肌肉酸痛緊繃、女生則有經期不規則等現象。

在此建議考生在生活作息方面要規律，不要熬夜，適當的運動、均衡的飲食、充份的休息，才能保持良好的體力以及清晰的頭腦來應付考試。如果真有任何身體上的不適，則應求助專業的中醫師。

從中醫養生的觀點，提供大家一些簡單而有效的保健之道。

藥膳方面：可用天麻、蓮子、黃精、玉竹、川芎、決明子等藥物加入排骨、雞肉或魚一同燉煮。此湯可補腦益智安神，補血養心。對於心神不寧、失眠健忘、記憶力衰退有一定功效。

茶飲方面：可用天麻、石柱參、麥門冬、五味子、遠志、菊花、枸杞、甘草、紅棗，以水煎煮後當茶水喝。可益氣養神，補腦明目。

大部分的考生因為長時間讀書，反而疏於運動，處於用腦過度，肢體怠惰的狀態下，如果沒時間運動，建議在家每天花30分鐘做做「小周天按摩」，在五官部位，像是眼睛，可以用中指指腹按摩眼頭處20次，或按摩眼睛周圍10次，讓眼球左右上下各移動各10次等，可以達到抒解壓力、活絡氣血的功效。

若出現反覆性頭痛，中醫科主任醫師鍾貴馨建議可用手指在穴位環形按揉或垂直按壓10分鐘，達到緩解頭痛之效。常見能緩解頭痛的穴位有百會穴、風池穴及合谷穴。熟知這三大治頭痛的穴位，在頭痛時，趕緊按壓或環形按揉，就能盡快舒緩身體不適。如果想要再找尋小穴位，他建議就診，以免誤判穴位。

其實不管藥膳或食補，都應視個人體質而定，不能一體適用。如果真有「考試症候群」出現，最好還是請教專業的中醫師，以便對症下藥，輕鬆度過考季。

●百會穴：位在頭頂正中心，兩耳交連線和頭頂正中線的交點處。百會穴位是全身氣之匯集處，頭痛時按壓此穴，能有效緩解疼痛。

●風池穴：位在耳後頭枕骨下，髮際附近的凹陷處。

●合谷穴：又名虎口穴，位在手背，第1、2掌骨間中點處。

以下提供一些醒腦、增強記憶、益智的藥膳供參考：

【天麻醒腦湯】
天麻三錢、人蔘鬚三錢、洋蔘二錢、枸杞五錢、黃耆十錢、川芎一錢、五彩龍骨三錢、雞翅一斤半。

1、雞翅洗淨置入沸水中汆湯去血水。

2、將雞翅，加入上列藥材，並加入少許米酒，餘加入水以淹沒，加蓋。

3、用電鍋或燉鍋，燉煮。外鍋八杯水。

【丹蔘養心湯】寧心安神，增強記憶
丹蔘二錢、遠志一錢半、黨蔘五錢、天麻二錢、麥門冬二錢、紅棗十枚、豬心一個。

加水四碗先煮藥材30分鐘，豬心切開洗去血水，再入燉鍋煮15分鐘即成。

【天麻燉烏雞】益智聰明
原料：天麻15克，烏骨雞1隻，薑、鹽、黃酒、味精各適量。

製法：將烏雞去除毛、爪、內臟後，洗淨，再將天麻切片裝入雞肚內放入鍋中，在鍋中加入薑、鹽、黃酒、清水各適量。先用武火將鍋燒開，再用文火將雞肉燉至熟爛，加入適量的味精即可食用。

用法：食雞肉飲湯。

功效：補益氣血，滋陰化痰。
