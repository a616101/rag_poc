# 1701600759_982

## 主旨  
本文件旨在提供預防腸病毒感染的相關注意事項，特別針對高風險群體（如三歲以下兒童）提供預防與應對措施，並說明何時應立即就醫。

## 重點整理  
- 「腸病毒」並非單一病毒，而是多種病毒引起的感染，感染後無法終身免疫。  
- 目前無疫苗可預防（小兒麻痺除外），主要透過口、飛沫及接觸傳播。  
- 三歲以下兒童為高風險群，易併發腦炎、類小兒麻痺症候群或肺水腫等嚴重併發症。  
- 預防關鍵：增強免疫力、個人與環境衛生、避免接觸感染者。  
- 家中若有病患，應特別注意排泄物處理與隔離，避免傳染給孕婦、新生兒及小孩。  
- 出現特定症狀時應立即就醫。

## 流程/步驟  
- 增強個人免疫力：注意營養、均衡飲食、適度運動及充足睡眠。  
- 加強個人衛生：養成正確洗手五步驟（濕、搓、沖、捧、擦），以預防自身感染，並避免傳染給嬰幼兒。  
- 注意環境衛生：保持環境清潔及通風。  
- 避免接觸感染者：避免出入過度擁擠的公共場所，不與病人（如家人或同學）接觸。  
- 家中出現病患時應注意事項：小心處理病人排泄物（糞便、口鼻分泌物），且處理完畢應立即洗手；多補充水分，學童應請假在家休息，以避免傳染給同學；對家中第二個病人需特別小心，其接觸的病毒量較高，嚴重程度可能提高，應避免與孕婦、新生兒及小孩接觸。  
- 出現以下任何情況應立即就醫：嗜睡、意識不清、活力不佳、手腳無力；肌躍型抽搐（類似受到驚嚇的突發性全身肌肉收縮動作）；持續嘔吐；持續發燒、活動力降低、煩躁不安、意識變化、昏迷、頸部僵硬、肢體麻痺、抽搐、呼吸急促、全身無力、心跳加快或心律不整。

## 注意事項/警示  
- 一般神經併發症於發疹後二至四天出現。  
- 家中出現病患時，應特別注意排泄物處理與隔離，避免傳染給孕婦、新生兒及小孩。  
- 對家中第二個病人需特別小心，其接觸的病毒量較高，嚴重程度可能提高，應避免與孕婦、新生兒及小孩接觸。

## 全文

屏基醫療財團法人屏東基督教醫院
預防腸病毒感染注意事項

編號 ND 0-020
製訂日期 2010.05
二修日期 2018.02

「腸病毒」不是一種病毒，而是一群病毒感染而成，由於腸病毒型別很多，無法得過一次就終身免疫，而且目前無疫苗可以預防（小兒麻痺除外），又可經口、飛沫及接觸等途徑傳染，控制不易。三歲以下小孩為高危險群，有較高比率得到腦炎、類小兒麻痺症候群或肺水腫等合併症。

一、預防方法：
1. 增強個人之免疫力：注意營養、均衡飲食、適當運動及充足睡眠。
2. 加強個人衛生：養成濕、搓、沖、捧、擦正確洗手五步驟，以預防自身感染，及避免藉由接觸傳染給嬰幼兒。
3. 注意環境衛生：保持環境清潔及通風。
4. 避免接觸受感染者：避免出入過度擁擠之公共場所，不要與病人（家人或同學）接觸。

二、家中有病患時應注意：
1. 小心處理病人之排泄物（糞便、口鼻分泌物），且處理完畢應立即洗手。
2. 多補充水分，學童儘量請假在家休息，以避免傳染給同學。
3. 對家中之第二個病人要特別小心，其所接受的病毒量往往較高，嚴重程度可能提高，病人應避免與孕婦、新生兒及小孩接觸。
4. 有下列情況需立刻就醫：
   (1) 有嗜睡、意識不清、活力不佳、手腳無力應即早就醫，一般神經併發症是在發疹二至四天後出現。
   (2) 肌躍型抽搐（類似受到驚嚇的突發性全身肌肉收縮動作）。
   (3) 持續嘔吐。
   (4) 持續發燒、活動力降低、煩躁不安、意識變化、昏迷、頸部僵硬、肢體麻痺、抽搐、呼吸急促、全身無力、心跳加快或心律不整等。
