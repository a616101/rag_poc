## 主旨
本文介紹類固醇的歷史、作用機制、臨床應用及潛在副作用，並強調應遵照醫囑使用，以達到最佳療效並降低風險。

## 重點整理
- 類固醇又稱「美國仙丹」或「黑狗丸」，於1940年代由美國合成，早期因止痛效果顯著而廣受歡迎。
- 台灣早期因醫療資源不足，類固醇常透過電台廣告或小販販售，但濫用導致多起副作用案例，引發公眾警惕。
- 類固醇是人體腎上腺皮質分泌的激素，具有調節免疫、抗發炎、維持血壓及心臟血管功能等作用。
- 合成類固醇結構與人體類固醇相似，臨床用於治療發炎、過敏、自體免疫疾病、濕疹、肌無力等。
- 類固醇劑型包括口服、注射、外用塗抹，使用時需遵照醫囑，不可自行停藥或調整劑量。

## 流程/步驟
1. **使用前評估**：由醫師評估病情，決定是否使用類固醇及選擇合適劑型。
2. **正確用藥**：
   - 口服藥物：不可自行停藥或更改劑量。
   - 外用藥膏：取約一粒花生大小的量，塗抹於手掌大小的面積，避免重複按摩患處。
3. **長期使用注意**：需在醫師指導下逐漸減量，以降低副作用風險。
4. **定期追蹤**：監測血壓、血糖、骨密度等指標，及早發現並處理副作用。

## 注意事項/警示
- **不可自行用藥**：應遵照醫師指示使用，避免自行調整劑量或停藥。
- **注意副作用**：
  - 長期、高劑量使用可能導致庫欣氏症（Cushing's Disease），症狀包括高血壓、水牛肩、滿月臉、肥胖、多毛症、皮膚變薄、易骨折等。
  - 男性可能出現性慾降低，女性月經不規則。
  - 糖尿病患者使用可能導致血糖升高。
  - 外用類固醇可能導致皮膚變薄、出現青筋、青春痘。
- **避免濫用**：過去因濫用導致多起副作用案例，應避免自行購買或使用來路不明的類固醇藥品。

## 全文

類固醇

文／陳慧娟 藥師

類固醇，早年認識此名稱的人不多，如果說起「美國仙丹」或「黑狗丸」，即有古早濃濃的台灣鄉土味。「美國仙丹」是在1940年代由美國發現合成的類固醇，當時此藥非常好用，對造成身體上的疼痛幾乎可改善，醫生非常喜歡用此藥，「赤腳仙」更愛用，將其混在中藥粉製成丸劑或粉劑販售。

台灣在經濟起飛前，當時務農者多，經年累月彎腰做事，飲食又多以米飯為主食，那時常會聽到農人說腰節骨痠痛或是膝蓋疼痛，也因醫療及院所的未普及，看病多須自費，當時最能深入鄉下及貼近民眾的就是電台或開著小發財車賣藥，尤其早年娛樂選擇不多，陪伴過日子的就是『收音機』，在電台內主持人問候的親切語詞及強力保證下很容易淪為忠實聽眾。這些常見販賣未有衛生署認證合格的宣稱有止痛療效的家傳秘方如痠痛貼布、白黑狗丸、黃色帶點亮亮的中藥粉等，幾乎用過後都說有效而繼續購買服用，經過一陣子就會看到或聽到因服用黑狗丸入院的報導或傳聞，內容多描述藥物引起的症狀包括虎背熊腰（水牛肩）、月亮臉、皮膚有如薄紙、骨鬆等，此種濫用引致的副作用住院及媒體多次的報導，導致多數人直至現在都聞之色變。

類固醇又稱『腎上腺皮質素』，人體可自行合成分泌的賀爾蒙（激素），其分泌量有一定規律，早上時最高，晚上量會降低，維持著人體正常生理運作，如調解蛋白質、脂肪、醣類及電解質，當一遇突發性壓力時會大量釋出來應付或改善，所以有時會看到醫生開的類固醇藥物服用劑量早上高於晚上。

合成的類固醇其結構與功能相似人體的類固醇，具調節免疫緩解發炎、維持中樞神經功能包括血壓及心臟血管功能。在臨床的治療用藥，其療效相當廣泛，劑型有口服、注射、外用塗抹，臨床上用塗抗發炎、抗過敏、慢性阻塞性肺病、關節炎、自體免疫疾病、濕疹、肌無力等症狀或疾病改善。

接受類固醇的治療時不管在服用或塗抹皮膚上務必遵照指示，口服藥勿自行停用或更改服用量；尤其用於皮膚濕疹、癢的過敏時的外用藥膏塗抹量及用法，取約一粒花生大小的量即可塗約手掌大的面積，塗抹後勿一直在患處重複按摩，遵照指示可降低副作用發生。因與多數的藥物作用一樣，為達到療效相對的副作用也會跟著發生，尤其是長期使用（短期用幾乎少發生）、高劑量，須等到劑量逐漸調降，副作用也會隨著減少。

副作用如1. 類固醇長期、大量使用，使的體內皮質醇過多發生庫欣氏症（Cushing's Disease）有高血壓、水牛肩、滿月臉、中廣肥胖體型（四肢瘦）、多毛症、皮膚薄如紙、易有骨折、背痛或有青春痘等，男性有性慾降低，女性月經不規則，甚至對於糖尿病人會增加血糖；外用者，皮膚變薄甚至透明可看到微血管，引起青春痘。2. 另一種副作用是「愛迪森氏病」，體內腎上腺功能不足，因使用太多的合成的類固醇，常見臨床症狀皮膚色素沉著、姿勢性低血壓，長期缺乏的症狀如疲倦、體重下降、脫水、腹痛等。

使用者應依照醫師指示用藥避免自行濫用，醫師會評估其利與弊，使期達到最大療效副作用降到最低。
