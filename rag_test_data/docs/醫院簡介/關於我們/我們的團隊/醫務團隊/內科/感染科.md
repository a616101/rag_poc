# 感染科

## 團隊簡介
本科與其他專科不同，其他專科專注於特定器官（例如心臟科專注於心臟血管、胸腔科專注於肺部、腎臟科專注於腎臟），而本科專注於治療感染性疾病，而非器官系統。感染性疾病定義為任何由病原體引起的人類疾病，包括細菌、病毒、真菌、分枝桿菌（結核病）及寄生蟲。本科診治的疾病包括：腦膜炎、腮腺炎、上呼吸道感染、胸腔感染、腹部感染、泌尿道感染、皮膚感染及其他感染性疾病。除了治療外，本科也參與院內感染控制及抗生素管理，預防院內感染並促進抗生素合理使用。

## 團隊編制
- 醫師：感染專科醫師
- 護理/照護：專業護理團隊
- 醫事技術/其他：與胸腔內科、整形外科、一般外科合作
- 特色分工：參與院內感染控制及抗生素管理

## 服務與治療
### 服務範圍（疾病/問題）
- 發燒、不明熱
- 各類感染症：腦膜炎、腮腺炎、上呼吸道感染、胸腔感染、腹部感染、泌尿道感染、皮膚感染
- 性病相關問題
- 愛滋病
- 結核病
- 南台灣常見感染症：恙蟲病、Q熱、地方性斑疹傷寒、鉤端螺旋體感染、登革熱

### 主要治療/處置
- 各類感染症之診斷及治療：專精於診斷與治療南台灣常見感染症，如恙蟲病、Q熱、地方性斑疹傷寒、鉤端螺旋體感染、登革熱，以及本地盛行率高的結核病
- 性病相關問題諮詢
- 愛滋病之診療
- 結核病之診療及接觸診追蹤檢查
- 可執行支氣管鏡、胸腔鏡、腹腔鏡等診斷性檢查，確認診斷並找出病原體
- 積極處理需要手術引流或處置的感染性疾病

### 特色門診/照護計畫（若有）
- 院內感染控制：預防院內感染
- 抗生素管理：促進抗生素合理使用
- 與胸腔內科、整形外科、一般外科合作，提供整合性感染症治療

## 檢查項目

- 胸腔鏡檢查：用於診斷胸腔感染
- 腹腔鏡檢查：用於診斷腹部感染
- 病原體鑑定：確認診斷並找出病原體
