# 胃腸肝膽科

## 團隊簡介
本科醫師皆為經醫學中心完整訓練之肝膽腸胃專科醫師。目前有主治醫師7名、技術員7名、衛教師1名。本科以堅強的醫護人員陣容、完備的檢查儀器，以及各種先進的治療設備，提供病患精確、詳細的醫療品質及服務。本科每週門診提供20診，內視鏡檢查每月平均約450-500人，腹部超音波檢查每月平均約1,500人。

## 團隊編制
- 醫師：主治醫師7名（皆為經醫學中心完整訓練之肝膽腸胃專科醫師）
- 護理/照護：專業護理團隊
- 醫事技術/其他：技術員7名、衛教師1名
- 特色分工：衛教師負責病患衛教

## 服務與治療
### 服務範圍（疾病/問題）
- 各種腸胃肝膽系統疾病
- 消化道出血
- 異物吞入
- 膽道疾病（膽結石、膽道結石、膽道及膽囊腫瘤、膽道囊腫、膽道狹窄及廔管或寄生蟲）
- 胰臟疾病（慢性胰臟炎、胰臟腫瘤、胰管結石）
- B型、C型肝炎
- 肝臟疾病

### 主要治療/處置
- 針對不同病因，提供各種內視鏡診療
- 全天24小時內視鏡治療服務：掌握消化道出血的治療先機以及異物吞入之移除
- 治療性內視鏡技術：多數膽道疾病病患可免於開刀之苦
- 腸胃道出血緊急治療：靜脈曲張出血治療（內視鏡結紮術、內視鏡硬化術）、胃潰瘍出血治療（注射止血術、熱探子燒灼止血術、止血夾止血術）
- 內視鏡異物取出術
- 診斷性經內視鏡逆行性膽胰管攝影術（ERCP）：可診斷膽道疾病及胰臟疾病
- 治療性膽道內視鏡：經鼻膽道內視鏡引流術（ENBD）、經胃內視鏡逆行性膽道引流術（ERBD）、十二指腸乳突切開術（EST）、膽道結石取出術、機械式膽道結石碎石術、可擴張式金屬支架置放術、內視鏡氣球乳突擴張術
- 內視鏡瘜肉切除術
- 經皮內視鏡胃造廔術
- 經內視鏡氣球擴張術、經內視鏡食道狹窄擴張術、腸胃道狹窄擴張術
- 食道狹窄、腸胃道狹窄支架置入術
- 超音波導引下肝臟切片：診斷肝炎的原因或程度、肝硬化的原因或程度、腫瘤性質等
- 經皮穿肝膽道膽囊膽汁引流術（PTCD）：超音波導引下經皮膚留置引流管做膽汁的引流
- 經皮穿肝膽道攝影術（PTC）：可將膽管系統分枝顯現出來，並找出阻塞的位置
- 經腹腔穿刺抽取腹水術：經超音波導引下抽取引流腹水以減輕腹脹不適
- 肝膿瘍引流術：超音波導引下經皮膚穿肝抽取膿液或放置引流管，膿液可進行細菌培養，依照培養之結果調整抗生素的治療

### 特色門診/照護計畫（若有）
- B型、C型肝炎帶原肝臟檢查追蹤計畫：透過腹部超音波，針對B型、C型肝炎帶原肝臟檢查，並建立完整資料以追蹤檢查治療

## 檢查項目
- 上消化內視鏡檢查：食道、胃、十二指腸部位內視鏡檢查
- 下消化內視鏡檢查：直腸、乙狀結腸、升結腸、橫結腸、降結腸部位內視鏡檢查
- 十二指腸、膽管、胰管、膽囊部位內視鏡檢查
- 肝臟、膽囊、膽管、胰臟、脾臟、腎臟部位超音波檢查
- 幽門螺旋桿菌吹氣檢查：無須做胃鏡可檢測幽門螺旋桿菌吹氣之檢查，較舒適，須自費
- 診斷性經內視鏡逆行性膽胰管攝影術（ERCP）
- 於開刀房檢查：以上、下消化道內視鏡檢查，須麻醉，且自費

## 設備與特色
- 內視鏡設備：提供各種內視鏡診療服務
- 腹部超音波設備：用於肝臟、膽囊、膽管、胰臟、脾臟、腎臟檢查
- 治療性內視鏡設備：提供各種治療性內視鏡技術
- 堅強的醫護人員陣容、完備的檢查儀器，以及各種先進的治療設備

## 品質與安全

- 透過腹部超音波，針對B型、C型肝炎帶原肝臟檢查，並建立完整資料以追蹤檢查治療
