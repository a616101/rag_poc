# 院區與樓層介紹：屏基醫院院區

## 概要

屏基醫院院區包含以下建築物：

- **恩慈大樓**：位於院區上方，獨立建築
- **約翰大樓**：位於院區左側，大型不規則建築，向下連接韓偉大樓
- **馬可大樓**：位於院區中央，不規則建築，連接馬太大樓
- **韓偉大樓(急診)**：位於院區左下角，大型矩形建築，標示為急診
- **馬太大樓**：位於院區中央，連接馬可大樓和路加大樓，內含第二門診區和第三門診區
- **路加大樓(門診)**：位於院區右側，大型矩形建築，標示為門診

**周邊道路：**
- 大連路60巷：橫貫院區上方
- 大連路：沿院區右側垂直延伸
- 莊敬街：從左下角向中右側延伸，經過主要建築群下方

## 重要動線（怎麼走）

### 從大連路入口到路加大樓門診
1. 從大連路進入院區
2. 往右側大型矩形建築（路加大樓）
3. 1F 設有掛號批價櫃台

### 從莊敬街入口到韓偉大樓急診
1. 從莊敬街進入院區左下角
2. 前往大型矩形建築（韓偉大樓）
3. 1F 設有急診入口、掛號、藥局

### 從路加大樓到馬太大樓
1. 從路加大樓往左側通道
2. 連接至馬太大樓
3. 馬太大樓1F 設有轉診中心、健康管理中心

### 從馬太大樓到馬可大樓
1. 從馬太大樓往右側通道
2. 連接至馬可大樓
3. 馬可大樓1F 設有放射科、電腦斷層室

### 從約翰大樓到韓偉大樓
1. 從約翰大樓向下連接
2. 可通往韓偉大樓（急診）
