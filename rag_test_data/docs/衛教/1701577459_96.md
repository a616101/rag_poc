## 主旨  
本文件由屏基醫療財團法人屏東基督教醫院提供，旨在介紹淋巴水腫的定義、症狀、預防方法及日常照護注意事項，特別針對乳癌治療後可能面臨上肢淋巴水腫的高風險群。

## 重點整理  
### 什麼是淋巴水腫？  
淋巴水腫是因淋巴系統受損，導致淋巴循環不良，使富含蛋白質的淋巴液堆積於身體某部位，引發腫脹的現象。乳癌患者在接受手術切除淋巴結或放射治療導致淋巴組織纖維化時，常因腋下淋巴組織受損而成為上肢淋巴水腫的高風險群。  

### 淋巴水腫的徵狀  
- 肢體有腫脹及沉重的感覺。  
- 皮膚粗厚乾燥或有緊繃的感覺。  
- 關節靈活度降低，肢體活動不便，外觀變粗及腫。  
- 皮膚抵抗力變差，容易破皮。  

### 預防淋巴水腫方法  
1. 避免穿戴過緊的衣物、首飾或戒指，以免影響淋巴循環。  
2. 避免在患側提掛手提袋或提重物（超過5公斤）。  
3. 避免在患側測量血壓，因氣囊可能對患肢施加過度壓力。  
4. 避免讓患側上肢過度用力或活動。  
5. 避免使血管擴張，如日曬、蒸汽浴、紅外線及熱敷等。  
6. 避免推拿或按摩，因可能造成血管擴張，增加淋巴液產生。  
7. 義乳重量不宜過重，內衣鬆緊度應適中，避免使用鋼絲。  

### 日常照護注意事項  
1. 保護皮膚，避免傷口或感染，切勿在患側進行抽血、注射、針灸或放血。  
2. 修剪指甲時，避免剪得太短。  
3. 若患側皮膚乾裂，應在沐浴後使用乳液塗抹。  
4. 睡覺時，將患側肢體墊高，使手肘位置高於心臟。  
5. 規律進行輕度運動，如走路、游泳、輕量有氧運動、騎腳踏車、瑜伽、太極、芭蕾等，但避免手臂過度勞累。  
6. 若患肢出現紅、痛、疹子、溫度上升或顏色改變，應立即諮詢醫生。

淋巴水腫預防與照護

什麼是淋巴水腫？

淋巴水腫是淋巴系統受到破壞，導致淋巴系統循環不良，使得富含蛋白質的淋巴液堆積在身體的某部位，產生腫脹的現象。乳癌病人在接受手術治療會移除部分或全部的淋巴結或放射治療時使淋巴組織產生纖維化，會無可避免的破壞腋下的淋巴組織，往往是發生上肢淋巴水腫的高危險群。

一、淋巴水腫的徵狀

1. 肢體有腫脹及沉重的感覺。
2. 皮膚粗厚乾燥或有緊繃的感覺。
3. 關節靈活度降低，肢體活動不便，肢體外觀變粗及腫。
4. 皮膚抵抗力變差，容易破皮。

二、預防淋巴水腫方法

1. 避免穿戴過緊的衣物、首飾、戒指，因會影響淋巴循環路徑。
2. 不要在患側揹掛手提袋或提重物（＞5公斤）。
3. 不要在患側測量血壓，測量血壓的氣囊會對患肢施加過度的壓力。
4. 避免讓患側上肢過度出力或過度活動。
5. 避免使血管擴張，包括避免患側的日曬、蒸汽浴、紅外線以及熱敷等。
6. 避免推拿、按摩因屬於較激烈的深部按摩，會造成血管擴張，增加淋巴液產生。
7. 義乳之重量不要太重，內衣的鬆緊度要作適當的調整，不要有鋼絲為佳。

三、日常照護注意事項

1. 保護皮膚，避免皮膚有傷口或感染，千萬不要在患側施行抽血、注射、針灸、放血等行為，這些均會在患肢上製造出微小的傷口。
2. 修指甲時不要修剪的太短。
3. 患側皮膚若有乾裂的現象，在沐浴之後用乳液塗抹。
4. 建議睡著時，將患側肢體墊高，把手肘放在高於心臟的位置。
5. 規律做運動，如：走路、游泳、輕量有氧運動、騎腳踏車、瑜伽、太極、芭蕾等，但不要讓手臂太勞累。
6. 患肢的皮膚若有紅、痛、疹子、溫度上升或者是顏色改變時，要立即諮詢醫生。
