## 主旨  
本文件為胃腫瘤手術前後之護理指引，包含術前準備、術後注意事項及相關護理措施，以促進患者恢復並預防併發症。

## 重點整理

| 項目 | 內容 |
|------|------|
| **術前準備** | 教導正確的深呼吸與咳嗽方法，預防術後肺部併發症；進行腸道準備（如甘油球灌腸）；術前一天晚上禁食，避免吸入性肺炎；準備束腹帶以減輕傷口疼痛。 |
| **術後照護** | 每小時進行 5–10 次深呼吸與咳嗽運動；採半坐臥姿勢，保持安靜環境；使用止痛藥時需間隔 4–6 小時，避免呼吸抑制；鼻胃管用於引流胃液、腸液與空氣，預防腹脹；傷口引流管用於排出腹腔內血水，預防化膿與紅腫；引流袋應保持低於傷口位置，避免滑脫。 |
| **術後居家照護** | 包含導尿管護理、傷口照護、飲食指導及活動建議，以促進恢復並預防併發症。 |
| **飲食管理** | 進食前需等待腸蠕動恢復並排氣，避免胃部受損；飲食應循序漸進，從流質（如米湯）→ 軟質（如稀飯）→ 正常飲食；少量多餐；水份應在兩餐之間飲用；進食後應坐著或斜躺 30 分鐘，再臥床休息。 |
| **飲食禁忌** | 避免辛辣、高脂肪、高蛋白、低糖及刺激性食物；避免牛奶及奶製品；採用低殘渣、體積小、易消化的烹調方式（如蒸蛋、魚湯）。 |
| **傾倒症候群預防** | 進食後避免立即躺下；進食後應坐著或斜躺 30 分鐘。 |
| **藥物與追蹤** | 依醫師指示服用藥物；定期門診追蹤病情變化；如有身體不適，應立即返院就診。 |
| **生活調理** | 每日觀察傷口有無紅、腫、熱、痛等發炎症狀；保持傷口乾燥；安排規律生活，保持心情愉悅。 |

## 流程/步驟

#### **術前準備步驟**
1. **深呼吸運動方法**  
   - 緩慢由鼻吸氣，腹部膨出。  
   - 再緩慢由口吐氣，腹部凹陷。  
   - 持續三秒鐘，重複一次。

2. **咳嗽運動方法**  
   - 坐立姿勢，身體往前傾。  
   - 以雙手壓住傷口部位。  
   - 閉氣三秒鐘，連續咳嗽兩次。

3. **腸道準備**  
   - 依醫師指示執行，如甘油球灌腸或清潔灌腸。

4. **禁食規定**  
   - 術前一天晚上開始禁食，不可進食任何食物，包括開水、口香糖、檳榔。

5. **準備束腹帶**  
   - 手術後使用，以減輕傷口牽扯疼痛。

#### **術後注意事項**
1. **呼吸與活動**  
   - 每小時進行 5–10 次深呼吸及咳嗽運動，促進肺部擴張。

2. **姿勢與休息**  
   - 依醫師指示採半坐臥式，提供安靜環境，並適度休息，減輕傷口疼痛。

3. **止痛管理**  
   - 可注射肌肉止痛針，通常每 4–6 小時給予一次，避免藥物引起呼吸抑制。  
   - 或自費並依醫師指示使用靜脈疼痛控制藥物。

4. **鼻胃管引流**  
   - 目的：引流胃液、腸液及空氣，預防腹脹、噁心、嘔吐。  
   - 留置期間可使用清水漱口或刷牙，維持口腔衛生。  
   - 若有不適，應立即告知醫護人員。

5. **傷口引流管管理**  
   - 目的：引流手術後腹腔內殘餘的血水，預防傷口化膿、紅腫發炎。  
   - 醫師會每日評估引流液量、性質與顏色，決定拔管時間。  
   - 翻身時應避免壓迫或反折引流管。  
   - 使用固定夾妥善固定，並保持引流袋位置低於傷口，防止滑脫。

#### **術後進食流程**
1. **術後至排氣前**：不可進食任何食物或水，僅可使用棉棒沾開水潤口。
2. **腸蠕動恢復後**：由醫師評估後，可開始進食。
3. **飲食進程**：  
   - 流質（如米湯）→ 軟質（如稀飯）→ 正常飲食。
4. **進食方式**：  
   - 少量多餐。  
   - 水份應在兩餐之間飲用，避免一次攝取過多。
5. **進食後姿勢**：進食後應坐著或斜躺 30 分鐘，再臥床休息。

## 注意事項/警示

- **禁食提醒**：術前一天晚上必須完全禁食，包括水、口香糖與檳榔，以防麻醉時嘔吐導致吸入性肺炎。
- **咳嗽與呼吸訓練**：術前應學習正確的呼吸與咳嗽方法，術後持續執行，以預防肺炎與術後發燒。
- **止痛藥使用**：止痛藥需按時給予，避免因疼痛導致呼吸抑制。
- **引流管護理**：不可壓迫、反折或脫落引流管，應保持引流袋低於傷口位置，並定期檢查引流情況。
- **導尿管護理**：置管期間，護理人員每日執行尿管護理，以降低泌尿道感染風險；待醫師評估病情穩定後，方可拔除。
- **傷口照護**：保持傷口乾燥，若紗布滲濕，應立即通知護理人員更換，以防感染。
- **活動建議**：若無頭暈、身體乏力等不適症狀，可提早下床活動，以促進腸蠕動及血液循環。
- **飲食禁忌**：  
  - 避免辛辣、高脂肪、高蛋白、低糖及刺激性食物。  
  - 避免牛奶及奶製品。  
  - 採用低殘渣、體積小、易消化的烹調方式（如蒸蛋、魚湯）。
- **傾倒症候群預防**：  
  - 進食後避免立即躺下。  
  - 進食後應坐著或斜躺 30 分鐘。
- **藥物與追蹤**：  
  - 依醫師指示服用藥物。  
  - 定期門診追蹤病情變化。  
  - 如有身體不適，應立即返院就診。
- **生活調理**：  
  - 每日觀察傷口有無紅、腫、熱、痛等發炎症狀。  
  - 保持傷口乾燥。  
  - 安排規律生活，保持心情愉悅。

一、手術前準備及注意事項

1. 教導正確的深呼吸、咳嗽，預防術後怕痛而不敢做深呼吸、咳嗽動作併發術後相關併症，如肺炎或術後發燒。

2. 深呼吸運動的方法：緩慢由鼻吸氣，腹部膨出，再緩慢由口吐氣，腹部凹陷，並閉氣三秒鐘。

3. 咳嗽運動的方法：坐立姿勢，身體往前傾，以雙手壓住傷口部位，開始慢慢吸氣，閉氣三秒鐘，連續咳嗽二次。

4. 教導您在床上使用尿壺及便盆。

5. 依醫師指示必要時執行腸道準備，如甘油球灌腸或清潔灌腸。

6. 依醫師指示於手術前一天晚上開始禁食，不可吃任何東西包括開水、口香糖、檳榔，預防麻醉時嘔吐造成吸入性肺炎。

7. 準備束腹帶於手術後使用，以減輕傷口牽扯疼痛。

二、手術後注意事項

1. 每小時做 5~10 次深呼吸及咳嗽運動，以利肺部擴張。

2. 依醫師指示採半坐臥式及提供安靜的環境，並適度的休息，減輕傷口疼痛。

3. 手術後傷口疼痛可注射肌肉止痛針，通常二次止痛劑給予的時間需要相隔 4~6 小時，以免藥物引起呼吸抑制等副作用，或自費並依醫師指示予裝置靜脈疼痛控制藥物。

4. 手術後放置鼻胃管引流：目的是引流胃液、腸液及空氣，預防腹脹、噁心、嘔吐，醫師會評估每日引流量及腸蠕動情形而決定拔管時間，若有不適請告知醫護人員，鼻胃管留置期間可用清水漱口或刷牙，以維持口腔的衛生。

5. 傷口放置引流管：目的是引流手術後腹腔內殘餘的血水，預防傷口化膿、紅腫發炎，醫師會評估每日引流液量、性質、顏色，決定拔管時間。翻身時避免壓迫或反折到引流管，傷口引流管會有固定夾，可作適當的固定，避免發生管路滑脫，應將引流袋保持低於傷口的位置。

6. 導尿管：導尿管放置期間，護理人員會每天執行尿管護理，以降低發生泌尿道感染，待醫師評估病情穩定，即可拔除尿管。

7. 手術後保持傷口乾燥很重要，如果傷口紗布滲濕，請告知護理人員換藥，以預防傷口感染。

8. 進食：手術後至排氣前，不可以吃任何食物、喝水，以免胃部傷口受到損害及腹脹不適，如果覺得口乾，可以棉棒沾開水潤口。當腸蠕動排氣後，醫師將告訴您可以進食那些種類的餐點。

9. 手術後如無特別的頭暈、身體乏力等不適症狀，可提早下床活動，以幫助促進腸蠕動的恢復及活絡全身血液循環暢通。

三、術後居家照護注意事項

1. 手術後進食有可能發生「傾倒症候群」，吃入的食物未經胃適當的消化、稀釋，就快速的進入空腸，會出現腹部疼痛痙攣、飽脹感、噁心、嘔吐等不適現象。飲食注意事項：

(1) 食物調配須採循序漸進方式，由清淡而濃稠，由流質（米湯）→軟質（稀飯）→正常飲食。

(2) 須少量多餐。

(3) 不要喝過多的水（含湯），水份應在兩餐之間飲用。

(4) 應避免刺激性的飲食，如辛辣食物，而以低殘渣、體積小、易消化、高蛋白、適量脂肪、低糖、溫和性的飲食為原則（烹調方式例如：蒸蛋、魚湯）。

(5) 應避免牛奶及奶製品食物之攝取。

(6) 進食時應細嚼慢嚥，採端坐姿勢。

(7) 進食後坐著或斜躺 30 分鐘才可臥床休息，以減輕不適感及減少「傾倒症候群」發生。

2. 每日觀察傷口有無紅、腫、熱、痛等發炎症狀，並每日清潔消毒傷口，保持傷口乾燥。

3. 安排規律的生活步調，並保持心情愉悅。

4. 依照醫師的指示服用藥物治療，並定期門診追蹤病情變化，如有身體不適，應立即返院就診。

屏基關心您 祝您早日康復
