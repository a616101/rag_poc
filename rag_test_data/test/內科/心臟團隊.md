# 心臟團隊

## 團隊簡介
本院心血管內科擁有6位專科心臟科醫師，配備2間心導管室，提供心血管疾病診斷與治療服務。多數心導管採經由橈動脈執行，冠狀動脈介入治療大幅降低術後併發症，讓病患術後可自由活動。心導管成功率極高，通過急性冠心症認證。自成立以來，本科致力於提供以病患為中心的服務，引進先進硬體設備如高速鑽石旋磨器、血管內超音波、彩色3D超音波、運動心電圖機，並整合「心臟保護守衛」通知群組，提升急性冠心症病患照護品質，縮短緊急心導管時間，達到國際標準。整合胸痛中心後，與心臟血管外科ECMO團隊合作，大幅提升急性冠心症合併心因性休克病患的存活率。24小時心導管治療服務，拯救許多生命。本科積極提供最佳醫療技術與品質，引進新設備，並積極參與透析廔管經皮血管成形術教育，教導其他醫院心臟科醫師如何治療阻塞的透析廔管。

## 團隊編制
- 醫師：專科心臟科醫師6名
- 護理/照護：專業護理團隊
- 醫事技術/其他：心導管室技術人員
- 特色分工：與心臟血管外科ECMO團隊、胸痛中心整合

## 服務與治療
### 服務範圍（疾病/問題）
- 心血管疾病
- 急性心肌梗塞
- 周邊血管疾病
- 心律不整
- 心臟衰竭
- 心室顫動

### 主要治療/處置
- 24小時緊急心導管治療：提供屏東地區急性心肌梗塞病患及時服務
- 一般心導管及治療（含氣球擴張術、支架置放術）：結合血管內超音波（IVUS）、高速鑽石旋磨器，確保更精準的心血管治療與病患安全
- 周邊血管疾病治療：頸動脈血管攝影及介入治療、透析廔管氣球擴張術及血栓清除術、周邊動脈狹窄介入治療及支架置放、男性血管性勃起功能障礙介入治療
- 心律不整治療：電生理檢查、射頻燒灼術、心律調節器植入、植入式心臟整流去顫器（ICD）植入、心臟再同步化治療（CRT）植入
- 葉克膜（ECMO）支援：2組ECMO、1組PCPS，配合治療性低溫，確保病患安全

### 特色門診/照護計畫（若有）
- 急性冠心症快速通道：整合「心臟保護守衛」通知群組，縮短緊急心導管時間
- 胸痛中心整合：與心臟血管外科ECMO團隊合作，提升急性冠心症合併心因性休克病患存活率
- 早期治療服務：結合放射科的64切電腦斷層掃描及核子醫學的鉈-201心肌灌注掃描

## 檢查項目
- 心導管檢查：血管攝影、冠狀動脈氣球擴張術、支架置放術、周邊血管氣球擴張術、電生理檢查、射頻燒灼術、心律調節器植入
- 超音波檢查：心臟超音波、肢體血流超音波、經食道心臟超音波
- 心電圖檢查：一般心電圖、運動心電圖、24小時霍特式心電圖
- 周邊血管：分段動脈壓力測量、血流超音波、頸動脈超音波
- 心臟功能評估：運動心電圖、24小時霍特式心電圖、周邊血管超音波、深部靜脈血栓檢查、經胸心臟超音波、經食道心臟超音波、24小時血壓監測

## 設備與特色
- 心導管室2間：提供24小時緊急主要經皮冠狀動脈介入治療（Primary PTCA）
- 葉克膜心肺氧合器（ECMO）：2組ECMO、1組PCPS
- 血管內超音波（IVUS）：2台，提供更精準的診斷與治療
- 高速鑽石旋磨器（ROTABLATOR）：1台，處理鈣化病灶
- 血流儲備分數儀（FFR）：1台，評估血管功能
- 二氧化碳血管攝影儀：1台
- 西門子血行動力監測儀：2台
- 波科電生理刺激器：1台
- 心臟超音波：含都卜勒及彩色都卜勒心臟超音波、經食道心臟超音波、頸動脈及周邊血管超音波
- 永久性心臟節律器（PPM）、植入式心臟整流去顫器（ICD）、心臟再同步化治療（CRT）
- 一般心電圖、運動心電圖、霍特式二十四小時連續心電圖、手腕式連續心電圖、植入式心電圖記錄器（ILR）
- 核子醫學檢查
