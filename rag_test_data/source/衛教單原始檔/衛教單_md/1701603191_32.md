# 1701603191_32

## 主旨  
本文件說明陰道乳劑的正確使用方法，提供詳細步驟以確保用藥安全與有效性。

## 重點整理  
- 使用前後應徹底清潔雙手。  
- 正確裝填藥膏並注意劑量。  
- 以舒適姿勢將給藥器置入陰道。  
- 注入藥膏後保持給藥器壓緊狀態。  

## 流程/步驟  
1. 使用前後，用肥皂及清水充分清潔雙手。  
2. 打開藥管蓋子，將給藥器裝上管口。  
3. 擠壓軟管，將乳劑藥膏充填到給藥器內，注意充填的劑量。  
4. 分開藥管與給藥器。  
5. 人躺下，小腿弓起，膝蓋朝上，輕輕將給藥器前端伸入陰道內，以舒適情況為原則，盡可能深入。  
6. 慢慢擠壓給藥器，將乳劑軟膏注入陰道內。給藥完全後，保持給藥器壓緊的狀況。  

## 注意事項/警示  
- 請確保藥膏完全注入陰道內。  
- 使用後請保持給藥器壓緊，以確保藥物充分作用。

陰道乳劑的正確使用方法

1. 使用前後，用肥皂及清水充分清潔雙手。
2. 將藥管的蓋子打開，給藥器裝上管口。
3. 擠壓軟管，將乳劑藥膏充填到給藥器內，注意充填的劑量。
4. 分開藥管與給藥器。
5. 人躺下，小腿弓起，膝蓋朝上，輕輕將給藥器前端伸入陰道內，舒適情況，愈進去愈好。
6. 慢慢擠壓給藥器，將乳劑軟膏注入陰道內。在給藥完全後，保持給藥器壓緊的狀況。
