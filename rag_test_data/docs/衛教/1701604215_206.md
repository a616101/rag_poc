# 1701604215_206

## 主旨  
輕鬆預防腸道疾病，保健康。

## 重點整理  
- 每日飲食應包含「3蔬2果」，有助腸道健康。  
- 每日攝取足夠水分，建議每公斤體重30c.c.。  
- 蛋白質攝取以白肉為主，如雞肉、魚肉、豆製品。  
- 烹調方式應以蒸、煮、燉、滷為主，避免油炸及燒烤。

## 流程/步驟  
1. **每天吃3蔬2果**  
   - 有助於腸內益菌生長，增加糞便體積，促進腸道蠕動。  
2. **攝取足夠水分**  
   - 建議每日每人每公斤體重攝取30c.c.的水，以增加體內代謝及腸道蠕動。  
3. **優先選用白肉**  
   - 蛋白質攝取建議以白肉為主，例如雞肉、魚肉類、豆製品等。  
   - 避免選用加工肉品，例如香腸、火腿、培根等。  
4. **烹調方式以蒸、煮、燉、滷為主**  
   - 避免油炸及燒烤，以減少腸道負擔。

## 注意事項/警示  
- 避免食用加工肉品，如香腸、火腿、培根等。  
- 避免油炸及燒烤的烹調方式。
