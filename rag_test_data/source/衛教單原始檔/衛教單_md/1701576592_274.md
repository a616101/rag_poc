# 1701576592_274

## 主旨
本文件為屏基醫療財團法人屏東基督教醫院所訂定的腹膜透析導管及輸液管安全照護指引，旨在提供患者正確的導管護理方法，以降低感染與併發症風險。

## 重點整理
- 每日洗澡後應立即進行導管出口護理。
- 導管應妥善固定，避免拉扯或扭轉。
- 嚴禁在導管附近使用剪刀，以防意外斷裂。
- 避免在導管出口周圍抓癢、塗抹藥物或泡澡。
- 如發生意外，應立即聯繫醫療單位。

## 流程/步驟
- 每日洗澡後立即執行導管出口護理，以環狀方式由內往外消毒。
- 使用膠帶將腹膜透析導管固定，防止拉扯或扭轉。
- 切勿拉扯或扭轉導管，以免造成損傷。
- 絕對不可在導管出口附近使用剪刀，以防剪斷導管，導致細菌入侵引發腹膜炎。
- 切勿在導管出口處周圍抓癢。
- 不可在導管出口處或周圍塗抹爽粉、乳液、藥膏或酒精。
- 切勿在導管出口處繫腰帶。
- 應避免泡澡，建議以淋浴方式清潔。

## 注意事項/警示
1. 每日洗澡後應立即執行導管出口護理，以環狀方式由內往外消毒。
2. 使用膠帶將腹膜透析導管固定，防止拉扯或扭轉。
3. 切勿拉扯或扭轉導管，以免造成損傷。
4. 絕對不可在導管出口附近使用剪刀，以防剪斷導管，導致細菌入侵引發腹膜炎。若發生此情況，應立即撥打公務手機（0988-289358）並返回透析中心換管。
5. 切勿在導管出口處周圍抓癢。
6. 不可在導管出口處或周圍塗抹爽粉、乳液、藥膏或酒精。
7. 切勿在導管出口處繫腰帶。
8. 應避免泡澡，建議以淋浴方式清潔。

## 全文

屏基醫療財團法人屏東基督教醫院

腹膜透析導管及輸液管安全照護原則

1. 每天洗澡完後立刻執行導管出口護理。在腹膜透析導管周圍的皮膚以環狀方式，由內往外消毒。使用指定的清潔方式消毒（圖 1）。

2. 請將腹膜透析導管固定好，使用膠帶貼好，可避免腹膜透析導管被拉扯或扭轉（圖 2）。

3. 不可以拉扯或扭轉腹膜透析導管（圖 3）。

4. 絕對不要在導管出口處附近使用剪刀，一不小心就有可能剪到導管而斷裂，因此讓細菌將會入侵至腹膜腔內導致腹膜炎，如發生這種事情時，請立刻撥打公務手機（0988-289358）後返回透析中心進行換管（圖 4）。

5. 不要在導管出口處周圍抓癢（圖 5）。

6. 不要在導管出口處或周圍塗抹爽粉、乳液或藥膏、酒精等（圖 6）。

7. 不要在導管出口處繫腰帶（圖 7）。

8. 不要泡澡（最好是淋浴）（圖 8）。
