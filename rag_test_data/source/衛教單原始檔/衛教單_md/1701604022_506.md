# 1701604022_506

## 主旨
屏東基督教醫院與7-ELEVEN合作推出「及早救智」公益計畫，自2015年起，結合門市通路資源與基層醫療網診所，推動失智症的認識、早期發現與多元照顧，幫助長輩活出健康自在的快樂晚年。本活動提供21份健康餐食組合，鼓勵透過均衡營養飲食降低失智風險。台灣每12位長者就有1位患有失智症，但根據研究，只要遵循地中海飲食原則，並搭配新鮮的本土食材，就能有效降低40%的失智風險。本計畫提供星期一至星期日的三餐建議，每餐份量為兩人份，並透過便利選購超商食材，讓冰箱空空、不想開伙或出門在外時，也能吃得營養均衡。

## 重點整理
- **活動名稱**：爸媽防失智 跟著營養師這樣吃
- **活動內容**：收錄21份健康餐食組合（1週 × 7天 × 3餐）
- **參與單位**：屏東基督教醫院、7-ELEVEN
- **活動目的**：透過均衡營養飲食，降低4成失智風險，提升長輩生活品質。
- **參與方式**：限量免費索取，可於7-ELEVEN公益DM架（107/2/1～2/28）領取。
- **飲食原則**：遵循地中海飲食原則，多攝取蔬果、魚類、家禽、奶蛋、豆類，並搭配新鮮本土食材。
- **營養均衡**：每餐應包含五穀雜糧、蔬果、蛋白質來源，並適量攝取甜點與加工食品。
- **適量咖啡**：適量飲用咖啡有助增加短期記憶力和認知功能。
- **餐食組合**：本週餐食可自由調換，每餐為2人份，食譜可依個人喜好調整，但需注意營養均衡。
- **食材取得**：部分餐食或食材可就近在7-ELEVEN取得。
- **營養重點**：強調抗氧化食物（如花椰菜）、維生素C的攝取，以及低鹽、低脂飲食原則。
- **特殊提醒**：長者可適度攝取甜點，但應以「充足」與「均衡」為原則，避免造成健康負擔。
- **營養重點**：富含DHA的深海魚類、優質蛋白質與不飽和脂肪酸，有助於腦部健康與心血管保護。
- **營養重點**：強調全穀雜糧的膳食纖維，有助促進腸道蠕動與降低膽固醇；咖哩中的薑黃素具有抗氧化與預防失智的功效。
- **營養重點**：強調每天攝取五彩蔬菜，特別是富含β-胡蘿蔔素、維生素C與E的蔬果，如洋蔥、番茄、花椰菜、紅蘿蔔、甜椒與菇類。
- **營養提示**：橄欖油富含單元不飽和脂肪酸，有助降低膽固醇，預防血管型失智症。
- **營養提示**：建議以五穀雜糧（如糙米、燕麥、地瓜、芋頭、薏仁）為主食，以攝取豐富的維生素、礦物質與膳食纖維。

## 流程/步驟
### 韓式泡菜豆腐鍋製作步驟
1. 將水煮開後，放入泡菜等鍋底食材煮滾。
2. 依喜好將配料陸續放入鍋中煮熟即可。
3. 搭配紫米飯食用。

### 野菇雞肉燉飯製作步驟
1. 食材洗淨後，將雞腿肉切塊、綠花椰菜切塊、香菇切塊、紅蘿蔔切絲、洋蔥切小丁。
2. 煮熟白飯、燙熟花椰菜備用。
3. 冷油下鍋，先將洋蔥炒香，再放入雞腿肉、香菇、紅蘿蔔翻炒。
4. 加入水、鮮奶、義式香料、鹽巴和飯，以小火燉煮攪拌。
5. 煮成稠狀後即可盛盤，撒上杏仁果和綠花椰菜。

### 日式炒板條製作步驟
1. 將洋蔥、紅蘿蔔切絲，青江菜切段後備用。
2. 熱鍋後加入芥花油，將洋蔥、紅蘿蔔及豬絞肉爆香。
3. 加入青江菜、板條及低鹽醬油拌炒均勻。
4. 盛盤時撒上柴魚片及腰果即可。

### 核桃吻仔魚鮮蔬拌飯製作步驟
1. 將食材洗淨後，高麗菜切段、甜椒切丁備用。
2. 鍋中倒入香油，中火拌炒吻仔魚、高麗菜、甜椒至半熟。
3. 將步驟2的配料平鋪於糙米上並加水，放置電鍋煮至熟透。
4. 取出後，將食材與糙米拌勻，再撒上核桃碎即可食用。

### 野菜咖哩湯拉麵作法
1. 將鴻喜菇根部切掉，用手剝成小塊；洋蔥切薄片；大番茄切塊；秋葵斜切對半；高麗菜切塊；青蔥切丁備用。
2. 平底鍋開中火，放入洋蔥拌炒至變金黃色，加入咖哩粉和水拌勻。
3. 將備好的蔬菜類加入鍋中煮滾，再加入拉麵。
4. 食材熟透後，打入雞蛋煮熟，再以味醂、醬油調味。
5. 盛碗時撒上蔥花。

## 注意事項/警示
- 失智症是不正常的老化，指大腦退化速度超過正常老化，引發認知功能和行為障礙。
- 若發現長輩出現上述任何警訊，應儘早諮詢專業人員。
- 建議定期進行健康檢查，並關注長輩的飲食與生活習慣。
- **熱量計算**：每日所需熱量可依身體基礎代謝率計算，並根據勞力程度調整。
- **年齡影響**：每10年，基礎代謝率平均降低2-5%。
- **建議**：30歲後應特別注意基礎代謝率的變化。
- **堅果攝取建議**：
  - 應選擇新鮮、無調味、無發霉的堅果。
  - 建議適量攝取，避免過量造成身體負擔。
  - 堅果富含蛋白質、膳食纖維、維生素與礦物質，其油脂主要為單元不飽和脂肪酸，有助於減少血脂異常。
- **關東煮食用建議**：營養師提醒，吃關東煮時建議少喝湯，以降低鈉攝取；可選擇「蒸」的關東煮，保留更多食材營養與原味。
- **食材選擇**：建議選擇低脂、低糖、高纖的食品，如低脂鮮奶、無調味核桃、低醣高纖豆漿等。
- **餐食組合**：本餐食組合皆為2人份。
- **七天餐食**：七天餐食可自由調換。
- **食譜調整**：食譜可依個人喜好調整，唯請注意營養均衡。
- **食材取得**：為便利食材採買，部分餐食或食材可就近在超商取得。
