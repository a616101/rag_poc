## 主旨  
本文件為屏東基督教醫院麻醉科提供的術後麻醉恢復室照護指引，說明手術後患者在恢復室的照護重點，包括疼痛管理、生命徵象監測、麻醉類型的照護需求，以及病患自控式止痛的重要性。

## 重點整理  
- 手術後患者將持續在麻醉恢復室接受照護，並進行與麻醉中相同的生理監測。  
- 疼痛是術後恢復期最常見的問題，通常使用強效嗎啡類止痛藥處理。  
- 可能出現的併發症包括高血壓、低血壓、心律不整、顫抖，極少數情況下可能發生呼吸困難、意識不清或嚴重電解質異常。  
- 所有接受全身麻醉、區域麻醉或靜脈麻醉的患者，均需在恢復室至少停留一小時。  
- 患者需在血液循環、呼吸、意識及體溫均恢復正常後，方可轉回病房。  
- 病患自控式止痛（PCA）提供患者自主控制止痛的方式，是現代人道醫療的重要進步。  

## 流程/步驟  
1. 手術後患者進入麻醉恢復室，持續接受生理監測。  
2. 醫護人員密切監控生命徵象，並針對疼痛、血壓異常、心律不整等症狀進行處理。  
3. 若出現嚴重併發症（如呼吸困難、意識不清），需立即轉至加護病房。  
4. 患者在恢復室至少停留一小時，直至：  
   - 血液循環穩定  
   - 呼吸正常  
   - 意識清楚且穩定  
   - 體溫正常  
5. 滿足上述條件後，患者可轉回病房。  

## 注意事項/警示  
- 患者應理解術後疼痛是正常現象，但不應忍耐，應善用「病患自控式止痛」系統。  
- 疼痛未妥善控制可能導致躁動、血壓上升，增加身體器官負擔。  
- 若出現呼吸困難、意識不清或嚴重電解質異常，需立即處理，必要時轉至加護病房。

# 術後麻醉恢復室

恢復室照顧：對病患手術後的照顧持續到恢復室，在恢復室中有和麻醉中一樣的生理監測器。

1、疼痛：麻醉後的恢復期間最常處理的是疼痛，常以強效的嗎啡止痛藥暫時止痛。也可能出現高血壓或低血壓、心律不整、顫抖等情形；極少部份會發生呼吸困難、發紺、意識不清、嚴重電解質異常等情形，本科將嚴密監控並在發現之第一時間積極處理，若非短期內可恢復時，則要轉至加護病房照顧。

2、無論是全身麻醉、區域麻醉或靜脈麻醉皆需接受恢復室照護，在恢復室至少停留一小時，當病患血液循環及呼吸恢復正常、意識清楚穩定、體溫正常就可回病房。

3、病患自控式止痛的必要性：以往手術後疼痛看作必然現象，要患者忍一忍這是不合生理及人道醫療。因疼痛會躁動、血壓上升、增加身體器官的負擔、現在有了「病患自控式止痛」，不只是手術患者的一個福音也給病人一個人道的自我選擇。
