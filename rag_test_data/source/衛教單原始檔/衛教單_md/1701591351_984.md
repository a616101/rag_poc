# 1701591351_984

## 主旨
屏基醫療財團法人屏東基督教醫院提供的腹腔鏡膽囊摘除術後照護指引，包含術後注意事項與返家後的護理建議。

## 重點整理
- 術後需禁食至排氣，並依醫囑漸進式進食。
- 重視肺部與下肢併發症預防，包括深呼吸、拍背、翻身及早期活動。
- 使用止痛藥及束腹帶以減輕疼痛。
- 多數患者術後可能有輕微發燒，通常在1至2天內消失。
- 返家後飲食無絕對禁忌，但應避免油膩食物以防腹瀉。
- 保持傷口乾燥，注意是否有感染跡象。
- 若有引流管，需每日記錄引流量與顏色，並依時複診。

## 流程/步驟
1.  **術後禁食與進食**：
    -   手術後需禁食，包括食物及開水。
    -   排氣後，依醫生指示進食。
    -   進食方式：先嘗試喝開水 → 運動飲料、蜜茶、無渣果汁 → 清淡、易消化的軟質食物（少量多餐）。

2.  **預防併發症**：
    -   **肺部併發症**：術後每2小時做深呼吸運動、拍背及翻身，以促進痰液排出，避免肺擴張不全。
    -   **下肢血栓**：手術當天即可下床活動，臥床時仍須翻身，以促進腸道蠕動。

3.  **疼痛管理**：
    -   醫師會開立口服止痛劑以減緩傷口疼痛。
    -   需用力時（如咳嗽、翻身、下床），應用手輕壓傷口，並可使用束腹帶以減輕疼痛。

4.  **術後返家注意事項**：
    -   **飲食**：可進食各式肉類、水果及蔬菜，但應避免油膩食物以防腹瀉。
    -   **傷口護理**：保持傷口乾燥，若有滲出液應立即更換敷料，避免感染。
    -   **引流管護理**：若出院時有引流管和引流袋，應每天記錄引流量和顏色。正常顏色為淡黃色，若顏色改變，應立即就診。
    -   **複診**：請依醫師囑咐時間至門診複診。
    -   **異常情況**：若傷口出現異味、紅腫、發燒，或有其他不適，應立即返回門診就診。

## 注意事項/警示
-   **發燒**：少部分病人術後可能有輕微發燒（體溫在攝氏37~38度之間），通常在1至2天內可消失。
-   **感染**：若傷口有異味、紅腫、發燒等感染跡象，應立即就診。
-   **引流管**：若引流管顏色改變，應立即就診。

一、膽囊切除術後注意事項

1. 手術後需禁食，包括食物及開水，須等到排氣後依醫生指示再進食。
2. 排氣後需醫囑採漸進式進食，一般可先嘗試喝開水，若無不適情形可嘗試運動飲料、蜜茶、無渣果汁；再改為清淡、易消化軟質食物，採少量多餐方式進食。
3. 預防肺部合併症，如肺炎、肺部擴張不全，術後每 2 小時做深呼吸運動、拍背及翻身，以促進痰液排出，避免肺擴張不全。
4. 預防下肢血塊栓塞，手術後當天即可下床活動，臥床時仍須翻身，以促進腸道蠕動。且 3~6 天後即可完全活動，沒有限制。
5. 手術後醫師會有口服止痛劑以減緩傷口的疼痛，如無法緩解疼痛，請告知護理人員。
6. 需要用力時（如咳嗽、翻身、下床…等），需用手輕壓傷口，以減輕疼痛，可使用束腹帶以減輕下床或咳嗽時所造成的疼痛。
7. 少部份病人於手術後可能有輕微發燒（體溫在攝氏 37~38 度之間），在 1~2 天可消失。

二、術後返家注意事項

1. 飲食無絕對禁忌，各式肉類、水果及蔬菜皆可進食，不可食油膩食物以免發生腹瀉。
2. 返家後，保持傷口乾燥，若有滲出液，應立即更換敷料，避免傷口感染。
3. 拆線後即不需換藥，若傷口有異味、紅腫、發燒…等情形，應立即返回門診，接受醫師診治。
4. 若出院時腹部有引流管和引流袋，請每天記錄引流量和顏色，正常顏色為淡黃色，若有顏色改變，請隨時到本院急診室或門診就診。
5. 請依醫師囑咐時間至門診複診。
