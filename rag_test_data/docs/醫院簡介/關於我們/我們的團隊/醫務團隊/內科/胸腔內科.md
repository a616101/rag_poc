# 胸腔內科

## 團隊簡介
本科成立於民國八十年，目前編制有5位醫師。本科設有：支氣管鏡室、肺功能室，皆由專業醫護技術人員負責，並成立胸腔衛教室，由專業衛教師負責；肺復原中心，由呼吸治療師協助肺部復健；病房部分有成人重症加護病房、呼吸照護中心、呼吸照護病房及一般胸腔病房，由胸腔專科醫師及專業呼吸治療師及護理師負責照顧病人。本科主要診療胸腔呼吸系統疾病之病人。

## 團隊編制
- 醫師：胸腔專科醫師5位
- 護理/照護：專業護理師負責照顧病人
- 醫事技術/其他：專業醫護技術人員負責支氣管鏡室、肺功能室
- 特色分工：專業衛教師負責胸腔衛教室、呼吸治療師協助肺部復健

## 服務與治療
### 服務範圍（疾病/問題）
- 胸腔呼吸系統疾病

### 主要治療/處置
- 支氣管鏡檢查：由專業醫護技術人員負責
- 肺功能檢查：由專業醫護技術人員負責
- 肺部復健：由呼吸治療師協助

### 特色門診/照護計畫（若有）
- 胸腔衛教：由專業衛教師負責
- 肺部復健：肺復原中心由呼吸治療師協助

## 檢查項目
- 支氣管鏡檢查
- 肺功能檢查

## 設備與特色
- 支氣管鏡室：由專業醫護技術人員負責
- 肺功能室：由專業醫護技術人員負責
- 胸腔衛教室：由專業衛教師負責
- 肺復原中心：由呼吸治療師協助肺部復健
- 成人重症加護病房：由胸腔專科醫師及專業呼吸治療師及護理師負責照顧病人
- 呼吸照護中心：由胸腔專科醫師及專業呼吸治療師及護理師負責照顧病人
- 呼吸照護病房：由胸腔專科醫師及專業呼吸治療師及護理師負責照顧病人
- 一般胸腔病房：由胸腔專科醫師及專業呼吸治療師及護理師負責照顧病人

## 品質與安全
- 專業醫護技術人員負責支氣管鏡室、肺功能室
- 專業衛教師負責胸腔衛教室
- 呼吸治療師協助肺部復健
- 由胸腔專科醫師及專業呼吸治療師及護理師負責照顧病人

## 就醫資訊
- 門診：提供一般門診服務
- 住院：收治胸腔呼吸系統疾病病患，包括成人重症加護病房、呼吸照護中心、呼吸照護病房及一般胸腔病房
- 急診/24 小時：提供會診服務
- 轉介：可接受院內外轉介
