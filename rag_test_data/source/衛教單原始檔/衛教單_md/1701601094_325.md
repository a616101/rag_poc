# 1701601094_325

## 主旨
本文件為屏東基督教醫院婦產科提供的衛教資料，說明陰道炎的常見類型、症狀及預防措施，並提醒女性應注意白帶的變化，以早期發現異常並及時就醫。

## 重點整理
- 陰道分泌物（俗稱「白帶」）是女性正常的生理現象，通常為無色、無味、量少。
- 當白帶出現顏色、氣味、質地或量的異常變化時，可能表示存在陰道炎或其他婦科疾病，需提高警覺。
- 常見的異常白帶類型包括：無色透明、乳白色豆腐渣狀、膿性、含血性及水樣白帶。
- 避免穿著緊身內衣褲、保持局部乾燥、勿使用陰道沖洗液，可降低感染風險。
- 不應濫用抗生素或自行灌洗陰道，以免破壞陰道正常菌叢，導致念珠菌感染。
- 平時可透過飲用優酪乳等含乳酸菌的食品，幫助維持陰道健康。

## 流程/步驟
1.  **觀察白帶變化**：注意白帶的顏色、氣味、質地及流量。
2.  **注意症狀**：如出現外陰瘙癢、灼痛、異味或出血等不適。
3.  **避免自行處理**：勿濫用抗生素或自行灌洗陰道。
4.  **就醫檢查**：若發現異常，應立即就醫，由專業醫師診斷並治療。

## 注意事項/警示
- **勿自行診斷**：白帶異常可能為多種疾病，需由醫師確診。
- **避免陰道沖洗**：會破壞陰道內的酸性環境，增加感染風險。
- **勿濫用抗生素**：可能殺死有益菌，導致念珠菌過度生長。
- **注意個人衛生**：保持外陰乾燥清潔，穿著通風透氣的棉質內衣。
- **定期檢查**：特別是對於有子宮頸糜爛、子宮肌瘤等病史的女性。

## 表格/比較
| 白帶類型 | 特徵 | 可能原因 | 建議 |
| :--- | :--- | :--- | :--- |
| **無色透明白帶** | 量多，無色，無味 | 子宮頸糜爛、卵巢功能失調、停經後雌激素缺乏、陰道萎縮 | 若無其他不適，可觀察；若持續或有症狀，應就醫 |
| **乳白色豆腐渣狀白帶** | 外陰瘙癢、灼痛感 | 念珠菌感染（黴菌性陰道炎） | 需就醫治療，避免自行用藥 |
| **膿性白帶** | 黃色或綠色，有異味，可能有泡沫 | 化膿性細菌感染（如滴蟲性陰道炎） | 需就醫治療，避免傳染 |
| **含血性白帶** | 性生活後出血 | 惡性腫瘤、子宮頸糜爛、子宮息肉、子宮頸癌等 | **立即就醫**，需排除惡性病變 |
| **水樣白帶** | 量多，稀薄 | 病變組織壞死或變性（如子宮黏膜下肌瘤脫出） | 需就醫檢查，確定病因 |

屏基醫療財團法人 屏東基督教醫院
PINTUNG CHRISTIAN HOSPITAL

婦科最常見的疾病—陰道炎

修訂日期：西元2018年04月
單位/作者：婦產科/劉德森醫師
備教編號：MD-婦產科-002

文／院長室發展組
審稿／婦產科主任劉德森

進入夏季，濕熱的氣候，讓許多病菌有機可乘，也是陰道炎病例上升的時節，呼籲婦女朋友避免穿著緊身內衣褲以保持局部乾燥且勿使用陰道沖洗液來灌洗，注意個人衛生清潔可減少感染的機會。

陰道分泌物增多或俗稱的白帶，是很困擾女性的問題，也是婦產科門診中常見的疾病。凡是女性都有陰道分泌物，統稱為「白帶」，白帶是從女性生殖器官各部位分泌出來的黏液及滲出物混合而成，生理情況下分泌物使陰道長期處於溼潤，了解白帶的來處之後，會知道沒有白帶，反而是不正常的，而這些液體量增多時，自陰道排出，即是生理性白帶，所以正常的白帶是應該是無色、無異味的分泌物。異常的白帶，通常帶有色有味有血，還伴隨其他不適，一定要提高警覺。

以下就陰道炎的幾種情況加以說明：

一、無色透明的白帶：與正常陰道中的分泌液較難區別，分泌量增多，常見於子宮頸糜爛、卵巢功能失調主要是停經後雌激素缺乏所致。陰道逐漸萎縮、變薄、乳酸桿菌減少，症狀有水樣惡臭分泌物或應用雌激素等藥物造成。

二、乳白色分泌物與豆腐渣狀白帶：是念珠菌感染即黴菌性陰道炎所特有的一種白帶，常有外陰瘙癢或灼痛感。由於念珠菌的感染與身體免疫系統抵抗力有關，所以包括孕婦、糖尿病人、肥胖、正在服用抗生素、使用口服避孕藥的人及正值發育的青少女較容易感染到。由於念珠菌較易在溫潮濕的環境中成長，避免穿著緊身內衣褲以保持局部乾燥可以減少感染的機會。

三、膿性白帶：由化膿性細菌感染所引起，多見老年性陰道炎、陰道內異物、慢性子宮頸炎、子宮內膜炎和子宮積膿的病人。若陰道分泌物呈黃色、膿樣兼有泡沫，且外陰瘙癢時，應考慮為「滴蟲性陰道炎」。

四、含血性白帶：有性接觸時出血，首先要排除惡性腫瘤的可能性，大都有重度子宮頸糜爛、子宮頸息肉、子宮頸癌等，需提高警覺。

五、水樣白帶：往往是由於病變組織壞死或變性所引起，如子宮黏膜下肌瘤脫出到陰道內。

陰道會分泌白色黏液呈酸性，由陰道乳酸桿菌負責維護，因此平時多食用乳酸菌食品如優酪乳，可幫助陰道健康。提醒婦女朋友正常的白帶，特別是在排卵及月經前後，是不用治療的，若亂投藥，如濫服抗生素，亂殺細菌，反而會引起念珠菌乘虛而入，若亂灌洗陰道，除了破壞原本的陰道生態，造成惡菌驅逐良菌，或把外來的病原，沖入骨盆腔內造成骨盆腔發炎，不得不注意。婦女平日應檢視其性質、流量及顏色的改變並注意個人衛生清潔，如有任何不適，應立即去看醫師，以免小病耽擱而成大病。
