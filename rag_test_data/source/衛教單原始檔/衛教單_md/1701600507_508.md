# 1701600507_508

## 主旨  
本文件說明正常體溫範圍、不同年齡層的體溫測量方法與注意事項，並提供具體操作步驟，以確保體溫測量的準確性與安全性。同時，針對兒童發燒時的特殊情況提供處理指引，包括不同體溫範圍的對應措施、注意事項與應急處理方式。

## 重點整理  
- 正常體溫範圍：36.5°C ~ 37.5°C。  
- 根據年齡選擇合適的體溫測量方式：耳溫或腋溫。  
- 測量時應讓寶寶處於安靜狀態，避免誤差。  
- 洗澡後需等待至少30分鐘至1小時再測量體溫。  
- 體溫低於 36.5°C：應增加蓋被、衣物，或提高室內溫度。  
- 體溫 37.5°C 至 38°C：減少衣物及棉被，調整室內溫度，保持空氣流通。  
- 體溫 38°C 至 39°C：進行「溫水擦身」，以毛巾沾溫水（約 27°C~37°C）擦拭全身，注意保暖，並就醫。  
- 體溫 39°C 以上：需經醫師診視後才可使用口服退燒藥物，不可自行服用。  
- 發燒前有發抖、寒顫、四肢冰冷時，應先用衣物或棉被覆蓋保暖；當觸摸四肢溫熱後，再減少衣物，以寬鬆為主。  
- 發燒處理後，應於 30 分鐘至 1 小時後再次量體溫。  
- 寶寶退燒時，體溫原則上不可低於 36.5°C。  
- 發燒過程中易流汗，應勤於更換衣物，保持乾燥。  
- 若出現活動力變差、劇烈嘔吐、嚴重腹脹、抽筋、便血、呼吸困難等症狀，應立即就醫。  
- 診療諮詢電話：08-7368686 轉。

## 流程/步驟  
1. 判斷體溫是否低於 36.5°C，若為是，則增加蓋被、衣物，或提高室內溫度。  
2. 若體溫在 37.5°C 至 38°C，減少衣物及棉被，調整室內溫度，保持空氣流通。  
3. 若體溫在 38°C 至 39°C，進行「溫水擦身」，以毛巾沾溫水（約 27°C~37°C）擦拭全身，請注意保暖，並就醫。  
4. 若體溫 39°C 以上，經醫師診視後才可使用口服退燒藥物，不可自行服用。  
5. 發燒前有發抖、寒顫、四肢冰冷時，先用衣物或棉被覆蓋保暖；當觸摸四肢溫熱後，再減少衣物，以寬鬆為主。  
6. 發燒處理後，約 30 分鐘至 1 小時後，需再次量體溫。  
7. 寶寶退燒時，體溫原則上不可低於 36.5°C。  
8. 發燒過程中，應勤於更換衣物，保持乾燥。  
9. 若出現活動力變差、劇烈嘔吐、嚴重腹脹、抽筋、便血、呼吸困難等症狀，應立即就醫。

## 注意事項/警示  
- 測量體溫時，宜於寶寶安靜狀態下進行。  
- 若洗澡前未測量體溫，建議洗澡後過30分鐘至1小時再測量，以避免溫度波動影響結果。  
- 發燒前有發抖、寒顫、四肢冰冷時，應先用衣物或棉被覆蓋保暖；當觸摸四肢溫熱後，再減少衣物，以寬鬆為主。  
- 經醫師診視後才可使用口服退燒藥物，不可自行擅自服用。  
- 若出現活動力變差、劇烈嘔吐、嚴重腹脹、抽筋、便血、呼吸困難等症狀，應立即就醫。  
- 診療諮詢電話：08-7368686 轉。

屏基醫療財團法人屏東基督教醫院
體溫測量及處理

一、正常體溫範圍：36.5～37.5℃。

二、體溫的測量方法

| 序 | 方法 | 適用年齡 | 方式及步驟 |
|---|---|---|---|
| 1 | 耳溫 | 三個月以上 | 三歲「以下」幼兒量測耳溫，耳朵應「向下」並向後拉。 |
|  |  |  | 三歲「以上」幼兒量測耳溫，耳朵應「向上」並向後拉。 |
| 2 | 腋溫 | 三個月以下 | 1. 將電子溫度計歸於起始設定值。<br>2. 將體溫計放置腋窩下，並使上臂緊貼於胸部皮膚測量。 |

圖片來源：由陳立欣醫師繪製

三、測量體溫的注意事項
1. 測量體溫時，宜於寶寶安靜狀態下測量。
2. 若洗澡前未測量體溫，洗完後過三分鐘至一小時，再測體溫。

四、特殊情況處理

1. 體溫偏低：體溫低於 36.5℃，給寶寶增加蓋被、衣物，或提高室內溫度。

2. 體溫偏高：

| 溫度 | 處置 |
| :--- | :--- |
| 37.5 至 38℃ | 減少衣物及棉被覆蓋，調整室內溫度，保持空氣流通。 |
| 38 至 39℃ | 予「溫水擦身體」：以毛巾沾溫水（以手肘內測試，溫溫的，約 27℃~37℃），擦拭全身，請注意保暖，並就醫。 |
| 39℃以上 | 經醫師診視後才可使用口服退燒藥物，不可自行擅自服用。 |

※發燒前有發抖、寒顫、四肢冰冷時，先用衣物或棉被覆蓋保暖；當觸摸四肢溫熱時，再減少身體衣物穿著，以寬鬆為主，並減少被蓋覆蓋。

五、發燒處理之注意事項

1. 給予處理後，約 30 分鐘至 1 小時後，需再次量體溫。

2. 寶寶退燒時，原則上體溫不可低於 36.5℃以下。

3. 發燒過程中，易流汗，需勤於更換衣物，保持衣物乾燥。

4. 寶寶若有發燒合併活動力變差，劇烈嘔吐，嚴重腹脹，抽筋，便血，呼吸困難等症狀時，必須馬上到鄰近醫療院所就醫治療。
