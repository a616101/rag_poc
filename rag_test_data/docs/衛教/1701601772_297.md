## 主旨  
本文介紹兒童嗓音異常的定義、成因分類、處置建議及預防措施，由復健科語言治療師林學民撰寫，旨在幫助家長與教育人員辨識與應對兒童聲音問題。

## 重點整理  
### 什麼是嗓音異常？  
在音質、音量、音調或共鳴表現上，與同齡、同性別者相比有明顯差異，例如：  
- 音質不佳：粗糙聲、氣息音、拉緊音  
- 音量過大或過小  
- 音調過高或過低  
- 鼻音過重或不足  

**常見例子**：剛從學校運動會回來的小孩，聲音變得低沉、沙啞。  

### 兒童嗓音異常成因分為三大類：  

| 分類 | 定義 | 常見原因 |  
|------|------|----------|  
| **一、器質性因素** | 構造上的問題，發聲器官受影響 | - 下嚥門狹窄<br>- 聲帶蹼、肉芽腫、乳頭狀瘤<br>- 聲帶囊腫<br>- 腦性麻痺、聲帶麻痺<br>- 顎咽閉鎖不全導致鼻音過重、鼻音不足、鼻氣流逸出 |  
| **二、功能性因素** | 發聲器官過度緊張或不當用力，雖原本構造正常，但長期使用不當導致變化 | - 在吵雜環境下大聲喊叫、尖叫<br>- 情緒不佳時大吼大叫<br>→ 可能導致聲帶結節、接觸性潰瘍<br>→ 症狀包括聲音沙啞、說話費力、喉嚨乾燥疼痛 |  
| **三、其他因素** | 其他生理或病理因素導致構造問題 | - 過敏、氣喘<br>- 胃食道逆流（GERD）<br>→ 可造成喉部乾燥、紅腫 |  

## 流程/步驟  
1. **初步評估**：若出現上述症狀，建議先尋求耳鼻喉科醫師協助，判斷是否有構造上的問題。  
2. **治療選擇**：  
   - 若有器質問題，給予適當藥物或手術治療。  
   - 若排除器質問題，轉介至語言治療師。  
3. **進一步干預**：  
   - 語言治療師評估用聲方式與習慣。  
   - 根據評估結果，進行衛教與治療。  

## 注意事項/警示  
- **避免在吵鬧環境下大聲喊叫**。  
- **生氣時大吼大叫於事無補**。  
- **感覺不適時應多喝水、多讓聲帶休息**。  
- **家中成員應以身作則，避免大聲喧嘩**。

## 全文

我的小孩又沙啞了—兒童嗓音異常

文／復健科語言治療師 林學民

壹、什麼是嗓音異常？

在嗓音的音質、音量、音調、共鳴的表現上，不同於同齡、同性別者或自己覺得嗓音有問題的人，像是可能會有音質表現不佳（粗糙聲、氣息音、拉緊音）、音量過大或過小、音調過高或過低、鼻音過重或鼻音不足等現象。例如：剛從學校參加運動會回來的小孩，聲音變得低沉、沙啞。

貳、為什麼會出現嗓音異常？兒童嗓音障礙成因分為以下三大類：

一、器質性因素

• 也就是構造上的問題，發聲器官受到影響。

• 例如：下聲門狹窄、聲帶蹼、肉芽腫、乳頭狀瘤、聲帶囊腫……等

• 腦性麻痺、聲帶麻痺

• 顎咽閉鎖不全導致鼻音過重、鼻音不足、鼻氣流逸出

二、功能性因素

• 說話時，發聲器官過度緊張或不當用力而造成異常，原本構造沒問題，但長時間的不當使用會造成構造上的變化。

• 例如：孩子在學校的吵雜環境下與同學玩時喜歡大聲喊叫、尖叫，或是常因情緒不佳就大吼大叫，這將造成喉部不當的緊張、用力；長久下來，可能會造成聲帶結節、接觸性潰瘍。

• 出現的症狀可能有聲音沙啞、說話費力、喉嚨乾燥疼痛等。

三、其他因素

• 例如：過敏、氣喘、胃食道逆流、聽損……等

• 可能是後天的疾病（胃食道逆流），造成構造上的問題（喉部乾燥、紅腫）

參、處置與建議

若出現上述症狀，建議先尋求耳鼻喉科醫師的協助，判斷是否有構造上的問題，如果有的話，先針對器官問題給予適當藥物或手術治療。若排除為器官問題，則可轉介至語言治療師，判斷是否因為用聲方式、習慣上的問題，進而導致嗓音問題的產生；治療師再根據評估結果，了解造成孩童嗓音障礙的成因，進一步進行衛教與治療。

肆、嗓音保健：需家長、學校老師、同儕多提醒

• 避免在吵鬧環境下大聲喊叫

• 生氣的時候大吼大叫於事無補

• 自己感覺到不舒服就要多喝水、多讓聲帶休息

• 家中成員需以身作則
