# 營養室

## 團隊簡介
營養室提供臨床營養、供膳管理及營養相關教學指導服務。單位願景為「營養專業達世界先進國家水準」。營養室每名成員均應秉持醫學倫理，以病人為中心的服務理念，提供住院病患美味、可口及衛生營養的食物，並持續改善以提升膳食管理服務品質。以實證醫學為依據，給予病患、家屬及營養照顧員，適切的營養治療服務。提供實習學生及營養師教育訓練，並培養臨床教師。積極配合醫院社區活動。積極推動臨床營養研究，以提昇營養照顧品質。

## 團隊編制
- 醫師：不適用
- 護理/照護：不適用
- 醫事技術/其他：營養師、廚師、供膳人員
- 特色分工：提供臨床營養、供膳管理及營養相關教學指導

## 服務與治療
### 服務範圍（疾病/問題）
- 各種疾病營養治療
- 糖尿病
- 高血壓
- 腎臟病
- 體重控制
- 各種營養問題

### 主要治療/處置
#### 臨床營養
- 住院、門診、居家及社區營養照顧服務

#### 供膳管理
- 病患訂餐流程管理、膳食設計及供餐品質衛生安全管理

#### 營養相關教學指導
- 提供營養相關教學指導服務

### 特色門診/照護計畫（若有）
- 糖尿病衛教門診：提供糖尿病衛教門診服務
- 各類疾病飲食調養：提供各類疾病飲食調養服務（糖尿病、高血壓、腎臟病等）
- 體重控制（減重）：提供體重控制服務
- 個人營養狀況評估：提供個人營養狀況評估服務
- 日常生活飲食知識教導：提供日常生活飲食知識教導服務
- 人生各階段營養調理：提供人生各階段營養調理服務（嬰幼兒、兒童、青少年、懷孕、哺乳、老年等）
- 特殊飲食設計：提供特殊飲食設計服務（管灌、流質、低蛋白點心等）

## 服務項目
### 住院病患膳食供應與營養諮詢服務
1. 病患訂餐流程管理、病患訂餐電腦連線系統作業管理
2. 訂餐餐飲製備供應、病患訂餐飲食問題處理
3. 病患普通、治療製備供應、管灌流質飲食製備供應
4. 食物製備品質監控、配膳正確性監控
5. 廚房安全衛生維護，食材、餐盤、餐具、水質衛生檢驗

### 對於住院之病患，營養室除供應各式餐點之外，亦針對病患提供以下服務
1. 各類疾病飲食調養（糖尿病、高血壓、腎臟病等）
2. 特殊飲食設計（管灌、流質、低蛋白點心等）
3. 體重控制（減重）
4. 個人營養狀況評估
5. 日常生活飲食知識教導
6. 人生各階段營養調理（嬰幼兒、兒童、青少年、懷孕、哺乳、老年等）
7. 其他營養相關知識教導（不定期於各病房舉辦與營養和疾病相關之團體衛教）

### 門診病患營養諮詢服務
1. 糖尿病衛教門診
2. 各類疾病飲食調養（糖尿病、高血壓、腎臟病等）
3. 體重控制（減重）
4. 個人營養狀況評估
5. 日常生活飲食知識
6. 人生各階段營養調理（嬰幼兒、兒童、青少年、懷孕、哺乳、老年等）
7. 特殊飲食設計（管灌、流質、低蛋白點心等）

- 供膳設備：提供供膳服務
- 病患訂餐電腦連線系統：管理病患訂餐流程
- 提供住院病患美味、可口及衛生營養的食物

## 品質與安全
### 單位願景
「營養專業達世界先進國家水準」

### 單位理念
1. 營養室每名成員均應秉持醫學倫理，以病人為中心的服務理念
2. 提供住院病患美味、可口及衛生營養的食物，並持續改善以提升膳食管理服務品質
3. 以實證醫學為依據，給予病患、家屬及營養照顧員，適切的營養治療服務
4. 提供實習學生及營養師教育訓練，並培養臨床教師
5. 積極配合醫院社區活動
6. 積極推動臨床營養研究，以提昇營養照顧品質

### 業務大綱
1. 臨床營養：住院、門診、居家及社區營養照顧服務
2. 供膳管理：病患訂餐流程管理、膳食設計及供餐品質衛生安全管理
3. 營養相關教學指導
